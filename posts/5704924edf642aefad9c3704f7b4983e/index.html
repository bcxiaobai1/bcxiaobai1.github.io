女，30+，已婚，未育的前端打工人2021年终总结

前言

看到标题4个标签的形容词，女，30+，已婚，未育，层层递进，焦虑感是不是油然而生并且越来越大。当这四个形容词组合在一起的时候，不知道猎头和其他公司的 HR 是怎么鼓起勇气给我打的电话，一定是因为我的简历太优秀了吧（快闭嘴，別把实话说出来）。减少接听这些电话的一个方法就是告诉他们，我打算要娃，果然清净了许多（看吧，果然还是要娃的理由最让人害怕）。这不，北京又多了30天的产假，女同志找工作是不是更难了。

好了，年终总结的正文开始吧。

“我自己是一名从事了6年web前端开发的老程序员，今年年初我花了一个月整理了一份最适合2021年自学的web前端全套培训教程（视频+源码+笔记+项目实战），从最基础的HTML+CSS+JS到移动端HTML5以及各种框架和新技术都有整理，打包给每一位前端小伙伴，这里是前端学习者聚集地，欢迎初学和进阶中的小伙伴（所有前端教程关注我的微信公众号：web前端学习圈，关注后回复“web”即可领取

工作

毕业以来的第三家公司，入职满了两年，从红度变成了蓝度，想想也是目前为止呆的最久的一家了（我就是那种不符合京东三五原则的履历）。

入职以来一直在做一个项目。这个跟我之前呆过的公司有所不同，之前都是大家一起维护，每个人只负责其中的几个模块，每个人都会接触到全组的项目。专人负责专项有好有坏，好的就是这个项目只有一个人负责，责任划分很明确。不好的就是万一这个人离职了，交接起来会有些麻烦。

虽然负责的是内部项目，但是项目复杂度还是有的。统计了一下，到现在为止已经有 17万+ 的代码量了。由于一些原因，项目组的技术同学算是当起了半个 PM，需求评审慢慢的变成了讨论如何实现这个需求。而且讨论来讨论去还是那些东西，无形中开了好多次的无效会议。也见识了 PM 自己的需求刚上线，复盘的时候不知道入口在哪里。算了，不说了。

但是吧，一直迭代的好好的项目，就在这两天被通知要停止迭代了。要整体迁移到另一个系统里。真是人生何处无惊喜。

技术

文章

今年在掘金就发了4篇技术文章（离组内每个Q两篇的要求渐行渐远），阅读量都是三位数以下，看着每天的阅读数统计，我果然是在吃老本（还是面试文章最受欢迎，谁不爱钱呢）。

与你项目相关的npm知识总结，总结了项目中 package.json 及 package-lock.json 的一些相关知识，对 node_modules 包里面的结构有了更深的理解。

TypeScript Interface vs Type 知多少， 团队开始用 vue3+TS 技术栈做新的项目，总结了 TS 中 Interface 和 Type的相同点和不同点，结论就是能用接口的时候优先使用接口。

AST抽象语法树，好多工具的底层都是基于 AST，挺有意思的。

走进chrome内心，了解V8引擎是如何工作的，跟 AST 也有关系，了解了 chrome 内部工作流程，有助于写出更丝滑的代码。

掘金翻译计划只校对了一篇文章，有点水。

掘金的文章会同步到 segmentfault 上，segmentfault 上的阅读量跟掘金根本不在一个水平线上。今年在 segmentfault 上收获了不少声望值和粉丝数，就是不知道有多少僵尸粉。

明年希望能满足组内每个Q两篇的文章输出要求，就算 flag 没倒。

瞎折腾

各种 APP 的每日签到薅羊毛活动，手动签到太麻烦还总忘，不如写个脚本，控制台手动跑一跑。平时上班还可以，手动执行一下。周末还得开电脑，还是没有解放双手解放劳动力（人就是这样，满足了一个需求后就会冒出下一个需求，啥时候才能满足呢）。咋能呢？腾讯云了解一下。

注册了个腾讯云，用了云函数的功能，把代码发布上去，写个定时执行命令，搞定。但是偶尔也有失败的情况，那就是登录信息过期了，还是得手动复制一下。每个月那点流量，完全可以白薅。但是突然有一天说我欠费了，我百般疑惑。啥东西用了这么多流量。后来发现是我把其中一个任务设置成了每分钟执行一次，害得我充了一块钱。在此也感谢某 APP 没有把我加入黑名单。

今年的技术也算是给自己做了点贡献，学以致用了。

读书

啥时候能买过等于读过就好了。每个季度买书经费买的书，有一些还没有拆封，就摆在桌子上，静静的看着我，只看了跟性能优化相关的。本打算写一篇性能优化相关的文章，但然后，好像就没有然后了，看来这个要留到明年的Q1了。没事，反正也经常这样（你快闭嘴吧，还好意思说出来）。

除了学习之外的杂书，果然还是自己感兴趣的东西愿意花时间。

没错，还有一本投资理财的书，在我的不懈努力下，基金理财到现在为止还赔了1000多，果然财还是越理越少。希望明年总盈亏能变成红色的（耳边似乎有人在说希望渺茫，我赶紧给他一巴掌）。

明年的读书 flag 就不立了，读书使人进步。

娱乐

某人去年年会抽奖得的 switch 也就新鲜了几个月。买了心心念念的胡闹厨房，在我俩刷完三星之后就没什么动力了。just dance 在部门运动打卡的一个月里派上了用场，之后也就一直吃灰了。

每周或者隔周的麻将局还是很固定，去了朋友家可以吃吃喝喝玩玩闲聊遛金毛的日子很美好。

疫情原因也没怎么出去玩。就十一跟朋友开车去了趟烟台。天气也不怎么美丽，时而还下场雨。去看了黄渤海分界线，也根本看不出来。但是卖苹果的是真的多。

前几天的团建，去了环球影城。周五不用上班还不算请假的日子，听着就开心。去环球影城还不用花自己钱的日子，更开心。本以为自己玩不了啥，结果还不错，每个主题基本上都玩了一两个项目。但是没见到威震天，有点遗憾。

想想去年的年假还剩4天没有休完呢，我可真是个热爱工作的人儿。

明年见

年底了，公司在想方设法的怎么能延迟或者减少发年终奖，各大互联网的裁员也在默默的进行中，不只是教育行业。大环境不好，咱也决定不了。

唯一能把握的就是未雨绸缪，提升自身软实力和技术能力。毕竟，还有好多公司在招人。毕竟，是金子，哪里都发光。

我们，明年见。我应该还是女，30+，已婚，xx。
