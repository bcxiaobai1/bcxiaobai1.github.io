前言

从入职第一份前端开发的工作至今，快满三年。我也在不断思考，将来的出路和制定职业生涯规划。

前端在最近七八年，充分展现出了它的活力，和各种可能性及发展方向：

  1. 专业做app，微信/支付宝小程序的移动前端开发工程师；
  2. 精通浏览器之间兼容性和各种响应式，做各种pc、移动端、ipad网站的web前端开发；
  3. 精通canvas、d3.js、echarts等前端动画，能够在页面上呈现各种3d效果的前端动效开发；
  4. 有一直做前端通用组件开发的前端组件开发工程师；
  5. 能够搭建不同前端框架，满足公司的项目需要的前端框架师等；

以上还只是从前端开发工程师所做的功能进行区分，实际上，我们还可以从技术栈上进行区分:

  1. 关于框架搭建的技术栈webpack/gulp/nodejs/vite；
  2. 关于通用开发框架的技术栈 vue/react/angular/vue3；
  3. 移动端的开发 react+taro搭配/react.native/uniapp+vue/原生的微信小程序/原生的支付宝小程序；
  4. vue + element-ui/ react + antd；
  5. 还有各种大屏项目方案(echarts/antv/d3.js/three.js/webgl)
    …
    不知道各位前端开发工程师尝试做过多少种。

2022年互联网大环境不行，大量的裁员，让人不禁发出疑问，中国是否真的需要这么多开发人员，而问题的背后所得到的结论，开发岗位在2023年只会越来越卷。我得不断在各个方向做出更多突破，以应对互联网行业的第一波寒冬。

2022年对我个人而言，是很稳定的一年

就职于一家世界五百强的公司，在IT部门担任一名前端开发工程师，主要负责现有页面的代码优化，功能迭代，和页面重构。年底参与了一个大屏数据展示项目，并搭建了最初的前端框架。

我自己仍然是一边学习一边写博客，虽然没有写满五十篇，但也还是获得了一万粉丝，和三十多万的阅读量；创建了vue3组件库的个人项目，继续努力吧；学习了部分nodejs，接下来一段时间仍然会继续研究，并通过博客记录下来。

2023年想要实现的计划

  1. 继续csdn创作，一边学习一边记录，目标50篇技术博客。
  2. 完善vue3搭建的组件库项目，目前已经写了十来个前端常用的通用组件。
  3. 前段时间，因公司业务需要，参与了一个大屏展示数据的项目，并搭建了前端开发框架，后续会总结更多echarts的使用，并尝试学习d3.js或者three.js的开发
  4. 我想成为一名前端架构师，之前自己搭建过一些不同技术栈的项目，以应对不同的项目需求，但还不够，我需要接触更多的项目，观察别人用过什么样的技术栈搭配去完成项目搭建。
  5. webpack，nodejs，gulp，vite 等搭建项目常用的技术栈，我需要更加熟练地使用，为将来熟悉原理打下基础。
  6. 关于移动端开发，在两年前，我参与过一些项目，但太久没有相关地项目经历，我需要重新熟悉，并进一步找项目练习。

除了工作，再说些个人生活相关的吧

过去的一年，我自己的感情是更加稳定了，在年中的时候，双方父母见面，也算是获得了对方父母的认可，一切会越来越美好。

在现在的公司，我的确实现了之前的一些愿望，工作生活的平衡，全年不加班。

2022年十月份，参加公司体检，查出一些身体的问题（以前从未检查过），甲状腺结节，还好是良性，并且很多人都有，虚惊一场，但在不了解之前，确实有些害怕。总之要照顾好自己的身体，注意饮食和睡眠，定期检查身体。

我今年比去年瘦了一点，脱离了肥胖，但还在快要肥胖的边缘，除了锻炼身体，减肥的事情真不能强求，千万不要通过节食等方式，去达成减肥目标。

买房的事情，还是遥遥无期，估计将来很长的一段时间，都得努力赚钱，节省开支，为了将来更美好的生活。
