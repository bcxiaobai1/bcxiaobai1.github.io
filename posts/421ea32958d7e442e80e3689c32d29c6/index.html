ChatGPT 完全颠覆了我今年的计划

仔细阅读本文需要超过 6 分钟的时间，建议您先通篇阅读粗体文字和图片。

上线近三个月，这把名为 “ChatGPT” 的火，点燃了科技圈。众多科技大厂纷纷入局，都在紧急开发类似的应用产品。昨天复旦 Moss 的发布，以及百度“文言一心”等产品的消息，也预示着今年的科技圈不会平静。

而对于我而言，现在的 ChatGPT 究竟意味着什么？先说结论，ChatGPT 对我而言是一个优秀的生产力工具。正是由于生产力的改变，ChatGPT 完全颠覆了我今年的计划！

1

在春节前，我已经制定好了今年的发展计划：

  1. 维持公众号的发文速度，保证至少 50 篇原创内容产出
  2. 业余时间开发一款独立游戏，题材是武侠

在当时制定这个计划，我做好了时间分配不过来的心理准备，也做好了放弃本职工作，全时间屯稿产出的预案。这个计划是非常冒险的，尤其是在当下这样的就业大环境下，并且眼下没有任何可能的收入来源。

我老婆在当时也提醒，得设置一个存款的告警线，过线了就得放弃计划，毕竟“生存是文明的第一需要”，我不可能靠我的游戏梦活着。

但在有了 ChatGPT 之后，这一切发生了改变。在“浪潮”过去后，ChatGPT 能带来的，是细水长流的生产力提升。大部分人会觉得，ChatGPT 的准确度不够高，自动产出的内容质量低，存在风险等。这是站在内容全部交给 ChatGPT 生产的角度出发思考问题，完全无监管，即完全意义上的 AIGC。

我赞同这样的看法，也不认为当下的 ChatGPT 能真正做到自动化的 AIGC，还处于前进的道路上。但我认为，思路不应该只局限在非 0 即 1 上。0 和 1 之间还可以有 0.5，甚至是 0.8，这是我最近在做的事。即，我和 ChatGPT 通过对话的方式“合作”，互为审核，从而更高效的产出内容。

虽然达不到 1+1≥2，但很轻松的，我们就能做到 1+1>1。而 ChatGPT 的这个 1，成本是很低的，算上 Plus 版本的成本，一个月 20 美金。就让我的生产力有了显著的提升。

我开始上线一些闲鱼业务（不是卖账号滥用割韭菜那种）；创建知识星球的频道，写更多的内容；以更高的速度产出各种代码，其中包含给公司项目写的验证过的代码，以及各种语言杂七杂八的代码，虽然还没能找到它们的用武之地。

因此，我将计划推翻了，我完全可以制定一个更具有野心的计划——我想在今年找到工作之外的稳定收入来源，在理想和现实中找到一个平衡点。

篇幅原因，我不会发出所有的截图。这个过程说明了一点，单靠 ChatGPT 完成不了这个需求，直接写 100% 是不能直接用的；单靠我来完成这个需求，按照以往的经验需要两周时间，排除掉 debug 测试的时间，压缩估计也至少需要 3 天共 24 小时的工时才能完成。

我以前没开发过 Twitch Drops，这里的时间加上了阅读文档提取项目需要的功能 API 的时间。

而通过我和 ChatGPT 一问一答这种方式，我们共同产出内容只花了不到 4 小时的时间，已经初步验证过完全可以运行，且几乎所有条件分支、错误异常处理都已经“写完”。代码产出时间甚至远远领先 Twitch 本身给测试账号的速度。换句话说，现在的产出瓶颈不在代码上，而是在商务上和人与人的沟通交流上。

这就是我认为 ChatGPT 是一个出色的生产力工具的原因。很多人初步尝试，让 ChatGPT 写代码，发现给出来的代码运行出错，有这样那样不合理的地方，所以觉得 ChatGPT 还不够好用。这完全是建立在自动生产代码的角度上思考这个问题。ChatGPT 目前只能保证生成最基本的代码不会出错，但在懂程序的工程师的“调教”下，它能发挥出巨大的潜力，部分开发场景，可以由对话的方式完成代码开发。

3

Github Copilot 于一年前发布。有不少大牛程序员发文称，项目中 80% 的代码都可以交给它来写，只用关注剩下的 20%。Copilot 是基于 GPT-2 的产品，而在当下，ChatGPT 背后是更强大的模型更大的训练数据集，ChatGPT 拥有更强大的代码生成能力。并且它拥有阅读理解 API 文档的能力。只要用好它，我们就能将代码的生产力成倍提升。

我认为这是一个有潜力与蒸汽机相提并论的工具。当下还达不到这样的程度，但是随着模型的进一步迭代优化，随着对它使用的进一步探索，我觉得这是有可能的。至少当下，写文章的这一刻，我就享受着 ChatGPT 带给我的生产力提升，让我有了更多的“时间”做更多的事，不是吗？

本文只是从 ChatGPT 写代码这个角度出发进行思考。目前在 Github 上有一个中文调教指南，已经有超过五十多种不同的场景案例，覆盖多行业。有趣的玩法就更多，也不可能完全列出来。

上周日和这周一，我很焦虑。但在我用上 ChatGPT 之后，我不焦虑了，它还不足以自动产出优质内容，所以读到这里的你也没必要焦虑。完全替代我们还在未来，现在我们能学会好好使用它创造价值。

4

最后，我想分享一下 ChatGPT 的诞生给我的感触。最近在 B 站上刷到一个视频，讲的是 ChatGPT 诞生的故事，OpenAI 如何从一个汇聚顶级研究人员但默默无闻的非盈利性公司，逐步转变成今天这个样子。“先捞钱，再谈理想”，我想 OpenAI 也处于理想和现实的激烈矛盾之中。

其实这次过年我也感受到了就业大环境的压力。家里的老人也问我，是不是只能做游戏，做别的程序相关的可不可以？我是一个结了婚成了家的人，有的时候真的很羡慕像螺舟这样的工作室，能把自己的理想坚持这么久，现实的引力太沉重了。我不想放弃自己正在做的事，但也得想办法在理想和现实之间寻求平衡。

知识星球是尝试之一。我闲下来喜欢刷刷贴吧，逛逛论坛，很喜欢发帖回复这种方式。我以前一直没下载过知识星球，但在老婆的手机上看过几篇“帅张和他的朋友们”星球里的帖子，很有收获。所以就在昨天创建了一个属于自己的星球【游戏匠心】，立意为“犟人有匠心”，也算是补完了公众号这个名字。名字也是 ChatGPT 建议的，我和老婆都很喜欢。

我希望我的知识星球能成为一个互相交流、分享技术的平台。公众号建的太晚，无法开放留言功能，很少能收到反馈和读者的想法，这让我很苦恼。我希望知识星球能改变这一点，多一些交流和分享。

我将星球设置为付费星球，但将付费和折扣价都设置为平台允许的最低。我希望付费能够挡住一些杂七杂八的事，分心在这些事上是无价值的；有一个最低的门槛，同时用户如果真的认真反馈或分享，我可以免除这个费用，将加入的费用原数退回，即：

  1. 乐于分享的人收获更多的技术分享，且无需任何经济上的代价，共同成长共同提升
  2. 其余人也能收获更多的技术分享，只不过需要支付一笔费用

别的不多说，如果你觉得我分享的内容对你有所帮助，你想看更多的技术分享和内容，或者是直接问我一些技术问题，讨论公众号文章细节，我都会以更高的频率在知识星球【游戏匠心】内回复产出。知识星球内的分享会更随意，分享很多不成文的小短篇、图片和代码资源，希望能通过类似发帖回复的方式呈现出来。
