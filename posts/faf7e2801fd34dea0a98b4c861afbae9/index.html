运算符与表达式

现在为止，C语言中的各种基本数据类型的特点和形式我们都已经了解了！

运算符和表达式是用来进行数据处理和运算的，包裹逻辑运输判断等....在程序中，运算符和表达式是进行数据计算求值的基本单位。

运算符与表达式

我们可以把运算符与表达式简单的理解为，我们小时侯学习数学时用于计算的公式，运算符与表达式由（如 + - * / =等等）组成，*可以理解为数学中的乘x    / 这个符号在C语言中是取整运算符，现在可以理解为除法，取整运算符顾名思义就是它在计算的时候计算的结果只取整数

具体多少运算符可以看上一章

赋值运算符  = 

在C语言中 = 并不表示相等，和我们数学中的=不一样

=在程序语言中 “=” 表示赋值运算符

a  =  100;

这个是赋值表达式语句

把值100赋值给变量a

读作：把值100赋值给变量a

直白点就是 把 = 号右侧的值赋值给左侧的变量a

被赋值的变量名 = 赋值给变量的值

“+”、“-”、“*”、“/”、“%”、“++”和“--”

在程序中经常遇到的符合就是 “=” 赋值运算符了

#include<stdio.h>

#include<stdlib.h>

int main(void) {

    int ChiNese, EngLish, Math;

        ChiNese = 80; //值80赋值给变量ChiNese         EngLish = 90; //值90赋值给变量EngLish         Math = 79; //值79赋值给变量Math     

        printf("%dn%dn%dn", ChiNese, EngLish, Math);     

    system("pause");

    return 0;

}
