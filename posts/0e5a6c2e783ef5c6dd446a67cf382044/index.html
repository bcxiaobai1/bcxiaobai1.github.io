4EVERLAND Hosting 现在支持 SNS+IPFS

 

EVERLAND的去中心化托管解决方案现已正式支持SNS（Solana Name Service），并提供4sol.xyz作为所有SNS用户的专属访问网关。

通过使用 4EVERLAND，用户现在可以部署专有的基于 IPFS 的 Web3 网站，并将他们的 SNS 域绑定到他们的网站，使其完全去中心化和无审查。4EVERLAND 甚至提供 IPNS 服务，支持自动 IPFS 哈希解析更新。

托管+SNS

自推出以来，4EVERLAND Hosting 得到了全球开发者的一致好评。它使开发人员能够基于 IPFS 快速构建网站、Web 应用程序或原生应用程序，并集成了隐私、加密和 p2p 功能。

4EVERLAND Hosting 是一个用于构建分布式应用程序的集成开发环境。它包括用于分散内容分发的 IPFS 点对点文件系统，这已成为现代 Web 应用程序的基本要求。集成的隐私、加密和 p2p/网络功能让您作为开发人员能够快速开发可以立即部署到其他浏览器或设备的安全分布式系统。

Bonfida 的 SNS（Solana 名称服务）为开发人员和用户提供了一个新的数字身份系统——将域名 (.sol) 映射到链上数据。这种链上数据可以是任何东西，从 SOL 地址到 IPFS CID、图像和文本。

此外，Solana 域名的应用场景非常广泛。用户可以通过域名（.sol）进行在线P2P资产支付，并将数据直接存储在链上。

对于 .Sol 用户，4EVERLAND Hosting 与 SNS 的集成提供了一站式 DWeb 部署服务。开发者通过Hosting一键部署IPFS站点并绑定其.Sol域名，可以方便、快捷、有效的方式快速实现网站真正的去中心化。

正如 Leland Kaiser 所说，“未来只是等待发生的无限可能。它等待的是人类的想象力来具体化它的可能性。因此，我们鼓励所有开发人员利用所有可用资源并构建更多分布式应用程序。

IPNS

星际名称系统 (IPNS) 由 IPFS 团队设计，旨在通过创建可更新的地址来解决此问题。在 IPNS 中，名称是公钥的哈希值。它与一条记录相关联，该记录包含有关它所链接的哈希的信息并由相应的私钥签名。可以随时签署和发布新记录。

由于 4EVERLAND 为 SNS 域用户提供 IPNS 服务，因此开发者无需在每次更新站点时都更新 SNS 域的 IPFS 哈希。

此外，4EVERLAND Hosting 支持 Deploy Hooks 功能，开发者只需更新授权的 repo 代码即可完成 SNS 域下 DWeb 的自动更新。这显着提高了效率，同时还降低了开发和工作负载成本。

4sol.xyz

4EVERLAND为所有 SNS 用户提供专属网关 4sol.xyz 。使用 4EVERLAND Hosting 服务部署 DWeb 的 SNS 用户可以通过Name.4sol.xyz访问他们的DWeb ，无需下载任何插件或更改任何配置。

因此，享受 4EVERLAND 的全球加速、可扩展性和 IPNS 服务，以满足您对 DWeb 的访问性能要求。

即使您不通过 4EVERLAND 托管服务部署您的 .sol 网站，也无需安装任何插件或对配置进行任何更改即可通过 name.4sol.xyz 访问它；4EVERLAND 将通过 IPFS 公共网关 ipfs.io 为您处理此问题。

如果您有 4everland.sol 并完成了绑定过程，那么您可以直接从浏览器访问4everland.4sol.xyz以查看解析后的 IPFS CID 的内容。
