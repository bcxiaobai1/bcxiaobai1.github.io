2、如何选择Spring cloud 的版本？

       

1、如何查看spring cloud的版本

        前面文章我们介绍了什么是spring cloud，那么我们该如何使用，选择哪一个版本来使用呢？我们打开官网Spring Cloud，点击官网首页的LEARN进入官网的版本列表

        spring cloud旧的版本命名方式为以伦敦的地铁站首字符A~Z来进行命名，最新的版本到Hoxton，后面更改为了日历式命名，如2020.0.6。版本号后面的-SNAPSHOT、-M5、-SR12等，代表的是该版本的快照版本、里程碑版本、发行版本等，后面的GA表示是稳定版本（可用于生产环境）。

  更多的spring cloud版本可以在Home · spring-projects/spring-boot Wiki · GitHub里面进行查找

2、选择cloud版本     

        一般我们选择的时候选择GA版本即可，此处我们选择最新的2021.0.4 CURRENT GA版本，后续我们的文章都将基于此版本来进行。

3、如何选择spring boot版本

        spring cloud要求与spring boot版本严格对应，所以我们选择好了cloud的版本之后，需要对应选择它所支持的boot的版本。cloud版本后面都会有一个Reference Doc，点击进去之后就可以查看当前cloud所支持的boot版本了。

 

 4、spring cloud alibaba

        为什么将spring cloud的时候需要讲到alibaba呢，因为我们后面的章节将会涉及到alibaba的东西来配合使用，比如注册中心nacos，熔断降级sentinel，配置中心alibaba config等等。我们可以打开spring cloud alibaba版本对应来查看后续我们所需要选择的版本。
