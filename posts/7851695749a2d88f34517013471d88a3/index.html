Image.new() 及 img.paste() 的用法记录

Pillow 是 PIL 的替代版本，PIL 软件包提供了基本的图像处理功能，如：改变图像大小，旋转图像，图像格式转换，颜色场空间转换，图像增强，直方图处理，插值和滤波等等。

软件包安装

pip install pillow

Image.new(mode, size, color)

作用：创建一个新的图像

参数：

1）mode：图片模式，常用模式有：'1' (二值图)，'L' (灰度图)，'RGB' (彩色图)，'RGBA' (彩色图+Alpha通道)；

2）size：生成图片的大小；

3）color：生成图片的颜色，默认值为0 (黑色)，也可以直接指定颜色，如：'white'。

应用示例：

from PIL import Image
import cv2
import numpy as np

img1 = Image.new("RGB",(480,640),'black')
img2 = Image.new("RGB",(480,640),'white')
img1 = np.array(img1)
img2 = np.array(img2)
cv2.imshow('img1',img1)
cv2.waitKey(0)
cv2.imshow('img2',img2)
cv2.waitKey(0)

Image.new() 常与 image.paste() 一起使用，在训练基于深度学习的图像分类模型时，这两个函数常用于数据增强 (打乱和重构)，

image.paste(img, box=None, mask=None)

代码如下：

from PIL import Image
import cv2
import numpy as np
import random

img_path = 'C:/Users/15989/Desktop/429a84b8517ba3a8ed9517f0c150e694.jpeg'
img = cv2.imread(img_path)
img = cv2.resize(img,[200,200])
cv2.imshow('img', img)
cv2.waitKey(0)

# box的格式为(l,t,r,b)，即边界框左上角和右下角的坐标
box_list = []
for i in range(0, 2):
    for j in range(0, 2):
        box = (100*i, 100*j, 100*(i+1), 100*(j+1))
        box_list.append(box)

img = Image.fromarray(img)      
img_list = [np.array(img.crop(box)) for box in box_list]

# 打乱img_list里面元素的顺序
random.shuffle(img_list)  
img_list = [i.reshape(100, 100, 3) for i in img_list]

# 创建新图片  
img_new = Image.new('RGB', (200, 200))

# 将img_list里面的crop_img粘贴到img_new
for i in range(2):
    for j in range(2):
        crop_img = img_list[i+j]
        img_new.paste(Image.fromarray(crop_img), (100*i, 100*j, 100*(i+1), 100*(j+1)))

cv2.imshow('img_new', np.array(img_new))
cv2.waitKey(0)
