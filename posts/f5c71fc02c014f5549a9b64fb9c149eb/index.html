被谷歌解雇一年后，AI 研究员 Timnit Gebru 自己成立独立人工智能研究所

Timnit Gebru 是谷歌的前员工，一年前在推特上发文称被谷歌解雇，原因是经理要求她从她合著的一篇研究论文中删除自己的名字，此事引发了一场关于员工质疑其公司技术影响和自由的争议。而在本周四，她独立成立了一个新的研究机构——分布式人工智能研究所 （DAIR），一个独立扎根于社区的研究所，旨在对抗 Big Tech 对人工智能研究、开发和部署的普遍影响。Gebru 认为，人工智能技术中蕴含的危害是可以预防的，当它的生产和部署容纳不同的观点和深思熟虑的过程时，它可以为人们服务，而不是反对他们。通过 DAIR，Gebru 旨在创造一个独立于激励利润而不是道德和个人幸福的结构和系统的环境。（DAIR 研究所、WIRED）

GitHub 宕机根源分析

11 月 27 日，GitHub 突发宕机事件，故障持续了 2 小时 50 分钟，影响了成百上千万的用户。12 月 1 日，官方回应宕机的主要原因是，在处理一个大型 MySQL 表的模式迁移时，遇到了一种新的故障模式。其表示，模式迁移是 GitHub 一项常见的任务，通常需要数周时间才能完成，迁移的最后一步是执行重命名，将更新的表移到正确的位置。然而，在这次迁移的最后一步中，GitHub 的 MySQL 只读复制的很大一部分进入了信号死锁。GitHub 的 MySQL 集群包括一个用于写流量的主节点，多个用于生产流量的读副本，以及几个用于备份和分析的内部读流量的副本。遇到死锁的只读副本进入崩溃恢复状态，导致健康只读副本的负载增加。

在事件缓解期间，为了增加容量，他们将所有处于健康状态的可用内部副本提升到生产路径，然而这种转变不足以完全恢复。他们还观察到，服务于生产流量的只读副本会暂时从崩溃恢复状态中恢复，只是由于负载可能再次导致崩溃。基于此崩溃恢复循环情况，他们主动选择通过从损坏的副本来删除生产流量，直到它们能够成功处理表重命名，得以优先考虑数据完整性而不是站点可用性。（GitHub）

估值高达 130 亿美元，开源科技公司、云基础设施平台 HashiCorp 计划 IPO

提供 Terraform 和 Vagrant 等云软件的供应商 HashiCorp 公司周一更新了它的 IPO 招股说明，招股说明披露了该公司估值高达 130 亿美元的股票出售计划，其预计在感恩节和圣诞节期间进行 IPO，且将以每股 68 美元至 72 美元出售公开发行的股票，这个IPO 定价区间将使其跻身估值最高的开源科技公司之列。但在股票正式出售之前，价格范围也存在变数。此前该公司宣布计划于纽约证券交易所上市、在纳斯达克交易，股票代码为“HCP”。（CNBC、Solidot）

开发人员呼吁希望将 Chromium 与谷歌独立开

12 月 3 日，谷歌的一个开发人员 Gabriel 发文称 Chromium 基金会需要将 Chromium 项目与谷歌分开。其认为，该浏览器当前的管理状态有些混乱，因为 Chromium 得到了业界很多大型企业的支持，但是却受谷歌公司的管理。Gabriel 表示，当很多开源爱好者和业界公司在对 Chromium 项目作出贡献的时候，谷歌依靠 Chromium 项目获得了浏览器的广告收入，也收集到了一些用户信息。从逻辑上讲，用户希望有更多样化的浏览器可供选择，不受 Google 的影响，且也应该有独立的 Chromium/V8/Blink 引擎。（Gabriel’s blog）

2021 年最佳 Linux 桌面版排行榜出炉，Red Hat 成首选

2021 年最佳 Linux 桌面版 Top 5 出炉，分别是：Red Hat、Arch Linux/Manjaro Linux、Gentoo/Sabayon Linux、Kali Linux、SystemRescue。

  • 其中 Red Hat 成开发商首选，Red Hat Linux 发行版 Fedora 是推动 Linux 极限的主流发行版，它由最新的 Linux 内核和最新的开源软件提供支持，是程序员的首选的 Linux 桌面系统；

  • Arch Linux 能让用户决定桌面环境以及进行自定义操作，Manjaro Linux 可以轻松切换操作系统齿轮；

  • Gentoo Linux 可以应用用户想要的所有定制，Sabayon 会做出大部分 Gentoo 设置决定，将 Sabayon Linux 与 Gentoo 一起使用，用户可以以优雅的格式提供最新的开源技术，从而打造“开箱即用的用户”体验；

  • Kali Linux 是一个 Linux 黑客发行版，专为渗透测试或黑客而设计；

  • 而修复型的 Linux 发行版中最好的是 SystemRescue，旨在修复损坏的计算机。

Linux 供应商固件服务达到 4000 万次

向 Linux 用户提供系统和组件固件服务的 LVFS 平台下载次数突破了 4000 万次，Red Hat 的 LVFS/fwupd 首席开发人员 Richard Hughes 在 12 月 1 日分享了这一好消息。LVFS 通过 fwupd 实用程序进行闪存，得以加速增长。由于安全问题导致 BIOS 更新增加，更多的 OEM / ODM 加入、扩展到支持服务器，以及 FWUPD / LVFS 周围的其他增加，这个数字可能会在 2022 年继续保持良好的增长。（Phoronix）

Open 3D Engine 的第一个主要版本来袭

图片来源：Open 3D 基金会
12 月 2 日，Open 3D 基金会（O3DF）及其成员 Adobe、AWS、华为、英特尔和 Niantic 宣布了基于 Apache 2.0 许可的开放 3D 引擎项目的 O3DE（Open 3D Engine）稳定版 21.11 版本。自 O3DF 成立并在 2021 年 7 月推出 O3DE 开发者预览版以来，来自各行各业的 250 多名开发者已经贡献了超过 2182 个拉动请求和 1801 个问题。在提高稳定性和性能的同时，还增加了开发者功能，以确保 O3DE 可以在现场游戏和模拟中使用。此外，O3DE 现在有一个 Windows 安装程序，给用户一个加速的上机体验，并支持 Linux 将 O3DE 带到更广泛的用户和贡献者社区。（Open 3D 基金会）

AWS Private 5G

AWS Private 5G 是一项托管服务，4G/LTE 和 5G 等蜂窝技术为越来越多的设备提供了更高的带宽、更低的延迟和可靠的远程覆盖，从而增强了现有网络，借助 AWS 提供的所有必需硬件和软件，以利用 5G 的技术优势，同时保持私有网络的安全性以及精细的应用程序和设备控制，可以轻松部署、操作和扩展自己的专用蜂窝网络。

AWS Private 5G 可用于运行智能制造设施，连接制造设施内的设备、传感器、机器和旧设备，通过跨连接机器的可靠数据链接实时跟踪装配线生产；它也支持业务关键型应用程序，支持一系列低延迟用例，例如用于设计工程的增强现实和虚拟现实 （AR/VR） 应用程序、医疗程序期间的图像分析以及履行中心的自动导引车；同时它可以提供可靠的园区连接，在活动场所、大学校园和企业设施中提供高质量、可靠的连接。支持数据和视频通信、安全、物流等。

图片来源：AWS

更多详情见： https://aws.amazon.com/cn/private5g/

grommunio 成 Exchange 开源替代，功能、外观与其类似

微软 Exchange 作为群件环境平台一直被广大开发者使用。在 2020 年末，一位奥地利开源软件开发人员推出了一个群件服务器（Groupware Server）和客户端产品——grommuino，它具有 Exchange 和 Outlook 用户熟悉的外观和操作体验。开发人员用和微软相同的方式将组件连接到平台，并且支持使用 HTTP 协议的 RPC（远程过程调用），同时 grommuino 还支持邮件，聊天、视频会议和文件同步等功能。感兴趣的小伙伴，不妨上手一试！

GitHub 地址： https://github.com/grommunio
