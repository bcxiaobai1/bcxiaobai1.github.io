⚽一、eNSP简介

? eNSP(Enterprise Network SimulationPlatform)是一款由华为提供的、可扩展的、图形化操作的网络仿真工具平台，主要对企业网络路由器、交换机进行软件仿真，完美呈现真实设备实景，支持大型网络模拟，让广大用户有机会在没有真实设备的情况下能够模拟演练，学习网络技术。

?二、数据链路层（交换机）

  1. 以太网MAC地址
    用来识别一个以太网上的某个单独的设备或一组设备。MAC地址为48个二进制位，一般用12个16进制位表示。MAC地址标识的设备全球唯一。

  2. 以太网帧格式
    数据链路层的协议数据单元

  3. 什么是交换机？
    交换机是用来连接局域网的主要设备。交换机能够根据以太网帧中目标地址智能的转发数据，因此交换机工作在数据链路层。

  4. 交换机转发原理：
    初始状态(一无所知)；
    MAC 地址学习(将源地址记录到 MAC 地址表中)；
    广播未知数据帧(广播目的地址，接收方回应，交换机记录接收方的源 MAC 地址到 MAC 地址表中)

  5. 交换机工作原理
    ?学习：MAC 地址表是交换机通过学习接收的数据帧的源 MAC 地址来形成的；
    ?广播：如果目标地址在 MAC 地址表中没有，交换机就向发送该数据帧的接口外的其他所有接口广播该数据帧；
    ?转发：交换机根据 MAC 地址表单播转发数据帧；
    ?更新：交换机 MAC 地址表的老化时间是 300 秒，交换机如果发现一个帧的源接口和 MAC 地址表中的源 MAC 地址的所在接口不同，交换机将 MAC 地址重新学习到新的接口；
    ?注意：断电，拔网线，过 300 秒都会使 MAC 地址表变化

?三、交换机的基本配置（指令）

??交换机总共有三种视图：用户视图<huawei>、系统视图[huawei]、接口视图，每种视图下可操作的权限不一样。

  • 配置主机名
    < Huawei >       #用户视图
    < Huawei > system-view   #进入系统视图
    [ Huawei ]         #系统视图
    [ Huawei ] sysname SW1为  #改名SW1

  • 查看版本号
    < Huawei > dispaly verson

  • 查看交换机的所有配置
    < Huawei > display current-configuration
    [ Huawei ] display current-configuration
    两种视图都可以

  • 使用账户和密码登录终端
    [ Huawei ] aaa
    #系统视图下进入aaa模式，此模式下才可以设置用户名和密码
    [ Huawei-aaa ] local-user 用户名 password simple 密码
    #simple表示密码明文
    [ Huawei-aaa ] local-user 用户名 password cipher 密码
    #cipher表示密码密文
    [ Huawei-aaa ] quit    #退回到系统视图
    [ Huawei ] #此时用户名和密码没有生效，需要进入console0
    [ Huawei ] user-interface console0   #进入console0接口
    [Huawei-ui-console0] authentication-mode aaa  #通过 aaa 认证

  • 保存交换机配置
    < Huawei > save

  • 重启设备
    < Huawei > reboot

  • 设置控制台时间
    [Huawei] user-interface console 0    #进入 console 0 接口
    [Huawei-ui-console0] idle-timeout 1   #设置控制台会话时间（1 分钟）

  • 恢复出厂设置
    < Huawei > reset saved-configuration     #恢复出厂默认值

?四、路由器

  • 什么是路由
    将数据包从一个网路发送到另一个网络，需要依靠路由器来完成。路由器只关心网络的状态，决定最佳路径。

  • 路由器如何选取最佳路径
    根据路由表。每个路由器都维护着一张路由表，每条路由表路径指明了：到达某个子网或主机应从路由器的那个物理接口发送，通过此接口可以到达该路径的下一个路由器地址。

  • 如何获取路由表
    直连路由：配置IP地址后自动生成
    静态路由：手工指定
    动态路由：由路由器协议自动设置

  • 配置直连路由

< Huawei > system-view

[ Huawei ] int g0/0/0

[ Huawei-GigabitEthernet0/0/0 ] ip address 192.168.1.254
255.255.255.0

#查看结果

[Huawei-GigabitEthernet0/0/0]display ip interface brief
#查看 ip 接口的摘要信息

[Huawei-GigabitEthernet0/0/0]in g0/0/1
