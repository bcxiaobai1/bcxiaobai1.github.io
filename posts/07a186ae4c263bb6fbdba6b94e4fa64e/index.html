很多人毕业之后都很迷茫，就只知道海投简历，生怕找不到工作。收到Offer什么也不考虑就去上班了，结果一朝选错，耽误终身。

其实，无论是迷茫还是赚不到钱，核心问题都与职业方向不清晰有很大关系！

大多数人，既不清楚自己是什么人，也不清楚市场需要什么样的人。随波逐流向前，最终难逃大浪淘沙。

职场发展不顺利本质上是因为不知道如何做职业选择。

我们可以做一个简单的想象：

如果你知道哪些行业未来几年十几年甚至几十年都会发展得很快；你还知道这些行业都有哪些岗位；这些岗位薪资水平如何；这些岗位的发展路径和成长空间是怎样的。

如果这些你都掌握了，你是不是很容易做出一个比较正确的职业选择？是不是也能更好地把控自己的人生呢？

所以我认为：迷茫、想进互联网大厂拿高薪的职场人，都应该进行一次系统性的职业规划，从而实现爆发式成长，成功收割大厂高薪Offer。

那么，什么互联网职业上手快，有前景，待遇高呢？

人生苦短，我学Python

Python学习（工作）方向：

Python的方向可以选一选，选好方向才能对应去学习内容，先了解下python可以从事哪方面工作，这里简单介绍一下，学习Python之后可以做什么工作：

  • Python web开发：学完Python可以做web开发，因为现在中国学习Python的比较少，而招聘Python的却非常的多。所以Python web是一个非常的选择方向。

  • 数据分析：现在无论是哪个行业的，做数据分析的人似乎都离不开Python，因为Python给他们带来的工作效率是非常的大。

  • 自动化测试：一切关于自动化的东西，似乎Python都可以满足，Python可以满足大多数自动化工作，提升工作效率。

  • 人工智能：下个时代就是人工智能时代，很多人都在关注，而我们的Python同样可以做人工智能，这是一个潜力最大的选择方向，所以学习Python不会错。

对于零基础的同学而言，要明确你学习 Python 仅仅是为了满足好奇心，还是兼职，还是有工作需要，转行需要。不同的目标，所选用的学习内容和相应的学习方法也就会有所不同。

比如，仅仅是好奇，其实就不需要系统学习 Python 的知识了，简单做个了解，能够根据老师的操作，做出一些能满足好奇心的小案例即可。但如果是工作的需要，那就需要踏踏实实打牢基础了。所以，决定自学时先想清楚为什么学，这很重要。
