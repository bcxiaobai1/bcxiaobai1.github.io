JavaWeb社区医院远程挂号系统【源码开源】可做毕设

大家好，今天小强给大家分享一个JavaWeb社区医院远程挂号系统，源码开源给大家了。

链接：https://pan.baidu.com/s/1NQDvGPcoE03tutcYoMhwDg 
提取码：htzg

该系统采用java来实现主体功能，且实现了普通用户与个管理员用户不同的权限操作功能，普通用户可以在登录后实现用户信息管理、挂号系统管理、挂号报表统计；管理员登录后可以实现用户信息管理，挂号系统管理，挂号报表统计，注销登录

数据库采用MySQL，分别设计有用户表，病人信息表，挂号系统表三个主要的表

服务器采用Tomcat进行部署

前端由原生html实现

整套系统基本实现了一个完整毕业设计该有增删改查功能，用来作为毕业设计，进行一些简单的修改与调试，用作毕业设计还是很nice的

package org.com.model;

/**
 * User类 id userName password
 * @author Administrator
 *
 */
public class User {

	private int userId;
	private String userName;
	private String password;
	private int role;
	
	public User() {
		super();
		// TODO Auto-generated constructor stub
	}

	public User(String userName, String password) {
		super();
		this.userName = userName;
		this.password = password;
	}
	
	public String getPassword() {
		return password;
	}
	public int getUserId() {
		return userId;
	}

	public String getUserName() {
		return userName;
	}
	public void setPassword(String password) {
		this.password = password;
	}
	public void setUserId(int userId) {
		this.userId = userId;
	}
	public void setUserName(String userName) {
		this.userName = userName;
	}
	public int getRole() {
		return role;
	}
	public void setRole(int role) {
		this.role = role;
	}
	
}
