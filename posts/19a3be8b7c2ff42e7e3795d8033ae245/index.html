开源大新闻

GitHub 首席运营官离职

12 月 4 日，GitHub 首席运营官 Erica Brescia 宣布离职。这也是继上个月 GitHub 换帅之后，其内部的又一大人事变动。Erica 在离职声明中对 GitHub 并肩作战的 Hubbers 表示感激，其写道，“我很高兴为 Hubbers 在@ashtom（GitHub 现任 CEO）的领导下肯定会继续实现惊人增长的下一阶段加油助威。” Erica 还表示将在 12 月做一些休整来陪伴家人，然后再投入新一轮的事业当中。

亚马逊前高管加入谷歌，并担任副总裁

12 月 5 日，亚马逊前智能语音系统 Alexa 负责人、LivePerson 首席技术专家 Alex Spinelli 宣布加入谷歌，担任副总裁一职。Spinelli 于 2018 年从亚马逊的智能语音系统 Alexa 部门加入 LivePerson，在那里，他致力于为电子商务公司的数字助理提供支持的软件。今年 5 月，LivePerson 宣布与谷歌云建立合作伙伴关系，以整合他们的工具，其旨在提高联络中心代理效率。Spinelli 目前正在致力于与谷歌的工作计划，包括为构建人工智能系统而开发的软件 TensorFlow。（彭博社）

小米四足机器人“铁蛋”（CyberDog）背后的智能大脑——Ubuntu

今年 8 月，小米在推出 Mi Pad 5、Mix 4 等硬件智能设备的同时，也发布了一款机器人——铁蛋（CyberDog）。近日，Ubuntu 开发的赞助公司 Canonical 透露，Ubuntu 18.04 将成为小米“铁蛋”的操作系统。小米项目经理表示，开源是“铁蛋”的核心，它确保了该产品的兼容性和可持续增长，让开发者可以自由地突破界限。

SiFive 开发新的 RISC-V CPU，向三星、苹果发起挑战

12 月 2 日，SiFive 的最新旗舰 RISC-V CPU P650 发布。P650 是继 SiFive 6 月份发布的 P550 之后的又一款产品，它在硬件中提供了虚拟化支持。P650 提供的是一个应用程序核心，用户可以授权将其放入单片系统，并在上面运行 Linux 和其他操作系统。比较有意思的是，SiFive 在 P650 中实现了 RISC-V 的虚拟机管理程序扩展，如果其物理内存保护 (PMP) 功能在内核中实现，则在所谓的机器层面上运行的虚拟机管理程序可以将可用的 RAM 拆分为多个分区，并在每个分区中运行一个主管级别的客户操作系统内核。值得一提的是，这一做法至少在硅谷企业中是第一次尝试。 （The Register）

华为与 Oniro OS 宣布将共建未来物联网生态
华为、Oniro OS、SFScon 和 NOI Techpark 宣布将共建未来物联网设备的生态系统。为了实现这一目标，物联网设备需要相互通信，并以强大的人工智能为用户提供设备。华为开源技术中心 (OSTC) 主任 Davide Ricci 解释说，目标是精确地创造兼容性，让国内外企业共同打造开源世界，提供平等的机会和精准的技术给大家，为社区做出贡献。据悉，该项目由 Eclipse 基金会管理，该基金会于 2021 年 10 月 26 日正式发布 Oniro 项目，供社区采用。据悉，Oniro 是 OpenHarmony 的独立开源实现，该项目和工作组立志成为已建立的物联网和边缘操作系统的透明、供应商中立和独立的替代方案。（HC、IT之家）

Davide Ricci

达摩院 AliceMind 开源体系再添新锐，首个中文表格预训练模型发布

12 月 2 日，达摩院深度语言模型体系 AliceMind 发布中文社区首个表格预训练模型 SDCUP。SDCUP 是用于表格理解的预训练模型，其表格是应用普遍的结构化数据，也是智能对话系统和搜索引擎的重要答案来源。新兴的表格问答技术，可将自然语言转换为查询语句，使用户能通过简单问句直接与表格数据库交互，具有广泛应用前景。本次，达摩院对话智能团队提出了首个中文表格预训练模型 SDCUP，其基于“模式依存”方法，通过模型直接预测自然语言与表格结构内容的关键词映射，提升了表格问答的准确率。（天下网商、电商报）


开源软件专区

JetBrains 为 Kotlin 推出跨平台 UI 框架

JetBrains 正式推出 Compose Multiplatform 1.0，其有望使用该公司的 Kotlin 编程语言加快桌面、Android 和 Web 应用程序用户界面的开发。该公司在 12 月 2 日表示，Compose Multiplatform 已达到稳定状态，可用于构建生产级应用程序。 Kotlin 的 UI 框架采用声明式和反应式方法来构建 UI，允许跨平台共享 UI 代码并节省开发人员从处理 UI 更新逻辑。（InfoWorld）

CentOS Stream 9 发布

近日，CentOS 官方宣布了 CentOS Stream 9 的可用，它也是 CentOS Stream 发行版的最新主要版本。CentOS Stream 是一个持续交付的发行版，提供 Red Hat Enterprise Linux (RHEL) 的每个单点发行版，它是与 RHEL 工程团队以及 CentOS 社区合作开发的。为了让 CentOS Stream 与 RHEL 本身一样基本稳定，Stream 的每个主要版本都从 Fedora Linux 的稳定版本开始。在 CentOS Stream 9 中，从 Fedora 34 开始，这是构建 RHEL 9 的相同代码库。

免费 3D 软件 Blender 3.0 已经提高性能

自上次主要版本发布十年后，开源 3D 软件 Blender 3.0 版出现了。Blender 是专业的自由及开放源代码三维计算机图形软件，现在 Blender 3.0 可以在 Windows、Linux 和 MacOS 上免费下载。Blender 3.0 的运行速度比其前身更快，该程序能更快地打开具有许多链接的项目。由于自适应采样的改进，Blender 3.0 中的开发人员推荐了一种工作流程，其中指定所需的质量，而 Cycles 可以根据现有的图像噪声自行决定每个像素发送的光线数量。Blender 3.0 最大的创新是现在可以使用音序器预览窗口中的小部件轻松旋转、缩放和移动图像和视频。通过这项创新，视频序列编辑器现在几乎可以像传统视频编辑程序一样轻松操作。

Code Runner for Web：浏览器下直接运行 Python 的插件

Code Runner 是一款免费的 Moodle 开源题型插件，可以运行学生提交的程序代码，以回答多种不同语言的各种编程问题。时至今日，它的下载量已经超过千万，支持 C、C++、Java、JS 等超过五十多种编程语言。它能运行当前活动文本编辑器的代码文件，同时可以通过文件资源管理器的上下文菜单运行代码文件，还有代码一键运行、通过在集成终端中运行代码来支持 REPL 等特征。同时该运行器最大的亮点是，在浏览器环境下可以直接运行 Python。
