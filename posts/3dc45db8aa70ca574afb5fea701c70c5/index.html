Eolink x CSDN｜战略合作，共建API全生命周期开发生态

近日，国内最早的 API 全生命周期管理（Full Life Cycle API Management）理念推动者和国内API整体解决方案的领军者Eolink与全球知名中文开发者社区CSDN签署战略合作协议，双方将在API全生命周期管理服务和研发效能提升方面开展深度合作，共同打造API开发者社区，引领API-First（API优先）时代。

API 作为不同应用程序和系统之间联通的桥梁，随着数字化浪潮的到来，以及研发前后端的分离，已经全面迎来了API-First时代。目前已有多数开发团队使用 API-First 设计理念，围绕着 API 来进行产品的迭代，包括API生成、管理、测试、运维、监控和发布等，对API进行全生命周期的管理。

API- First是一种设计理念，更是一套研发方法论。此次战略合作，Eolink与CSDN双方正是基于对API- First的一致看好，利用各自在技术、知识、内容和资源方面的优势，共同推广从需求到实现交付过程以API为中心的设计理念，以及打造围绕API的设计、实现、测试、上线、在线化文档、维护升级等全生命周期方法论和最佳实践，持续传递Eolink平台价值，共同推动API-First理念深入各行各业，全面提升中国数字化转型中面临的效率挑战。

Eolink CEO刘昊臻表示“API 已经成为企业不可或缺及推动业务高速增长的数字资产，无论是企业数字化转型、大数据、区块链，还是如今备受关注的人工智能，API作为获取数据和服务的渠道在其中发挥了至关重要的推动作用。很高兴 Eolink 能够和CSDN 达成战略合作，并且在接下来的时间里利用各自在技术、知识、内容和资源方面的优势，共同为全球开发者提供领先的API理念和产品、共同引领API-First时代。”并表示让我们在一个更好的数字化时代相遇，而API能够成为我们之间的桥梁。

CSDN战略合作负责人闫辉表示：“CSDN一直致力于为开发者提供最前沿的技术资讯和最优质的内容，Eolink既是 API 全生命周期管理领军的厂商，也为开发者提供全面的API 生命周期管理的解决方案。两家公司的战略合作将共同探讨和打造专业的开发者社区，让更多开发者能够通过社区来了解和使用Eolink的技术产品，更好地管理和开发API。”

关于CSDN

CSDN（中国开发者网络）创立于 1999 年，是全球知名中文开发者网站。秉承成就一亿技术人的使命，为 IT 技术人成长及科技企业发展，提供开发者生态的全方位服务。CSDN 在社区基础上，通过知识云、人才云、开发云三大服务，赋能开发者、研发团队及科技企业在 IT 知识学习、人才招聘、研发效能与协同管理等方面的高速成长与发展。

关于Eolink

Eolink 成立于2017年，是国内API整体解决方案的领军者、国内最大的API在线管理服务平台。秉承着“万物互联”的理念，Eolink致力于打造一站式、智能化的API全生命周期解决方案，为企业用户提供完善的API规范化治理、API研发流程优化、API性能和安全保障、API数据服务开放及交易等创新服务。 截至目前，Eolink已为全球超过50万开发者及3万家企业用户提供高效、 安全、快速的API管理服务，客户已覆盖金融、电信、汽车、先进制造等多个行业，感谢泰康人寿、易方达、小鹏汽车、理想汽车、奇安信、商汤科技等客户选择并实施Eolink提供的API解决方案。
