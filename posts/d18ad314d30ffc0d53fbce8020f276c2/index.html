开源项目推荐

Kamaji

Kamaji 可以大规模地部署和运行 Kubernetes 控制平面，而只需承担一小部分操作负担。Kamaji 的特别之处在于，控制平面组件是在一个单一的 pod 中运行，而不是在专用机器中运行。这种解决方案使运行多个控制平面的成本更低，更容易部署和操作。

Robusta KRR

Robusta KRR（Kubernetes Resource Recommender）是一个用于优化 Kubernetes 集群中资源分配的 CLI 工具。它从 Prometheus 收集 pod 使用数据，并建议 CPU 和内存的请求和限制。这降低了成本并提高了性能。

Kadalu

Kadalu 是一个在容器生态系统中提供持久存储的项目。Kadalu 操作员根据配置部署 CSI pod 和 gluster 存储 pod。你将通过在 CSI 中实现的 API 获得你的 PV 服务。

云原生动态

Dapr v1.11 发布

日前，Dapr v1.11 发布。该版本有诸多功能更新，以下是该版本的亮点：

  • 配置 API 现在达到 v1 稳定版
  • 服务调用现在可以调用非 Dapr 端点（预览功能）
  • Dapr 工作流更新（预览功能）
  • 加密构建块（预览功能）
  • 选择 Dapr sidecar 的构建方式：所有组件或仅有稳定组件
  • Dapr 仪表盘不再与控制平面一起安装
  • Windows Server 2022 容器 Image
  • 用于改进本地开发的多应用运行进行了改进
  • Actor 状态 TTL（预览版）

Azure Linux 发行版 GA

微软最近宣布面向 AKS 的 Azure Linux 容器主机全面上市。在 GitHub 上可用 CBL-Mariner 项目代号，轻量级 Linux 发行版仅包含在云环境中运行工作负载所需的包。

适用于 AKS 的 Azure Linux 容器主机是一种开源 Linux 发行版，可作为 Azure Kubernetes 服务（AKS） 上的容器主机使用。新发行版旨在为 AKS、AKS-HCI 和 Arc 产品提供可靠性和一致性。Azure Linux 节点池可以部署在新的集群中，也可以添加到现有的 Ubuntu 集群中。也可以将现有的 Ubuntu 节点迁移到 Azure Linux 节点。

Amazon EKS 支持 Kubernetes v1.27

AWS 宣布支持 Kubernetes v1.27，称为 Chill Vibes，用于 Amazon EKS 和 Amazon EKS Distro。在这个版本的 Kubernetes 中，有很多新的功能，其中一些功能对集群有潜在的破坏性。
