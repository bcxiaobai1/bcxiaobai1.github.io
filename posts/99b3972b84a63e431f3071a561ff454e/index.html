Element Black 打造 NFT 新形式

大家好，我是晴天defi，今天给大家分享一款 Social-Fi 平台游戏 Pixel-Infinity。

Pixel-Infinity 是 NFT 开发者平台 Element.Black 目前推出的第一款游戏，Element.Black 是一个以明星社交为基础的 NFT 开发者平台，在这里，用户不仅可以自行创作元宇宙中所有天马行空的虚拟物品，还能通过去中心化存储智能合约来完成用户创作内容的 NFT 标记和存储，以及通过为名人和粉丝创建的 DAO，完成去中心化的社交管道构建。Pixel-Infinity 受到 Reddit 上爆火游戏“像素格子”的启发，并在原始游戏的逻辑基础上新加了区块链技术和 Token 经济学的概念。

Element.Black 生态系统有包含 NFT 游戏、Defi、电子钱包、公链等。目前已推出了 Social-Fi 平台游戏 Pixel-Infinity，并在香港完成了两次公测，该游戏可以用邮箱登录，也可以用钱包登录，如使用小狐狸、TP等钱包登录，首次登录的用户都可以获得100个点的奖励。用户和明星或者是艺术家的共同创作是在画布上完成的，我们还可以在个人画布上进行创作，同样可以 Mint 成 NFT。

Pixel-Infinity 除了可以创作和 Mint 成 NFT 之外，还可以对 NFT 进行拍卖，也可以在聊天室和其他用户进行交流，还可以在 NFT 画廊欣赏以往每隔8小时自动保存的共同创作 NFT。

让我们再来看一下 Pixel-Infinity 怎么实现“边玩边赚”机制，系统将会生成一个全尺寸的 NFT 并在 OpenSea 上拍卖，拍卖所得将分配给所有创作者，系统还将发放50000个 ELT 代币以奖励所有创作者。

那 Steaking ELT 代币有什么好处呢？也就是质押代币有什么好处呢，Steaking 类似于挖矿，你可以获得高达16.5%的年化回报率。例如，如果你质押了10000个 ELT 代币，你就可能每8个小时收到100个免费像素点，对此是没有上限的。
