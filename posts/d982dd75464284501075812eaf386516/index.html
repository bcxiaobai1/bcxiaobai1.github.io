NFT 的 10 种实际用途

NFT最终为我们提供了证明数字资产所有权的解决方案。在过去两年中获得了广泛的欢迎，机构、公司和组织继续采用 NFT 作为筹集资金的工具。毫无疑问，大品牌、公司甚至政府机构都看到了实施区块链技术的潜力。科技巨头 Meta将很快将 NFT 整合到他们的平台中，许多其他知名科技公司也想分一杯羹。NFT 在 2021 年成为一个价值 400 亿美元的市场，进入了投资者和创造者的眼中。到目前为止，这项技术似乎令人印象深刻，毫无疑问，NFT 是科技行业的一项有意义的进步。我们甚至听说市场分析师预测到 2030 年 NFT 市场的价值将超过 2300 亿美元。

 事实是……现在还为时过早。世界上不到 1% 的人口拥有 NFT。这仍然是一个新领域，而且 NFT 很可能会经历许多试验、错误和错误，直到该技术成为主流。换个角度来看，在上一次牛市的高峰期，有 2860 万个活跃钱包交易 NFT。这只是人口的 0.03%（或者更少，考虑到一些用户有多个钱包）。

NFT 就像一段浪漫的关系。它仍然是新的，我们正处于“彩虹和蝴蝶”阶段。我们这些 NFT 爱好者深深地爱上了这项技术的潜力，有时我们看不到它丑陋的一面。就像在一段关系中一样，在炒作消退和现实袭来之后，爱是不够的。我们需要一个更深刻的理由来保持这种关系。有理由坚持它，拥抱它，让它成为我们日常生活的一部分。这同样适用于 NFT。在炒作结束后，NFT 需要一个有效的理由来保持相关性：实用的、现实生活中的效用。

NFT 的价值与它的效用直接相关（或者换句话说，拥有它的主要好处）。为我们日常生活中遇到的问题提供解决方案的少数 NFT 项目最终将推动这项技术更接近主流采用。

当 NFT 成为一种社交需求而不是奢侈品或爱好时，我们就会体验到 NFT 的真正潜力。

市场分析师推测，它们可能需要长达十年的时间才能成为我们日常生活的一部分。然而，随着数字世界不断呈指数级扩张，它所花费的时间可能比这要少得多，而且这项技术的起飞速度可能比你预期的要快。

在那一刻到来之前，重要的是对 NFT 今天的位置以及 NFT 的实际用途有一个宏观的概述。

NFT 的 10 种实际用途将在未来 10 年内改变我们的社会

1. 法律文件和认证

我们是一个日益无纸化的社会。数字文档、数字签名和在线收据在过去十年中变得非常流行。区块链与法律文件的真实性完美契合，如果实施得当，前景广阔。法律文件将变得更快、更高效，并有望更加透明。我们看到最近的一个案例，英国法官允许通过 NFT 空投进行诉讼。假设护照和国民身份证等身份证件成为存储在区块链中的完全数字化资产。在这种情况下，很容易将我们的身份与法律文件相关联，例如 NFT 形式的结婚证书，通过访问个人的官方数字 ID 即可公开查看。

2. 通行证

活动票务和 NFT 就像花生酱和果冻一样。航空公司、商业活动和娱乐业等行业最终可以依靠安全、透明的系统，更好地控制机票、通行证和会员资格。最终，我们将使用我们的智能手机无缝地以 NFT 的形式交易票证。

比如说，皇家马德里，最大的足球俱乐部之一。由于需求量大，他们的门票相对昂贵，二级市场存在欺诈和假票的问题。最终，像皇马这样的俱乐部可能会以 NFT 的形式提供门票，买家将能够在区块链中验证其真实性，避免在二级市场上被骗。此外，由于智能合约，俱乐部可能会在二级市场上对每笔销售设定 2% 的特许权使用费。这样一来，每次将球票无限期地转让给新所有者时，俱乐部都会从版税中额外获得 2% 的收入。这开辟了一个全新的收入来源，如果没有 NFT，这将是不可能的。

3. 不可转让代币

无法从钱包转移到钱包的 NFT 在学术文件验证方面具有巨大潜力。大学最终将能够以 NFT 的形式授予完整的学位，该学位不能转让给他人。其真实性在区块链中得到官方验证，并得到其他机构的认可。也许最终，我们将看到一个 web 3 版本的 Linkedin，专业人士将在其中公开展示他们的教育 NFT 成就。

4. 出席证明

出席证明协议 ( POAP，Proof Of Attendance Protocol ) NFT 是一种特定类型的代币，可证明个人参加了物理或数字世界中的活动或体验。POAP NFT 的主要目的是证明人们参加了特定的学术课程、商业研讨会，甚至是体育马拉松等，仅举几例。最终，出席证明将成为普遍现象。我们将把它作为参加现实生活中的活动的一项要求，在这些活动中，人们可能需要在活动入口处扫描他们的智能手机上的数字身份证。作为回报，他们将自动获得一份空投到他们钱包的出席证书，该证书可能会也可能不会转让给其他人。

5. 实物资产代币化

当今最大的问题之一是证明物理和数字资产的真实性。假冒行为正在增加，预计到 2022 年将超过 3 万亿美元。实物资产的代币化对于这项技术的长期成功至关重要，而 NFT 充当了物理世界和数字世界之间的桥梁。在接下来的几年里，我们将能够验证一件昂贵的珠宝是真还是假。大型奢侈品牌可能会在其供应链中实施区块链技术，以跟踪库存和实物收藏发布。珠宝品牌可能会以 NFT 的形式出售与数字所有权证书相关联的产品，实物资产将能够通过加密钱包在店内购买，收据将被记录为 NFT。

6.身份确认

出示所有权证明与个人身份直接相关。NFT 公开显示特定资产的所有者以及购买该资产的金额。一旦区块链成为常态，知识产权、专利和商标将与个人数字身份证相关联。目前，法律文书工作缓慢、效率低下，而且在许多情况下非常官僚。让我们以房地产为例。假设一个人决定购买一套漂亮的公寓，他将能够使用个人的加密货币钱包在一次交易中支付全部金额，并立即以 NFT 的形式收到财产所有权的合法证明。一旦受到监管，政府就可以验证其是财产的实际所有者，交易将变得非常透明。

7. 治理与投票

在许多国家，投票过程效率低下、不诚实，在某些情况下还存在腐败。选民在访问投票中心时必须携带带照片的身份证件和居住证明。NFT 可以帮助解决这个问题，因为它们可以为没有实物文件证明他们是谁以及他们在该国居住的地方的人提供数字身份。将官方数字 ID 存储在其市政当局合法认可的区块链中，将促进投票过程并产生更加透明的投票系统。NFT 将减少欺诈并作为投票者及其投票的记录。

多亏了区块链，去中心化自治组织（DAO）的创建作为一种治理系统变得非常流行。在 DAO 中，没有中央权威；相反，权力分配给代币持有者，他们集体投票与特定决定相关。这将成为管理项目、公司甚至政党的规范，其中项目的方向将由特定组织的代币持有者的集体决定决定。展望未来，政府甚至可能以 DAO 的形式运行，变得更加去中心化，权力将分配给选定的人。

8. 物流与供应链

物流是关于跟踪资产的，而区块链是完全透明和高效供应链的理想解决方案。公司不断采用 NFT 来跟踪他们的实物库存。最终，我们将生活在一个产品数据、跟踪号和物流都在区块链上完全管理的时代。借助 NFT，公司可以发布与货物和交付服务相关的智能合约，从而实现从原材料到最终用户的供应链中每个项目的完全透明和可追溯的记录。所有这些数据对于在全球范围内提高与物流和运输相关的生产力和效率至关重要。

9. 房地产

NFT最吸引人的方面之一是它们能够与高价值的实物资产相关联，例如私人和商业房地产。我们在房地产市场面临的最大问题之一是文书工作、许可证、公证人以及银行和代理等许多中介机构。可以在两方之间的单一交易中简化的事情有许多额外的成本。NFT 将通过在单笔交易中直接转让所有权来帮助加快购买或出售实物资产。

碎片化一词在房地产市场将变得非常普遍。要了解碎片化的工作原理，让我们使用一个简单的示例。例如，假设您决定投资房地产市场，并且想以 100 万美元的价格在海滩上购买一座漂亮的别墅。但是你没有足够的资金，所以你决定请你的四个朋友和你一起买别墅。该财产现在可以分成 5 个 NFT，每个价值 25 万美元（总计 100 万美元）。这样，您就可以成为业主之一，而无需自己购买整栋房子。换句话说，它降低了进入房地产市场的成本，让其他人更容易进入。另外，如果你们都同意的话，你将来可以通过购买他们的 NFT 来扩大自己对房子的份额。

此外，该物业可用于出租，产生被动收入，平均分配给所有代币持有者。所有所有权和财产历史都可以记录在区块链中，任何人都可以公开验证。NFT 在房地产市场上的可能性是无限的，并且可能在未来对银行系统产生重大影响。

10. 游戏和游戏内资产

据估计，游戏行业的市值为 700 亿美元。那是在没有实施 NFT 的情况下。通过增加数字游戏资产的货币化和跟踪，游戏行业将成为未来十年 NFT 擅长的领域之一。随着元宇宙的整合，游戏行业可能是年轻一代中采用率最快的行业。

Play-to-Earn等游戏内激励措施在发展中国家越来越受欢迎。借助 NFT，游戏玩家可以通过玩游戏获得报酬，而不受位置限制。这为发展中国家的千禧一代带来了巨大的机会。我们已经看到玩家平均每月收入约为 1,500 美元的例子，超过了他们在本国的全职工作工资。

  • 尝试一些事情来获得经验。随着区块链的成熟，有经验并且之前被证明知道如何适应和改进的人将在 web3 公司中占据主导地位。经验将在 NFT 领域变得极其宝贵。现在我们仍处于这项技术的早期阶段，现在正是学习的好时机。

总结一下

NFT 还有很长的路要走，在被主流采用并成为我们日常生活的一部分之前还有很大的增长空间。只有当 NFT 的实用程序具有现实生活中的实际用途时，我们才会看到 NFT 的真正潜力，这在社会中具有重要的目的和社会需求。在那之前，我们必须经历大量的发展、艰苦的工作、汗水和泪水。但最终，它会得到回报。
