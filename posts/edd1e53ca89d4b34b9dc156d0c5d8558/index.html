MEXC Global科普：Polkadot波卡平行链插槽拍卖是什么

Polkadot波卡平行链插槽拍卖在即，一文带你了解波卡的架构以及插槽拍卖的机制。

波卡区块链架构

波卡之所以成为如今备受关注的跨链明星项目，主要与其创始人Gavin Wood博士带领的技术团队以及其宏大的Web3.0愿景有关。

2015年，Gavin Wood离开以太坊之后创建了Parity Technologies公司，并且提出了关于“Web3.0”的构想。

2016年，随着波卡白皮书《波卡：可扩展的的异构多链框架》的发布，以及平行链的概念产生，标志着波卡项目的推出和诞生。2018年，Gavin针对波卡推出了可扩展、模块化、通用的区块链技术框架Substrate。

波卡在架构上主要有三部分，分别是平行链、中继链、转接桥。中继链（Relaychain）是一条基于substrate创建的区块链，作为波卡网络的中心，负责信息交互的验证，保护和协调平行链，维护整个网络的安全；平行链（Parachain）通过插槽接入中继链运行，并与中继链进行交互，负责处理各种“琐事”，目前来看，这个平行链既可以是智能合约平台、DeFi中心，也可以是任何专注于不同业务的协议。

转接桥主要用来帮助波卡系统链接外部的区块链系统，允许波卡网络和独立的区块链网络通信。这些独立的区块链网络不通过波卡中继链进行安全验证，而是使用自己的安全机制，例如比特币和以太坊网络。而与拍卖相关的还有一个关键构成——平行链插槽（Parachain Slots），它是中继链和平行链的接口。要想共享波卡网络的安全，与其他平行链进行业务交互，必须通过插槽来实现。

插槽分为社区测试型插槽（community beta slots）、商用插槽（commercial slots），20%的槽位预留给 Web3 基金会来部署和支持，作为社区测试型平行链促进整个波卡生态的发展，余下80%的插槽作为商用插槽，通过拍卖方式来开放。

基于这些，波卡是一条可扩展的异构多链的区块链，波卡网络的主要作用其一是为区块链提供可操作性，其二是为区块链提供安全性。

波卡网络的治理主要有四种角色：提名者（Nominator）、验证者（Validator）、核对者（Collator）、钓鱼人（Fisherman）。

持有DOT的用户可以以提名者的角色参与治理，通过质押代币并投票支持验证者；而验证者则负责封装波卡网络上的新区块，要成为验证者需要质押一定的DOT代币，也可以通过由提名者委托的方式当选；核对者主要帮助验证者生产有效的平行链区块，负责完全同步数据库，运营全节点（可以把验证者理解为PoW矿池，核对者理解为矿工）；钓鱼者则有点类似于轻节点，主要通过发现行为不端的验证者来获得丰厚回报。

总的来说，波卡遵循去中心化原则主要靠链上投票。

除了以上几个重要角色，波卡还成立了理事会，理事会的成员由选举产生，可以优先投票，也有权直接否决提案。

插槽拍卖机制和方式

从波卡的架构可以看出，区块链项目可以通过平行链或者是转接桥的方式接入波卡进行交互。要成为波卡的平行链，则必须通过插槽来接入。但平行链数量只有100 条左右，且一次只释放一条，故而波卡设计了拍卖机制允许项目方通过竞拍来获得插槽的使用权。

波卡的平行链竞拍所采用的是“蜡烛拍卖+随机参数”机制，在区块链上模拟“蜡烛拍卖”的过程中，还结合了“二阶密封价格拍卖”的思想。那么具体怎么运作呢？

“蜡烛拍卖”起源于16世纪出售船只时，从决定拍卖开放期的“一寸蜡烛”而名。当蜡烛火焰熄灭时，拍卖会突然终止，此刻的竞价者将获胜。当蜡烛拍卖在网上使用时，通常需要一个随机数来决定终止时间。

但平行链插槽拍卖又与普通蜡烛拍卖略有所不同，因为它不使用随机数来决定其开盘阶段的持续时间。而是有一个已知的开放阶段，“在正常结束时，追溯性地确定在过去的某个时候结束”。

波卡将使用一个基于VRF的随机信标，来追溯确定拍卖的“结束时间”。也就是说，蜡烛拍卖的过程中每个人都知道竞拍的状态，包括其它平行链的出价情况，但没人能提前知道拍卖结束的时间。这样能有效防止“拍卖狙击”，且促使项目方尽早参与出价。

整个拍卖过程被分为安全期和随机期两个阶段，两个阶段内都可以自由质押DOT，在安全期内，项目方有更大可能会竞相出价，等进入随机期后，拍卖会根据链上随机数随时落锤结束拍卖，届时，谁质押的DOT数量多，谁就能租到这个插槽。

插槽持续时间上限为2年，每个租赁周期为6个月。参与出价的项目方必须竞拍单个或相邻的多个租赁期，这意味着它们可以通过租赁一个连续的插槽将其租约延长到2年的插槽期限。中标人需要在相应周期内锁定（质押）其DOT。

 

关于波卡 Crowdloan （众贷）

平行链在竞拍插槽时，项目方可以通过 Crowdloan 的方式向各自的社区筹集 DOT，DOT 持有者可以选择将自己的 DOT 投（Contribute）给任何自己看好的竞拍项目。

众贷项目及奖励机制一览

Acala

Acala 是全球首个去中心化开放式金融联盟、波卡 Polkadot 生态金融中心，旨在联合波卡 Polkadot 生态系统创建开放式金融框架，为全球加密资产带来金融稳定性，流动性与可及性。

奖励机制

总竞拍奖励：ACA 总量的 17% (1.7 亿枚 ACA、15% 的基础奖励+ 2% 的其他奖励）

Astar Network

Astar Network 从 Plasm 改名，由 Staked Technologies 开发支持，允许开发人员轻松无缝地将 Plasma 功能添加至 Substrate 当中。

奖励机制

总竞拍奖励：ASTR 总供应量的 20%（14 亿枚 ASTR 代币）

Clover Finance

Clover Finance 是一个跨链智能合约平台，致力于提供易于使用的区块链基础架构，并为基于 Substrate 的应用程序创建一站式兼容 EVM 框架，以降低开发人员的门槛和成本。

奖励机制

总竞拍奖励：Clover 总供应量的 20%（2 亿枚 CLV 代币）

Crust Network

Crust Network 目标是建立去中心化的云生态系统，实现了分布式存储的激励层协议，适用于多种存储层协议（例如 IPFS），并为应用层提供支持。Crust 体系结构还具有支持去中心化计算层，和构建去中心化云生态系统的能力。

奖励机制

总竞拍奖励：Crust 总供应量的 5%（1 百万枚 CRU 代币）

Darwinia Network

达尔文网络是基于 Substrate 开发的跨链桥接网络，专注于建设未来资产互联网络，包括非标资产拍卖市场、稳定币跨链、资产交易兑换等领域。

奖励机制

总竞拍奖励：RING 总供应量的 10%（2 亿枚 RING），8000 KTON

Equilibrium

Equilibrium 是多合一的可互操作 DeFi 集合，首个将信用借贷、合成资产生成与交易相结合的跨链数字资产市场。

奖励机制

总竞拍奖励：EQ 总供应量的 10%（1200 万枚 EQ 代币）

Litentry

Litentry 是一个隐私保护的 DID 聚合器，在 Polkadot 生态系统中发挥独特的功能，为多链身份数据提供 Liquidity、互操作性和一致性。

奖励机制

总竞拍奖励：Litentry 总供应量的 20%（2 千万枚 LIT 代币）

Moonbeam

Moonbeam Network 构建跨链连接的 DApp，提供与以太坊兼容的智能合约平台，可轻松构建本地可互操作的应用。其由波卡 Polkadot 网络提供支持，由 PureStake 进行开发。

奖励机制

总竞拍奖励：GLMR 总供应量的 10%（1 亿枚 GLMR 代币）

Manta Network

Manta 的首个产品 MantaPay，是能够将波卡生态整个中继链和平行链如 Acala 上的稳定币，Moonbeam 上的 EVM 数字资产隐私化，将波卡的互操作性发挥到极致。Manta 最终会建立基于 zkSNARK 的隐私智能合约平台，可以搭建和部署各种去中心化应用，例如隐私交易平台 MantaSwap 以及隐私 NFT 平台。

奖励机制

总竞拍奖励：MANTA 总供应量的 15.6%（1.56 亿枚 MANTA 代币）

Parallel Finance

Parallel Finance 是建立在波卡生态系统上的去中心化贷款协议。其独特的方法将使用户通过同时抵押、借贷，获得双重奖励。

奖励机制

总竞拍奖励：PARA 总供应量的 15%（15 亿枚 PARA 代币）

SubDAO

SubDAO 是专注于解决 Substrate 链多样化治理的 DAO 平台，基于 Substrate，致力于创建可让任何组织创建和管理跨链 DAO 组织的平台，也是 PAKA Labs 加速器孵化的首个项目。

奖励机制

总竞拍奖励：GOV 总供应量的 10%（3 亿枚 GOV 代币）
