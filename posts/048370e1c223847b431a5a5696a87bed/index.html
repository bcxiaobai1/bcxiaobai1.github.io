破局 NFT 困境：实用型 NFT 是什么？

自 2021 年以来，NFT 作为一个风靡全球的名词，开始频繁出现在大众眼前。然而在 2017 年，**第一个在以太坊上引起流量的 Cryptokitties 应用程序就属于 NFT 概念。**到目前为止，其已发展成为一个快速发展的行业并被推向了更多的普通大众。

但在 NFT 火爆的背后，有一个缺陷一直被大众所诟病和讨论 —— 大部分人认为NFT 基本没有实际用途的。事实上，这或许是一种“偏见”。NFT 不仅可以为资产提供流动性载体，也能作为各类实物资产和数字资产出现，在商业方面、供应链和零售领域拥有众多机会，以其多元的应用场景和丰富的功能特性打破了大众对其无实际用途的“偏见”。

实用型 NFT 的定义

首先，实用的定义是有用、有利可图、有益，这种事物性质是功能性的。放到 NFT 上来说，是指 NFT向其买家和持有者提供的价值，利益或奖励类型。一旦将 NFT 与数字会员、DeFi 项目概念等联系起来后，NFT 就能发挥其实用性成为数字领域的重要资产载体。

除此之外，NFT 还有很多用例来作为许多不同用例或应用程序的数字标记，今天就来一同探索不同的实用型 NFT。

实用型 NFT 的分类

参与型 NFT：社区互动与身份交流

以大热的 SocialFi 赛道作为契机，参与型 NFT 借助其特有的数字资产属性，成为社交平台和用户身份验证通道的重要载体，用于品牌和社区内的其他用户的互动需求。在社交上来说，NFT 以徽章，GIF 和表情符号的形式出现，成为个人的独特标记；在身份上来说，NFT 使个人能够跨平台展示和拥有身份，实现身份自如变换和交互。

BAYC（无聊猿游艇俱乐部）就是较为知名的参与型 NFT，以 NFTs 的形式将**“无聊猿”角色植入元宇宙世界**，让热爱区块链的用户可以凭借这一平台实现新的身份塑造和和谐的交流沟通。

游戏化 NFT：穿戴设备与体育运动

以 NFT 赋能体育产业，产生了大量火爆的新晋体育消费品，例如建立在 Flow 链上的 NBA Top Shot已经拥有接近 200 万注册用户，交易流水接近 10 亿美金。这些火爆的体育项目 IP 结合 NFT 数字资产实现游戏交易、快闪挑战等一系列用户活动。实际上，游戏化 NFT 主要应用在可穿戴 NFT 和 游戏 NFT。

前者以虚拟服装或数字配件等形式出现，装扮用户在虚拟世界中的个人形象，同时高度仿真的成像特点可以让用户在虚拟世界中也仿佛有着真实的“打扮”；后者以**“Play-To-EarnNFTs”**的形式应用，为玩家提供特殊的角色、能力、武器和各种其他游戏内物品，通过交易盘活用户个人资产。

此外，还有一些结合 AR 技术的游戏化 NFT 应用，将增强现实的表现形式投入数字虚拟世界，让用户置身于一个全新的数字消费情境，带来新的虚拟体验享受。

资产型 NFT：独特载体与交易属性

这类 NFT 其实作为发挥 NFT 本身优势和特性而存在，可以出售、租赁、授权或转让给其他用户。任何实体产品或商业内容都可以借助资产型 NFT 实现交易流通，成为促进商品市场发展的重要手段，在这种思路下，NFT 将成为巩固数字与真实物理世界融合过程的一部分。通过添加额外的功能，如元数据、智能合约的执行，以及使用这些功能来真正推动数字资产拥有更多的多层和动态设置。

比较多的应用来自版权 NFT 和实体 NFT，版权 NFT 是数字资产的重要证明，包括商标权和转让许可等，赋予了 NFT 持有者商业使用权；实体 NFT 指的是直接跟数字资产和物品所有权相联系的保证。

2021 年 3 月 11 日，美国艺术家 Beeple 的作品**《Everydays — The First 5,000 Days》** NFT 在纽约佳士得拍卖行以将近 7 千万美元的价格售出，这场拍卖活动的背后就是实体+版权型 NFT 发挥作用的关键，这一作品也成为新晋买主的重要资产认证。

实用型 NFT 发展展望

综上分析的实用型 NFT 类型，不难看出当下区块链行业对 NFT 持续发挥数字资产优势充满了期待和想法。但是关于实用型 NFT 发展的未来，我们仍需思考如何从打造品牌心智的概念**，树立实用型 NFT 的社会地位**，为项目的实用性以及路线图设定明确、现实的期望。

一方面来说，**实用型 NFT 应当进一步扩大其访问权限与扩展空间。**可以考虑与一些 Web3 品牌合作，为代币持有者提供一系列产品访问权限。比如，Surge Women NFT 就是利用与其他品牌的合作伙伴关系为其代币持有者提供实用性 NFT 的例子，以购买 NFT 域名的优惠券、获得发展资助金的机会和访问网络峰会作为吸引点引发关注。

另一方面，**实用型 NFT 还可以利用承诺吸引突出数字虚拟货币持有者的未来价值。**拥有 Bored Ape Yacht Club (BAYC) 意味着持有者不仅有资金，还收获了虚拟世界的社会身份地位——一个头像或一件可能稀有的艺术品，等于拥有了俱乐部的会员资格，这能够充分吸引消费者打开数字世界的大门。

最终，这些实用型 NFT 发展所指，便是在用户心中树立关于 NFT 的全新概念与使用价值。

无论目前用例如何，NFT 都将成为未来生活的一部分。在不久的将来，实物资产可能会出现在元宇宙中，它们的所有权和存在在网上和现实中都可以获得证明。我们依旧有理由相信 NFT 还将在构建真实世界的问题逻辑和实用程序方面发挥重要作用。实用型 NFT 充分赋予了 Web3 时代更多应用发展的机遇和优势，让社交、交易、游戏等领域的发展与真实世界的使用场景结合，带给用户更灵活的操作体验和更实用的需求满足。
