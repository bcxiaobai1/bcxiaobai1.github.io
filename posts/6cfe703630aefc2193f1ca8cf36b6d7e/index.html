MySQL如何卸载干净以及下载、安装教程

第一次下载MySQL的同学，直接从第二步看即可

一、卸载

1、cmd下停止mysql服务

net stop mysql

2、从控制面板中找到这两个文件，双击进行删除


3、删除MySQL安装时产生的两个文件夹以及一个隐藏文件

如果当时是默认安装在c盘，没有修改地址，则文件夹在该处位置



显示隐藏的项目，点击ProgramDate

删除mysql文件

4、 进入注册表（运行 regedit），删除以下目录内容

将每一条复制，并粘贴到上方进行搜索，如果未找到，说明已经删除

HKEY_LOCAL_MACHINESYSTEMControlSet001ServicesEventlogApplicationMySQL

HKEY_LOCAL_MACHINESYSTEMControlSet002ServicesEventlogApplicationMySQL

HKEY_LOCAL_MACHINESYSTEMCurrentControlSetServicesEventlogApplicationMySQL

HKEY_LOCAL_MACHINESYSTEMCurrentControl001tServicesMYSQL

HKEY_LOCAL_MACHINESYSTEMCurrentControl002ServicesMYSQL

HKEY_LOCAL_MACHINESYSTEMCurrentControlSetServicesMYSQL

二、下载

1、进入官网
https://www.mysql.com/

2、点击downloads
3、往下拉，点击最后一句话

4、点击MySQL installer for Windows

5、点击下载，64位也是下载该32位的

三、安装

1、双击下载好的安装包

2、点击第二个安装服务器版本就好
点击Next
3、直接点击Execute执行
4、直接点击Next
5、直接点击Next

6、输入数据库root密码，要牢记该密码
点击Next

7、记住mysql服务名字为MySQL80
点击Next

8、直接点击执行，等待安装

9、安装完毕，点击Finish

10、直接点击Next

11、直接点击Finish，到此就安装完毕

四、配置环境变量

在系统变量中找到path，双击打开

点击空白处，找到bin文件夹的目录位置，粘贴进去

五、运行

用管理员身份执行cmd
查看mysql版本

mysql -V


启动mysql服务

进入MySQL

mysql -u root -p

第一次下载mysql，输入密码后会出现该情况，是因为没有初始化

可以不输入密码，直接按回车便可进入

下面我们初始化，设置密码
输入该指令，注意后面的分号

use mysql;

修改密码

ALTER USER ‘root’@‘localhost’ IDENTIFIED BY ‘密码’；


离开重新进入即可

exit
