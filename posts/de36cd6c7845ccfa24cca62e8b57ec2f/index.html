计算机网络第二章 物理层

2.1 物理层的基本概念

  • 物理层解决如何在连接各种计算机的传输媒体上传输数据比特流，而不是指具体的传输媒体
  • 机械特性：指明接口所用接线器的形状和尺寸、引脚数目和排列、固定和锁定装置等
  • 电气特性：指明在接口电缆的各条线上出现的电压范围
  • 功能特性：指明某条线上出现的电压范围
  • 过程特性：指明对于不顶用功能的各种可能事件的出现顺序

2.2 数据通信基础知识

2.2.1 数据通信系统的模型

  • 模拟信号（连续信号）：代表消息的参数的取值是连续的
  • 数字信号（离散信号）：代表消息的参数的取值是离散的

2.2.2 有关信道的几个基本概念

  • 信道分类：按照传输信号分模拟信道与数字信道，按照传输介质分无线信道与有线信道

  • 模拟信道：传送模拟信号

  • 数字信道：传送数字信号

  • 从通信的双方信息交互的方式来看：单工通信、半双工通信、全双工通信

  • 从信道上传送的信号的角度分类：基带信号、宽带信号

  • 基带信号：来自信源的信号，放在数字信道上传输（基带传输）

  • 宽带信号：把基带信号调制后，放在模拟信道上传输（宽带传输）

  • 从信号的加工手段的角度分类：数字信号、模拟信号

  • 数字数据经过数字发送器==》数字信号（这个手段叫编码）

  • 模拟信号经过PCM编码器==》数字信号（这个手段叫编码）

  • 数字数据经过调制器==》模拟信号（这个手段叫调制）

  • 模拟信号经过放大器调制器==》模拟信号（这个手段叫调制）

  • 常用编码方式

  • 基本的调制方式

2.2.3 信道的极限容量

  • 码元：一个固定时长的信号波形，代表不同离散数值的基本波形。时长内的信号称为k进制码元，该时长称为码元宽度。码元的离散状态M个时候，被称为M进制码元，一码元可以对应多个比特
  • 4进制码元==》码元状态4个==
    一个码元携带2比特（而2位比特可以表示4种状态）
  • 码元传输速率：1s传输多少个码元
  • 信息传输速率：1s传输多少个比特

2.2.3.1 奈式准则

  • C = 2Wlog2V

2.2.3.2 香农定理

  • 信噪比（dB）= 10log10(S/N)
  • C = Wlog2(1+S/N)

2.3 数据交换方式

  • 数据交换方式电路交换、报文交换、分组交换

2.3.1 电路交换

  • 在数据传输期间，源结点与目的结点之间有一条由中间结点构成的专用物理连接线路，在数据传输结束之前，这条线路一直保持
  • 建立连接==》通信==》释放连接

2.3.2 报文交换

  • 无需在两个站点之间建立一条专用通路，其数据传输的单位是报文，传送过程采用存储转发方式

2.3.3 分组交换

  • 分组交换与报文交换的工作方式基本相同，都采用存储转发方式，形式上的主要差别在于，分组交换网中要限制所传输的数据单位的长度，一般选128B。 发送节点首先对从终端设备送来的数据报文进行接收、存储，而后将报文划分成一定长度的分组，并以分组为单位进行传输和交换。接收结点将收到的分组组装成信息或报文。
