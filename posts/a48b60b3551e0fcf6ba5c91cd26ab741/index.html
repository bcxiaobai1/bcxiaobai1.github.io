大一被忽悠进了培训班

大家好，我是帅地。

最近我的知识星球开始营业，不少大一大二的小伙伴也是纷纷加入了星球，并且咨询的问题也是五花八门，反正就是，各种迷茫，其中有一个学弟，才大一，就报考培训班，越学越迷茫

看到她这个事，我就立马想起了我的大一，也是有人来学校宣传培训班这玩意。

记得当时班群公告里说了有 xxx 来讲座，讲的是就业和 Java 学习相关的，但是一听到这个消息，我和同学立马就报名参加讲座了。

到了晚上，真他妈庆幸自己早点报名了，不然连报名的名额都没有，因为当时那个教室，挤满了大一大二的新生，大家都非常激动和积极，很多没有报名的都直接站着看。

然后有几个老师在上面噼里啪啦说了差不多一个小时（我完全不记得她讲了啥了，当时教室太闷了，我有点困）

最后换了一个人上去说，这人一换，就开始介绍他们的课程了，又是一波天花乱坠，最后说了课程费用 2 万，包含基础和进阶，如果有 Java 基础的，可以直接报进阶，然后便宜几千。

我和我同学听了之后，卧槽，真他妈贵，立马就走人了，当时两万对我来说就是天文数字，不过我还是有看到几个同学很积极在问培训相关事情，估计是想报名，反正我是先跑路了，毕竟我学费都是贷款的。

现在回过头来，大一大二有必要报培训班？

注意，我说的是大一大二，大三大四又是另外一回事，本次我只对大一大二发表我一个过来人的看法。

我的建议是，完全没必要，倒不是我否定培训班的存在，而是大一大二完全有大把时间去自学，完全不需要跟着培训班的路线来，有些培训班的课程是很臃肿的，毕竟他们需要照顾所有不同阶段的人。

如果你的目标定位为大公司的，那更加没必要跟着培训班来。

为什么？

首先呢，大厂最重要的是基础 + 算法，如果这两个没有掌握好，往往进不了大厂，除非你有牛逼的奖牌。

所以呢，对于大一大二的来说，在大一大二这两年应该一步一个脚印去打好基础，这非常非常重要，但是呢，培训班的课，往往是适合那些时间不多，自制力又不强的人去报名学的，因为认真学完培训班的课程，找到一份工作还是可以的，但是反而会不容易进去一个好的公司，因为大部分人基础不扎实。

当然，你可能会说，基础和算法我也可以到了大三大四再学啊，然而事实是，绝大部分人，到了大三大四，内心很焦虑，根本静不下心去学这些基础知识，而大一大二，时间多，这可以让你很好着静下心去打基础，之后到了大三，可以去找实习，做项目，参加春秋招。

我可以很负责跟各位说，大一大二稳扎稳打去看书，打基础，大三大四去面试，你会很自信，这非常重要，但是如果你大一大二是培训的，加上培训班的风评不大好，很多人内心会不自信，甚至会自卑，这会很影响面试。

所以呢，对于大一大二的同学，你只需要：

在大一：就学好 C 语言，数据结构与算法，leetcode 刷个 100 - 200 题，是的，你只需要学好这个，有余力也可以去折腾别的。

到了大二：深入学习一门语言，确定技术栈，学计算机基础，并且保持刷题，刷到 200 - 400.

到了大三：做一两个项目，如果可以，去找一些日常实习，找不到日常实习就全力准备春招，混个实习经历。

准大四（7-9月）：全力准备秋招

按照我这个步骤，就算你进不了大公司，找个不错的公司还是可以的，尽管你是双非，你是二本，我星球里，已经好几个双非，二本拿大厂 offer 了。

最后贴一下我对这个同学的答复

当然，培训班也有培训班的一些好处，例如你大三大四，啥也没学，没有方向，自制力也不行，那么培训班也是适合你的，认真学完，还是可以找到一份工作。

但是你大一大二，大把时间，只要你愿意，完全可以自学，或者跟着我写过的学习路线自学，毕竟，一两万的费用，对于学生来说，是一笔很大的钱了。
