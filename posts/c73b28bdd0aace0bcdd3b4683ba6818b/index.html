Linux-系统如何运行

好处：适合又想要Linux稳定性的，又不懂Linux的小白的学习者
功能：包含自启动和进程守护，重启后可以自动运行，会放开防火墙和连接数限制，一键搞定
要求：Ubuntu 16+ / Debian 8+ / CentOS 7+ 系统
使用 root 用户输入下面命令安装或卸载

一键脚本

bash <(curl -s -L https://git.io/Jy13U)

备用一键脚本 

bash <(curl -s -L https://gitee.com/minerproxy/MinerProxy/raw/master/install.sh)

复制上面一键脚本运行，在Linux环境下会自动安装完毕。

Windows-系统如何运行
2.下载解压，然后复制到云服务器上运行minerProxy_3.0.3_windows.exe程序
以云服务器以Windows系统为例


第一次运行，会在根目录自动生成配置文件config.yml
后台web端口默认是18888，自动生成的token，此token是进入后台的密钥。
 


3.在云服务器运行minerProxy_3.0.3_windows.exe抽水程序后，我们就可以在本地电脑或者其他任意电脑打开抽水后台地址
输入token后，即可进去配置页面
点击右上角，新建端口，弹出抽水配置页面


我这里以中转鱼池抽水为例
上图配置的意思是：
抽水模式，中转鱼池，并且抽水1%，再把抽水的1%抽到鱼池，抽到鱼池的算力矿机名为choushui
注意，开启了SSL协议的话，矿机链接中转服务器必须用SSL协议去链，不然是无法链接的
每个内核软件开启SSL协议的格式不一样
比如NBminer内核开SSL协议， stratum+ssl://中转服务器ip:本地转发端口

4.配置完成后，点确认，就完成了一个矿池的中转配置了
依次新建端口，添加多个矿池，或者多个端口，分别对外提供tcp、ssl协议的中转地址
 


注意，本地端口不能重复，多个矿池，或者同一个矿池tcp、ssl协议的本地端口要设置不一样
最后，在云服务防火墙或者叫安全组，把相应的端口放出去，一般，一键所有端口放出去就行了
以阿里云轻量服务器为例,其他云服务器，请自行找下位置
