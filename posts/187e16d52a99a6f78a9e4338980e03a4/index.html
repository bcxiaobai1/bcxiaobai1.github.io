4.3 我的提问：如何用Python实现抓取网页内容？

ChatGPT答案：
如果你想用Python来抓取网页内容，你可以使用Python的第三方库来实现这个功能。一个常用的库是requests，它提供了一个简单的方法来发送HTTP请求并获取网页内容。

下面是一个简单的例子，展示了如何使用requests库来抓取网页内容：

插入
新建

import requests

# 要抓取的网页地址
url = 'https://www.example.com'

# 发送HTTP GET请求
response = requests.get(url)

# 获取网页内容
html = response.text

# 打印网页内容
print(html)

在上面的代码中，我们首先导入了requests库，然后使用requests.get()方法来发送HTTP GET请求，并获取网页内容。我们可以通过调用response.text来获取网页的文本内容，然后打印出来。

请注意，上面的代码仅供参考，实际应用中还需要考虑很多其他因素，比如处理HTTP响应状态码、处理网页编码等。
