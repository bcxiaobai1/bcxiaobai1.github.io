Wordpress搭建（初学者）

一、安装Apache HTTP服务

1.执行下列命令，安装Apache服务

yum -y install httpd httpd-manual mod_ssl mod_perl mod_auth_mysql

2.执行如下命令，启动Apache服务

systemctl start httpd.service

3.测试Apache服务（在Linux系统下的火狐浏览器访问）

http://ip地址

出现如下说明成功安装并且启动

二、安装MYSQL数据库

1.执行下列命令，下载安装MYSQL官方的Yum Repository

wget http://dev.mysql.com/get/mysql57-community-release-el7-10.noarch.rpm
yum -y install mysql57-community-release-el7-10.noarch.rpm
yum -y install mysql-community-server

2.执行如下命令，启动MySQL数据库

systemctl start mysqld.service

3.查看MySQL运行状态（出现running说明安装成功）

systemctl status mysqld.service

 4.执行如下命令，查看MySQL初始命令

grep "password" /var/log/mysqld.log

5.    执行如下命令，登录数据库。

mysql -u root -p

 

 复制“localhost:”之后的内容，输入到Enter password之后

6.执行如下命令，修改MySQL默认密码。

ALTER USER 'root'@'localhost' IDENTIFIED BY 'NewPassWord1.';

7.执行如下命令，创建wordpress库。

create database wordpress;

8.    执行如下命令，创建wordpress库。 执行如下命令，查看是否创建成功。

show databases;

9. 输入exit退出数据库

exit

三、安装 PHP 语言环境

1.    执行如下命令，安装PHP环境。

yum -y install php php-mysql gd php-gd gd-devel php-xml php-common php-mbstring php-ldap php-pear php-xmlrpc php-imap

2. 执行如下命令创建PHP测试页面。

echo "<?php phpinfo(); ?>" > /var/www/html/phpinfo.php

3.    执行如下命令，重启Apache服务

systemctl restart httpd

4.    打开浏览器，访问

http://IP地址/phpinfo.php

5.出现这个界面表示安装成功

 四、Wordpress安装和配置

1.安装epel源

yum install epel-release

2.执行如下命令，安装wordpress

yum -y install wordpress

3.修改WordPress配置文件

1）执行如下命令，修改wp-config.php指向路径为绝对路径

cd /usr/share/wordpress
ln -snf /etc/wordpress/wp-config.php wp-config.php

2）执行如下命令，移动wordpress到Apache根目录

mkdir /var/www/html/wp-blog
mv * /var/www/html/wp-blog/

3）执行以下命令修改wp-config.php配置文件

sed -i 's/database_name_here/wordpress/' /var/www/html/wp-blog/wp-config.php
sed -i 's/username_here/root/' /var/www/html/wp-blog/wp-config.php
sed -i 's/password_here/NewPassWord1./' /var/www/html/wp-blog/wp-config.php

4. 执行如下命令，重启Apache服务

systemctl restart httpd

5.关闭防火墙

systemctl stop firewalld
systemctl disable firewall

五、测试Wordpress

1.浏览器访问

http://IP地址/wp-blog/wp-admin/install.php。

2.    根据以下信息完成wordpress初始化配置

3. 单击Install WordPress完成Wordpress初始化

4.单击Log In进行登录。

5. 输入设置的用户名和密码

 6.    登录后，出现如下界面表示安装成功
