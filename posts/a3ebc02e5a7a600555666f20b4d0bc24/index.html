11.7版本--Audio DJ Studio for .NET是 MultiMedia Soft 开发的 .NET Windows Forms 自定义控件，可以轻松地向使用Microsoft Visual Studio编写的Winform和WPF 应用程序添加声音播放和混音功能；由于与 DirectShow 编解码器和 BASS 库的集成，两者都允许对最常见的声音格式（如 MP3、WAV、Ogg Vorbis 和许多其他格式）进行解码，Audio DJ Studio for .NET允许创建具有惊人播放功能的强大多媒体应用程序.

Audio DJ Studio for .NET可以在不同的虚拟面板上同时加载不同的声音，称为“播放器”；播放器可以比作 DJ 控制台上的物理平台，即放置要播放的黑胶唱片/CD 的地方。您可以创建一个带有许多虚拟面板的控制台，这些面板可以同时在一张或多张声卡上播放不同的歌曲。一定数量的播放器（面板）的可用性将使容器应用程序能够在不同的输出通道上混合多首歌曲，例如在不同的输出通道上播放/混合歌曲时能够播放广告位：这非常有用用于多频道无线电台自动化软件。

每个播放器都可以有单独的音量、速度、播放速率、音调和均衡器设置；每个播放器还配备了一组嵌入式和可配置的视觉反馈组件，例如 VU 表、频谱分析仪、示波器和波形。

嵌入式自动推子可以轻松地将播放列表中加载的音轨与准确且可配置的淡入/淡出混合。

.NET API 版本可用
