Java入门第一个程序：HelloWorld

JAVA入门第一个程序 HelloWorld

  1. 新建一个文件夹，存放代码
  2. 新建一个JAVA文件
    • 文件后缀名为.java
    • Hello.java
    • 【注意点】系统可能没显示文件后缀名，需自己打开显示后缀名
  3. 编写代码
public class Hello{
	public static void main(String[] args){
		System.out.println("Hello World!");
	}
}
  1. 编译 javac Java文件
  2. 运行class文件，java class文件

可能遇到的问题

  1. 每个单词的大小写不能出现问题，Java是大小写敏感的

  2. 尽量使用英文

  3. 文件名必须和类名保证一致，并且首字母大写

  4. 符号使用了中文类型也会出现错误

  1. 把程序打包成jar包并生成文档

打包成jar包

jar -cvfe Hello.jar Hello Hello.class

运行生成的jar包

java -jar Hello.jar

生成文档

javadoc -encoding UTF-8 -charset UTF-8 -d HelloDoc Hello.java

  1. 查看文档

在文档中可以查看各个方法的详细资料

总结

当然以上只是Java的简单基础操作，但也是我们应该熟悉了解的内容，不过后续学习开发都会使用到各种各样的开发工具如Eclipse、IDEA之类的工具。

个人比较推荐IDEA，功能比较丰富而且好用又好看，不过选择开发工具都是看个人喜好的。
