0x01 `LeetCode

https://leetcode-cn.com/`

这个网站还是很不错的，题型多题目多而且试题经典。在国内绝对是相对较好的刷题网站了。进入后可以看到各种类型试题，还有各种大企业的面试题，绝对看的你眼花缭乱。当然不能乱，按照自己的计划，该看哪个看哪个就行了，如果今天没有看完，明天继续接着看，可以看刷题记录。你也可以根据网站整理的热题和经典题型来入手，在 http://leetcode-cn.com/problem 页面的右侧。先刷热题 HOT 100，再刷精选 TOP 面试题，之后刷其他的题。

算法一直是企业面试必备的试点，毕竟算法的使用还是挺多的，如果不想做一个搬运工，还是需要好好补补算法。在上面每天刷两道算法题，面试哪家都不怕，绝对算法方面顺利通过。

0x04 SQL 练习平台

https://sqlzoo.net/wiki/SQL_Tutorial

是SQLZOO包括了 SQL 学习的教程和参考资料，支持 SQL Sever、Oracle、MySQL、DB2、PostgreSQL等多个 SQL 搜索引擎。

通过自己的实践，给如下建议：

按难度分组刷(从基础 到高级)按分类刷需要定期复习，这样加深记忆，理解更深入建议只是我自己的经验，如果你认为其他方式更好，那你就按照自己的规划。掌握正确方法是非常重要的。

刷题方法：

  • 第一遍：先思考，之后看参考答案刷。思考、总结并掌握本题的类型，思考方式，最优题解。

  • 第二遍：还是需要先思考，回忆最优解法，并与之前自己写过的解答作比对，总结问题和方法。

  • 第三遍：速度，速度，速度，拿出一个题，就能够知道其考察重点，解题方法，在短时间内写出解答。

定期总结：善于思考善于总结加强锻炼
