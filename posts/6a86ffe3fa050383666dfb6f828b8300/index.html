4. 京东云上线博客

到了这里，想必大家已经有了自己的云服务器，也对云服务器有了一些知识的了解，接下来我们就直接使用它来快速上线一个个人博客

4.1 进行服务器的配置

  1. 首先我们要进入京东云的控制台去设置 root 用户的密码，这就是我们连接服务器的密码

  2. 找到自己的服务器IP，在远程连接时需要用到

  3. 对服务器的一些端口进行开启，如：80、21、22端口等等

4.2 手动搭建服务器环境

此时我们就可以搭建我们项目上线的一些开发环境了，这里直接手动搭建，大家也可以使用宝塔

4.2.1 安装 Xshell 和 Xftp

为了方便远程操控服务器，我们可以安装 Xshell，为了进行服务器和本地的文件传输，我们可以安装 Xshell，这两款软件都可以直接去官网安装。

安装好之后，我们可以通过服务器的IP，来进行连接

4.2.2 安装 JDK

Xshell 连接服务器成功后，通过下面的命令安装 jdk

yum -y install java-1.8.0-openjdk.x86_64

4.2.3 安装 Tomcat

由于 yum 上没有 tomcat 的源，因此需要以 wget 的方式来下载

# 首先切换到 opt 目录
cd /opt

# 再运行下面的命令进行 tomcat 的下载
wget https://mirrors.cnnic.cn/apache/tomcat/tomcat-8/v8.5.73/bin/apache-tomcat-8.5.73.tar.gz

# 解压
tar xzf apache-tomcat-8.5.73.tar.gz

下载完成后，运行 tomcat

/opt/apache-tomcat-8.5.73/bin/startup.sh

运行成功后，就可以通过外网进行访问了，在浏览器上直接访问服务器IP端口号，例如：http://120.76.203.209:8080

4.2.4 安装 MySQL

通过以下命令来进行 MySQL 的安装

wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm
rpm -ivh mysql-community-release-el7-5.noarch.rpm
yum install mysql mysql-server mysql-devel -y

安装成功后，通过下面的命令来启动 MySQL

service mysqld start

通过 yum 安装的 MySQL 的管理员账户是没有密码的，这里通过命令设置其密码为 XXX

mysqladmin -u root password XXX

4.3 上线个人项目

首先我们要登录数据库，并创建我们需要的 SQL

mysql -u root -p

之后我们将我们项目的打包好的 war 文件通过 Xftp 上传到 tomcat 的 webapps 目录中

然后再重启 tomcat，使得我们上传的 war 文件被自动解析

# 关闭
/opt/apache-tomcat-8.5.73/bin/shutdown.sh
    
# 打开
/opt/apache-tomcat-8.5.73/bin/startup.sh

此时我们的项目就已经上线到云服务器上了，可以通过外网来访问
