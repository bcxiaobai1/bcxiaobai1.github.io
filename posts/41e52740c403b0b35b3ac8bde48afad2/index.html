使用 Footprint Analytics, 快速搭建区块链数据应用

Nov 2022, daniel@footprint.network

如果你有一个处理 NFTs 或区块链的网站或应用程序，你可以在你的平台上直接向用户展示数据，以保持他们在网站或者应用内的参与，而不是链接以及跳出到其他网站。

对于任何区块链应用或者媒体、信息网站来说，通过在网站上展示数据图表（如 Coinmarketcap 或 DappRadar），可以体现业务趋势，同时让平台给用户带来更高的价值。这种数据应用，在过去需要大量的开发者资源来构建，但现在只需要几分钟就能够实现。

Footprint Analytics 支持任何网站快速、简便地嵌入我们强大、灵活的可视化功能。此外，任何人都可以用 Footprint Analytics 的数据 API 构建他们的自定义 Web3 应用。

我们的愿景是区块链分析变得简单；每个人都可以使用 Footprint 工具建立分析工具和应用。

目前支持3种方式： 

  • iFrame （嵌入网页）

  • widget（嵌入组件）

  • API （直接调用数据接口）

在未来，我们计划创建更多的基础设施工具，供分析开发人员/分析师构建以及定制自己的应用程序。

1. 网站嵌入

开发人员使用 iFrame（内置嵌入网页）在自己的网站内加载一个 HTML 文档。这个文件也可能包含 JavaScript 或 CSS。

要使用 iFrame 整合 Footprint Analytics 的数据面板，只需要选择一些图表并创建自己的仪表盘，或者，你可以选择社区已经创建好的合适面板。然后点击分享按钮，复制代码进行嵌入。仅需一行代码，即可把数据面板插入你的网站。

点击此链接 可以参考一个使用 Footprint Analytics 快速嵌入网站的样例。

Footprint Analytics 嵌入式组件

Footprint Analytics 已经开发了一些 NFT 嵌入式数据组件，支持任何平台在网站上展示头部收藏品的数据。你指需要从嵌入式组件列表中，选择你需要的指标，并简单地将代码嵌入你的网站，即可在网站上展示对应的数据。

目前，该小工具可以显示前 6 名或前 12 名的NFT收藏，以及每个收藏的三个指标之一 —— 成交量（7天）、底价（7天）或市值。用户还可以选择白色或暗黑模式。

为了确保小组件显示的数据是有意义的，Footprint 从 7 天的交易量指标中过滤掉了洗盘交易。头部的 NFT 集合反映了实际的链上市场活动。

点击此链接，可查看使用 Footprint Analytics 嵌入组件以及贴牌面板的样例。

2. Footprint Analytics 数据 API

Footprint Analytics 拥有最全面的链上数据索引仓库，涵盖 22 个公链，14 个NFT市场，超过 1900 个GameFi项目，以及超过 10 万个NFT收藏。

现在，Footprint Analytics 通过统一的数据 API 将所有这些数据放在世界的指尖上。它允许开发人员从包含所有这些协议的结构化数据的数据库中进行调用，只需一次查询，或在需要时轻松定制数据API。由于该API是跨域的，用户可以通过一个工具访问NFT、GameFi和地址数据。洗涤交易过滤器为用户提供了过滤掉无意义和欺诈性数据的选择。

灵活的API，有SQL和REST版本，是一个企业级的Web3工具，将任何技术栈连接到区块链网络。

要了解更多关于Footprint Analytics 数据 API 的信息，可以点击本文档进行查看。

3. 使链上数据分析变得更加触手可及

正如我们在上周的一篇文章中所述，链上数据显示 FTX 在崩盘之前就已有端倪。如果更多的人，而不仅仅是专业分析师，能够探索区块链数据，也许问题可以更早被发现。

尽管目前市场情绪消极，但随着区块链开发者继续建设，我们相信该行业会继续发展。但易于使用的数据平台对于重建对加密货币和 NFT 的信任将是至关重要的。

我们的 iFrame、widget 和 API 集成，使得开发人员可以快速、轻松地利用来自22个不同链、数千个协议和超过 10 万个 NFT 集合的历史数据。Footprint 旨在成为开发者在我们的平台上建立应用程序的基础设施，提供数据洞察力，而无需在前端进一步开发，也方便产品开发者能快速推出以及测试产品概念。而使用 Footprint Analytics 的数据 API，则可让你根据产品设计，定制一个更好的用户体验和用户界面。
