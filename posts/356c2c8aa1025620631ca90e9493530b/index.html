Linux上杀毒软件有哪些?

Linux平台上的杀毒软件不如Windows平台上那么普及，因为Linux操作系统本身设计得就比较安全，而且不容易感染病毒。但是随着技术的进步和恶意软件增多，Linux平台上的安全软件也越来越多，以下是一些Linux平台上比较知名的杀毒软件：

ClamAV: 开源的杀毒软件，采用 ClamAV 引擎检测病毒、恶意软件等，支持多种系统，还可作为邮件和网关扫描器使用。

Eset NOD32 Antivirus for Linux Desktop: 专门为 Linux 桌面而设计的杀毒软件，使用 Eset 公司的杀毒引擎对系统进行保护，并支持管理控制台、防火墙等功能。

Sophos Antivirus for Linux: 针对企业的杀毒软件，能够防御威胁、监视系统进程等，可保护拥有数以千计设备的组织。

Comodo Antivirus: 支持 Linux、Mac、Windows 等系统，采用自家研发的引擎查杀病毒、木马等。

F-Prot Antivirus for Linux Workstations: 同时支持文件和邮箱等多种类型的扫描，可检查包括 zip、 rar 等压缩文件中的病毒。

McAfee VirusScan Enterprise for Linux: 支持服务器和桌面系统，可快速检查病毒、木马等，并进行威胁处理和防卫。

Dr.Web Antivirus for Linux: 针对网关、邮件服务器等使用的杀毒软件，可实时检测并清除病毒、木马等，还支持启发式扫描。

AVG Antivirus: 可检查邮件及其附件、HTTP/FTP、文件等，还能够扫描压缩文件、动态链接库和可执行文件等。

Kaspersky Antivirus for Linux: 采用高效的扫描引擎，可检测恶意软件、病毒、木马等，并为大型企业提供全方位解决方案。

Bitdefender Antivirus scanner for Unices: 支持多平台检查、实时防护、病毒扫描等，可保证数据安全与防御系统袭击等。

Trend Micro ServerProtect for Linux: 支持多种版本、危险等级的病毒扫描，还拥有动态更新和即时查杀等功能。

Symantec Endpoint Protection for Linux: 支持多种扫描方式、威胁防御、漏洞修复、文件离线检测等多项功能。

Avast Linux Home Edition: 提供 Linux 用户病毒、木马等威胁的防护，支持多种查杀方式和扫描模式。

Panda Antivirus for Linux: 提供实时保护、定期扫描、彻底清除病毒等功能，采用 Panda 自家杀毒引擎。

Norman Antivirus for Linux: 支持文件检测、定期检查、完全清除病毒、木马等，并提供定时扫描与扫描脚本。

Rootkit Hunter: 一款防止根套工具（rootkits）入侵的探测软件，能够扫描系统文件、程序等，以实现即时检测。

rkhunter: 同样是一款根套工具检测软件，具有检查文件系统、进程、端口等能力，并可以更新本地恶意软软件库以及查找可疑文件等。

chkrootkit: 又是一款专门检查根套工具的软件，支持多种 Linux 发行版，包括 Red Hat、Debian、SUSE 等，能够检查多种类型的挂马。

Tripwire: 可以监视文件、目录、设备的变化（添加、删除、修改等），一旦有异常就会立刻报警，可以有效避免潜在的安全威胁。

Logwatch: 监视系统的系统日志文件，如 syslog，能够排查是否有异常 IP 连接等网络活动，当有重要警报或者入侵痕迹时，可以自动邮件通知管理员。

AIDE: 文件完整性检查工具，它可以保存系统目录和文件的加密哈希值，并在后续比对中检测任何不同之处，从而找出任何被恶意修改的文件。

OSSEC: 一款免费开源的入侵检测和防御软件，支持实时监视日志、进程、文件等变化，能够保卫服务器不受攻击。

Snort: 免费开源的网络入侵检测系统（NIDS），可以监视网络流量、包括内部的数据交换，以检测攻击、防止入侵等网络活动。

SpamAssassin: 实现反垃圾邮件的过滤器，可以从邮件头、正文、附件等多方面评估邮件的可能是垃圾邮件，同时还提供自定义的规则配置。

Amavisd: 支持多个邮件协议的反病毒和反垃圾邮件软件，支持多种引擎和协议，拥有虚假邮件检测等多项强大功能。

DansGuardian: 用于过滤 Web 内容的反垃圾软件，可以通过多种方式控制不良和危险网站的访问，从而保障网络安全。

Bastille: 提供了一些自定义设置，实现操作系统的安全维护，针对不同的场景给出定制化的建议。

SELinux: 权限设置框架的开源实现，它是一组内核安全模块，可用于检查进程能否访问网络、文件、设备等资源，可以实现系统安全审计。

AppArmor: 第三代 Linux 安全模块，可以保障一切程序的安全、稳定，并阻止非法访问，是一种较为灵活的安全控制软件。

Firewalld: Linux 防火墙，提供了动态管理和代替 iptables，通过网络区域和端口管理，可快速建立起安全的防火墙。
