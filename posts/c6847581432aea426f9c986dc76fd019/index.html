UI设计用什么软件做？

1、即时设计

即时设计是一款国内的在线协同设计工具，提供原型设计、UI/UX 设计和设计交付等核心功能。它无需第三方插件，拥有丰富的组件样式、中英文字体库和本地化资源，受到专业设计师的好评。与其他国外的 UI 工具相比，即时设计在国内服务器运行，没有平台限制，用户可以在任何时间、任何地点使用浏览器打开并开始工作。它提供全中文语言环境，让用户能够无障碍学习界面操作、官方教程以及与其他用户的交流，快速提升设计技能和效率。

 

即时设计具有以下优点

（1）团队高效协同

支持多人在线编辑和查看，无需下载，云端同步更新修改内容，可以在项目中轻松进行评论和修改校正。

（2） 智能设计工具

自带专业设计工具，具有与 Figma、Sketch 等类似的创作体验。只需修改主组件，所有组件都会同步变化，快速形成统一的规范。组件级自动布局，增加或删除内容都无需手动调整，具有智能排版功能。

 

（3）企业级数据安全

提供内网本地服务器部署，确保敏感数据和业务数据的私有化。还提供定制化功能开发，可以享受更多高效的体验。同时，具有自定义团队管理权限的功能，可以规避数据泄露风险。

 

（4）海量设计模板

集成了腾讯、阿里、字节、蚂蚁设计等众多大厂设计系统。可以直接拖拽复用组件素材，省去大量重复劳动。即时设计的设计资源每月都会进行更新，让你能够更快地获取创意灵感。

即时设计对个人用户承诺永久免费，并提供了内置的所有功能。这使得它成为一个经济实惠且功能全面的选择。

 

2、 Sketch

适用于所有设计师的矢量绘图应用，具有轻量便捷和高效的特点。作为 UI 设计的核心工具之一，Sketch 帮助全球设计师创造了许多令人惊叹的作品。它提供直观的矢量编辑工具和可编辑的布尔运算，实现灵活的迭代设计。拥有无限的设计画布、灵活的画板、设计预设和可自定义的网格等功能，让你可以将设计缩放到任何屏幕尺寸。然而，Sketch 需要安装一些插件，例如 InVision、Abstract 和 Zeplin。此外，它需要手动保存和传输文件，将本地设计稿上传到 Sketch Cloud 上存在传输慢、沟通难、生产效率低等问题。此外，对于 Windows/PC 用户而言，并不太友好，因为 Sketch 是一款 Mac 工具，且不能在浏览器中运行。

3、 Adobe XD

一款功能强大的基于矢量的 UI 工具，包含协作设计和原型制作的功能。Adobe XD 具有快速的速度和强大的功能，能够帮助你完成整个 UX 和 UI 设计过程。与 Sketch 相比，Adobe XD 在 Windows 和 Mac 上均可使用，这是它稍微领先的一点。通过 XD Starter 计划，你可以免费使用 Adobe XD 的基本功能。但如果想要获得完整的 XD 功能，就需要付费使用或订阅完整的 Creative Cloud 套件。Adobe XD 具有基于矢量的拖放编辑器工具，无限的画板和智能指南，可帮助你对齐设计中的不同对象和元素。它还提供了用于 Apple Design、Google Material Design 和 Amazon Alexa 等的 UI 套件，可直接使用现成的组件。

4、Marvel

Marvel 是一款基于 Web 的原型制作工具，适合初学者入门。它支持从 PS 和 Sketch 等设计稿导入制作交互原型，并提供线框图、原型制作和交付设计规范等核心功能。Marvel 具有拖放式线框模板、丰富的预制资产、静态设计导入等优点，帮助用户快速构建可视化想法。同时，它还具有热点、交互和层的功能，创建交互式原型，并内置用户测试功能，方便获取反馈。此外，Marvel 还提供设计移交工具，将设计转化为代码、规范和资产，便于开发人员使用。它还支持与 Jira、Maze、Confluence、Lookback 等工具的集成，提供更多的功能和协作选项。然而，需要注意的是，Marvel 的付费计划较为昂贵，且目前只支持英文语言环境。此外，使用 HotSpot 连接目标画板并设置页面转场和操作手势可能较为复杂，容易导致逻辑混乱。综上所述，Marvel 是一款功能全面的原型制作工具，初学者可以借助其简洁直观的操作界面快速入门，但需考虑其价格和语言环境对使用体验的影响。
