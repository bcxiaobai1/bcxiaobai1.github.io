Linux的Web服务器配置

准备工作：

1.准备两台虚拟机（CentOS）：一台作为服务器，一台作为客户机（选择仅主机模式进行连接）

2.检查是否安装好了httpd：rpm -q httpd

3.如果没有安装好，安装步骤：

①.cd /run/media/root/CentOS 7 x86_64/Packages (使用自动补全)

②.查看目录下的文件

ls | grep httpd 选择第一个下载

出现了依赖情况的话，按要求下载即可

4.嫌麻烦，可以连上网络，使用命令：yum install httpd下载

5.安装好后，不要忘记使用命令rpm -q httpd查看是否成功

6.设置为仅主机模式，配置好ip地址

7.关闭防火墙和安全策略：

systemctl stop firewalld.service

setenforce 0

服务器端

1.进入web服务配置文件路径：vim /etc/httpd/conf/httpd.conf

2.打开web服务器：systemctl start httpd

3.检查打开状态：systemctl status httpd

4.编辑web文档：

进入文档路径：cd /var/www/html

编辑新增文档：vim index.html （里面随意写一些东西）

客户机端

1.首先检查是否和服务器端能够ping通

2.打开火狐浏览器：输入服务器端的ip地址

出现了服务器端的文档就表示成功了
