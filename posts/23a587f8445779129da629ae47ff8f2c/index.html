青龙面板 搭建：

action！！

首先，我们安装宝塔，访问宝塔官网，或者直接复制以下代码，终端运行，等待两分钟左右就可以成功安装

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

运行等待2分钟左右就会出现这张图

 

复制这些信息进入浏览器，登录就安装好宝塔了

ps：前提开放8888端口，要不然服务器不会让你通过防火墙，你也就进不去了

进入宝塔之后点击软件商店，进入我们去安装docker容器

 这里点击安装之后 就慢慢等待吧

安装好docker容器之后，我们进入终端输入命令

拉取青龙面板镜像

docker pull whyour/qinglong:latest

等待下载完成之后，输入一下代码，创建容器，直接全部复制，终端粘贴

docker run -dit 
  --name ql 
  --hostname ql 
  --restart always 
  -p 5700:5700 
  -v $PWD/ql/config:/ql/config 
  -v $PWD/ql/log:/ql/log 
  -v $PWD/ql/db:/ql/db 
  -v $PWD/ql/scripts:/ql/scripts 
  -v $PWD/ql/jbot:/ql/jbot 
  whyour/qinglong:latest

这里要开放端口 5700

当然我要讲的是另外一个方法，因为最新版的青龙面板镜像安装好之后，就算你开放5700端口，你也进不去，估计是作者写的小bug，因此，我们直接导入以前备份的能用的镜像使用！！！
