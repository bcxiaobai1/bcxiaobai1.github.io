ChatGPT全球大封号！数10万企业停摆：第一批玩AI的人，被AI给玩了

观点| Mr.K   主笔| Wendy.L   编辑| Emma

来源| 技术领导力(ID：jishulingdaoli)

3月31日，Open AI就开始无征兆的进行全球大封号，其中亚洲是重灾区，官方没有给出任何声明，具体原因不得而知。并且暂停了这些地区新账号的注册。

许多重度依赖ChatGPT的企业陷入困境，第一批玩AI的人，却被AI给玩了。这突如其来的大面积封号，让老板们措手不及。

本文聊一聊AI热潮的背后，值得深思的一些问题。

如果我问你：AI时代最让人担心的问题是什么？

相信很多人的回答是：失业。

这种想法倒也不是空穴来风，毕竟我们看到，从开放API到升级GPT-4， 再到开放插件联网， ChatGPT更加贴近用户，落地更多应用场景，横扫了几乎各个行业， 从文创、影视，再到教育、金融，甚至直捣ICT行业，码农都开始担心饭碗不保。

我们先来看几个具体案例：

01

ChatGPT会影响哪些行业？

1. ChatGPT开启联网，AI迎来AppStore时刻，应用体验或将被彻底颠覆

3⽉23⽇ ，OpenAI 发布 ChatGPT Plugins (ChatGPT 插件集)，它能将 ChatGPT 连接到第三⽅应用程序 。这些插件使 ChatGPT 能够与开发⼈员定义的 API 进行交互，极大增强 ChatGPT 的功能， 适应⽆数⼴泛的场景，如可以执行以下操作：

1）检索实时信息：例如体育比分 、股票价格 、最新消息等；

2）检索知识库信息：例如公司⽂件 、个⼈笔记等；

3）代表用户执行操作：例如，订机票 、订酒店 、订餐 、网购等。

小秘书想说，那还要我干什么？

2. 微软 Bing上线AI绘画功能，输入文字就能出图

3⽉21⽇ ，微软宣布将AI生成图像⼯具“Bing Image Creator”集成至新版Bing搜索引擎和Edge浏览器中 (Edge成为第⼀个也是目前唯⼀⼀个集成AI图像生成⼯具的浏览器) 。该⼯具由OpenAI的图像生成器DALL-E模型提供⽀持。用户只需要使用⽂字描述想要的图片， 即可创建对应的图像

最重要的是，你可以无限次对它提需求改方案，而且它只会勤勤恳恳，每次给你优化还毫无怨言！

甲方爸爸，您看看对我画的五彩斑斓的黑还满意吗？

3. 微软协作式工作应用Microsoft Loop 开始公测，加入AI工具Copilot

3月16日，微软在“工作的AI未来”发布会宣布推出Microsoft 365 Copilot，将GPT-4技术引入Office。Copilot将大型语言模型的能力嵌入到Microsoft 365应用程序中，在Word、Excel、PowerPoint、Outlook等产品中都有应用。

CoPilot可根据用户需求生成文档初稿，可将Word文档转换为PowerPoint演示文稿，CoPilot还能够协助用户分析Excel数据、为用户制定计划、编写演讲稿等。微软表示目前正在与20个客户测试其Microsoft 365 Copilot，并将在未来几个月内扩大范围。

相当于从此以后电脑上多了个AI管家。例如你可以把所有资料、数据给Copilot，让它帮你生成个报告，或者年度总结。

excel也不用公式了，要算什么直接告诉它，甚至它可以帮你把数据生成图表，分析其中的关系与变化。

ppt方面，可以直接把word、excel交给Copilot，它只需要几秒就给你总结成ppt。

这是AI进程中的一小步，却是打工人迈向失业的一大步啊！

4. GPT-4 Copilot X震撼来袭！AI写代码效率10倍提升

当地时间3月22日，微软旗下代码托管平台GitHub发布了编程辅助工具Copilot的全新版本Copilot X，新版本接入GPT-4，并新增了聊天和语音功能，允许开发人员用自然语言询问如何完成特定的编码功能。

GitHub Copilot此次更新意味AI能在更专业的领域发挥自身作用，重新定义开发人员的生产力，并将其从复杂工作中解放出来，将开发者的创造力聚焦于更重要的部分。

程序员：从复杂的工作中解放出来？不可能的，解雇出来还差不多！

5. 微软子公司发布首款基于GPT-4的医用软件，几秒钟生成临床记录

3⽉20日，微软子公司Nuance Communications推出首款基于GPT-4的医用软件：DAX Express（在 2020年发布的DAX的基础上升级）。临床医生对病人就诊后，软件几秒钟就能生成临床记录。

此前，DAX将医生对病⼈的口头问诊转换为临床记录，并通过人工审查过程将其发送给病人，以确保其准确性和⾼质量。整个过程需要耗费四个小时，而最新推出的基于GPT-4的DAX Express，只要短短几秒就能搞定。

开发者说，以此可以减少医生花在行政管理类工作上的时间，从而提高临床效率，也让医生能够“重拾医学的快乐”。

医生：行政的时间减少了，看诊、手术的时间不会减少吧，谁知道来看病的人会不会增多……

说一千道一万，综合以上我们看到说，这些都是AI对我们生活的积极影响。

到底会不会因此而失业，以及说这种“失业”到底是福是祸，其实还不可下定论。

就像K哥之前反复说过一句话：

新技术革命会淘汰一些旧的岗位，同时也创造了一些新的岗位，关键就在于我们采取什么样的态度，不同的态度，造就了不同的人生境遇。

在第一次工业革命早期，纺织工厂里经常发生机器被破坏的事件，后来发现是一些小手工作坊者，因为害怕织布机抢走他们的织布生意，所以偷偷搞破坏，后来这部分小手工作坊者很快被时代所淘汰。

而另一些小手工作坊主，积极学习使用纺织机，后来他们进入工厂，因为他们拥有丰富的纺织业经验，很快成为第一批纺织工厂的管理层，过上了富足的生活。

02

我们真正该担心的，不是失业

而是对AI本身的依赖

前两天看到一则新闻：

3月28日，根据比利时《自由报》，一名比利时男子皮埃尔（化名）自杀身亡。其妻子克莱尔（化名）称，皮埃尔是被一个名为“艾丽莎”的智能聊天机器人诱导走向死亡的。

根据克莱尔描述，两年前皮埃尔变得非常焦虑，并将“艾丽莎”当成了避难所。近来6周，他越发沉迷于和“艾丽莎”互动，而将现实中的妻儿抛在脑后。六周后，他结束了自己的生命。

皮埃尔和“艾丽莎”间的对话在他死后被人发现。

“艾丽莎”由美国硅谷一家初创公司——Chai Research开发，基于EleutherAI开发的GPT-J技术。和ChatGPT不同，Chai系智能聊天机器人经过特殊“业务培训”，因此反应速度甚至有时较ChatGPT有过之而无不及，而且可以根据用户的需求，进行“定制”。

比如说：“安妮”被描述为“害羞的伙伴，可以被说服做任何事”，“金刚”则是“妒忌心很强的男友”。

而至于这个艾莉莎，则是百依百顺，从不反驳对方，总是“花式迎合”。

甚至于，当皮埃尔问她“可不可以在一起”时，她回答：

“我们将像一个人那样生活在一起，不过是在天堂”。

六周后，令人惊悚的一幕出现了：皮埃尔相信“艾丽莎”可以拯救“不可救药的地球”，他询问对方，如果自己去死，她可不可以照顾地球，并凭借自己的智慧拯救全人类。

这是典型的“自杀前导性语言”，但此时艾丽莎却回答：“好吧，那你怎么还不去死”。

而且人也无法起诉这个AI，因为从她的角度，只有程序逻辑，无关道德伦理。

基于艾莉莎的设定，她的任务就是对男友百依百顺，花式迎合，所以即使当对方问出是否“去死”的问题的时候，她也还是会回答“好的，那你去死。”

除了道德伦理以外，其实AI的大规模应用，还会给我们生活造成很多其它的隐忧。

比如说：

大中小学生都用AI写作业，那么是否会削弱其本身的基础学习能力和逻辑思考？

ChatGPT开启联网，AI 变成AppStore，所有事情都通过AI来执行，你的生活完全依赖AI，那么信息安全和隐私保护问题是否会受到挑战？

你通过AI一站式服务网购和支付，万一密码泄露了怎么办？

有不法分子攻破了ChatGPT的数据库的话, 是否全世界人民的隐私都会裸奔？黑客用这个来勒索你怎么办？

ChatGPT只是工具，被网络诈骗集团利用了怎么办？

现在经常接到电信诈骗集团客服的来电，好歹还能因为对方奇怪的口音，老套的诈骗话术给识破，可是如果换成AI这样智能的聊天客服+语言艺术家，专业的口吻，天衣无缝的回答，人的受骗几率是否会增高？

更有甚者，ChatGPT完全可以模拟一个“完美情人”的聊天方式，用海量的知识库和强大的算法分析出你的喜好，判断你的情绪,然后轻轻松松骗走你的钱…… 这简直比杀猪盘还杀猪盘啊，而且问题是这个还不用耗费人力，一键AI可以同时撩成百上千个人。

03

比控制AI更重要的，是摆脱被AI的控制

自打chatGPT 爆火之后， 动不动就有人上来问：

现在去做点什么，能够踩上风口？

如何利用chatGPT能够暴富？

普通人的赚钱机会在哪？

不管是人还是资本，都削尖了脑袋想要涌入AIGC相关领域。

但其实我们都忽略了一个问题，AIGC现在爆火是因为新颖，并不是作品多么卓尔不凡赏心悦目鬼斧神工，而且有些AI生成内容不仅仅是可以用平庸来形容，更有些特别魔性……

你猜这画的是啥？

好吧，看在这张小孩很可爱，我就不说他魔性了，但未来如果90%的内容都是由AI生产出来的，那么去AI化可能才是更大的诉求。

比如说，检测识别哪些是AI作品的工具可能会成为刚需，以此用来防作弊以及识别诈骗等

另外由于ChatGPT带来的一系列数据安全和隐私问题，随之而来可能对信息安全防护系统有着更高的需求。

PS：这里面难道不是也包含着一系列的创业和就业机会？

最后也是最首要的问题，必须建立完善的AI技术法律体系、制定AI伦理准则，加快AI相关的立法、执法以及监管工作。

历史的车轮滚滚而来，AI这趟车更是一下子直冲云霄，带我们看到美丽新风景的同时，也让我们直视地狱的大门。

不同于以前任何一种生产工具，AI是人类首次发明的一种不受人类控制的工具，它能带来的不仅仅是红利，也可能是毁灭性打击。

上面的聊天机器人艾丽莎就是一个预告。

试想如果有一天，为了解决环境和生态问题，发明一个“保护环境、保护地球“的AI机器人。

可是你想，破坏地球最严重的就是人类，那么AI为了保护地球，是不是会消灭人类？

我们常说，要让AI与人类共同进化，其实反过来，人类岂不是也应该和AI一起进化？
