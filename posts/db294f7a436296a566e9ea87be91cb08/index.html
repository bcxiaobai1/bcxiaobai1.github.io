首批 RISC-V 手机要来了，你准备好了吗？

近日，国内一家科技公司 Sipeed 在 Twitter 上发布了一则短视频，演示了运行 Android 10 的 RISC-V RV64 智能手机原型。如果进展顺利的话，预计在 2022 年，搭载 RISC-V 的商用机将成功面世。

Android 10 在 RV64 手机原型上操作很活泼（图片来源：Sipeed）

RISC-V 是一种基于已建立的简化指令集计算机（RISC）原则的开放标准指令集架构（ISA）。 与大多数其他 ISA 设计不同，RISC-V ISA 是根据开源许可证提供的，不需要付费使用。许多公司正在提供或已经公开 RISC-V 硬件，可以支持 RISC-V 的开源操作系统使用，并且在几个流行的软件工具链中支持该指令集。


图片来源：RISC-V

作为一种负载存储架构，RISC-V ISA 浮点指令使用 IEEE 754 浮点，采用了架构中立的设计，可以简化 CPU 中多路复用器的位模式，将最重要的即时值位放置在固定位置上，得以加速符号扩展。该指令集设计用于存储、微控制器、边缘计算和人工智能应用等领域上。基本指令集具有固定长度的 32 位自然对齐指令，ISA 支持可变长度扩展，其中每条指令可以是长度任意数量的 16 位包裹。子集支持小型嵌入式系统、个人计算机、带矢量处理器的超级计算机和仓库规模的 19 英寸机架式并行计算机。

指令集规范定义了 32 位和 64 位地址空间变体。该规范包括对 128 位平面地址空间变体的描述，作为 32 位和 64 位变体的外推，但 128 位 ISA 仍然有意“未冻结”，因为使用如此大的内存系统的实际经验还很少。



RISC-V ISA 的开发每年都在以越来越快的速度发展，越来越多的企业也开始深度拥抱 RISC-V 架构。今年，半导体公司 SiFive 推出了第一个面向 PC 的 RISC-V 平台，随着 Linux 支持在许多 RISC-V 芯片组时（例如来自 Alpine 和其他单板生产商的最新 RV64 封装）越来越趋于稳定，现在能看到整个行业也在计划推动着基于 Android 的智能手机采用 RISC- V 。

Sipeed 的推文也暗示了想要在 2022 年发布 RISC-V 的决心，但是，随着一些短缺的出现，发布可能会推迟到 2023 年。 正如 TuxPhones.com 所指出的，基于 RV64 的手机应该允许移植完整的 Linux ，并且移植过程比现代任何基于 ARM 的芯片都容易。就性能而言，Sipeed 指出，配备第一款 RISC-V 手机的 RV64 处理器应该比一些四核 ARM Cortex A73 处理器更强大，可能与高通的 Snapdragon 662 / 663 型号相匹配。

放眼望去，RISC-V 在业界主流的科技行业的成就可谓是叹为观止，不少人认为 RISC-V 是全新的指令集，是时代背景下新的机遇。不管是电子产业，还是芯片科技领域，大家对 RISC-V 架构的认同可谓是一致的。它呈现了可观的商业前景，同时发展模式也不受限制。随着 RISC-V 架构在智能机的应用，相信它在中国产业带来的机遇也会越来越多。你期待基于 RISC-V 架构的手机设备吗？
