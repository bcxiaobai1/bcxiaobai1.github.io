2. 系统选择

3.安装组设置

linux安全组默认开放22，把后续要用的到80，443，8888，4700，4701，都开了吧！IP限制填写 0.0.0.0/0，参照22端口的ip填写即可。

4.安装堡塔远程工具

堡塔软件下载链接：

 5. 安装宝塔面板

复制下面代码到ssh界面并回车,当页面内代码不再滚动的时候会显示[y/n]，输入 y 继续安装。一般网速快一分钟左右就完成了！！（因为公众号限制，建议大家先复制到记事本上，调整下代码尽量不要随意换行！！！！）

完成后显示默认的面板登录地址和默认账号及密码！！这里一定要用记事本保存着！！！（忘记了也没事，SSH登录服务器，输入字母 bt 再输入14 即可获取到刚才的信息。这里借用一下网上的截图。） 

6.安装Docker管理器

安装完宝塔面板后，复制生成的登陆链接到浏览器打开，简单的安装下宝塔面板，然后登陆，到软件商城下载Docker管理器

7. 创建青龙面板

我这里用的是2.10.9指定的版本，感觉比较稳，大家根据自己的需求选择版本哈！！！（我自己创了两个容器，所以还需要开下5600，5601端口，小白不会的就都开着！！！）

提醒：一定要把5600,5601开着（如果只建一个容器开一个就行了！！）

复制下面的代码到ssh界面然后回车！！！（注意：进入到远程连接界面，复制如下代码，鼠标右键选择粘贴，注意直接Ctrl+V 是粘贴不上的。）

docker run -dit 
--name QL 
--hostname QL 
--restart always 
-p 5600:5600 
-v $PWD/QL/config:/ql/config 
-v $PWD/QL/log:/ql/log 
-v $PWD/QL/db:/ql/db 
-v $PWD/QL/scripts:/ql/scripts 
-v $PWD/QL/jbot:/ql/jbot 
246771270/jdwxx-ql:2.10.9

多容器创建方式，小白不会就省略这个步骤！！！！（千万别看！！！）

服务器配置高可以创两个面板，还是像上面的步骤重新复制代码新建一个容器！！！！

docker run -dit 
--name QL2 
--hostname QL2 
--restart always 
-p 5601:5600 
-v $PWD/QL2/config:/ql/config 
-v $PWD/QL2/log:/ql/log 
-v $PWD/QL2/db:/ql/db 
-v $PWD/QL2/scripts:/ql/scripts 
-v $PWD/QL2/jbot:/ql/jbot 
246771270/jdwxx-ql:2.10.9

 安装好青龙面板后，青龙面板地址是你服务器 IP+端口 :5600(示例：192.168.1.1:5600) 访问青龙面板,进行初始化。（小白别把我的示例粘贴进去了！自己的服务器加端口号访问你自己的）
