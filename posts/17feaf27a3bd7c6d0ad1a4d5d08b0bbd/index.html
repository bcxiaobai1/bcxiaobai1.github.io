开源大新闻

Linux Mint 与 Mozilla 建立合作伙伴关系

1 月 10 日，Linux Mint 的首席开发人员 Clement Lefebvre 宣布与 Firefox 网络浏览器的制造商 Mozilla 建立合作伙伴关系，以改进 Linux Mint 发行版中的 Firefox。Clement 表示，Firefox 仍将以 .deb 包的形式在 Linux Mint 存储库中分发，同时 Firefox 将在 Linux Mint 中提供与在其他操作系统中相同的体验。Clement 还说通过这种商业和技术合作，Linux Mint 可以满足 Mozilla 在使用其知识产权（其名称、品牌和身份）方面的要求。

Signal 首席执行官辞职，WhatsApp 联合创始人代理运营

1 月 10 日，Signal 创始人 Moxie Marlinspike 在他的博客文章中宣布，将辞去公司首席执行官一职，“Signal 的无限潜力令我震惊，我想引进一个充满活力并愿意奉献的人来挖掘这份潜力。”Marlinspike 还表示，他将继续留在 Signal 的董事会并寻找新的 CEO，同时在找到接替人选之前，WhatsApp 联合创始人 Brian Acton 将担任代理首席执行官。

英特尔 30 年老将 Gregory Bryant 离职

1 月 10 日，英特尔执行副总裁兼客户端计算事业部总经理 Gregory Bryant （GB）宣布将离职，于 1 月底离开公司并寻求新的机会。自 1992 年加入英特尔以来，Bryant 担任过许多职务。他领导的团队负责共同设计、交付英特尔领先的消费者和商业 PC 平台，包括英特尔 Evo 和英特尔博锐平台。

图片来源：CSDN 下载自东方 IC

英特尔首席执行官 Pat Gelsinger 说，“我要感谢 GB 在英特尔 30 年的杰出领导和服务，他帮助实现了连续五年的业务增长，最近还成功推出了第 12 代英特尔酷睿系列。我们祝愿他未来一切顺利并取得好成绩。”同时英特尔宣布客户端计算事业部将由执行副总裁（EVP） Michelle Johnston Holthaus 接手。

开源办公套件 LibreOffice 7.3 将引入现实中不存在的语言

1 月 7 日，LibreOffice 团队宣布，在即将于 2 月初发布的 LibreOffice 7.3 版本中引入两种现实中不存在的语音—— Interslavic 和 Klingon。Interslavic 是一种人工语言，旨在促进不同斯拉夫国家代表之间的交流，是泛斯拉夫的辅助语言；而 Klingon（克林贡语）是虚构的外星种族 Klingons（克林贡人）在星际迷航宇宙中使用的人造语言。LibreOffice 团队表示，要将免费和开源软件带到 Nekrit Expanse（星际迷航中的太空区域名），甚至外太空或更远。

王者荣耀、QQ 和微信使用的动画神器即将开源

王者荣耀、QQ、微信都在用在动画神器 PAG，将于本周五（2022 年 1 月 14 日）正式对外开源，采用 Apache 2.0 协议。PAG 支持 Android、iOS、web、windows、mac OS、Linux 各平台。1 月 14 日开源时会提供 web 版本，基于源码可以构建 windows、mac OS 和 Linux 版本相关库文件。据悉，PAG（Portable Animated Graphics）是一套完整的跨平台渲染动画工作流，支持 AE 动画实时渲染、运行时编辑，可以广泛应用于 UI 动画、贴纸动画、视频编辑、模板设计等场景。


Linux 5.17 中 Btrfs 有良好的性能改进

Linux 5.17 内核的 Btrfs 较之前内核版本有一些性能优化。在 Linux 5.17 的 Btrfs 的性能方面，目前目录记录所需的元数据较少，因此目录删除速度提升了 20-40%，同时可用空间树条目按大小索引和搜索可以减少约 30% 的延迟和搜索时间。在性能工作之外，Btrfs 的发送代码已经被调整与并发块组重定位一起运行；有一种新的“平衡暂停”模式，允许在平衡暂停的情况下向文件系统添加设备。同时还有一些其他修复和代码改进等。（Phoronix）

Apache Flink ML 2.0.0 发布

1 月 7 日，Apache Flink 社区宣布 Flink ML 2.0.0 发布。Flink ML 是一个提供 API 和基础设施的库，用于构建流处理、批处理统一的机器学习算法，它易于使用且具有（接近）实时延迟的高性能。此版本涉及对早期 Flink ML 库的重大重构，并引入了扩展 Flink ML API 和迭代运行时的主要功能，如支持具有多输入多输出的阶段、基于图形的阶段组合以及新的流处理、批处理统一迭代库。同时，此版本还添加了五个算法的实现方式。

具体详情见：https://flink.apache.org/news/2022/01/07/release-ml-2.0.0.html

UbuntuDDE Remix 21.10 Impish 发布

UbuntuDDE Remix 21.10 新版本发布，代号为“impish”（Impish Indri）。当前版本是非 LTS 版本，将在接下来的九个月内提供支持。新版本的主要特点：带有 Linux 内核 5.13.0-22 的 Ubuntu 21.10 基本系统；更新到最新可用的深度桌面环境和软件包；Firefox 95.0.1 作为默认网络浏览器；LibreOffice 7.2.3.2 作为默认办公包等。

具体详情见：https://ubuntudde.com/support

谷歌推出 Chrome 98 测试版，支持颜色渐变矢量字体

继上周发布Chrome 97 之后，谷歌将 Chrome 98 升级为 beta 版本。在 Chrome 98 中，有一些小的改进，主要面向开发人员。Chrome 98 beta 的一些亮点包括：支持 COLRv1 颜色渐变矢量字体作为新的字体格式；使用动态范围和视频动态范围的 CSS 进行 HDR 彩色媒体查询；支持控制 window.open() 行为，无论是新窗口中的弹出窗口还是新选项卡等。

具体详情见：https://blog.chromium.org/2022/01/chrome-98-beta-color-gradient-vector.html

Codeberg：基于 Gitea 的软件锻造器

Codeberg 是一个社区驱动的非盈利软件开发平台，用于免费和开源软件、内容和项目。该平台由 Codeberg e.V. 运营，以 Codeberg.org 为中心，是一个基于 Gitea 的软件锻造器。Codeberg e.V. 的使命是建立和维护一个免费的协作平台，用于创建、归档和保存代码并记录其开发过程。

具体详情见：https://codeberg.org/
