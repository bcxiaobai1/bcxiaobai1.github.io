1.1 修改原图中的错误

原图中存在在两处错误：

1是九宫格上面的蓝色标头，原先的hoc的两头位置反了。整个大纲视图的出入方向是从右到左(九宫格的上侧，rfc文档的的方向 )和从下到上（九宫格的左边，nod文档的方向）。所以代表了事先分组的propter hoc 在右，代表了事后分析的post hoc在左。原先位置放反了，上图中已经改过来了。显然，hoc是时间上（而非物理上的）的分断，对应前面给出的 “Figure 4. Types of Processes”上 的竖条bar。

2是图的左上角处，最外面的一横一竖的黄色条带，虚线连接到隐藏层的引线处的两处文字标签也标反了。标有“数据库”和“概念域”的位置现在已经对调过来，其它不变。

需要注意的是，数据库是面向应用的，数据库中的数据很难避免重叠，“概念域”是面向服务的，不同概念域之间难免会有在交叉。这些交叠是由左上和它们虚线相连的防腐层通过关系型来处理和控制的。处理过程如下图：

1.2 增加rfc和nod共同的主题词-众生相续和世界相续--所需要的落点(作为其根源)

为rfc和nod增加的主题词包括：

1) 完整了rfc标示的根和 nod标示的源。这包括三部分：

a）两者相交的左上角 在原先的词标 旁边 增加它门各自对应的词牌: 三个 词标(表示主题角色，a tag )-- 数据库database、关系型relational和概念域(conceptual domain) 分别对应各自的 词牌(指出主旨意义，a name )  --索引词indexical 、词汇词lexical和概念词conceptual。

b）在各自内侧，增加标出了三列和三行主体标头(表明主体身份，a value)  （对应图中黄色横条下方的蓝色条带和纵向砖色条带右侧 ，也就是Piece三分法图的标头)对于aaas系统的地位。分别是体现 可移植性的 导入、port和导出，以及 可跨平台迁移的输入、put和输出。

c）在各自外侧的位置上，增加 rfc和nod标示 的各自的 起源（陈述方法statement的虚子，名称标签）和缘起(表达方式上expression的实子，共指标签)。（图中分别标有"s" 和"e" 字样)。

  • 陈述方法s【语句statement，独立于特定符号的抽象语句】：陈述自然的物理世界的两种方法。用一种语言后者使用语言所表达的知识来陈述的 关于时间的推理。词汇lexicon 是一种语言与该语言所表达的知识之间的桥梁。这需要一个语言处理器来处理不同语言在语法，字和它们表达的概念上的缺口（差异，a gap）。语言处理器通过不同语言级别的索引词，不同知识水平的概念词以及不同个体习惯的词汇词来处理差异的三个不同来源-意外的accidental、系统的systematic和文化的cultural 。
  • 表达方式e【表达式expression，使用实现无关的特定符号的具体表达】：自然规律（关于因果关系的推理）的两种表达方式，现代物理学家描述世界的方式e1和人们通常谈论世界的方式e2，两者之间存在着巨大的分歧（a  sharp break）。为了容纳各种可能的理论，无论是正式的还是非正式的，科学的还是经验法则的，提出了一个法治过程（ law-governed processes）的连续统（a continuum ），它弥合了完全随机的混沌和完全可预测的决定论之间的鸿沟。因果关系的各种理论可以根据它们所假定的法律种类和它们所施加的约束的严密性来解释。

2) rfc标示的内容。这也包括三部分：

  • 右侧的三个表示实体本身的性质的三分法的三个标示：表示了“物自体的独立性”的“Independent”、表示了“结果之间自相关性”的“Relative”和表示了“原因之间互相干性”的“Mediating”。它们分别和中下的代表了“因果关系的控制域 ”部分的“(time= 1~3”)相对-其图示为顶点在p点的光锥的横截面（下图中右下的图C ）。
  • 下中、右中 、和 右下的有关时域和空域的标示。代表“因果关系的作用域或控制域 ”的“(time=-2~0 | 1~3”)，代表“结果之间自相关性”的“(space=1~3)“”和代表了“因果结构理论”的 (time=4,space=0)-其图示为在t=4的一个时钟的petri网（下图中右上的图B ）
    • 九宫格的左上、左下和右上 显示观察者观察实体的方式的三分法的三个标示。显示了“概观总表上中线”的“orrelative”、显示了“概观总表右对齐线”的“Composite”和显示了“概观总表左对齐线”的“Component”。它们分别和中下的代表了“因果关系的作用域 ”部分的“(time= -2~0”)的相关项。它们是右下的“概观总表下中线”的“M系”作为它们的主体的起点(一体，观察者)的 "三位”- 其图示为因果影响锥的离散逼近（下图中左边的图A ）。
