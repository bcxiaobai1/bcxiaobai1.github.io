前言

本文主要就在CentOs下yum配置做详细介绍，同时也介绍了一些yum常见命令，供大家参考。

一、Yum仓库的作用是什么？

基于RPM包构建的软件更新机制
自动解决软件包依赖关系
所有软件包由集中的YUM软件仓库提供

二、Yum相关知识

1.软件仓库的提供方式

import numpy as np

repodat仓库数据文件

2. Yum配置

   mkdir /dvd/
   mount /dev/cdrom /dvd/
   vim /etc/fstab 
   rm -rf /etc/yum.repos.d/*
   vim /etc/yum.repos.d/dvd.repo 
   yum clean all 
   yum repolist

影响yum使用的主要文件
基本设置: /etc/yum.conf
仓库配置: /etc/yum.repos d/.repo
日志文件: /var/log/yum.log
注：客户端文件路径：/etc/yum.repos.d/.repo，在此路径下，错误的配置文件会影响正确的配置文件
.repo基本配置项

  [源名称]:自定义名称，具有唯一性
  name:本软件源的描述字串
  baseurl:指定YUM服务端的URL地址
  enabled:是否启用此频道
  gpgcheck:是否验证待安装的RPM包
  gpgkey:用于RPM软件包验证的密钥文件

3.Yum常见命令

yum search [关键词] ——查找关键词
yum info [软件名] ——查看软件描述信息
yum provides 文档路径 ——查看哪些软件可以提供该文档路径
例子：
yum search httpd //查找关键词httpd
yum info httpd //查看软件httpd的描述
yum provides /etc/passwd //查看哪些软件能提供/etc/passwd文件
yum -y install:安装指定软件包，类似于rpm -i
yum -y remove：卸载指定软件包，类似于rpm -e
