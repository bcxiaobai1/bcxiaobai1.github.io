一、什么是NFT

1. NFT概念

一般说来，加密数字货币可分为两类：同质化代币(FT)和非同质化代币(NFT)。

同质化货币FT：指的就是ETH、BNB等代币，在交易转账等过程中每个代币可以互换、无限拆分，无法区分。

非同质化货币NFT：指每个代币拥有独特且唯一的标识，两两不可互换，最小单位是1且不可分割，如加密猫，MEME等应用。

由于其不可替代性，NFT使得虚拟货币世界拥有了实物价值，因此多用于收藏艺术领域。

2. NFT发展历史背景

NFT的诞生基于2017年以太坊中一个叫做CryptoPunks的像素头像项目，这些像素头像总量上限为1万，任何两个人物都不能相同，拥有以太坊钱包的人都可以免费领取，领完可以放到二级市场交易。

6个月后，区块链小游戏Cryptokitties（迷恋猫）迅速流行，这是一种在以太坊撸猫的游戏。

《加密猫》的游戏突然蹿红网络，反复出现在各大媒体头条，甚至价格最高的创始猫竟然卖出了77万元的高价，令早期的参与者赚的盆满钵盈。

Cryptokitty是虚拟猫，买家拥有两个及以上，就可以培育新猫，培育出稀有特征的价格会更贵。

为什么一只虚拟猫可以卖出77万？

答：稀缺性，每只猫都对应链上的一个NFT代币,有唯一的ID和基因，不同猫的基因进行组合得到新NFT代币。

除了加密猫，NFT经过2年多的发展，已经形成了游戏道具、通证化的电子门票、收藏品、加密艺术品等主要场景，这些场景有一个共同特征，就是有价值的事物具有不可替代的差异性，非常适合用NFT代币来标记它在区块链上的数字所有权。

这种虚拟猫经历了几轮价格的暴涨和暴跌后，同时也让NFT被更多人认识。

Uniswap一双袜子卖16万美元，推特创始人五个单词拍出250万美元，加密艺术家Beeple在佳士得的NFT拍卖作品获得975万美元出价……

在美国艺术圈，越来越多的主流艺术家也已经将作品和NFT绑定。

在上个月，美国乐队林肯公园的创办者、联合主唱Mike Shinoda将自己制作的一段音乐作为NFT拍卖，最终以3万美元成交，Shinoda将拍卖获得的资金建立了一个奖学金，用来资助有经济困难的艺术生。

NFT的应用现在也已经进入了NBA，用来购买标志性篮球比赛中的著名精彩片段，比如勒布朗·詹姆斯（Lebron James）的大灌篮。

3. NFT应用领域

首先，可能是知识产权领域。NFT可以代表一幅画，一首歌，一项专利，一段影片，一张照片，或者其他的知识产权。在这个领域，NFT起到的是专利局的作用：帮助每一个独一无二的东西进行版权登记，帮助其识别专利。

第二，实体资产。房屋等不动产等其他的实物资产，也可以用NFT来表示进行代币化。可以用作资产的流通等金融市场。

第三，记录和身份证明。NFT独一无二，因此，也可以用来验证身份和出生证明，驾照，学历证书等方面。这些可以用数字形式进行安全保存，而防止被滥用或篡改。

第四，金融票据。各类金融票据在流通和交易过程中承载大量信息，如果与NFT结合，不仅能够确权，还便于追踪。另外，未来各类NFT资产的交易本身就可以形成一个细分的金融市场。

最后，票务。演唱会门票、电影票、话剧票等等，都可以用NFT来标记——看起来似乎长得一样的票据，实际上有不同的座位号，自然NFT也是可以在此得到应用。。所有的票都一样，但是座位号不同。

2020 年 11 月，总部位于里斯本的 NFT 平台 Mintbase 在种子轮融资中共获得投资 100 万美元。

本轮融资由中国风投公司 Sino Global 领投，D1 Ventures、Block Oracle Capital、Arweave 和多位天使投资人跟投，旨在帮助 Mintbase 从众多日益壮大的 NFT 平台中脱颖而出。

一个购买、销售和铸造 NFT 的市场。Mintbase 的主要区别在于，它提供了在其他平台上不常见的 NFT。例如，Mintbase 涵盖了利基 NFT 类别，如音乐、会员资格、服务、门票、新闻以及摄影。

团队: Nate Geier 是 Mintbase 创始人兼首席执行官。自 2018 年 3 月，一直参加世界各地的任何区块链社区活动，包括里斯本、巴厘岛、新加坡、曼谷、清迈、布拉格、布达佩斯、开普敦、柏林等。在期间获得了 Devcon4 Status Hackathon 一等奖。

项目亮点：

  1. 立足NFT交易市场，NFT市场潜力巨大。如Epic Games仅在2018年就在其免费游戏《Fortnite》中靠销售游戏内的服饰获得了24亿美元的收入，但用户尝试在eBay出售一款Fortnite服饰，目前非常困难，而NFT交易将切入这个潜力领域，变革这个市场。
  2. 懂得差异化竞争。与竞品Opensea相比，具有自己的独特之处。Mintbase提供在其他平台不常见的NFT，例如音乐、会员资格、服务、门票、新闻以及摄影等。
  3. 注重用户体验，有利于吸引普通用户进入NFT生态。Mintbase提供Minting工具致力于让普通用户也可也轻松创建属于他们自己的NFT，且可以批量生产。
  4. 产品去中心化。用户可以部署自己的智能合约创建商店；Mintbase整个Ui通过IPFS上的TheGraph驱动，当用户与区块链进行交互时，TheGraph会记录购买、转移、刻录和铸造事件。
  5. 使用NEAR公链分片技术进行扩容。目前在Mintbase上部署合约创建商店、铸造NFT代币等操作都会耗费非常昂贵的手续费，Mintbase会利用NEAR公链的分片技术来降低手续费和提升交易速度，有利于吸引新的用户和流量进入NFT领域。

作为Opensea竞品，Mintbase发展思路清晰明确，通过上架差异化NFT产品、采用NEAR分片技术来改善用户体验，个人认为是一条正确的发展路径。NFT交易市场发展潜力巨大，所以建议对这一赛道持续关注，进行布局。

Mintbase 是一个全球平台，允许任何人创建无需担心技术复杂性的 nft。艺术家们可以创建 nft 来销售数字艺术，音乐家们可以用它们来制作音乐，活动组织者可以用它们来销售他们下次活动的门票。可以把 Mintbase 想象成“非商业技术公司的商店”。人们正在创作和销售音乐、艺术、门票、摄影等等。简而言之，除了钱以外的任何东西都可以被标记。

过去部署一家商店大概要花费 2 美元，价格为 2gwei，现在这一成本已飙升至 250 美元，价格为 350+gwei，单是铸造 5 枚 NFT 也要花掉大约 50 美元。我们希望有一天可以以 NFT 的形式销售体育馆门票，但照目前的情况看，这个方法还行不通。即使明天 gas 费的价格下调至 2gwei，DApp 开发者还是会经常担心 gas 费再度经历大幅涨价。如果今天还有哪个以太坊项目没有寻找替代方案，那它肯定是在自欺欺人。

在 Mintbase 铸造 nft 的第一步是创建一家销售 nft 的数字商店。这要求 Mintbase 用户创建一个智能合约，并支付gas费用以太空。除非创建多个存储，否则用户只需要创建一个存储，而这往往是流程中最昂贵的部分。最初，这笔费用是可以管理的，花费用户大约2美元的gas费用来部署一个商店。随着以太坊网络越来越堵塞，gas费用上涨，这一成本开始迅速上升。

创建一个商店迅速上升到成百上千美元的gas费。情况已经到了这样的地步: Mintbase 积极地告诉用户，在gas费下降之前，不要开设商店。对于一个旨在简化创建、购买和销售 nft 的平台来说，这已经成为一个难以维持的局面。

Mintbase 团队在 NEAR 中发现了一个更好的区块链平台。最重要的是，Mintbase 附近的gas费用明显低于以太的。这是因为 NEAR 有一个更高效的合约执行模型，需要更少的计算量，并且因为它使用了一种名为 Nightshade 的动态切分方法。因此，在 Mintbase 上部署一个商店和铸造 nft 将变得非常便宜。

但是，Mintbase 转向 NEAR 还有很多其他原因:

  • NEAR uses NEAR 的用途Rust or AssemblyScript 生锈或组合脚本 for its smart contracts, making it easier for developers to write secure code. 为其智能合约，使开发人员更容易编写安全代码

  • NEAR has human-readable addresses for both accounts and contracts. There’s no need for an ENS workaround. NEAR 对帐户和契约都有人类可读的地址。不需要使用 ENS 工作区

  • NEAR Drops 近距离水滴 allows developers to easily onboard users by paying for their NEAR account and sending them NEAR tokens. 允许开发者通过支付他们的 NEAR 帐户和发送 NEAR 令牌来方便地登录用户

  • The 这个ETH <> NEAR Rainbow Bridge ETH < > 近彩虹桥 allows developers and users to move assets and data between both blockchains without added complexity. 允许开发者和用户在两个区块链之间移动资产和数据而不增加复杂性

  • Big DeFi players such as Balancer and Aave are building on NEAR, which will make it easier for Mintbase to integrate with them further down the line. 像 Balancer 和 Aave 这样的大 DeFi 玩家正在 NEAR 上构建，这将使 Mintbase 更容易与他们进一步整合

超过13,000人已经注册了 Mintbase，数百家商店已经建立，艺术家们正在越来越多地创新他们如何使用 nft。NEAR 是 Mintbase 所需要的坚实基础，它致力于实现一个任何人都可以创建数字资产并与之进行交易的互操作世界的愿景。

其他选择
Mintbase 未来计划添加多个链，以下是我们为何一开始没有选择这些链的一些原因（这一情况之后可能会发生变化）。

  • xDai 去中心化的程度还不够（甚至连 EOS 都不如）。12 个精心挑选的验证节点中，仅有 4 个已知的验证节点为跨链工具签署了多重签名，根据我们的理解，这意味着四分之三的签署人可以从 xDai 将 DAI 撤至主网。同时，POA （和 xDai 属于同一家族）的一些用户已经遇到扩容瓶颈了。类似的理由还有很多，不过我们这篇文章不会深入讨论这些技术上的细节。像 xDAI 这种类型的平台适合一些低于 500 美元的 POC (proof of concept，意思是实验用的原型来证明一些假设，而不是可量产的完成品)，但我绝对不会把我的 401k 之类的大额财产托管给他们。（译者注，401k 是美国的养老保险，此为隐喻 xDAI 这种平台不够去中心化，安全性不足。）

  • Matic 在指导用户入门时遇到了和使用（假设没有彩虹桥的） NEAR 同样的困难：1. 获得新通证；2. 使用拥有 RPC 切换功能的钱包；3. 指导用户切换 RPC。我并不想使用另一个第三方（如 biconomy）运行元交易。NEAR 拥有元交易和系统内置的种子账户。**随着无需信任的 NEAR 彩虹桥工具的推出，Mintbase 切换至 NEAR 的理由变得更加充分。**人们都说 Matic 正在强化以太坊的性能，不过它的最终目标和任何其他的 L1 是一样的：成为主导链。

  • Flow：**该平台的风险是只局限于游戏和 NFT 这两个领域，我们希望能和所有的平台（Defi、游戏、去中心化交易所）互动。我并不想学习他们自己发明的编程语言。**老实说，我们还没有在这方面做太多的研究工作，不过 makoto_inoue 有写过这方面的文章。

  • Rollups：我们目前的交易吞吐量还不够高，使用 Rollup 对我们来说成本太高。

  • 波卡：如果我们认为“做一条自己的链”是最好的方法，波卡会是我们的首选。

  • Cosmos： 我不想学习 Go 这一编程语言，“自己保证自己的安全”似乎是一场噩梦。

  • 以太坊 2.0：资金终有一天会面临枯竭，EF 打算在后 ETH2 阶段持续更新吗？完整版可用的以太坊 2.0 可能也是 2 年之后的事情。以太坊开发团队似乎经常加班而且报酬很低。我的观点可能是错的，只是在参与大会和与开发者交谈的过程中我的印象是这样的。

  • 以太坊 1： 状态膨胀仍然是个问题，有一天人们将无法在任何机器上同步节点。我们每个人都该在我们的计算机上有一份互联网的拷贝吗？过去关于以太坊 1x 的讨论有很多，不过现在似乎已经逐渐平息了，希望可以听到关于这个话题的更多信息。

  • Elrond： 基本元素似乎和 NEAR 很接近，不过他们需要更多的启动伙伴。

  • EOS/WAX：21 个区块生产者的基本理念适用于帮派心态，我并不想学习 C++，开发者对开发基础设施也没有太大兴趣，甚至 EOS 的创始人也是如此：block.one 的 voive 应用都是在侧链上开发的。
