【ManageEngine卓豪】移动终端管理解决方案,助力中州航空产业数字化转型

ManageEngine卓豪签约中州航空有限责任公司，助力航运产业领导者建立“移动终端管理解决方案”，提升移动终端管理和信息安全。

中州航空总部位于郑州，于2016年6月成立，是一家以郑州新郑国际机场为主运营基地的本土航空公司。中州航空致力构建“以郑州为枢纽，覆盖国内、辐射全球”的现代化航空物流网络，目前，已获得郑州-无锡-泉州，郑州-深圳，郑州-海口等航线。中州航空计划开航3年内，机队规模达12架；开航5年内，机队规模达20架；开航10年内，机队规模实现60架。

2020年统计数据显示，郑州机场完成的货邮吞吐量5.2万吨，同比增长30%，单月增速位居全国机场首位。1-4月份，累计完成货邮吞吐量同比增长4%，增速在全国机场中排名第二，成为国内仅有的2家货运量正增长的大型机场之一。截至5月，郑州机场累计完成货邮吞吐量15.6万吨，同比增长8%，其中国际地区货运量10万吨，同比增长19%；全货机货运量11.5万吨，同比增长21%。目前在郑州机场运营的定期货运航空公司已达到25家，开通货运航线44条，其中国际和地区货运航线37条，货运通航点58个，其中国际和地区通航点45个。中州航空扎根郑州、放眼全球，充分发挥基地航空公司作用，加大运力投放，拓展航线网络，做大运输规模，争取短期内成为国际货航在国内分拨衔接的主要力量，中远期成为亚太地区物流集散分拨的主力航空公司。

树立航运产业数字化转型标杆

在航空信息安全管控下，数字化转型已经成为推动企业管理的内在牵引力，作为货运航空公司，中州航空在ManageEngine卓豪的帮助下，不断探索数字化转型新模式，建立“移动终端管理解决方案”。

Mobile Device Manager Plus是一款全面的移动设备管理解决方案，旨在增强企业员工的移动性，在不影响企业安全的前提下提高员工生产力。它可以让你管理智能手机、笔记本电脑、平板电脑和台式机以及多种操作系统，如iOS、Android、Windows、macOS和Chrome OS。

许多组织的公司设备可以员工共享，或者轮流分配给团队或部门。医疗机构、物流公司、零售商和学校经常部署共享的移动设备，简化设备管理流程，降低库存成本，解决移动办公的需求，并有效管理员工和学生。

虽然使用共享设备很方便，但多个终端用户必须访问同一个公司设备，设备中可能包含大量业务关键数据，为企业带来安全风险。

中州航空所有相关人员，都需要存储和访问设备上的物流信息或其他数据，并保护敏感数据的安全。但在设备传递的过程中，并没有人关注数据是否存在篡改或丢失的情况。

不同的行业，如物流、零售和教育，也是类似的情况。IT管理员该如何部署供多个终端用户使用的设备，保护重要数据和设备呢？中州航空IT管理员可以在组织中部署移动设备管理(Mobile Device Manager Plus)软件，管理允许多用户访问的公司移动设备（包括iPad的应用程序、内容和帐户，等等）。

部署移动设备的挑战

1)用户管理

跟踪所有需要使用某个移动设备的员工相当困难，其中涉及到不同的用户在不同的时间登录不同的设备。在航运场景中，设备需要在多名人员之间传递以存储数据。这就是给中州航空带来诸多问题。

2)数据安全

由于一台设备可以被多个用户访问，所以很难确保移动设备中存储的所有公司数据的安全性。公司设备如何采取合适的安全保护策略，防止未经授权的访问和数据泄露，以免危害组织?

3）存储问题

所有用户都需要一定的存储空间来执行任务，无论是存储物流记录人员，还是在本地检测人员。如何为每个终端用户设定存储上限，以免影响其他用户的使用？

4）隐私问题

非被管共享设备缺乏隐私性，登录的用户仍可以看到前一个用户的任务，或者会话结束后仍保留着用户数据。如果确保隐私安全性？

中州航空如何借助ManageEngine卓豪的Mobile Device Manager Plus(移动设备管理软件)应对这些挑战的呢？

启用共享设备设置

使用Mobile Device Manager Plus，您可以通过启用共享移动设备设置，在允许多个用户登录的同时，您也可以对设备（包括其功能和用户帐户）进行控制。共享移动设备设置允许您像在PC或笔记本电脑上一样创建本地账户，同时为您提供设备的完全管理权限。

可用性和安全策略

为移动设备应用可用性和安全策略，以保护公司数据不被传输或滥用。例如，可以限制移动设备上的功能，如AirDrop、DataSync或MobileNetworks。假设IT管理员想限制人员只连接到特定的Wi-Fi网络，那么可以将特定Wi-Fi配置远程推送到所需移动设备上，以限制对未经批准的网络的访问。

远程清除数据

移动设备注册到Mobile Device Manager Plus后，将自动为每个用户分配相同的存储空间，从而解决了用户存储空间的使用不均问题。如果需要重新分配或重置设备，则可以使用Mobile Device Manager Plus中的安全命令远程清除数据。

简化应用程序处理流程

因为每个用户在设备上都有一个本地帐户，他们可以在不干扰其他用户的数据、电子邮件等的情况下完成日常任务。为了简化应用程序处理流程，您可以在无需任何用户干预的情况下将应用程序分发到共享的移动设备上，移动设备的所有用户帐户都将安装相同的应用程序。

为了有效地应对管理组织中共享移动设备的挑战，中州航空最终选择了ManageEngine卓豪的Mobile Device Manager Plus(移动设备管理软件)，保证数据安全的同时保护用户隐私。Mobile Device Manager Plus提供了自动化设备引导流程，简化商店和企业应用管理，执行安全策略，远程排除设备故障等多种功能。可以根据用户过去的行为以计算他们正常活动时间范围。如果在正常登录时间范围之外的登录成功，则会触发异常登录告警并将通知管理员。确保企业活动目录的安全性和合规性。
