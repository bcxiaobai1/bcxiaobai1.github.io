青龙面板一键安装自用2022年1月22日更新（复制诺兰2.4）

第一步：点击购买服务器

38和74的都可以

系统选择centOs7.6 

第二步：重置密码 

前提：服务器提前放行端口,然后登录finalshell，输入命令后回车运行。

安装宝塔面板

1，一键安装宝塔面板，需要选择的选y和yes

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

①安装之后根据提示进入宝塔在软件商店里搜索docker管理器和Linux工具箱

②在Linux工具箱里把地址改成114.114.114.114点击保存（如果你是腾讯服务器）

安装青龙面板

一键安装青龙面板代码，需要设置的直接全部回车。

wget -q https://raw.githubusercontents.com/shufflewzc/VIP/main/Scripts/sh/ql.sh -O ql.sh && bash ql.sh

拉库

一键拉库脚本，按照要求选择。（GitHub拉库速度感人，大家耐心等待）

docker exec -it qinglong bash -c "$(curl -fsSL https://ghproxy.com/https://github.com/shufflewzc/VIP/blob/main/Scripts/sh/1customCDN.sh)"

安装依赖

一键安装青龙依赖

curl -fsSL https://ghproxy.com/https://raw.githubusercontent.com/FlechazoPh/QLDependency/main/Shell/QLOneKeyDependency.sh | sh

安装诺兰

1拉源码 国内

git clone https://github.com/tracy1949/nolanjdc.git

 2 拉取基础镜像以后不需要拉取镜像了 

sudo docker pull liqinghuan/nvjdc:latest

3 执行命令

apt-get install wget unzip -y

yum install wget unzip -y

4创建一个目录放配置

 cd /root/nolanjdc
mkdir -p  Config && cd Config

 5手动建立Config.json 配置文件 （根据需要修改）

现在需要挂代理才能访问诺兰获取cookie 设置代理方法：在 13行输入 ip+端口
 格式

"Proxy": "ip+端口",

比如"Proxy": "149.56.44.70:17270",

下面写几个能用的随便输入一个即可

64.235.204.107:8080
149.19.224.36:3128
159.65.43.19:32283
149.56.44.70:17270

6 回到nolanjdc目录创建chromium文件夹并进入

cd /root/nolanjdc && mkdir -p  .local-chromium/Linux-884014 && cd .local-chromium/Linux-884014

7下载 chromium

wget https://mirrors.huaweicloud.com/chromium-browser-snapshots/Linux_x64/884014/chrome-linux.zip && unzip chrome-linux.zip

8删除刚刚下载的压缩包

rm  -f chrome-linux.zip

 9回到刚刚创建的目录

cd  /root/nolanjdc

10启动镜像(注意这里的端口为5701可以自行更改）

sudo docker run   --name nolanjdc -p 5701:80 -d  -v  "$(pwd)":/app 
-v /etc/localtime:/etc/localtime:ro 
-it --privileged=true  nolanhzy/nvjdc:latest

 至此诺兰已经配置完成ip+5701即可访问。

附：诺兰更新命令

cd /root/nolanjdc

docker stop nolanjdc
git pull
docker start nolanjdc
