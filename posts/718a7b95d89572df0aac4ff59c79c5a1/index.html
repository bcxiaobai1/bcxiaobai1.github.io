VMware vSphere简介

VMware vSphere 是领先的服务器虚拟化平台，可帮助虚拟数据中心实现一致的管理。它从一开始就能实现业务价值，同时提供强大的服务器虚拟化、突破性的高可用性、安全的自动化管理，以及可适应您环境的智能运维洞察力。此外，它还支持基于可自定义的预置模板自动进行工作负载安置和资源优化。

VMware vSphere功能介绍

VMware vSphere with Operations Management 是业界领先的虚拟化平台，同时提供洞悉运营状况和优化容量的能力。 此解决方案是云计算体系结构的关键促成因素。 vSphere 使 IT 部门能够以最低的总体拥有成本满足要求最严苛的关键业务应用的 SLA。

计算

虚拟化 x86 服务器资源并将其聚合到可以分配给多个工作负载的逻辑池。

vSphere ESXi

可提供功能强大且经过生产验证的高性能虚拟化层，对服务器硬件资源进行抽象化处理，并允许多个虚拟机共享这些资源。

DRS

通过跨主机自动平衡负载，根据业务优先级调整计算资源。 在低负载期间关闭主机以降低能耗。

vMotion

可在主机之间迁移正在运行的虚拟机，因此在计划内服务器维护时无需中断应用的使用。

Big Data Extensions

在 vSphere 上运行 Hadoop 工作负载，以实现更高的利用率、可靠性和敏捷性。

NVIDIA GRID vGPU

GRID vGPU 能够为虚拟化解决方案带来 NVIDIA 硬件加速图形的所有优势。

网络

vSphere虚拟网络连接提供针对虚拟环境优化的网络服务，并且简化了管理。

网络 I/O 控制

根据既定的业务规则定义对网络资源的访问优先级。

Distributed Switch

使用整个数据中心范围内的网络聚合来集中进行网络资源调配、管理和监控。

存储

存储服务通过抽象化处理使后端存储系统的复杂性不复存在，并支持在虚拟环境中最高效地利用存储。

Storage DRS

存储 DRS 可基于 I/O 延迟和存储容量提供智能的虚拟机放置和负载平衡机制。

基于存储策略的管理

通过策略驱动的控制层，跨存储层实现通用管理以及动态存储类服务自动化。

Storage vMotion

执行主动的无中断存储迁移，消除虚拟机存储 I/O 瓶颈，释放宝贵的存储容量。

存储 I/O 控制

通过持续监视存储卷的 I/O 负载并根据业务需求向虚拟机动态分配可用 I/O 资源，来确定存储访问的优先级。

VMFS

使用针对虚拟机优化的高性能集群文件系统来简化虚拟机调配和管理。

Storage Thin Provisioning

通过动态分配及智能部署物理存储容量，大幅度提高虚拟机的存储利用率。

存储 API

增强您的存储识别、数据保护和阵列集成功能，并可利用第三方存储供应商提供的多路径功能。

Virtual Volumes

可对外部存储（SAN 和 NAS）设备进行抽象化处理，使其能够支持虚拟机。

Flash Read Cache

可虚拟化服务器端闪存，从而提供一个可大幅降低应用延迟的高性能读缓存层。

安全性

VMware vSphere提供了目前最强健、最安全的虚拟化平台。

vShield Endpoint

通过将防病毒功能卸载到经过强化的安全虚拟机，使这项功能不占用虚拟机空间，从而提高防病毒扫描性能。

可用性

提高基础架构各方面的可用性，包括应用、存储资源、基础架构和管理。

High Availability

在整个虚拟化 IT 环境中实现高可用性，而不像传统集群解决方案那样花费高成本或实现起来过于复杂。

Fault Tolerance

在服务器出现故障时确保应用的持续可用性，实现数据零丢失。

Data Protection

通过快速的无代理磁盘备份保护您的数据，使用重复数据消除最大限度地减少备份磁盘空间的使用量。

复制

使用内置的 vSphere Replication 消除第三方复制成本。

自动化

自动化可提供准确、一致和可重复的解决方案，节省 IT 管理员的时间和精力。

Auto Deploy

仅需数分钟即可部署和修补 vSphere 主机。

主机配置文件

只需创建一次配置文件，然后即可用该文件设置多个 vSphere 主机。

Update Manager

通过自动跟踪、修补和更新 vSphere 主机以及应用和操作系统，减少日常补救所花费的时间。

内容库

从一个中央存储库轻松管理虚拟机模板、vApp、ISO 映像和脚本。

管理

Management 可在 vSphere with Operations Management 环境中提供核心控制点，同时还能通过添加工作流、监控和性能数据以及容量优化，提供真正的托管虚拟化解决方案。

vCenter Server

集中管理 vSphere with Operations Management

vCenter Orchestrator

使用简便的拖放式界面组建工作流

vCenter Operations Standard

洞悉环境的运行状况，并允许进行容量优化

  • 通过整合服务器降低资金和运营成本。 通过更充分地利用现有的服务器并限制您需要管理、供电、存储和购买的物理资源来获得更多收益。 许多小型企业在利用 vSphere 进行虚拟化之后，x86 服务器利用率从 5-15% 增长到了 60-80%，整合率超过了 15:1。
