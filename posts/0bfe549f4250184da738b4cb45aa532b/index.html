MEXC Global分享：一文了解参加波卡插槽拍卖的项目情况

2015年，Gavin离开以太坊并与以太坊基金会的前安全主管Jutta Steiner创办了一家名为Parity Technologies的区块链软件开发公司，之后又创立了Web3.0基金会，并于2020年5月上线了Polkadot。

经过了长时间的发展，波卡生态系统已经成为了全球知名的公链之一，并拥有了庞大的社区基础，下面我们来盘点一下目前提出要参加波卡插槽拍卖的项目情况。

Acala

Acala Network是波卡生态系统的一站式DeFi中枢，由去中心化开放金融协议公司Laminar和PolkaWallet共同创立，旨在为开发者提供一系列开箱即用的DeFi基础模块，使其便捷的打造一系列供终端用户使用的应用程序。致力于为全新的Web3.0世界提供低门槛的金融稳定性、流动性，以实现更加开放公平的未来。

在早先Kusama的平行链插槽拍卖中，Acala的先行网Karura作为第一个获得Kusama平行链插槽租赁权的项目为Kusama提供了巨大的流动性，成为了Kusama中必不可少的明星项目。

在本次波卡平行链插槽拍卖中，Acala Network拿出了ACA总量的17%作为竞拍奖励，即1.7亿枚ACA，以求获得96周的平行链插槽租赁期。100%的ACA会在网络上线支持转账后进行分发，其中20%的ACA将在上线后支持转账，剩余的ACA在96周内以区块为单位线性释放。同时还有额外奖励和限量版的NFT吸引着社区用户的积极参与。

同时通过IcDOT进行Crowdloan的用户，100%的IcDOT将与ACA同时在上线后发放，100%的IcDOT将是流动的并可以在Acala DeFi协议内使用。让用户在享有Crowdloan激励的同时，最大程度上获取流动性。在96周平行链租赁期结束之后，IcDOT可以1：1赎回DOT。

Moonbeam

Moonbeam是波卡上的以太坊兼容智能合约平台，让多链项目可以轻松的扩展到波卡生态当中，获得波卡带来的互操作性、可扩展性和安全性。而对于中继链本身不具备智能合约功能的波卡生态系统而言，扮演波卡生态智能合约链角色的Moonbeam的重要性不言而喻。

在早先Kusama的平行链插槽拍卖中，Moonbeam的金色雀网络Moonriver成功的获得了Kusama第二个平行链插槽的租赁权，并在一个月之内完全的启动。由于其良好的性能，获得了以太坊中众多知名项目的支持。

而在本次波卡平行链插槽拍卖中，Moonbeam将初始供应量的10%作为总的竞拍奖励，即1亿枚Glimmer。30%在平行链上线时解锁，70%在96周的租赁期内以区块为单位线性分配。

Astar Network

Astar是波卡生态早期的技术性项目，是基于Polkadot的多虚拟机、多链、Layer2可拓展化智能合约平台，获得了超过7次Web3基金会的Grant资助，是波卡生态系统中的明星项目，致力于打造波卡原生智能合约生态系统。

在早先Kusama的平行链插槽拍卖中，Astar的先行网Shiden作为Kusama上的多链去中心化应用层，获得了第三个插槽的租赁权，为Kusama提供了多样化的功能和巨大的可扩展性。

在本次波卡平行链插槽拍卖中，Astar将总供应量的20%用于Crowdloan，15%作为基础奖励，5%作为额外奖励池。Astar会在启动转账功能后分发10% ，剩下90%会在22个月内线性释放。除此之外，10ASTR/DOT的推荐人奖励、20%的早鸟奖励和10%的Shiden支持者奖励等等额外奖励也非常的诱人。

Parallel Finance

Parallel Finance是建立在波卡生态系统上的去中心化贷款协议，在Polkadot生态系统中可提供杠杆抵押和拍卖借贷，使社区能够提高资本效率、安全性和可访问性。
通过Liquid Staking、Margin Staking和Money Market协议的创新性设计，允许用户在Parallel Finance系统中同时进行抵押和借贷，获取流动性的同时赚取双重利率收益。

目前Parallel的先行网络已于10月6日成为了Kusama网络的第10个平行链。

Auction Loan贡献通道已经开启，用户可通过Auction Loan产品Contribution来赢取Parallel的40%早鸟奖励。Auction loan是Parallel Finance为Crowdloan参加的用户推出一款可以用来提高资金利用率来获得更高收益的产品。

Parallel将总量（100亿PARA）的10-15%用于Crowdloan贡献（15亿枚PARA通证）

  • $PARA供应总量：10,000,000,000

  • Crowdloan奖励分配/供应总量：10-15%

  • Parallel's 平行链租期：96 weeks

  • $PARA解锁周期：24个月线性释放

Parallel提供5种奖励方式，通过Auction Loan产品贡献Parallel Crowdloan，用户可以获取所有奖励种类。

  • 基础奖励：1 DOT = 33 PARA

  • 推荐奖励：10%（推荐人获得5%；被推荐人获得5%。

  • 社区早鸟奖励：高达40%

  • NFT奖励：Parallel将会为参与Parallel Crowdloan的用户分配1000个NFT作为激励。这些NFT在未来将可以在Parallel的平行链进行使用

  • cDOT作为流动性资产：通过Auction Loan贡献任何参与Crowdloan项目，用户都会1DOT可以得到1cDOT作为在2年DOT质押过程中的流动性资产。

  • 在Parallel的平行链上线后，可以在Parallel的借贷市场和AMM市场中使用。

Menta Network

Manta Network是波卡首个链上隐私保护项目。Manta Network致力于通过隐私保护构建一个更好的Web3世界。Manta的产品设计从第一性原理出发，通过zkSNARK 等领先的密码学架构为区块链用户提供端对端的隐私保护。在保障隐私的同时，Manta兼具互操作性、便捷性、高性能以及可审计性，允许用户进行任意平行链资产间的隐私转账和交易。

目前Manta Network先行网Calamari已于9月15日成为了Kusama网络的第七个平行链。

Manta Network最新Crowdloan策略：

  • 不再采用1DOT：5MANTA固定奖励比例，而是将15.6% MANTA奖励全部按贡献比例分配给首轮波卡平行链Crowdloan参与者，其中包括12%基础奖励、3%流动性挖矿奖励和0.6%的Binance贡献奖。

计算公式为：基础奖励=（您的 DOT 贡献 / 对 Manta 的 DOT 总贡献总额） * 120,000,000 MANTA；流动性激励奖励= （您的 DOT 贡献 / 对 Manta 的 DOT 总贡献总额） * 30,000,000 MANTA。其中，流动性激励奖励在去中心化平台参与均可获得。

  • Binance贡献奖保持不变，11月4日至11月11日，通过Binance质押的贡献者在基础奖励之外，可瓜分 6,000,000 MANTA，同时获得额外10% MANTA的早鸟奖励。据估算，目前Manta在Binance的DOT质押量已超过150万。

  • 在波卡首条平行链竞拍开始前，通过所有平台参与的贡献者均可以获得额外10%MANTA，第三条平行链竞拍结束前可获得额外5% MANTA。

Phala Network

Phala是Polkadot生态中基于Substrate的隐私保护云计算服务。大家所熟知的数据隐私只是 Phala Network白皮书中提到的基本功能之一。Phala的愿景是成为全球最大的P2P计算网络，即基于Web 3.0的标准去中心化计算云。

实现这一目标的一个重要前提是实现计算网络本身的去信任化：Phala网络上的所有计算都运行在安全的飞地中。安全飞地保证计算负载的安全，包括程序执行的正确性，以及数据的机密性和隐私性。Trustless不仅是Phala社区认可的第一个特性，也是Phala成为可靠的分布式计算网络的基石。

此外，“共享经济”模式也是Phala的Web 3.0特色。为了保证网络用户和算力提供者不会被中心化的互联网公司控制，Phala采用了区块链协议，允许任何计算节点加入网络并提供可用和稳定的计算服务。这也是Secure Worker Mining机制的基础。

目前Phala已于7月13日成为了Kusama网络的第四个平行链。根据Phala Network的上线规划，其将经历三叠纪、侏罗纪时代、白垩纪和全新世四个时期。目前，Khala Network已进入全新世时期。

而关于波卡插槽的计划，Phala在早前的计划是在赢得Kusama插槽的后一年争取赢得波卡的插槽。Phala将划拨占总发行量一定比例的原生PHA 作为竞拍成功奖池。大致设计为Kusama插槽：0.5%-1% （500~1000 万），Polkadot插槽：3%-5% （3000~5000 万）。

不过最近Phala网络的联合创始人在推特上发文表示：希望Phala社区公开讨论，共同做出决定。他建议Phala不获得第一个插槽，而是加入12月23日开始的第二批Polkadot插槽拍卖。

SubDAO

SubDAO是波卡的DAO基础设施，专注于解决Substrate链多样化治理的DAO平台。它允许任何去中心化组织快捷地创建和管理DAO。SubDAO不仅可以链接DAO与DApp，实现DAO的跨链管理，还将为Web2.0连入Web3.0世界提供入口。

此前SubDAO宣布将参与Polkadot第一轮平行链竞拍，SubDAO一直以来积极筹备已推出平行链网络Starfish（海星网络）并发行GOV（Governance）作为网络通证，将从第1个插槽开始（11月11日）全力参与Polkadot平行链竞拍并赢得插槽，为波卡的中继链和平行链提供基于区块链的电子协议签署、DAO社交、资产管理等工具与服务以及DAO管理功能。

SubDAO提供多种奖励：

  • GOV竞拍总奖励：GOV首次发行总量的30%，即3亿GOV。

  • 基础奖励：基础奖励为20% ，即2亿GOV。每支持质押1枚DOT获初始奖励20GOV，预期筹集10,000,000 DOT投票。除了2亿的基础奖励，还有1亿GOV作为其他奖励。

  • 早鸟奖励5%：SubDAO对前1000名质押用户，将会奖励额外5%的早鸟加成福利。）

  • 邀请奖励5%：SubDAO官方Crowdloan页面可获取邀请链接与专属推荐ID，使用推荐码成功邀请用户，推荐人与被推荐人均将获得被推荐人的 5% 邀请奖励。）

  • “Strong Supporter” 最高10%：投票贡献100 DOT或以上的用户，将获额外的加成奖励

  • “冲刺计划” 最高6%：当支持竞拍锁定的DOT总数达到某个数量级，所有竞拍支持者都将获得奖励加成

  • 在Starfish Network成功拍得插槽且作为平行链顺利运行后，GOV奖励将立刻释放34%；剩余 66% 的GOV奖励将分为 96 周线性释放。

目前，确定支持SubDAO竞拍的渠道包括Polkadot-JS、Bifrost、Zenlink、Math钱包等去中心化平台，以及Huobi、Gate、MEXC等中心化交易平台。

Bifrost

Bifrost是Polkadot生态DeFi基本协议。致力于成为质押资产提供流动性的基础设施。通过Bifrost用户可以随时将PoS币种通过Bifrost协议兑换成vToken获得Staking收益和流动性。

目前Bifrost已于7月20日成为了Kusama网络的第五个平行链。截至目前，Bifrost网络已经支持Kusama和Bifrost的双向资产跨链，KSM可在Bifrost与Kusama网络中进行互转操作。

Bifrost团队计划在Polkadot上部署插槽竞拍流动性协议SALP，将于北京时间11月9日18时在Bifrost Dapp Dot Slot模块开启Derivatives vsDOT+vsBond的铸造作为参与波卡平行链竞拍的支持。
参与Contribute的用户将拥有解锁平行链锁仓DOT Liquidity释放的优势，而通过Bifrost Dapp DOT Slot投票支持Bifrost的用户，将在获得的候选平行链提供的奖励的同时，获得Derivatives: vsDOT和vsBond。

最新消息，Bifrost团队正在酝酿波卡平行链插槽计划，将会晚些时候发布，敬请期待。

KILT

KILT是最早采用由Parity Technologies创建的模块化系统Substrate的公司之一。KILT是一个开源的区块链协议，用于为Web 3.0发布自主、匿名和可验证的凭证。与对等解决方案不同的是，KILT使用区块链技术提供了自独占数据和可撤销凭证。

KILT将真实世界可验证凭证（护照、驾驶证、教育证书等）的传统信任过程带入数字世界，同时保持你的数据私密和所有权。KILT还可用于为机器、服务和任何可以建立身份的东西创建标识符。

目前KILT协议已于9月8日成为了Kusama网络的第六个平行链。KILT协议已经上线，并进行Runtime升级，引入治理（内部测试）和国库。接下来会陆续添加KILT功能，如去中心化标识符（DID）和可验证凭据。当KILT功能测试和完整系统结束时，将删除Sudo模块并实现去中心化。

KILT此前宣布计划参与波卡的插槽竞拍活动，但目前尚未公布细则。

HydraDX

HydraDX是基于Substrate创建的，完全去中心化和开放的协议，采用了一种受纽带曲线启发的通证模型，旨在处理多种资产池，并帮助协调不同参与者和协议之间的关系。
相较传统的AMM，HydraDX的目的是降低滑点、提供更深的流动性（最小化流动性分散）、更好的内部可组合性和执行和用户体验，以此希望为跨链DeFi提供“如同海洋般”的流动性。

HydraDX协议已于9月22日成为了Kusama网络的第八个平行链。HydraDX此前宣布计划参与波卡的插槽竞拍活动，但目前尚未公布细则。

Interlay

Interlay的愿景是打造一个区块链无缝连接和交互的未来：任何人都可以在任何区块链上使用任何数字资产，不受信任且不受限制。
其目前重点打造的产品是InterBTC，将成为去中心化和去信任的比特币的“桥”，首先会连接到波卡和Kusama。InterBTC允许用户1:1铸造BTC支持的资产PolkaBTC到波卡区块链上，并在广泛的应用中使用这些资产，包括去中心化交易所、稳定币和贷款协议。

目前Interlay先行网Kintsugi已于10月13日成为了Kusama网络的第十一个平行链。

由于Interlay先行网Kintsugi刚刚赢得Kusama插槽，在Interlay在Polkadot上做任何事情之前，Kintsugi将上线所有功能。
Interlay的interBTC来自XCM。只要其他平行链和XCM还没有100% 准备好，interBTC就无法发挥其全部潜力。所以Interlay Network将不会参与波卡平行链的首次插槽竞拍，而是参与2021年12月23日的第2批竞拍活动。

Litentry

Litentry是一个去中心化的身份聚合器，以基于Substrate构建的区块链和分布式的去中心化身份（DID）验证网络为核心特色。在Polkadot生态系统中发挥独特的功能，为多链身份数据提供Liquidity、互操作性和一致性。
可以给不同系统的DID数据提供互操作性和流动性，并通过分布式去中心化身份（DID）索引和基于Substrate构建的信用计算网络，为DID用户和DID 应用程序提供了提供了包括DID验证、信用计算及评级等身份聚合服务。

由于考虑到时间、成本和资源，Litentry将直接参加Polkadot平行链插槽拍卖，跳过Kusama插槽拍卖。

Litentry会拿出总供应量的20%来激励那些帮助Litentry在拍卖中获得优势的支持者。所有拍卖的支持者可以瓜分总共两千万个LIT的奖励池。每锁定1DOT，参与者将获得至少2.5LIT的奖励。最终奖励将以每位参与者对Crowdloan池贡献的百分比计算。
除此之外，用户还可以获得最高20%的额外奖励，11月11日前，在Binance参与可以瓜分额外50万LIT，同时用户还可以通过Parallel的Auciton loan来解锁DOT流动性并获得额外的PARA奖励。

Efinity

Efinity是一个建立在波卡上可拓展的NFT跨链网络，将帮助人类创造一个真正自由的全球数字经济。由前沿的区块链科技公司Enjin开发，他们曾在以太坊上创建ERC-1155 NFT标准。
Efinity 通过与波卡的合作，将能够提供一个可访问、可扩展的解决方案，使每个人都能参与新兴的NFT经济。

Efinity此前宣布计划参与波卡的插槽竞拍活动，在11月11日之前参与Efinity Crowdloan便有机会在1,500,000美元FEI的奖池中获得奖励。
