过去、现在、未来：UGC 将如何点燃元宇宙？

自互联网早期以来，我们今天居住的数字空间已经发生了很大的变化。下一个重大的进化转变正在迅速展开，许多世界上较大的科技公司宣布他们打算开创所谓的 “元宇宙”。

正如风险投资家 Matthew Ball 所描述的，“[ The metaverse ] 是一个具象化的互联网，由实时渲染的 3D 虚拟世界的大规模且可互操作的网络组成，实际上无限数量的用户可以同步且持续地体验个人的存在感。”

与互联网的先前迭代不同，元宇宙将主要由其用户构建和拥有，而不是大型集中实体。因此，用户生成的内容 ( UGC ) 将是其核心。

什么是用户生成的内容 (UGC)，为什么它对元宇宙很重要？

UGC 是由用户而非专业雇用的内容创建者创建的内容（例如图像、视频、文本和音频）。

 

UCG 并非数字领域独有，但了解它如何影响互联网、网络和社交媒体对于了解它如何塑造元宇宙至关重要。

元宇宙和最新的万维网迭代（称为 Web 3.0）是同一枚硬币的两个方面。元宇宙展示了用户将如何体验未来的互联网，而 Web 3.0 则是关于谁将拥有、共同创造和控制未来的互联网。

Web 迭代和用户生成内容的时间表

UGC 在 Web 的三个主要迭代中的作用，以更好地解释它将如何塑造元宇宙。
 

Web1.0

早期的 Web 由少数拥有先进技术技能的个人和组织创建的静态只读网页组成。这样的网页几乎没有提供交互性。尽管如此，他们的创作者仍然控制着内容，因为这些网页要么托管在创作者自己的服务器上，要么直接租用。引入了诸如 Microsoft FrontPage 和 Macromedia 的 Dreamweaver 等工具来使网站创建民主化，但需要一些 Web 开发和编码技能。

Web2.0

我们当前的迭代 Web 2.0 的特点是集中式平台，如 Facebook、YouTube 和 Medium。除了平台之外，这个时代还被赋予了新的移动工具，如 iPhone 和 Android 手机，它们提供连接和媒体捕获功能。连接到社交平台的智能手机和移动应用程序的迅速普及使 UGC 的数量猛增到了曾经难以想象的数量。这些工具和平台使用户可以轻松地创建内容并与之交互，但许多工具和平台限制了内容创建者对内容的控制程度。中间人（平台）真正从 UGC 中获利并从中受益，同时提供有限的财务参与到创作者的成果分配制中。

Web3.0

得益于区块链技术和建立在其上的各种解决方案（例如不可替代的代币 [NFT]），Web 3.0 承诺让内容创建者能够在不使生产复杂化的情况下控制其内容。它旨在通过将内容从集中式平台转移到分散的分类帐和类似内容的聚合器中来实现这一目标。
 

“无论元宇宙如何形成，UGC 都将成为赋予其个性、真实性和规模的核心构建块。通过 UGC 鼓励创造力并允许表达自由是所有元宇宙体验都应该努力做的事情。”

——Scott Reismanis，ModDB 的创始人

内容的去中心化和保证所有权对于元宇宙至关重要，因为它激励用户花费大量时间为其创建 UGC。内容创建者可以对其产品进行更精细的控制，甚至可以通过可配置的版税享受初始销售之外的补偿。在 Web 3.0 上，创作者还将受益于更大的互操作性，由此他们的单一产品可以在各种元宇宙中看到和使用。

UGC已经在塑造早期的元宇宙项目

没有 UGC，元宇宙永远不会成为互联网的具体化版本，因为没有这种类型的内容，互联网就不会是今天的样子。

它不再是人们消费内容、学习、社交和分享他们的想法和最深刻感受的互动场所，它更像是那些基于文本的电视服务的现代版本，称为图文电视或广播图文电视。

UGC 对元宇宙的重要性可以在早期的元宇宙项目中观察到，例如 Somnium Space、The Sandbox、Roblox 等。

这些原始元宇宙的成功很大程度上归功于玩家创造的内容。这与开发人员的归因形成鲜明对比，他们努力为内容创建者提供易于使用的工具，并通过根据他们的创作使用量来赚钱并激励他们。

毫无疑问，Roblox 的顶级 UGC 会产生大量参与度。例如，有史以来最受欢迎的 Roblox 游戏宠物模拟器 Adopt Me ，截至 2022 年 2 月，访问量超过 273.9 亿次。母公司背后的独立工作室雇佣了大约 40 名员工，年收入 5000 万美元，其中大部分来自微交易。

但是，诸如 Adopt Me 之类的 UGC 高调示例仅仅是个开始。元宇宙中的大部分 UGC 将更加平凡和无关紧要，就像今天网络上的大部分 UGC 是盲目的评论和用户因兴趣而产生的自转发一样。然而，这种平凡的 UGC 正是虚拟世界需要让人感觉鲜活和真正身临其境的东西。

我们所知道的互联网正在发展并逐渐变得身临其境。很快，我们将体验到它是一个实时渲染的 3D 虚拟世界网络，用户生成的内容将在这些世界的外观以及可以在其中做什么方面发挥重要作用。几乎所有年龄段的人学习的方式将在未来十年发生根本性变化。

与 Meta 的中央控制元宇宙相比，世界各地的建设者正在共同努力开发替代生态系统的基础：多元宇宙。也称为 Web 3，hyperverse 是一个逐步去中心化的网络，也是一个互连的、可组合的、协作的互联网。它是一个非线性数字空间。它包括多媒体、社区和金融协议，并以作为永久和无需许可的共享基础设施层的区块链技术为基础。

像 DAO（去中心化自治组织）这样的新组织模式正在出现，以支持这种演变，重新调整平台激励措施以建立集体价值。目标是使用协作经济学重新开放互联网 —— 通过将传统实体与 DAO 相结合、通过传统激励措施与代币化模型相结合以及通过传统技术与区块链系统相结合。

炫弛介绍

炫弛世界是一家以区块链技术为核心，自主创新研发线上线下相结合的元宇宙电商、社交平台。炫弛世界将构建一个平行于传统的物理世界之外的虚拟世界，后期增加一比一映射北京、上海、深圳等现实中的一二线城市，而现有的中心城是通往这些城市的“传送门”。用户不仅可以创建自己的商城小店，也可以租赁给其它入驻的商户，用来搭建商城小店，开启元宇宙商城购物、社交娱乐、办公等。
