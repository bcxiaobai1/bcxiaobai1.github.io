章鱼加速器每年会提供100万美金，资助20个优秀的 Web3.0 应用链创业团队。我们将会和每一位 Web3.0 创业者深度碰撞，把项目打磨到最佳状态。

章鱼加速器 

由创业者组成的 Web3.0 社区

章鱼加速器，是由 Octopus Network 章鱼网络发起，由创业者组成的 Web3.0 全球创业社区。我们的目标是：帮助 Web3.0 创业者迈出第一步，并且帮助应用链项目成长到足够优秀。

章鱼加速器的 Web 3.0 课程已经全部免费公开，涵盖「应用链创业导论」、「产品设计」、「社区建设」、「应用通证工程」、「加密世界的监管」和「 融资指导与冲刺」等主题。

章鱼未来之星 

100万美元的章鱼奖金/年

章鱼网络「Octopus Appchain Showcase Day」，旨在为 Web3.0 应用链创业团队搭建展示和交流的平台。章鱼加速器本期创业营项目整体质量极高，经过3个多小时现场演讲和展示，评委团在一周内提名和反复讨论，终于评选出5个项目授予「章鱼未来之星」称号，排名不分先后，他们分别是：

Atocha Protocol

https://atocha.io

Atocha Protocol 为益智游戏社区提供创建、解答、赞助和挑战谜题的基础设施。

您是否正在寻求冒险并想通过解谜赢取奖品？除了 P2E 模式，我们还有 C2E 模式，让创作者的创造力得到更多奖励。欢迎成为我们社区的一员，一同创造并沉浸于令人兴奋、充满挑战的解谜体验中！

Discovol

Discovol

Discovol 是一个去中心化的优质内容发现与分发平台。

我们帮助那些想在社交网络中分发高质量内容的人，高效地找到自己需要的内容；我们也帮助创作者，将自己精心制作的内容，通过分发者群体，分发给更多的目标受众。

Fusotao Protocol

FUSOTAO

Fusotao Protocol 是一个采用「链下撮合，链上验证」范式的撮合验证协议，项目发起人基于它可以创建订单簿模式的 DEX。使用基于 Fusotao Protocol 开发的 DEX，社区用户可以交易获取奖励，或者通过为这些 DEX 质押代币而直接分取交易手续费。

PNS

Polkadot Name System

Polkadot Name System ，Polkadot 生态的去中心化域名系统，是基于区块链、开源、去中心化的新一代互联网域名系统。

PNS 将会以波卡区块链开发框架 Substrate 的 Pallet 的形式实现，来为多链生态所有平行链提供域名服务。PNS 将会提以.dot后缀的全局唯一域名，此域名可以解析多种链上链下信息，包含钱包地址、智能合约地址、IPFS 地址、NFT 地址、传统网址、SSL 证书、社交账号、邮箱等。域名的管理和使用都全权交给用户,从而为用户提供去中心化的数字身份，安全、透明、开放的多链域名系统，助推 Web3.0 愿景的实现。

Web3Go

web3go

Web3Go 是一个开放的数据分析平台，用户可以通过强大而易用的工具提取、管理和可视化数据。

Web3Go 扎根于 Polkadot 生态系统，提供一系列与Web3数据相关的服务，包括：跨链资产交易、平行链拍卖、DeFi 交易历史的微观层面分析和可视化；完善的地址标注系统，根据链上的行为数据创建用户肖像，积累用户身份库；利用 Substrate 智能合约部署奖励系统，激励用户在 Web3Go 上发布数据任务和悬赏数据分析者。
