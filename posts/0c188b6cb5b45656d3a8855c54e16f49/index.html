一、虚拟机概述

1、什么是虚拟机？

●运行在计算机上的一款软件程序

●能模拟计算机硬件功能

●一个独立的计算机环境

2、软件运行的架构

①传统运行模式

●一台计算机同时只能运行一个操作系统

●在系统之上运行软件

②虚拟机运行的模式

2.1 寄居架构

●作为应用软件安装在操作系统上

●可安装多个操作系统

●宿主（Host OS）与客户（Guest OS）

2.2 原生架构

●虚拟机软件直接安装在计算机硬件上

●虚拟机本身就是一个操作系统

●虚拟机中可以同时运行多个操作系统

2.3 寄居架构和原生架构的区别

原生架构相对寄居架构缺少宿主，运用程序访问底层硬件时通过操作系统对虚拟机直接就可以访问到底层CPU等，但是寄居架构中间还需要通过宿主才能去访问底层硬件，耗时较长。并且单宿主系统发生问题时，会直接影响到虚拟机上的其它操作系统。

3、虚拟机产品

①VMware（威睿）

●VMware Workstation

目前所使用的就是此虚拟机，主要为寄居结构。

●VMware vAphere

 此类为原生结构

②Micro Soft（微软）

●Virtual PC

●Virtual Server

●Hyper-V

原生架构程序，效率很高

③Citrix（思杰）

●XenDesktop

桌面虚拟化产品（简单理解：就是运行的不是一台整配的电脑，而是一个瘦客户机，瘦客户机配置很低，主要作用就是远程主服务器，瘦客户机也可当成一个连接设备，有输入和输出的设备。一台主服务器可以进行多台瘦客户机连接，合理完整进行资源的利用，对于数据的管理，安全维护等操作，也是极为方便）

●XenServer

原生架构

二、虚拟机的安装与简单操作

在前面操作系统的的文章中已经描述了如何安装

1、启动和停止虚拟机

2.1.1 打开虚拟机电源

2.1.2 右击选项卡，找到电源，选择关闭虚拟机电源，或者直接将虚拟机内部系统关机

2.1.3 挂起虚拟机

将虚拟的运行的所有程序，进行暂停，相当于按下了暂停键。

2.1.4 重启虚拟机

当虚拟机无法声场运行时，可以进行重启虚拟机。

三、虚拟机的管理

1、VMWare Tools的作用

●这是VMware Workstation的一个工具

主要作用：

●增强虚拟机硬件性能的驱动程序

●支持虚拟机使用宿主机打印机的功能

●可以实现宿主机到虚拟机的文件复制粘贴

2、安装VMware Tools

在操作系统那一个文章中已经描述了如何安装

3、虚拟机快照特点

3.3.1 快照功能可将系统恢复到指定的状态

3.3.2 输入快照的名称以及描述，点击拍摄快照，即可开始拍摄快照

3.3.3 在左下方会显示快照的状态进度信息。

 

 3.3.4 右击选择快照，点击快照管理器，即可查看所有快照

3.3.5 上面显示拍摄的快照，以及当前位置，可以拍摄多个快照，双击任何一个快照即可恢复成快照时所保存的样子。

4、虚拟机克隆特点

●克隆功能可将快速创建另外一台虚拟机

●创建克隆时，虚拟机要关机状态才可以完成创建。

①创建链接克隆

链接克隆是寄存于被克隆的系统的，当以前的系统发生删除时，连接所创建的克隆将无法正常运行。

4.1.1 右击选项卡，找到管理，点击克隆。

4.1.2 点击下一步

4.1.3 点击下一步

4.1.4 选择链接克隆后点击下一步

4.1.5 选择链接克隆的位置，连接克隆的名字，即可完成创建

②创建完整克隆

完整克隆相当于将被克隆的系统完全复制一份，计算机名，IP地址等都一样，创建完成后两个系统是相对独立的，不论另一个系统如何操作，都不会影响到另一个系统。

 4.2.1 在这一步时，选择创建完整克隆

4.2.2 修改路径跟名称点击完成即可开始创建
