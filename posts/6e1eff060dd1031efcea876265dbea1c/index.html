1.安装epel-release

epel（xtra Package for Enterprise Linux）是为“红帽系”的Linux系统提供的额外软件包（centos属于红帽系），相当于是一个补丁，大多数rpm包在官方repository中是找不到的。

[root@]# yum install -y epel-release
