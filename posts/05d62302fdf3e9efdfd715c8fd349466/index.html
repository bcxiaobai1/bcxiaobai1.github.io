IDEA常用快捷键和IDEA自定义模板设置（一个专业程序员所需要掌握的快捷方式）

1.注释

单行注释：Ctrl + /
多行注释：Ctrl + Shift + /

2.撤销

Crtl + Z

3.复制整行

快捷键自己设置的，可以点击File -> Setting -> Keymap -> 搜索duplicate -> 找到后要右键先移除原来的快捷键然后再右键增加快捷键

4.删除整行

快捷键也是自己设置的，要查看系统快捷键或者更改为自己的快捷键，方法同上

5.运行

也是自己设置的快捷键，可以按照上面操作去设置

6.代码快速排版（超级好用）

这个可以使代码全部快速排版，是有强迫症的同学福音，用了都说真香，所以赶紧去设置一个自己觉得方便的快捷键吧

7.alt + Enter (多用处)

  1. 当一个类没有进行导包的时候，光标在该红色的类上，按下 Alt + Enter 可以快速导包
  2. 自动分配变量名，例如 new Properties()按下 Alt + Enter 可以自动生成为

还可以通过在后面加 .var ，例如 new Properties().var来自动分配变量名，看个人习惯吧

8.生成构造器（很有用）

Ctrl + Insert
不仅仅可以快速生成构造方法，还能生成实现接口方法等等

9.查看一个类的层级关系（有用）

Ctrl + H

右键该类 -> Diagrams(最下面) -> show Diagrams
可以可视化图表展现类级关系

10.自定义模板

File -> Settings -> Editor -> Live Templates 进入模板页面
