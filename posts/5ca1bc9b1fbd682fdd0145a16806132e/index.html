科普|什么是元宇宙

作者概述：Zoey | NFTScan 研究员，NFTScan 是一个多链 NFT 浏览器。

在过去一年多的疫情期间，人们的出行多多少少受到限制，因此许多人际互动从现实世界中转移到了网络上。一些人通过元宇宙参加各种活动。那最近大热的元宇宙是什么呢？元宇宙「Metaverse」这个词，应该大家都多少有所耳闻，之前以改名为 Meta 的 Facebook 为首，有不少的公司在营销这个概念，并且视其为网络的未来。然而，如果大家仔细去看所谈的元宇宙实际上为何物的话，就不难发现其实每个人所说的都莫衷一是，而是以各自的强项为出发点，去阐释其心目中的元宇宙。

元宇宙的内涵是吸纳了信息革命（5G/6G)、互联网革命（web3.0）、人工智能革命，以及 VR、AR、MR，特别是游戏引擎在内的虚拟现实技术革命的成果，向人类展现出构建与传统物理世界平行的全息数字世界的可能性；引发了信息科学、量子科学，数学和生命科学的互动，改变科学范式；推动了传统的哲学、社会学甚至人文科学体系的突破；囊括了所有的数字技术，包括区块链技术成就；丰富了数字经济转型模式，融合 DeFi、IPFS、NFT 等数字金融成果。来自———数字资产研究院学术与技术委员会主任朱嘉明教授。

一. 元宇宙概念

2021 年被称为元宇宙元年，但元宇宙的设想、理念、概念、技术等并不是 2021 年才突然出现的，而是 70 多年来随着人类的想象、研发、实践逐渐产生和发展起来的。

Metaverse 一词最早来源于尼尔·斯蒂芬森 1992 年的科幻小说《雪崩》，这个词开始是来描述一个基于虚拟现实的互联网世界。在本书中，Metaverse 是一个虚拟的城市环境，沿着一条道路发展，覆盖了一个无特征的、黑色的、完美球形星球。这个虚拟世界中的土地可以被购买，并在上面开发建筑。Metaverse 用户可以通过高质量的个人虚拟现实眼镜，或通过高品质的公共虚拟现实眼镜进入，并与彼此产生互动。在 Metaverse 中，人们能够在其中以 avatar (虚拟人物角色) 自由生活，步行或车为交通工具，就像现实中一样。

元宇宙本身不是一种技术，而是一个理念和概念，利用科技手段进行链接与创造，与现实世界映射与交互的虚拟世界，具备新型社会体系的数字生活空间。当前，元宇宙被认为是 Web 3.0 的实现，依赖于区块链的存在，最终区块链和 NFT 等技术创造了一个真正去中心化的数字世界。

 

二. 元宇宙具备特征

1.沉浸式体验

如果给元宇宙打一个最典型的标签，那无疑就是参与和沉浸式体验。一方面用户通过 AR 增强现实、VR 虚拟现实和 MR 混合现实技术体验到元宇宙无处不在的沉浸体验，且把互联网的 2D 平面式体验提升至 3D、4D 甚至更高的层级上，除了使用在游戏、社交等娱乐中，还将逐渐的扩展到各种线上线下一体化的应用场景。

另一方面，元宇宙是一个能够完整运行的、跨越现实和虚拟世界、始终实时在线的世界，无穷无尽的人们可以同时参与其中，并且能够连接互通。

2. 去中性化

当然，目前元宇宙必须借助区块链等去中心化手段，来形成全新的数字资产、智能资产的获取和分配方式。元宇宙强调去中心化，即避免出现大的中心节点而影响普通参与者的权益，实现去中心化的技术。目前市场上被质疑蹭热点的“伪元宇宙”、炒概念的大有人在，正因为有这些模糊概念的观点存在，许多人认为我们现在所处的网络世界就是元宇宙。

所以元宇宙一定是基于去中心化的基础上才能构建，如若不然跟 QQ 空间有什么区别呢？开源、上链、隐私缺一不可。核心和基础即基于去中心化理念，元宇宙才能使更多创作者来一起构建开放的 Metaverse。

3. 隐私性

此外，元宇宙强调隐私性，因此以加密货币为基础的经济系统也是元宇宙的重要构成要素。既然元宇宙基于区块链，必然承载了区块链的特征之一，即分布式账本。区块链、隐私计算和分布式身份等是元宇宙隐私保护的技术基础。

除了技术，监管也是保护隐私必不可少的路径，分布式技术和自发治理将有助于元宇宙实现有效果的隐私保护和有效率的监管。

4. 开放性

构成元宇宙的系统非常开放，既要做到对用户的开放，让世界各地的用户可以随时随地的自由进出，还要做到对不同领域全方位的开放，兼收并蓄。这样可以吸引更多的开发者参与其中，实现生态的丰富和多样化，接入元宇宙的设备也需要有很好的兼容性，低门槛的操作将会带来更多的用户实时在线。

开放的虚拟形象设计让用户摆脱现实中外形，而追求更强的自我认同。不同的化身使得一方面增添了社交的趣味性，也增强了用户呈现的自信，另一方面能表达自由、交易自由等等。

 

三. 元宇宙与 NFT

NFT 可说是进入元宇宙的关键，正因为元宇宙也是使用区块链技术所打造的公开、公平虚拟世界。因此每个人可以透过 NFT 虚拟资产，在元宇宙玩游戏、买东西、社交互动。 不仅如此，NFT 还将帮助更好创建元宇宙游戏。随着新的加密经济模式如雨后春笋般涌现，NFT 在元宇宙中的作用将变得更加突出。

1. Avatar NFT

Avatar 的起源，是印度教中的一个概念，字面意思是“降临”，《雪崩》中写到："人在软件中的化身称作 Avatars。Avatars 是包含视听觉的躯体，让人们用来在元宇宙中相互沟通"。Avatar 是引入 Metaverse 现实意义的接口。Avatar 是用户的线上代表，是因人而异的数字身份，简而言之，就是虚拟身份。

目前市场上的 Avatar NFT 有不少，较为出名的便是由 CryptoPunks 开发者 Larva Labs 团队推出的 Meebits。作为 3D 形态，Meebits 可以对接各类虚拟世界，用户可以使用其虚拟化身进行交互。在蓬勃发展的元宇宙中起到至关重要的作用。另外，由 RTFKT Studios 推出的虚拟形象 NFT 项目 CLONE X，不仅具有三维模型文件，可以引入多个元宇宙平台，而且持有者可以将所持有的 NFT 运用在未来的 AR 滤镜、Zoom 视讯会议、游戏等场景中。

 

2. Gamefi

游戏能快速将用户代入一个虚拟场景中，无疑是普及元宇宙的最佳方式。玩家可以使用 NFT通过游戏增加对元宇宙的参与，此外，通过赋予玩家对资产的全部所有权和控制权，向玩家提供了公平的游戏体验。

GameFi 是目前用户了解元宇宙一个非常直观的窗口，例如，Axie Infinity 是目前该领域最大的游戏项目，也是最大的 NFT 项目之一。一度爆火而吸引外界广泛关注的 P2E（play to earn，边玩边赚）为核心的一种商业模式。Axie infinity 虽然仅仅是一个页面简陋的宠物养成类 PK 游戏，但其致胜的关键正是其颠覆性的重构了游戏的生态架构。另外以 Sanbox 为例，Sandbox 中玩家可以购买装备来定制自己的Avatar，并且使用 Avatar 参与建设空间、发行 NFT、创造游戏、进行交易。

 

3. 虚拟土地

元宇宙的载体为一个虚拟数字城市土地，试想我们想在线上元宇宙简单举办一场画展，一场音乐会，谁可以提供场地呢？目前市场上做的比较好的项目有 Decentraland 和 Sandbox Land。“Decentraland”是一个网络虚拟空间，也就是所谓的元宇宙的其中之一，使用者可以在该平台上面购买虚拟土地，建造房子，装修自己的房子，开设店面，也可买卖交易房地产，以虚拟化身在其中漫游。

LAND 是 The Sandbox 元宇宙中的一块数字房地产，与实体房地产类似，是基于区块链的虚拟世界中代表一块数字土地的 NFT，且已被证明是可以存储价值的资产，已跻身领先的 Metaverse 区块链项目之列，你可以在这块土地上举办音乐会展览会，建立商务数字办公室等等。

 

4. 品牌

Adidas Originals 去年与多方单位合作推出的 NFT 项目 Into the Metaverse（进入元宇宙）为名，正式进军元宇宙上架，拥有者将获得对各种 Adidas Originals 体验与产品的独家访问权，此次购买为虚拟与实体结合，购买者不仅会获得 NFT商品，还会获得一系列实物，包括运动服、卫衣、帽子等等。随着 NFT 作为虚拟土地与免费联乘单品的通行证，拥有者能够在基于区块链的游戏 The Sandbox 穿戴相关虚拟设备。

另一个就是上面提到过的 RTFKT Studios，从开始就立足于元宇宙，使用最新的游戏引擎，结合制造专业知识、NFT、区块链身份验证和增强现实技术而创建并售卖数字产品，火爆虚拟时尚行业。

 

四. 元宇宙与现实应用场景

看过《头号玩家》的朋友对元宇宙有一定的认识，影片中人们通过 VR 头盔进入这个名为“绿洲” 的虚拟世界。绿洲既是一个大型多人线上游戏，又是一个虚拟社会，完美地诠释了元宇宙。这种自定义角色个体、技能，探索包罗万象的赛博朋克风格城市，让我们对未来充满幻想。那么目前或是未来元宇宙在现实生活中那些方面得到应用呢？

1. 游戏领域：游戏作为现实世界的模拟和延申，有可能成为元宇宙最先应用的场景。游戏作为虚拟电子形式存在，与元宇宙都具有相互吸引力。元宇宙将现实生活的真实感带入游戏中，虚拟游戏中的装备和游戏币都是真金白银，玩家能够在游戏当中随意构建自己的地盘，甚至可以通过持有游戏代币改变游戏的模式和未来的走向。

2. 商业领域：越来越多零售业者、时尚品牌（Gucci、Balenciaga）、运动品牌（Nike、Adidas）等开始默默布局元宇宙。例如零售龙头沃尔玛在去年底，向美国专利商标局申请与元宇宙销售有关的商标，甚至开出数位货币产品策略、加密投资。

3. 教育领域：元宇宙可被用来在任何地方和任何历史点进行实地考察。元宇宙可以让学习变得更加简单、更加有趣。进入元宇宙，教学不再依赖于时空条件和真实设备，教育资源缺少和不均衡的问题得到改善，且各种创意课堂能够被激发，教学效果即将获得质和量的提升，让所有的人不再束缚于校园。

4. 房地产领域：元宇宙可用于拟真的虚拟房屋参观。购房者将可以在自己家里通过元宇宙参观位于世界哪方的房屋。当然 NFT 虚拟房地产亦为一个新市场。

5. 音乐领域：元宇宙中的演唱会对于粉丝来说是一种新的追星方式。传统演唱会中，粉丝的观看视觉有限，与偶像互动方式单一。元宇宙中的演唱会可以突破限制，粉丝能够建立虚拟身份的方式参与，与偶像亲密互动，NFT 化的演出门票、虚拟道具、数字周边等也能够满足粉丝的想象。近年有部分艺人在 Decentraland、Sandbox 、堡垒之夜等游戏中举行元宇宙演唱会，令人印象深刻。

6. 艺术领域：创作者或商家可以在元宇宙创建线上画廊，供游客参观，同时可以直接跳转到 Opensea 等平台进行交易，所有者在拍卖会上展示和出售他们的 NFT 艺术品。

7. 办公领域：元宇宙可用于虚拟办公平台，用户可在模拟办公环境的 3D 环境中进行虚拟协作。 Crypto 企业可以在元宇宙中建立数字办公室，员工可以在那里见面和协作。远程办公在新冠疫情的影响下变得更加普遍，但传统的远程办公仍然面临一些问题，比如缺少实时互动、沟通效率低等，而元宇宙能够使得虚拟办公以面对面互动的方式进行。

8.影视领域：传统影院 2D 和 3D 的视觉感受已不能够满足大众，虽然目前已知的 4D 动感影院的出现弥补了一些短板，但当元宇宙与电影的结合生成沉浸式影院，观影的身临其境感就会再提升一个层次，通过虚拟世界打造多人参与的沉浸式影院将提升观众的体验。

虽然元宇宙的发展依然处于非常早起的阶段，但伴随着数字技术的不断发展，计算机性能的提升和游戏的繁荣，使得现实与数字世界深度融合。无论其过程有多么艰难和漫长，沉浸式体验一定会替代传统界面，而结合 NFT、去中心化技术，未来的元宇宙中，任何有价值的个体和事物都将被发现。元宇宙拓宽了消费场景，重塑了数字资产的价值，各方面都带来重大升级。
