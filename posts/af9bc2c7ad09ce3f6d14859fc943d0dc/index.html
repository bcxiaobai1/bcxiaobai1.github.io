3.git clone http:xxx 失败

将git clone https:/ 换成git clone git://

如下：

git clone https://github.com/**.git
git clone git://github.com/**.git

4.编译caffe时，报错/usr/bin/ld: cannot find -lhdf5_hl，

我们对Makefile.config文件进行修改，将原本的如下所示的代码

LIBRARY_DIRS := $(PYTHON_LIB) /usr/local/lib /usr/lib

改成

LIBRARY_DIRS := $(PYTHON_LIB) /usr/local/lib /usr/lib /usr/lib/x86_64-linux-gnu/hdf5/serial/

5.VMware虚拟机启动后出现黑屏或紫屏幕，无法进入系统的解决办法 

这个问题我在csdn里找寻了很多博主的回答。在多次尝试后，才得以正常运行。

由于每个人在虚拟机里都可能出现不同的问题，所以我把所有的解决方案都列写在下面吧

1、最简单的方法，重启虚拟机。

2、如果重新启动后，仍然不行,这时建议修改虚拟机原有处理器内核数量，降至1个，继续重新启动。
