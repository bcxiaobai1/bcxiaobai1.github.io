讲解

1请在/tmp目录下创建student目录,并在student目录下同时创建t1,t2,t3文件

既然是创建,那么肯定就是touch或者mkdir了,不过一个是目录一个是文件
注意好哦

答案

cd /tmp
mkdir student
touch /tmp/student/t1 /tmp/student/t2 /tmp/student/t3
ll /tmp /student/

2请在/tmp目录下递归创建test1/test2/test3目录

递归,那就是mkdir下要跟上-p的参数
答案

mkdir -p /tmp/test1/test2/test3
cd /tmp/test1/test2/test3/
pwd

3切换到/tmp/test1/test2/test3目录下,并且打开查看当前所在目录

切换就是上cd了,然后查看的命令上面也展示过,pwd

因为上面进入了,就已经在了哈

cd /tmp/test1/test2/test3
pwd

4请同时在/opt，/media目录下创建upload文件

创建文件,那就是touch了,请同时在,那就是不要切换目录
直接做

touch /opt/upload /media/upload
cd /opt
ll
cd /media
ll

5请将/opt目录下的upload文件移动到/tmp/test1/test2/test3目录下,并改名为upload.bak

移动,那就是mv啦

cd /opt
mv upload /tmp/test1/test2/test3/upload.bak
ll /tmp/test1/test2/test3/upload.bak

6请将/etc/passwd文件拷贝到/opt目录下,改名为passwd.bak.并且源属性不变

拷贝,那就是cp了,至于扩展属性,看答案~

cp -p /etc/passwd /opt/passwd.bak
ll /opt/passwd.bak

7请将/etc/fstab文件拷贝到/opt目录下,改名为fstab.bak

拷贝,改名,还是一样的cp -p

 cp -p /etc/fstab /opt/fstab.bak
 ll /opt/fstab.bak

8请将/etc/sysconfig/network-scripts/ifcfg-ens32文件拷贝到/opt下,改名为ens32.bak

不多说了,上操作
答案

cp -p /etc/sysconfig/network-scripts/ifcfg-ens33 /opt/ens32.bak
ll /opt/ens32.bak

9删除etc/yum/repos.d/目录下所有文件

删除嘛,那就是rm了,强制就是-rf

rm -rf /etc/yum/repos.d/*

注意这个地方,带和不带的区别大了去了
带*则是目录下的文件,如果不带,就是把这个目录都给干掉了

10在/etc/yum.repos.d/目录下创建local.repo文件

创建就是还是touch了

touch /etc/yum/yum.repos.d/local.repo
ll /etc/yum/yum.repos.d/local/repo

11请查看/etc/sysconfig/network-scipts/ifcfg-ens32文件末尾五行的内容

到了查看呢,还是末尾,末尾对应的是啥呢?是tail
答案

tail -5 /eyc/sysconfig/network-scripts/ifcfg-ens33

12请查看/etc/passwd第一行内容

前面是后面,现在是前面了,那就直接cat就完事了,当然也可以用head

cat -n /etc/passwd

13请查看/etc/hostname内容

又是查看了,查看真滴老多了
cat,less,head,tail

cat /etc/hostname

14请查看/etc/hosts文件内容
直接上答案了

cat /etc/hosts

15说出硬链接和软链接的特点
这是说的哈
那我就打字吧
1.软链接=windows里面的快捷方式
2.软链接是复制了源文件的快捷方式,源文件失效软链接也会失效
3.软链接要写最详细的路径,哪怕是在当前路径下
1.硬连接,不能跨分区,不能跨目录
2.虽然他不是复制,但是实际上的效果跟复制一样
3 硬连接过来之后就是独立的,哪怕源文件失效也不会变
4.颜色和软链接不一样

16.请在/opt目录下创建hello.soft文件,并创建软链接到/tmp目录下


touch /opt/hello.soft
ll /opt/hello.soft
ln -s /opt/hello.soft /tmp

17,请在/opt目录下创建hello,hard文件,并创建硬连接到/tmp目录下,并查看链接文件详细信息

touch /opt/hell.hard (我才发现打少了一个o)
ln /opt/hell.hard /tmp
