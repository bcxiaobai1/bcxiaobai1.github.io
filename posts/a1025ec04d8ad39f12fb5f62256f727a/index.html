ChatGPT 支持的搜索引擎 Bing 究竟什么样？

微软于2月8日北京时间凌晨在 Redmond 线下举办一场媒体活动，围绕微软的产品以及 AI，公布最新消息。这里我们先回顾一下微软在 AI 上的布局。

2019年，微软向 OpenAI 投资10亿美元，成为了 OpenAI 紧密的合作伙伴，而微软的 Azure OpenAI，于2021年推出，微软云服务客户可以使用各种 OpenAI 工具，如 ChatGPT 基于的 GPT-3.5 系统，以及可以通过文本提示生成图像的 Dall-E 模型。2022年11月底 ChatGPT 推出，迅速席卷互联网，用户迅速上了百万。

2023年1月初，微软已经在讨论将 OpenAI 的技术纳入 Word、PowerPoint、Outlook 和其他应用程序，这样用户就可以用简单的提示自动生成文本，2023年1月下旬，微软宣布即将会把 ChatGPT 加入 Azure 云服务。

此外，新版微软必应+ChatGPT 的测试版会推出，在面对谷歌刚刚宣布发布 Bard 聊天机器人的时刻，这场发布会就更容易引人注目。

之前网上流传的新版必应的特点是有一个聊天框，而不是搜索框，你可以像人一样与搜索引擎对话，用自然语言查找信息和搜索网络查询。

 

微软MVP实验室研究员


张雅琪（阿法兔）

微软最有价值专家（MVP），毕业于外交学院和香港大学，IT 技术社区创始人，中关村互联网研究院兼职研究员，多次受邀在微软 Reactor 进行公开演讲，也曾在很多知名大学、多家知名基金进行过课程的讲授，热爱研究计算机历史和科技公司的商业模式。

 

发布会说了啥?

 

发布会提出，新版微软搜索引擎，在桌面上推出了测试版，移动版也即将推出。

微软首席执行官 Satya Nadella 先上台，和大家开玩笑说我们要谈论 Kubernetes，所有人都笑了（哈哈）。Nadella 介绍了当前微软的情况，包括过去微软经历过的所有，和取得的业务里程碑，Nadella 提到了卡特总统和当时所面临的经济、通货膨胀问题。

他说："微软将如何克服这些挑战？”

之后，Nadella 在分享了一些关于微软和技术的轶事后，（负责回答发布会问题的微软成员们）就开始回答关于 Bing ChatGPT 和 OpenAI 的问题了。

发布会

新版微软必应 Waitlist 时，用户可以通过安装一个浏览器扩展，更快地获得 AI 工具，新的微软必应不是 Tay 2.0，今天起，微软由 ChatGPT 支持的搜索引擎 Bing，会开放给所有人试用，大家很快就可以试用新的必应了。

新的 Bing 搜索引擎特性

  1. OpenAI：下一代语言模型。
  2. 普罗米修斯（Prometheus ）模型：提高搜索结果的相关性，对答案进行注释，显示最新结果，了解地理位置，增加安全性。
  3. 核心搜索索引：应用了 AI 模型，也是二十年来相关领域最大跳跃。
  4. 用户体验（集成）：搜索汇集了答案、聊天和浏览器。

The New bing:

  1. OpenAI: next gen language model.
  2. Prometheus Model: Improve relevancy, annotate answers, up to date results, understand geolocation, increase safety.
  3. Core search index: We applied the AI model and we saw the largest jump in relevance in two decades.
  4. User Experience (unified); search coming together answers, chat, and the browser.

Q:关于广告：有没有计划纳入广告？
回答：广告一开始就会有。

Q: 对终端用户是否有任何费用？展示的很多东西都是基于事实的。这能帮助人们进行创意写作吗？
回答：不收费，它是免费的，它可以帮助人们进行创造性的举例。

Q：开发者会得到一个 API 吗，可以对这些查询进行限制吗？
回答：我们正在继续评估我们的产品，限制这块还会有一些区别。

Q:在 Chrome 或其他浏览器中能否使用？
回答：我们希望把这个功能带给所有的浏览器，只是从 Edge 开始。Chrome 需要设置一些功能，才能使用，但我们的目的是所有的浏览器。

Q:ChatGPT 有时会产生幻觉，乱编乱造，你们是否战胜了这个问题，或者说用户可能会看到这个问题？
回答：从一开始我们在努力解决这个问题，我们正在测试模型，并且努力保持接地气的语气，但是目前它还不完美（ We've been working on this since the beginning. We are measuring where the model is not staying grounded in the search results, but it's not perfect. Users will see where it's changing a small number or something.）

Q：在使这个产品可持续发展和计算优化方面，你们做了哪些努力？
回答：我们希望它可以区分，在什么时候会使用严格的计算，而什么时候可以不进行那么重的计算，这一点很智能。

问：生成的内容，是否完全会被标记为 AI 生成的？
回答：我们的愿景是辅助，而不是希望 Bing 完全生成和写东西。

问：为什么不重新塑造一个新的品牌名称（Why not rebrand Bing?）？
回答：我们认为，（ChatGPT 的这场革命）完全是对搜索的重新认识，我们喜欢 Bing 这个品牌，所以我们也要坚持它继续叫 Bing。"

问：在 GPT 上运行查询与传统搜索的成本是什么？
回答：我不打算回答这个问题。

问：在这里，数据的相关性和新鲜性将非常重要。比如说，Bing 给出的是新鲜的训练数据、现场新闻来源，还是两者的结合？
回答：我们马上会对它进行测试，看它是否知道这个事件。

问：关于 Bing 的准确性。
回答："我们不会总让它得出100%的正确，Bing 一直在学习和迭代。而这里的关键其实是，如何将信息落地？要给它提供更多的数据。可以提高准确性。

问：关于模型中信息的时效性，比如说如果一台新电视刚刚发布，机器人会知道吗？
回答：We'll be impressed with the recency.

问：是否会产生大量的不良内容？被返回到模型中，让模型学习？
回答：将搜索与模型结合起来，意味着可以用搜索来进行事实核查。把这两者结合起来时，能够进行检查，产生一种平衡。
