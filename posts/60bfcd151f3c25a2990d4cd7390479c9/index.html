利用anaconda安装pytorch(含GPU且从官网下载的方法)

还是从土堆老师那学的，土堆老师yyds！
如果你从官网下载失败或者是cpu没有gpu可以去佬的视频里找其他的方法下载，链接如下（记得三连）：软件下载教程
1.确定显卡算力

单机右键

查看自己电脑的显卡型号

需要谷歌爬一下（也可以不爬，大佬说后续会分享出来这个表，这里因为表太长了，我只截图了我的设备的部分）

我的3060显示算力是8.6

这个表显示了各个版本适应的算力

8.6在这些版本的范围内，也就是说等下下载cuda要是这些版本才行

2.建议更新最新的英伟达驱动
网页地址输入nvidia.com，进入官网

要对应自己电脑的显卡型号找到正确的版本

这个是对应我显卡型号的版本

启动，忘记截图了，就是放在默认的位置不需要动什么，往下继续安装就行了




注意这里都不勾选，然后点关闭
打开anaconda的命令窗口anaconda prompt

输入nvidia-smi，看到版本更新了就说明成功了。

3.开始安装pytorch

进入官网

我之前确定了我可以下载cuda11.7

进入你创建的虚拟环境中，将复制的链接粘贴上
【如何创建一个虚拟环境：
1.创建镜像
A.用conda
conda create -n 虚拟环境名字(eg:kpytorch1) python=版本(eg:3.8 3.9,最高应该是不超过下载的anaconda能用到的版本)
B.镜像加速
conda create -n 虚拟环境名字 python=版本 -c 镜像地址(eg:http://mirrors.aliyun.com/anaconda/pkgs/main)

清华镜像：https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
北外镜像：https://mirrors.bfsu.edu.cn/anaconda/pkgs/main
阿里镜像：http://mirrors.aliyun.com/anaconda/pkgs/main
】
然后它会显示一个plan，注意检查一下列表里面有没有pytorch（这里忘记截图了）

问你继续吗？打 y 就是yes

然后，就会开始下载一堆东西，官网下的慢一些（我下的时候还好我觉得不算慢），如果觉得很慢可以去小土堆老师的视频里找镜像下载的方法。（其实我没有用主要是我这个指令里面应该是有两个channel，我不清楚有一个channel的镜像网站地址是什么）下载完了之后就会显示done

进入你的虚拟环境，输入conda list

注意找一下有没有torch或者pytorch有的话，就是下载成功了。
输入python

输入import torch
torch.cuda.is_available()

有true说明电脑有gpu pytorch可以正确使用gpu。

成功安装好啦！
