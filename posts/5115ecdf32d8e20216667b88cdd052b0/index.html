一、基本介绍

随着互联网的快速发展，越来越多的企业都在慢慢的向云时代方向发展，主要原因是因为 云主机能够快速构建更稳定、安全的应用，同时也降低了开发和运维的难度以及整体 IT 成本，使企业可以更专注于核心业务的创新。

1.为什么现在都在使用云服务器

现在越来越多的企业选择云服务器主要是因为云服务器有以下几点优势：

1）成本低：

  • 相比传统的服务器，云服务器采用 即用即收费，减少购买成本；
  • 并且云主机还提供了 灵活的扩展能力，可以很方便的实现升配和降配（可以有效降低硬件维护的运维成本）

2）性能高：

  • 云服务器实现了 硬件上的隔离以及宽带上的独享，不会受到地域、流量等的限制；
  • 也就是说可以持续的进行业务交流，不会因为中断而影响效果。

3）强大的管理能力：

  • 云服务器可以实现 远程同步管理，共享，各种业务的备份；
  • 像传统的服务器，是需要在某一网络区域内，同时还有可能受到网络影响导致资料缺失。

4）安全性更高：

  • 为了保证云平台的可靠性，云服务器可以通过配置安全组（防火墙）来保证云主机的安全性；
  • 安全组与传统的防火墙不同，安全组配置简单，在页面上操作即可，不像传统的服务器，需要通过命令实现防护。

2.我们可以使用云主机做什么

1）搭建网站或个人博客

  • 使用云主机最常见的应用就是搭建网站，一般程序员都会有个人博客，来发布个人的专业知识，对未来找工作也是有加成的。

2）搭建个人网盘

  • 我们可以使用 NextCloud 工具来搭建属于自己的个人网盘（NextCloud 是一款开源免费的私有云存储网盘程序）

3）搭建自己的仓库

  • GitLab 是一款使用 MIT 许可证的基于网络的 Git 仓库管理工具，使用云主机结合 GitLab 可以搭建出自己的 Git 仓库。

总结： 可以看到使用云主机，我们可以干很多事情，包括搭建网站或个人博客、搭建个人网盘、搭建自己的仓库等等，基本上想的到的我们使用云主机都可以做到，而且 每个云主机都会携带一个公网 IP，也就是说我们只要有云主机，便可以通过互联网来访问。
