Windows 11 将迎来首次重大更新：终于要支持 Android 应用了

自去年 6 月微软打破“Windows 10 是 Windows 最后一个版本”的 Flag、官宣 Windows 11 的那天起，它就引来了无数争议：“苹果味儿”的 UI 设计、高门槛的硬件要求、各种小 Bug 层出不穷…

尽管如此，在微软正式推出 Windows 11 后，依旧有很多人没有逃过“真香”现场：据微软官方表示，用户升级 Windows 11 的速度是当年升级 Windows 10 的两倍，在 Windows 11 PC 上花费的时间相较于 Windows 10 也增长了 40%。

距离去年十月 Windows 11 正式推送已经过去了 3 个月，微软也开始着手于 Windows 11 的第一次重大更新：本周三，Windows 负责人 Panos Panay 发布了一篇官方博文，预告微软计划将于 2 月推出支持 Android 应用程序的 Windows 11 公开预览版。

一、PC 新时代

这篇博文标题为“PC 新时代”，因此在正式介绍微软对 Windows 11 的下一步计划前，Panos Panay 首先回顾了过去两年 PC 的变化。

Panos Panay 感慨道，过去两年人们的生活方式发生了深远而持久的变化，PC 在这之中扮演着比以往任何时候都更有意义的角色。而归根结底，造成 PC 发生这种结构性转变的根本原因是混合工作和学习的兴起、娱乐习惯和分销模式的转变以及消费者日常习惯的改变。

PC 市场也在这两年得到了近十年来的最大增长：据权威市调机构 Canalys 数据统计，2021 年全球 PC 出货量超 3.4 亿台，较 2019 年增长了 27%。与此同时，Windows 每个月也在为超 14 亿台活跃设备提供支持，各种计划将 PC 用于创意、游戏或工作的用户人数也增加了近 50%。

二、 Windows 11 的产品满意度历代最高

为了顺应 PC 新时代下的变革与挑战，Windows 11 由此诞生。Panos Panay 指出，Windows 11 的推出承载了很多期望，为此微软动员了整个行业的芯片、OEM、零售商和其他合作伙伴，结果也不负众望：

  • Windows 11 的质量得分和产品满意度，高于微软曾经发布的所有 Windows 版本。

  • 用户接受 Windows 11 升级的速度是曾经 Windows 10 的两倍。

  • 人们在 Windows 11 PC 上花费的时间比 Windows 10 多 40%。

  • Windows PC 的多任务处理和生产力优势的使用大幅增加，近一半的 Windows 11 用户都在使用新的 Snap Layouts。

  • Windows 11 为新设计的 Microsoft Store 带来 3 倍流量。

三、将于下个月迎来重大更新

这样的成绩即便已十分可观，但微软深知这并不是终点，也并不代表 Windows 11 已完美无缺， 紧跟用户需求进行重要迭代、继续提供丰富和启发人们生活的 Windows 体验必不可少。为此，微软决定在下个月推出万众期待、支持 Android 应用程序的 Windows 11 公开预览版。

除了支持 Android 应用，下个月 Windows 11 还会对任务栏进行一些改进，包括静音和取消静音功能、天气小部件的回归，并引入两个重新设计的新应用程序：记事本和媒体播放器。Panos Panay 表示：“Windows 将继续成为创新的支柱，成为游戏、创造力和探索的目的地，以及通往元界的门户。”

不过关于其他更新细节 Panos Panay 并未在博文中过多描述，但仅“支持 Android 应用”这一点相信已让许多 Windows 11 用户充满期待。
