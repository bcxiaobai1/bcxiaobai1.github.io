RISC-V “异军”突起，英特尔：消灭不了就加入它

伴随着近年来 RISC-V 的异军突起，被业界戏称为“牙膏厂”的英特尔处境也似乎变得愈发严峻：许多人开始期待着未来 CPU 领域形成 x86、ARM、RISC-V 三分天下的格局，而这无疑将对如今市场份额占大头的 x86 造成一定影响。

面对发展势头如此强劲的 RISC-V，英特尔采取了怎样的应对措施呢？国内有网友将其精准概括为：“消灭不了就加入”策略。

一、曾欲以 20 亿美元收购 SiFive 被拒

早在去年 6 月，业内就有传闻称，英特尔想将当时估值为 5 亿美元的 RISC-V 芯片公司 SiFive，以溢价 4 倍的 20 亿美元收入麾下，同时还有多家公司向 SiFive 同时发出了收购要约。

（注：对于有能力自己定制芯片的科技公司，RISC-V 是免费开源的；但同时也有很大一部分资源不足的中小型公司，会有偿委托 SiFive 公司来采用 RISC-V 架构设计量产芯片。此外，英特尔早已是 SiFive 的投资者之一。）

虽然英特尔与 SiFive 当时均未正面回复这一传闻，但多位半导体业界人士均认为该消息的真实性很高：

  • 彼时，英伟达计划收购 ARM、AMD 也全资收购了 FPGA 方面的领头羊赛灵思科技——面对威胁，英特尔可能坐不住了。

  • 现任英特尔 CEO 帕特.基辛格拥有凭借一己之力同时掌控 CISC 与 RISC 两大阵营芯片路线的实力，不难理解其想要借助 RISC 带英特尔重回巅峰。

彼时，该传闻引起了多方高度重视，有人说英特尔可能会借 SiFive 的 RISC-V 来增强自身在 AI 训练推理专用芯片方面的实力，也有人说英特尔未来可能会依靠 RISC-V 架构与英伟达进行抗衡，甚至有人预言这可能会对未来的芯片发展带来新变革。

但这些猜测在四个月后似乎都变得没有意义了：去年 10 月，据彭博社报道，因 SiFive 管理层不喜欢英特尔将其整合到自己路线图中、打断其经营理念，拒绝了英特尔报价 20 亿美元的收购要约，计划保持独立并寻求首次公开募股。

二、宣布加入 RISC-V 国际基金会

对 SiFive 的收购失败并没有令英特尔停止布局 RISC-V，反而换了一种方式拥抱它：本周一，英特尔宣布加入 RISC-V International 组织，成为其 Premier 级别会员，并承诺投资 10 亿美元创建一个新基金，用于支持各晶圆代工初创企业、为代工生态系统构建颠覆性技术。


许多人对此感到惊讶：为何有着 x86 这张王牌，英特尔还要对 RISC-V 如此上心？

英特尔 CEO 帕特.基辛格在公告中表示：“代工厂客户正在迅速采用模块化设计来区分他们的产品并加快上市时间，而英特尔代工服务（Intel Foundry Services，简称 IFS）在引领这一重大行业转折方面处于有利地位。凭借我们的新投资基金和开放式小芯片平台，我们可以帮助推动生态系统在全系列芯片架构中开发颠覆性技术。”

若进一步深究，我们会发现此举其实与帕特.基辛格上任后提出的首个战略 IDM2.0 相符：英特尔既使用第三方代工厂的先进产能，也会自己代工——即英特尔代工的芯片不会仅限于 x86，不论什么指令集或架构，只要愿意与英特尔合作就都是客户。

为了融入 RISC-V 领域，英特尔对此可谓是“诚意满满”：

  • 与 RISC-V 生态系统中包括 Andes Technology、SiFive 等领先合作伙伴联手，IFS 计划提供一系列经过验证的 RISC-V IP 内核，并针对不同的细分市场进行了性能优化，确保 RISC-V 在 IFS 芯片上跨所有类型的内核（从嵌入式到高性能）运行最佳。

  • 将提供三种类型的 RISC-V 产品：基于 IFS 技术制造的合作伙伴产品；RISC-V 内核被许可为差异化 IP；利用先进的封装和高速芯片间接口，提供基于 RISC-V 的小芯片构建块。

  • 还有丰富的开源软件生态系统，这对加速 RISC-V 处理器增长以及芯片设计人员充分释放其价值至关重要，为此 IFS 将赞助一个允许自由实验的开放源代码软件开发平台。

三、国内也将加大对 RISC-V 的投入

手握闭源 x86 的英特尔，选择拥抱其他架构的举动也引起了诸多业内人士的关注。

RISC-V CEO Calista Redmond 表示：“RISC-V 的开放合作已经引发了半导体行业的深刻转变，这种合作伙伴关系将加速开放计算的创新。”

加州大学伯克利分校名誉教授、谷歌杰出工程师和 RISC-V 国际董事会副主席 David Patterson 说：“我很高兴 50 年前开创微处理器的英特尔公司现在成为 RISC-V 国际的成员。”

中国科学院计算技术研究所研究员、中国开放指令生态（RISC-V）联盟秘书长包云岗也为此发博表示，这将对国内形势产生一定影响：“Intel 加入 RISC-V 国际基金会，个人判断将会进一步促进国内企业加大投入。相信国内企业和机构也会像通讯领域那样成为 RISC-V 领域的核心贡献者和引领者。”
