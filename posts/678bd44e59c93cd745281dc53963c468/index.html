前言

        拥有一个属于自己的个人博客，先讲讲整个流程吧，这样大家后面也比较好理解，个人博客想要被人访问到，那么必须有一个服务器，现在很多的服务商都提供云服务的租用。如果是学生的话优惠力度是比较大，这里我建议去租大厂的云服务器，像阿里，腾讯等等，这些大厂的服务还是性能都比较有保障。那么服务器需要怎么样的呢，如果是做一个个人博客，CPU1核、2核，内存2G、4G的差不多了，配置高一点的价格也会相应的贵一些，我个人建议前期可以租一个配置低一些的，后期再去升级啥的。云服务器租好了，服务商会提供服务器的IP地址，我们就需要一个能远程连接云服务的工具。毕竟服务器在人家哪里，总不可能跑人家哪里去上传文件吧，不过一般在提供云服务的服务商在网页提供控制台也可以远程访问，我介绍一个我用的叫Xshell,后面也会给出网盘的链接。使用Xshell连接云服务器后通过输入命令安装宝塔面板，宝塔是一个支持Linux与Windows系统，轻松管理云服务。在宝塔面板可以安装相应的软件，安装Wordpress,一键部署，然后就在Wordpress里选择主题，添加文章。

一、云服务器选择

        提供云服务的服务商有很多，我选中的是腾讯云，租了一年的轻量级应用服务器，配置为：CPU: 2核 内存: 2GB40GB SSD云硬盘 带宽：4Mbps 操作系统是CentOS7 。在最新活动里可以找到比较优惠的服务器，阿里云通过学生认证可以领取2个多月的服务器。可以在阿里云去尝试搭建，后面考虑好后可以去长久的租用。租好服务器后服务商会提供服务器的IP地址，腾讯云的是在控制台里查看。

         租好服务器后，如果你是想租个服务器玩玩就可以不需要去考虑域名，购买域名后要备案后才可以使用，而且租服务器不能小于3个月。像腾讯云里也是提供域名购买服务的，我建议你在哪里买的服务器就在哪里购买域名，因为不同的地方购买可能还需要域名转入转出，这些操作应该都是要收费的。选中一个未注册的域名，有的域名也是比较便宜的，活动期间有1元的，买好域名后要过72小时才能备案这些都是要实名认证的，域名买好后在腾讯云的平台还需要解析域名，在腾讯云的首页有IPC备案，根据要求提供相应的材料，一般是腾讯云平台会有初审，初审过后提交管局，管局的审核可能比较久，大概20多天。反正就等通知就好了。这20天的时间就可以自己去搭建网站。先使用服务器的IP地址去访问服务器。

二、Xshell的使用

        Xshell：是一个强大的安全终端模拟软件，它支持SSH1, SSH2, 以及Microsoft Windows 平台的TELNET 协议。Xshell 通过互联网到远程主机的安全连接以及它创新性的设计和特色帮助用户在复杂的网络环境中享受他们的工作。

        安装好后，就可以使用Xshell远程连接服务器；也可以使用服务商提供的远程登录访问。这里两种方法都行。

       到服务器的控制台——找到自己的服务器——找到重置密码。就个步骤就为了下面远程连接的时候需要密码或者秘钥。记住密码

        Xshell新建会话，名称自定义，协议默认，主机就是云服务的IP，端口默认就22。

        确定后，输入登录名：root，然后输入密码。登录成功后。

        这个就登录好了。

三、安装宝塔面板

        宝塔面板：让运维简单高效。面板支持Linux与Windows系统。一键配置:LAMP/LNMP、网站、数据库、FTP、SSL,通过Web端轻松管理服务器。

宝塔面板 - 简单好用的Linux/Windows服务器运维管理面板

        在宝塔面板的官网下

 点击“立即免费安装”

        选择Linux还是Windows 看你服务器的操作系统。我安装的liunx的操作系统，点击安装脚本。根据自己选择的选择安装脚本。我用的是第一个安装脚本。在Xshell输入后等待安装就好了。

Centos安装脚本

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh ed8484bec

Ubuntu/Deepin安装脚本

wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec

Debian安装脚本

wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && bash install.sh ed8484bec

万能安装脚本

if [ -f /usr/bin/curl ];then curl -sSO https://download.bt.cn/install/install_panel.sh;else wget -O install_panel.sh https://download.bt.cn/install/install_panel.sh;fi;bash install_panel.sh ed8484bec

        中途宝塔还会询问你，直接y就好了，安装好后，他会出现宝塔面板登录地址和登录用户名、密码。复制宝塔面板的登录地址，输入用户名和密码。这里的地址和账号信息要记一下。如果忘记了，在xshell命令行输入，会重新显示出来。

/etc/init.d/bt default

我在给几个命令，其他的linux命名我不详情介绍，网上都有相应的资料。

bt //这个命令可以对宝塔的一些操作，例如修改密码
netstat -lntup   //查看开发的端口

        这里如果出现无法访问面板，就是因为没有放行宝塔面板的端口号。解决办法：

1.找到服务商提供的控制面板，找到自己的服务器。

2.在“防火墙”——“添加规则” 

        应用类型：自定义，协议TCP，端口就是宝塔面板的端口，这里要和宝塔的端口对应，我宝塔面板的端口是8888，所以这里填8888，备注你可以备注一下宝塔面板端口。添加成功好后应该就解决了面板无法访问的情况。

        登入面板，面板会自动推荐你安装环境套件，这里有两种选择，第一种是LNMP套件，第二种是LAMP套件。LNMP和LAMP环境主要的区别在于web服务器上面，一个套件使用的是Apache服务器、一个套件使用的是Nginx服务器。如果你不了解两种的区别和差异，请使用宝塔官方默认的推荐安装（LNMP套件）。选择“极速安装”，等待安装即可。应该还要让你注册一个宝塔的账号，不然不让使用。

四、部署WordPress

        安装好后，选择“软件商店”，在这里你可以选择安装软件。找到WordPress，一键部署。         

域名填服务器的IP，等域名备案成功后在修改，提交好后，会出现部署成功，和数据库账号资料，数据库名用户名、密码需要保存一下。

         在浏览器中输入服务器的IP，这时候会出现wordpress的安装，和安装软件一样，需要填数据库的时候把上面保存的数据相应的输入到不同输入框就好了。好了之后就会出现wordpress帮你部署的网页，这时候你完成全部的部署操作。
