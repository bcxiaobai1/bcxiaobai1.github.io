如何制作NFT指南

不可替代的代币或NFT最近引起了很多关注。这并不奇怪，特别是当你听到关于艺术家出售单个NFT赚取数千万美元的故事时。

当你观察市场时，仅在过去一年中，NFT就飙升了。根据跟踪公司DappRadar的数据，NFT的交易从2020年的1亿美元激增到2021年的令人难以置信的220亿美元。前100名NFT的总价值高达167亿美元。

但什么是NFT，你怎么能做一个？

这个概念可能有点令人困惑。但不要担心，我们将引导您完成您需要了解的所有基础知识，以及创建和铸造自己的NFT的步骤。

什么是NFT？

NFT是具有唯一身份的数字资产，在区块链（数字分类账）上保存和验证。每个NFT都是不可替代的，这意味着它不能轻易地换成另一种具有类似价值的资产。常见的 NFT 资产包括图像、视频、音乐、GIF 和收藏品。

NFT具有价值，因为只有一个版本的资产。换句话说，一个NFT一次只能有一个所有者——没有人可以在区块链上改变其所有权或铸造相同的NFT。鉴于这种稀缺性，NFT的创建者/所有者有能力为其资产设定自己的利率。

NFT如何工作？

NFT需要通过一种称为密码学的技术创建数字令牌。在加密世界中，这种令牌创建过程被称为铸造。通常，要铸造NFT，您必须为使用加密货币创建智能合约付费。

智能合约是允许区块链以安全和透明的方式存储信息的软件代码片段。最终，这些代码是管理NFT的所有权和可转让性的原因。

谁可以创建 NFT？

任何有少量钱可以花钱的人都可以铸造NFT。今天的NFT平台提供的工具可让您创建各种NFT，并将可解锁的内容附加到您的文件。无论您是艺术家，音乐家，收藏家，企业家还是公司，您都可以通过使用NFT市场上提供的简单铸造工具来创建自己的NFT。

在进入NFT制作的细节之前，让我们来看看一些流行的NFT例子，让你得到启发。

就像收藏家为邪教玩具和稀有商品支付巨额资金一样，Doge NFT展示了数字稀缺如何使标志性作品变得有价值。2021年6月，一只名叫Kabosu的微笑的柴犬的邪教模因以400万美元的价格售出。有趣的是，购买NFT的NFT投资集团现在出售了它的部分所有权，这意味着任何人都可以以低至1美元的价格拥有一块。

如何创建和铸造 NFT

让我们来看看制作自己的NFT的基本步骤，请记住，这并不是一个全面的指南。根据您使用的工具，有不同的方法可以执行此操作，因此请使用这些提示作为一般准则。

1. 确定要创建的 NFT 类型

NFT用途广泛 - 您有很多选择来选择要创建的类型。您的选项包括艺术品，活动门票，模因，媒体和音乐，游戏，虚拟物品等。您甚至可以为现实生活中的物品创建NFT，例如有价值的收藏品或名人签名的照片。

它们通常采用图像，音频或视频文件的形式，如JPG，PNG，MP4，MOV等。

如果您决定制作一个简单的NFT，则不一定需要开发人员。但是，如果您的NFT是一个更复杂的项目，如游戏，或者具有复杂的智能合约（销售条件），那么您可能需要开发人员的帮助。

在制作第一个NFT时，请考虑如何为受众提供价值。如果您已经拥有业务，则可以尝试创建会员卡或独家促销代码。您可以随时聘请艺术家为您创建自定义艺术。

2. 选择一个 NFT 市场

在寻找在哪里铸造您的作品时，有大量的NFT市场可供选择。每个NFT市场对卖家都有自己的优缺点，因此请确保在做出选择之前进行研究。其中最受欢迎的三个是 OpenSea, Rarible, 和 SolSea.

这些平台使新创作者更容易潜入NFT的世界。有些要求你在区块链上验证或写入你的NFT，而另一些像OpenSea和Rarible这样的允许“快捷方式”，如懒惰的铸造。在懒惰的铸币中，您可以通过将NFT出售而不将其写入区块链，然后将该费用转嫁给您的买家（如果购买了）来避免一些费用。

对于初学者来说，最重要的考虑因素之一是费用，通常被称为“汽油”。每个市场都有自己的费用，具体取决于您计划如何铸造和销售NFT。例如，OpenSea需要付费才能初始化您的帐户，在某些情况下，这可能高达100美元。

3. 设置加密钱包

下一步是创建一个数字钱包，您可以在其中存储您的加密货币和NFT。在选择钱包时，您必须考虑的一个重要因素是它是否与您打算用于铸造NFT的区块链和NFT市场兼容。

虽然那里不乏加密钱包，但大多数NFT创建者都使用MetaMask浏览器扩展钱包。该钱包易于设置，并连接到大多数用于创建和交易NFT的区块链，包括以太坊和币安智能链。其他常见的钱包是 Enjin, AlphaWallet, 和Trust Wallet.

**注意：**大多数NFT钱包都是基于浏览器的，这意味着它们与底层浏览器应用程序一样安全。为了获得最大的安全性，我们建议投资像Ledger Nano X这样的硬件钱包。硬件钱包离线存储您的私钥，用于管理区块链上的各种硬币和NFT资产。

4. 通过交易所购买加密货币

一旦你创建了一个钱包，是时候购买一些加密货币了，这样你就可以支付汽油来铸造你的NFT。最好的方法是在信誉良好的交易所（如币安，Kraken或 Crypto.com）创建一个帐户。该过程非常简单，可以在智能手机上完成。

现在您有一个帐户，您可以登录并购买加密货币。对于NFT铸币，您需要购买以太坊（ETH）或Solana（SOL）硬币，具体取决于您要铸造的区块链。完成后，将您的硬币转移到您在上一步中设置的钱包中。

要从交易所转移到您的加密钱包：

  1. 登录到交易所。
  2. 打开顶部或底部导航栏中的“钱包”选项卡。
  3. 点击提款，然后选择相关货币。
  4. 在地址字段中输入您的钱包地址。
  5. 选择合适的网络（ETH 为 ERC-20，Solana 为 SOL）。
  6. 输入金额并点击 提款.

5. 将您的钱包连接到NFT平台并造币厂

充值钱包后，您需要将其连接到将用于创建数字资产的NFT平台。让我们说明您需要采取的步骤（我们将以市场Rarible为例）。

1、转到 Rarible.com，然后单击屏幕右上角的“连接钱包”按钮。

2、选择您的数字钱包（如 MetaMask ），并授予“稀有”权限以查看您的帐户。

3、单击“连接”并接受平台的服务条款和年龄确认。

4、返回Rarible的主页，然后单击右上角的蓝色“创建”按钮。

5、在创建NFT的单个数字副本和创建多个副本以多次销售同一商品之间进行选择。

6、将要铸造的数字文件上传为 NFT。

7、接下来，系统将要求您选择要如何出售NFT文件。在创建资产时将其出售并不是强制性的。您也可以将其出售并在铸造后设置最低定价。

8、选择是要提供完整的高分辨率版本的NFT，还是通过秘密下载链接或网页添加可解锁的内容。

9、选择“ Rarible ”作为 NFT 的集合。

10、为 NFT 图稿添加标题和说明。

11、指定要为二次销售收取的版税的一定百分比。

12、指定文件的属性，如颜色或大小（以像素为单位）。

13、单击“创建项目”。

14、批准您的加密钱包中的交易（支付 gas 费）。

15单击确认>上传文件和铸币厂令牌。

16、在您的钱包中确认合约集成（创建智能合约将收取少量gas费用）。

17通过单击“登录钱包”来签署卖出订单。

就是这样。您的 NFT 现已铸造。要查找创建的 NFT，请单击“配置文件”>“我的项目”。您的NFT收藏品会在屏幕上弹出。

当您第一次听说NFT时，似乎过于复杂或不可能自己真正进入市场。但是，通过一些彻底的研究，勤奋和创造力，它比看起来更容易实现。

如何创建 NFT 常见问题解答

制作NFT需要多少钱？

铸造NFT的成本通常根据天然气和场地费用而有所不同。例如，在以太坊区块链上，您可以期望支付大约70美元来保护令牌。网站费用平均约为300美元，但有些网站允许您免费列出NFT。

您可以免费铸造NFT吗？

如果您选择的NFT平台支持惰性铸造，则有可能。此过程为 NFT 生成元数据，而不实际创建 NFT 本身的令牌。当有人购买该NFT时，它就会被正式铸造，并且产生的汽油费将转嫁给买方。请记住，费用可能会让试图购买您的作品的人感到惊讶。因此，虽然您可以避免铸造费用，但您最终可能会因错过的销售和投资组合机会而付出代价。
