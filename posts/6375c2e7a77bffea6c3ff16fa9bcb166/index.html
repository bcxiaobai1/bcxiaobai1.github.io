HMS Core电商与游戏行业解决方案，全流程赋能开发者创新

2021年12月29日，“华为云&华为终端云服务创新峰会2022”在北京柏悦酒店成功举办。华为HMS Core电商与游戏行业解决方案亮相本次峰会的线下展区，为行业开发者们解密电商应用增长之道及游戏应用创新之法。

HMS Core 电商行业解决方案 助力打造智能购物新体验

HMS Core 电商行业解决方案在商品智能搜索、立体展示、无忧购买、精准营销等方面提供核心能力，帮助开发者的应用丰富视觉展示，实现购物体验创新，进一步提升流量，促进转化。智能搜索方面，机器学习服务提供了智能语音识别能力和拍照购物能力，随时随地查找商品，提升搜索效率；立体展示方面，AR Engine带来创新的AR购物沉浸式体验，视频编辑能力提供的一站式视频处理能力，使商品展示更生动直观；还有最新开放的3D建模服务，用户只需使用普通的RGB相机，即可快速生成高精度3D模型，用于商品360°全方位展示，细节更丰富。开发者们在本次活动3D建模展台中亲自体验了高效精准的建模服务。

无忧购买方面，安全检测服务能够随时检测设备、应用、网络环境的安全状态，为购物流程保驾护航；精准营销方面，华为帐号服务、华为分析服务及华为推送服务是助力运营的“三剑客”。华为帐号一键登录，避免繁琐验证过程造成用户流失；精准推送，使高质量用户活跃度提升，十分钟在线到达率高达99%；分析服务可以帮助开发者针对应用运营中的问题进行深度数据分析，进而有针对性地制定营销和应用优化方案，提升用户黏度。

HMS Core电商行业解决方案针对精准用户触达、应用效率提升、智能用户搜索、丰富视觉展示与创新购物体验等场景，提供创新的能力组合，帮助应用快速获客、提升转化率，实现业务增长。

HMS Core 游戏行业解决方案 助力游戏应用高品质创新

HMS Core游戏行业解决方案在图形渲染开发、游戏交互体验、网络加速、高效运营等方面带来一站式服务方案。图形渲染方面，图形引擎和图形计算服务提供了高性能、低功耗的渲染框架和插件，可便捷实现专业级渲染表现；3D建模服务提供了物体建模和材质生成能力，简化了游戏建模和场景开发的流程；同时在图形调测工具链的加持下，大幅提高图形开发和测试效率。交互创新体验方面，AR Engine具备了业界先进的环境、人体和运动跟踪能力，帮助开发者轻松快速地构建AR游戏；音频编辑服务通过AI辅助算法，实现个性化音色变化，让游戏语音更有趣味。网络加速方面，近距离通信服务利用蓝牙和WiFi组成双通道通讯能力，让玩家随时随地面对面快速组队，游戏体验畅通无阻；Network Kit实现游戏资源下载速度的大幅提升。高效运营方面，应用内支付和广告服务助力应用开发者商业成功；分析和游戏服务提供了多样的游戏基础服务和运营功能，大幅提高游戏测试、发布和管理效率。

HMS Core游戏行业解决方案针对游戏基础开发、网络加速、游戏创新、高效运营、游戏安全等场景，通过HMS Core各领域的开放能力，打造全流程解决方案，帮助开发者增强游戏表现、打造创新玩法，进一步提高开发效率，助力商业成功。

HMS Core全方位定制化的行业解决方案，基于丰富的华为端、云开放能力，提供适用于各行业的产品能力组合，提供全生命周期的行业应用集成服务，以满足开发者快速集成上线、应用体验创新、商业成功等需求。未来，我们将聚焦更多垂域行业的移动应用创新解决方案，与更多的行业开发者一起携手，实现商业成功。
