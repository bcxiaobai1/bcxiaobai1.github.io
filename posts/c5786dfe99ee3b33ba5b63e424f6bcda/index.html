永远相信美好的事情即将发生

这么一想，虽然最终结果难以接受，但却是自作自受，不过上大学后已经收敛了，这大概是因为遇见了自己的热爱–计算机

这个大一没参加啥社团，有点遗憾，大学可不能不精彩，人生只有一次读大学对吧

想起了高中时想参加 OI，但是家人不支持，遗憾满满，但好在读了大学，所以，尽情追求自己想要的吧哈哈哈

关于学习

这学期可谓是学的很舒服，大学有一点很重要的好处就是可以尽情放飞自我去学习自己感兴趣的东西，这学期既有新知识的学习也有就知识的复习

  • 学习了 Rust 语言
  • 学习了 Go GIn 框架，顺便复习了一遍 Go 语言
  • 学习了 Redis，顺便复习了 Mysql
  • 更熟练地使用 linux
  • 复习了数据结构，唉，蓝桥杯加油
  • 学习了区块链相关理论，感觉很有意思，用 C 实现了个小型区块链
  • 写了个单线程版的 MapReduce 小 demo
  • 学习了 Element UI，并写了个简单后台模板

以后的职业目标是 Rust 工程师，为什么不是 c++？个人原因呗，不喜欢 C 系语言的语法，觉得很丑，而且对 rust 有很浓厚的兴趣，这语言符合我的一切要求

  • 内存安全（讲到这个我就不得不想对 C 写工程代码经常出现的内存问题说声 ***）
  • 语法漂亮（个人喜好问题）
  • 只要能通过编译，运行就没有成功（想起来 C 的 segmentation fault，真想说 fuc***）
  • 等等

但是 rust 工程师要求很高，这个语言想要玩起来所需要的前置知识不少，尤其是偏向原理性的知识，唉，加油！！！

下学期目标是把 c++ 水平更上一层楼，尽量做个能写上简历的项目（正在做一个分布式系统）
