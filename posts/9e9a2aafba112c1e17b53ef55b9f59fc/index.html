关于MySql服务不见了怎样解决

前言

在某天使用数据的时候突然发现MYSQL(可视化工具)报了2003错误

这是因为我们的mysql服务没有开的缘故，随后就当我准备去打开时才发现mysql服务不见了。

导致服务不见的原因有很多，我们可以去到Mysql官网进行服务下载。

一.下载MySql服务

服务下载地址：MySql服务

1.选择MySQL Community Server

2.选择需要的版本进行下载

下载好服务的压缩包放在我们C盘的Program Files目录下，接着我们需要新建my.ini配置文件，如果有就不需要新建了。

my.ini配置文件内容（路径记得修改成自己存放服务的路径）:

[mysqld]
#skip-grant-tables
#datadir=C:Program Filesmysql-8.0.31-winx64
# 设置3306端口
port=3306
# 设置mysql的安装目录
basedir=C:Program Filesmysql-8.0.31-winx64
# 设置mysql数据库的数据的存放目录
datadir=C:Program Filesmysql-8.0.31-winx64\Data
# 允许最大连接数
max_connections=200
# 允许连接失败的次数。
max_connect_errors=10
# 服务端使用的字符集默认为utf8mb4
character-set-server=utf8mb4
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
# 默认使用“mysql_native_password”插件认证
#mysql_native_password
default_authentication_plugin=mysql_native_password
federated
sql_mode=NO_ENGINE_SUBSTITUTION,STRICT_TRANS_TABLES
# 设置时区(mysqld下)
default-time_zone='+08:00'
[mysql]
# 设置mysql客户端默认字符集
default-character-set=utf8mb4
[client]
# 设置mysql客户端连接服务端时默认使用的端口
port=3306
default-character-set=utf8mb4

二，配置环境变量

1.在系统变量当中新建变量名：MYSQL_HOME，变量值：C:Program Filesmysql-8.0.31-winx64

注意：此处的变量值是你存放服务的目录

2.接着在系统变量中的path中新建**%MYSQL_HOME%bin**

三，安装MySql服务

1.以管理员身份运行命令行(cmd)

2.从命令行中进入C:Program Filesmysql-8.0.31-winx64bin目录下

3.输入安装命令：mysqld -install，若出现Service successfully installed，证明安装成功；如出现Install of the Service Denied，则说明没有以管理员权限来运行cmd
4.然后继续输入命令：mysqld --initialize，此时不会有任何提示，等它跳转就好

5.再输入启动命令：net start mysql，便会出现服务启动成功。
