1. 前言

几年前，视频会议系统还是政府和一些大企业的“专属”，需要专线、整套服务器和专业视讯终端，价格往往在几十万到上百万不等，而且系统组成复杂，维护成本高昂，让大多数中小企业望而却步。

疫情的影响，驱使远程视频会议成为市场近年来的新宠儿，成为企业保持正常沟通和业务往来的刚需，也让企业深刻体会到了远程跨地域沟通协作的便捷高效和成本节约。

如今随着社会面的全面放开，视频会议的市场竞争并没因此而减缓。随着过去三年对用户习惯的全面培育，大家都习惯了远程开会，感受到了因此带来的便捷和实用。同时随着云技术的发展，云会议快速普及，资源按需购买，初期投资成本低，且无需专人维护，中小企业也享受到了生产力的提升。

市场机会窗的打开，直接加速了云会议产品的发展和成熟。无论是Cisco、Polycom等传统视频会议大厂，还是腾讯、阿里等互联网厂家都纷纷加速布局视频会议赛道，而华为通过结合多年来在软硬件领域及云计算方面的能力，推出了多种端云结合的新型会议室解决方案。

新一代华为办公宝IdeaHub S2系列，适配原生华为云会议，实现“软硬一体、端云协同”，打造出更“丝滑”的办公体验，为企业客户带来专业的数字化办公解决方案。

3. 全方位功能体验

经过这两天的使用，这对组合给我留下了深刻的影响。首先我觉得IdeaHub的颜值非常高，不管是放在会议室还是办公室，不管是壁挂还是使用支架，都能很好的和周围环境契合；而在功能和实用性方面，接下来我从以下几个使用场景来描述IdeaHub结合华为云会议是如何提升我们团队的协作效率的。

3.1 IdeaHub接入华为云会议

一台大屏要实现云会议功能，首先需要搭配云会议资源使用。这里给大家推荐一个我目前从市面上能找到最优惠的云会议接入套餐（限新用户购买），包含一个IdeaHub接入账号+一个50方VMR（云会议室），价格只需2999元/年。

使用这个套餐，我们就能召开一个支持1个会议室+49人在线接入的会议了。

如果企业的员工随时开会需求比较多，还可以搭配并发端口。云会议室和并发端口是两类不同的资源：

并发端口可由任意企业员工召开会议，同一时间可以召开多个会议，只要最大同时与会数≤您购买的并发数，会议均可正常召开。并发会议更适合日常小快灵的会议，而且支持叠加购买

云会议室需要分配给个人或会议终端，由会议管理员召开会议，并且一个云会议室只能同时召开一个会。云会议室更适合固定的会议，不支持叠加购买。

3.2 随时随地一键开会

过去，开会需要跑到公司，跑到会议室，所有人在规定时间规定地点出现。在出差、在途、居家的情况下实在是不方便。对会议组织者来说，也是一项大工程，预定时间，逐个沟通，逐个督促与会人按时到会……随着云会议的发展，随时随地开会成为了一种可能，只要有互联网的地方，便可以开视频会议。

华为云会议支持电脑、平板、手机、小程序会议终端等各类终端接入，通过WEB和小程序，无需安装客户端也可加入会议，随时随地，便捷参会。下载华为云会议APP，注册并升级为企业后可获得50方云会议室免费体验。无论是免费版还是付费版，华为云会议都能支持用户加入企业通讯录，支持基于通讯录的批量添加与会人、会议通知、会议日程、邀请入会、仅本企业入会等功能。通讯录功能看似不起眼，但实用性非常强，而这也恰是很多竞争对手所不具备的。

3.3 天生高清

3.3.1 高清视频

博主在测试过程中，首先就被进入会议后，专业级视频摄像机所带来的高清画面所狠狠震撼到！值得一提的是，华为云会议在共享画面时，可以达到4K的清晰度，在这个基础上，搭配IdeaHub的4K屏幕，画面效果清晰细腻，可以更好地展示材料、分享图像，这在一些内容较多的课件分享，设计方案评审，医疗看片等场景下可以发挥重要作用。此外，华为云会议采用H.265感知编解码算法技术，提高了图像压缩比，结合华为IdeaHub新一代硬件平台硬编解码，在同样带宽下，画面会更清晰、更流畅。

3.3.2 纯净音质

面对在开会经常出现的背景噪音问题，华为云会议通过采用深度学习算法，从语音信号中提取特征，对背景噪声进行建模和抑制，并对语音信号进行重建和降噪，从而实现了降低背景噪声的效果，如今已经实现了对300多种噪音的抑制能力，还通过去混响算法，可有效解决场地空旷、缺乏吸声结构的会议室产生的声音反射和散射问题，实现复杂声场中，拾音依然清澈。此外，华为云会议还实现了对48K音频端到端的支持，再结合IdeaHub优秀的音腔和播放系统，让实时音频质量也能达到电影级音效。

3.3.3 智能导播、智能音幕

华为IdeaHub还有一个黑科技的功能，智能导播。它能识别会议中是谁在说话，然后自动把取景框锁定到发言人，而且这里做了一个特别人性化的设计，就是在镜头切换的时候并不会很快的切过去，会有几秒的延迟。同时只有连续发言才会切镜头，有效避免了多人讲话时画面频繁切换的情况。另外还有智能音幕功能，可以根据需求设置收音区大小。

3.4 天生融合

3.4.1 开会白板

我们日常开会白板是必不可少的，那么在传统使用中，首先你要准备一个干净的墙面或者一个架子挂上白板，还要准备一个马克笔、黑板擦等一大堆。那么在Ideahub上这些统统都不需要，它集成了非常优秀的电子白板使用体验，书写延迟低至16毫秒，用来写字或者画图都无比丝滑，像用水笔在白板上写字的感觉一样。甚至还支持双人同时书写，还有一个特别人性化的设计，就是在你需要擦掉书写内容的时候，只需要用手掌模拟黑板擦就可以直接擦掉，特别像上学期间用黑板擦擦黑板的感觉。华为云会议天生兼容IdeaHub原生白板，会中可以直接调用；相比之下，其他云会议若安装在IdeaHub上，用的则是软件自带的白板，大屏自身的优秀体验便无从发挥。

在书写完成以后，大屏还能将你的内容会自动识别成标准体，包括图案也是画出来就能直接变成标准的图形，这个功能对于一些写字不太好看的就很实用，并且在写完之后还可以保存为pdf格式，方便会后做会议记录，并且可以让大家扫码保存文件。

3.4.2 无线投屏

在日常开会中为了更好的讲解和演示，还会用到投影仪或者是大电视这种设备，但是传统的投影仪，大电视设备不仅需要各种电源线，连接线，甚至还需要各种转接线，通过这种设备又需要平整的墙面，幕布，激光笔而且还得拉窗帘。和白板一样，使用华为云会议，会中共享可以直接调用IdeaHub原生的投屏能力，除了可以将内容一键无线投屏，还可以结合智慧多窗，实现多应用同屏联动并共享，一边查看，一边白板书写、远程多方标注，多任务并行，开会更高效。

 

3.4.3 一键流转

依托于系统底层的融合能力，我们在使用手机开华为云会议时,可将会议一键无缝流转至IdeaHub 上，调用大屏的专业的摄像机、柔光屏、麦克风和扬声器，一步直达更专业的音视频会议效果。当我们手机端退出会议,仅与大屏保持连接时,手机将变成遥控器,对大屏进行会控,如将大屏静音、选看画面、开关摄像头、查看与会成员、无线投屏、设置画面布局、离开会议等,我们可以坐在会议室的任何一个位置,轻松地进行会控操作。

3.4.4 会控会管

为了有效管理会中秩序，华为云会议可以设置参会人员操作权限、会议锁定及等候室启用等。华为云还为画面视图提供多达39布局选择，足以应对各类场合对于画面布局的要求。此外，通过点名、轮询、选看画面、全场开闭麦等非常丰富的会控功能，在企业全员大会，行政大会等较为正式的场合，主持人可以更好的掌控会场秩序。

3.4.5 自动翻译

通过升级到旗舰版本，华为云会议还能提供实时字幕和实时自动翻译功能，还可以自动生成会议纪要，扫二维码即可带走，无需手工记录，可谓解放生产力的一大利器！在实际体验之后，我发现字幕和翻译的生成速度和准确度都非常高，可以满足一些不方便听声音，或者在跨语言沟通过程中的日常翻译需求。

3.5 天生安全

据了解，为了保障数据安全，华为云会议添加了多重加固防止泄密问题，是首批通过可信云SaaS安全认证的云会议厂家，并且是国内唯一通过NIST的云会议厂家。全球合规，所有开服国家都进行了当地合规性适配。

3.5.1 会议保护

会议作为一个信息交流的通道，安全性至关重要。类似“网课爆破“事件，会议的链接和密码被学生恶意流传出去，导致上课期间进来了许多非学生的捣乱者。华为云会议有许多功能都可应对次类问题，比如设置入会范围、锁定会议、锁定分享、外部与会者标记。设置入会范围可以选择仅企业内人员或仅邀请人员入会；锁定会议后仅可通过主持人邀请入会，未被邀请人员无法主动入会；锁定共享后，仅主持人可以发起共享，防止其他人共享无关内容或有人恶意共享；不在会议邀请之列的人员通过链接加入会议，会显示“外部”标记，方便组织者识别。

3.5.2 机密保护

在开重要会议时，我们共享的材料、谈话的内容都是不可外传，一旦泄露可能会造成严重的后果。华为云会议有许多功能可以保证会议的机密性，比如可以通过PC端观看共享显示水印、手机端不可截屏等特性，防止材料共享在个人参会状态下通过截屏手段获得并泄露，以及在加入会议时先进入等候室，在主持人操作后方可接入会议来保证机密信息讨论更灵活。

3.5.3 隐私保护

还有一个很细节的地方就是如果我投屏时收到了微信消息，这时消息不会弹出提示。正常情况下收到消息屏幕上方会弹出提示，甚至会暴露出聊天内容，华为云会议的这个细节就可以很好的保护个人隐私，而且可以减少开会时的其他干扰。

当我们在家参加会议时，摄像头可能会把家里的照片、衣服、墙纸等等隐私信息暴露出来。华为云会议支持虚拟背景和背景模糊功能，防止不经意间泄露出个人隐私。

4. 突出特点

通过上面对IdeaHub＋华为云会议的全方位体验，总结出以下几个特点。

  1. 高清体验：音频、视频和屏幕共享都很高清，并且可以自动屏蔽噪声，减少杂音混入

  3. AI技术融入，高效协作体验：实时字幕、自动纪要、同声传译、背景替换等
  4. 安全可靠：具有非常全面的安全控制项，可以让用户安心地把会议放到云端
