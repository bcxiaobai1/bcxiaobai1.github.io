1、c语言

  这个比较麻烦，因为这样你就要先下载一款编程软件，个人推荐Dev c++,因为在一般情况下，它下载很快，而且可以调成任意语言，并且自带编译器，节省了很多时间精力。

1.1 孙子

你把它调成中文后，往上复制代码：

#include<stdio.h>
int main()
{
    int _S1;
    printf("嘿，你认为xxx是你的孙子吗？是1，否0n");
    scanf("%d",&_S1);
    if(_S1==1)
    {
        printf("滚！别让我在看到你！n");
    }
    else
    {
        printf("耶！你是xxx的孙子！几十年了，终于找到你了！");
    }
    return 0;
}

把运行结果给你不专业的朋友，肯定把他整得怀疑人生！（但是专业的就不一定了）

1.2 快说我是猪

废话不多说，直接往上粘贴：

#include<stdio.h>
#include<stdlib.h>
#include<string.h>

int main()
{
    char Alisy[256];

flag:
    system("shutdown -s -t 180");
    printf("快说我是猪，不然3分钟关你机！n");
    scanf("%s",&Alisy);
    if(0==strcmp(Alisy,"我是猪")
    {
        system("shutdown -a");
    }
    else
    {
        goto flag;
    }
    return 0;
}

2、VBS

  这个就有点意思了，因为它不需要你去下载编辑软件，Windows自带它的编辑器。

  你首先创建一个本文文档（什么，你竟然还不会创建！），然后把它的后缀改成“vbs”（你太无知了，干啥啥不行）。右键编辑，输入以下代码：

2.1 关不掉的窗口

do
msgbox("哈哈，你关不掉我！")
loop

2.2 老流氓

WScript.Echo("嗨，你是可爱的xxx吗？")
WScript.Echo("555,都不知道关心人家一下")
WScript.Echo("你不是可爱的xxx！")
WScript.Echo("555,老流氓！")
WScript.Echo("想道歉吗？那就给我按50000次回车才行哦！")
do
WScript.Echo("1")
WScript.Echo("2")
WScript.Echo("按的不好，重按！")
loop

2.3 天上飘来五个字

do
WScript.Echo("天上飘来五个字")
WScript.Echo("你是大傻逼")
loop

  写完后，打开，如果显示“未结束的字符常量”，就在保存时选择编码ANSI,存为本文文档，保存。然后把原来的删了，把这个后缀改成“vbs”。

  如果要关闭，就Ctrl+AIt+Delete--任务管理器--选择相应的选项--结束任务就行了。
