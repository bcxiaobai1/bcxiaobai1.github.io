在Linux系统下安装软件分为三种方式：

  • rpm安装
  • 解压缩安装
  • yum在线安装

这里只演示rpm安装和解压缩安装两种方式

一、rpm安装：

1.1下载JDK的rpm包

这里我提供了jdk8版本的（也可以到官网下载）
链接：https://pan.baidu.com/s/1BR-ULB2TC1hqU9KqyhhnQQ
提取码：5hsa

1.2用xftp上传rpm包到Linux /opt目录下


到 /opt目录下查看是否有这个文件

1.3、检测当前系统是否存在java环境

输入指令 java -version

如果有已经安装了openjdk

  • rpm -qa|grep jdk 检测jdk版本信息
  • rpm -e --nodeps java版本信息 卸载jdk


1.4安装JDK

输入指令 rpm -ivh jdk-8u221-linux-x64.rpm （输入jdk后按Tab自动补全）


到这里就安装完成了
注：rpm安装无需配置环境

二、解压缩安装

2.1、用xftp将压缩包上传到Linux


检查是否上传

2.2检测当前系统是否存在java环境

2.3、将刚上传到压缩包解压到/opt目录下

输入指令 tar -zxvf jdk-8u121-linux-x64.tar.gz -C /opt

2.4、配置环境变量

输入指令 vim /etc/profile

在profile文件最后输入：
JAVA_HOME=/opt/jdk1.8.0_121（jdk安装目录）
PATH=$JAVA_HOME/bin:$PATH
CLASSPATH=$JAVA_HOME/lib:.
export JAVA_HOME PATH CLASSPATH

如何使用vim编辑器的可以参考这篇文章

2.5 关机重启

如果不想关机重启可到/etc目录下使用指令source profile
