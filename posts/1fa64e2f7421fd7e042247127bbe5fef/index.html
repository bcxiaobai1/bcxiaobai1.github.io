开源大新闻

PyTorch 公开发布五周年

2022 年 1 月 20 日是开源 Python 机器学习库 PyTorch 公开发布 5 周年的纪念日。为此 PyTorch 在 Twitter 上发布博文庆贺道，“今天是 PyTorch 公开发布 5 周年的日子。我们没有想到会走到今天，当前在这一开源库的基础上，我们拥有——2K 的贡献者、90K 项目、GitHub 上 390 万行的 “import torch”。更重要的是，我们收获大量的支持，也有一个伟大的旅程。为未来干杯!”

Deepmind 更改 Alphafold2 许可证，现已允许商业用途

Google 旗下的人工智能 Deepmind 公司近期修改了 Alphafold2 的许可，当前“AlphaFold 参数根据知识共享署名 4.0 国际版（CC BY 4.0）许可条款提供”，而在之前 Alphafold 权重和数据仅在知识共享属性-非商业4.0国际(CC BY-NC 4.0)许可证的条款下可用于非商业用途。如今随着协议的更改，Alphafold2 也可以用于商业用途。

NGINX 之父 Igor Sysoev 从 F5 离职

1 月 19 日，NGINX 在官方 Twitter 上宣布 NGINX 创始人 Igor Sysoev 从 F5 离职，以便将更多的时间留给陪伴家人和朋友，以及个人项目。NGINX 团队表示，“感谢他所做的一切让全世界的网站变得更好。”

Apache Hop 达到顶级项目状态

作为一个编排平台，Apache Hop 完全用 Java 编写，旨在提供广泛的数据编排工具，包括可视化开发环境、服务器、元数据分析、审计服务等。据外媒报道，Apache Hop 在 2020 年底达到 Apache 孵化器状态后，Apache Hop 在近日被提升为顶级状态，目前已经成为 Apache 软件基金会的顶级项目。

Linux 恶意程序数量在 2021 年增长 35%

近日，全球知名安全厂商 Crowdstrike 发布了一份 2021 年攻击数据报告。根据调查显示，与 2020 年相比，2021 年期间针对 Linux 系统的恶意软件增加了 35%，这些恶意程序主要通过感染物联网设备组建僵尸网络发动 DDoS 攻击。其中 XorDDoS、Mirai 和 Mozi 恶意程序家族占到了 22%；其中尤以 Mozi 数量最多，Mozi 恶意程序样本数量同比增长了 10 倍，XorDDoS 同比增长了 123%。除了 DDoS 之外，Linux 物联网设备还能用于挖掘加密货币，发送垃圾邮件等等。

开源软件专区

英特尔媒体驱动程序 22 首次亮相，支持 Alchemist / ATS-M、ADL-N

英特尔媒体驱动程序（Intel Media Driver） 22.1.1 发布，是 22.x 系列开源 GPU 视频编码/解码软件的第一版。此季度更新引入了新的硬件支持和许多功能更新：增加对 DG2/Alchemist（也称为 ATS-M）的支持；添加了 Alder Lake N (ADL-N) 平台支持；提高了 AV1 视频解码的稳健性；增加了对 VA-API vaCopy 上限报告以及各种优化等。

具体详情见：https://github.com/intel/media-driver/releases/tag/intel-media-22.1.1

Chrome 99 Canary 恢复了可移除预置搜索引擎选项的功能

在早前发布的 Chrome 97 版本中，Google 悄然引入了一项变化。作为搜索引擎设置页面改造的一部分，新版竟然不允许用户删除默认的搜索引擎提供商选项。但是在引发了舆论的一片哗然之后，Google 又承诺在浏览器设置 UI 中恢复该选项。在一段时间的等待之后，我们终于在 Chrome 97 Canary 分支中见到了该功能的回归。

Java XMPP 服务器 Openfire 4.7.0 正式发布

Openfire 4.7.0 版本正式发布，此版本是一年多以来的第一个非补丁版本，它带来了大量的新功能以及错误修复。主要更新包括允许从管理控制台下载日志文件、更新通知应包括集群节点参考等等。

具体详情见：https://download.igniterealtime.org/openfire/docs/4.7.0/changelog.html

前端开源文本编辑神器——Brackets

今日份推荐的开源工具是 Brackets，小编也是在学 SQL 课程时被老师安利，这个编辑器比电脑自带的文本编辑器好用多啦！

Brackets 是一个主要专注于 Web 开发的、轻量级但功能强大的现代文本编辑器。它由 Adob​​e Systems 创建，是根据 MIT 许可证获得许可的免费开源软件，目前由开源开发人员在 GitHub 上维护。它是用 JavaScript、HTML 和 CSS 编写的。

Brackets 的好处可多呢，比如即时编辑即时显示，能快捷地修改 css，以及使用插件很便捷等等。赶快去试试吧，诚不欺你哦！

GitHub 地址：https://github.com/adobe/brackets
