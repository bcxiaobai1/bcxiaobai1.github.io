?查找

⭐find命令

find [搜索范围] [选项]

?按文件名

根据名称查找/home目录下的hello.txt文件

find /home -name hello.txt

?按拥有者

查找/opt目录下，用户名称为nobody的文件

find /opt -user nobody

?查找Linux系统下大于200M的文件

（+n 大于，-n 小于，n 等于）
查找整个Linux系统下大于200M的文件（M是大写）

find / -size +200M

(因为是整个Linux系统，所以find后面是 / )

⭐locate命令

相比于find命令，locate命令可以快速定位文件的位置

?使用方法

使用locate命令前必须要先使用updatedb命令

updatedb
loacte 文件名

⭐which命令（查找指令的，而不是文件）

?使用方法

比如要查找 ls 命令在哪个目录下

⭐grep命令和管道符号 |

grep是过滤查找
管道符“|”是将前一个命令的处理结果输出，传递给后面的命令处理

?使用方法

请在hello.txt文件中，查找“hello”所在行，并且显示行号

cat hello.txt | grep -n "hello"

grep -n "hello" hello.txt

?日期时间

⭐date命令

?显示当前时间

data

?显示当前月份

date +%m

(+和%之间没有空格)

?显示当前年份

date +%y

(+和%之间没有空格)

?显示当前是哪一天

date +%d

?显示年月日时分秒

date "+%y-%m-%d %h:%m:%s"


?使用date命令来设置日期

?使用方法 date -s 字符串时间

比如把系统的时间设置为2021-11-11 11:22:22

date -s 2021-11-11 11:22:22
