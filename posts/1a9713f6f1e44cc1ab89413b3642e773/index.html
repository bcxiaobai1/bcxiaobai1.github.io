安卓抓取JD wskey + 添加脚本自动转换JD cookie

一、准备工具:

       1. app：VNET（抓包用）、京东；

         安卓手机需要下载VNET软件。下载官网：https://www.vnet-tech.com/zh/https://www.vnet-tech.com/zh/

       2. 已安装部署好的青龙面板；

二、VNET抓包wskey:

    1. 打开已下载的VNET软件，第一步先安装CA证书；

        点击右下角三角形按钮（开始抓包按钮），会提示安装证书，点击确定即可，app就会将CA证书下载至手机里，随后在手机设置里进行安装，这里不同手机可能安装位置不同，具体根据自己的手机型号百度查询如何安装CA证书即可；

      2. 第二步安装完CA证书后，点击上面的菜单按钮，选择【添加应用】（需要先停止抓包），然后点击右上角的加号，在弹出来的页面找到【京东】，然后返回；

   3. 点击开始抓包按钮，然后打开京东app首页刷新一下，再返回VNET软件，点击停止抓包；

4. 可以看到有很多链接，找到GET https://im-x.jd.com开头的链接，点进去往下滑，找到cookie里的pin和wskey，这样就算获取成功了；

三、青龙面板使用wskey转换脚本

1. 青龙面板拉取wskey转换脚本，GitHub仓库地址：GitHub - Zy143L/wskey: wskeywskey. Contribute to Zy143L/wskey development by creating an account on GitHub.https://github.com/Zy143L/wskey    终端拉取指令：

国外鸡：
docker exec -it qinglong ql repo https://github.com/Zy143L/wskey.git "wskey"

国内鸡：
docker exec -it qinglong ql repo https://ghproxy.com/github.com/Zy143L/wskey.git "wskey"

2. 拉取成功后，登录到青龙面板后台应该就能看到wskey转换脚本； 

3. 按照官方说明，增加用户wskey的环境变量，即通过刚才VNET抓取的wskey添加到青龙的环境变量中，格式如下：

 

变量名: JD_WSCK 参数: pin=xxxx;wskey=xxxx; # 注意分号 不要用中文分号!

3. 将wskey的环境变量添加完成后，便可以去执行一遍wskey的脚本了，执行后在环境变量里就能看到转化后的JD_COOKIE了：
