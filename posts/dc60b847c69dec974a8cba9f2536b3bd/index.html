用C语言基础及数学公式来简单实现土味表白

前言：

       在学习了一些编程基础以后，尤其学习了scanf函数以后，我觉得scanf函数还挺有意思的。所以就想着通过所学习的一些简单C语言基础知识，尝试着去写出一个有意思的程序。于是我就开始想有哪些有意思的数学公式，忽然就想到初中的时候玩过的一个小游戏，就是将一个任意数字放入公式中计算，结果都会是固定值，也就是5201314。所以我就去百度搜索到了这个公式。于是就写出了这个程序。

（PS：只是一名小小白，有什么错误的地方请大佬多多指教，谢谢！！！）

一、首先简介一下所需要用到的知识点以及各个知识点的用法

（1）输出函数：printf();（printf()是C语言标准库函数,用于将格式化后的字符串输出到标准输出。）

其用法如下：

#include<stdio.h>//库函数 
//定义主函数 
int main(){
	printf("HelloWorld!n");/*输出函数printf的
用法，其中 n是转义字符，用于换行。*/
	return 0;//程序结束标志 
}

编译运行结果为：

（2）宏定义：#define（C语言中，可以用 #define 定义一个标识符来表示一个常量。其特点是：定义的标识符不占内存，只是一个临时的符号，预编译后这个符号就不存在了。）

其用法如下：

#include<stdio.h>//库函数 
#define PAI 3.1415 //这里宏观定义了标识符PAI的值为3.1415 
//定义主函数 
int main(){
	printf("%f",PAI);/*输出PAI的值，其中%f是对PAI的
格式控制，保证它输出来的数字的精准*/
	return 0;//程序结束标志 
}

编译运行结果为：

（3）输入函数：scanf();（用于程序获取你从键盘上输入的数据，并赋值给你在程序中所定义的变量）

其用法如下：

#include<stdio.h>//库函数 
//定义主函数 
int main(){
	int a;//定义一个整型变量a 
	scanf("%d",&a);/*从键盘任意a的值 ，因为定义和%d控制
的都是int型，所以输入的值要为整数 */
	printf("a=%d",a);//整型输出a的值 
	return 0;//程序结束标志 
}

编译运行结果为：

 ps：我从键盘输入的是3，也就是给a赋值3，所以输出a=3

（4）浮点型定义：float（属于四大类型之一，用于定义更为精准的数字，范围在-3.402823466×10的38次方到3.402823466×10的38次方）

其用法如下：

#include<stdio.h>//库函数 
//定义主函数 
int main(){
	float a=3.134122;//将a定义为单精度浮点数 
    printf("a=%10f",a);//单精度输出a的值 
	return 0;//程序结束标志 
}

                                                              编译运行结果为：

 （5）数学公式：((X+10402628)*10-X*10)/20;（其中X是任意的两个数字,但公式结果值都会是5201314）

—————————以上就是全部知识点了，肥肠滴基础和简单

二、最后我们来上程序

#include<stdio.h>   //头文件 
#define JZZ 10402628  /*宏定义JZZ的数值，这里给大家一个小tip，
就是JZZ可以换为任意字母，懂我的意思吧哈哈 */
int main()   //主函数 
{
    float fNumber;   //定义一个变量 
	float fResult1;  //由于公式太长，下面分为两次计算，所以定义两个结果值 
	float fResult2;  // 将最后的结果值赋给这个变量 

	printf("请输入一个数字:n");  //输出提示词，并换行 
	scanf("%f",&fNumber);  // 将输入的值赋给所定义的变量fNumber
    
    fResult1=(fNumber+JZZ)*10-fNumber*10;  /*将输入的数字带入公式并进行计算，
	其中用到了宏定义JZZ，它代表的值也就是上面的10402628 */
    fResult2=fResult1/20;    //将最终结果值5201314赋值给fResult2

	printf("不管你输入的是什么我想对你说的是：%fn",fResult2); 
	 //输出语句和结果值 
	return 0; //程序结束 
}
