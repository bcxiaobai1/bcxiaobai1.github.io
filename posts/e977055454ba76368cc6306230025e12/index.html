  1. 检查 HDFS 文件系统的整体状态
命令： hdfs fsck /

  1. 使用 ls 命令查看文件和目录，输出是空的吗？如果是空的表示什么？
    命令：hdfs dfs -ls /

是空的。表示文件系统里面没有任何文件。
4. 使用 mkdir 创建两个文件夹，文件夹名称为 test1 和 test2.


命令：hdfs dfs -mkdir -p /gs/test1
      hdfs dfs -mkdir -p /gs/test2

  1. 使用 touchz 在第 3 步的文件夹 test1 中创建空白文件，文件名称是 ”text+你的 学号”
    命令：hdfs dfs -touchz /gs/test1/text180703021043

6.在本地 Home 目录下创建文件 test1.txt, 任意写入内容。使用 copyFromLocal 将 test1.txt 复制到 HDFS 上的 test1 目录下。/home/ubuntu

命令：touch test1.txt 
      ls test1.txt
	  gedit test1.txt
      cat test1.txt
      hdfs dfs -copyFromLocal /home/ubuntu/test1.txt  /gs/test1
      hdfs dfs -ls /gs/test1


  1. 使用 mv 命令将 test1.txt 从 test1 目录移动到 test2 目录
命令：hdfs dfs -mv /gs/test1/test1.txt /gs/test2

  1. 使用 rm 命令删除 test2 文件夹中的 test1.txt
命令：hdfs dfs -rm /gs/test2/test1.txt

  1. 使用 get 命令将文件 test1.txt 复制到本地 home 目录下

10.使用 text 命令输出 hdfs 上的 test1.txt 文件的内容，比较 cat 命令

命令：hdfs dfs -text /gs/test1/test1.txt
命令：hdfs dfs -cat /gs/test1/test1.txt

text命令检测文件的编码，并在可能的情况下将其解码为纯文本，而cat则不这样做

  1. 使用 rm 命令删除 test2 文件夹，如果 test2 文件夹中有文件，如何递归删除？
