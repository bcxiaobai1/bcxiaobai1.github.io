DeFi还没搞明白，DeSci又来了？

区块链的发展日新月异，新概念层出不穷。最近，看一个很新的概念“DeSci（Decentralized Science）”，顾名思义，是“去中心化科学”。去中心化科学是什么意思？今天一起来了解下这个新概念。

// 什么是DeSci

”DeSci是一个区块链应用的实例，旨在建立能够公平公正地创建、资助、审查、存储和分发科学知识的可信公共基础设施。在DeSci这个生态系统中，科学家和研究者可以通过分享他们的研究和知识来获得奖励，并且网络上的任何人都可以公开访问这些研究和知识。DeSci的基本思想是，科学研究的过程都应该是透明的，以及人人都可以获得科学知识。这是一个相当具有革命性的Web3运动，DeSci或将改变学术界科学研究和资助的传统做法。此外，DeSci还能够为创新公司和优秀科学家建立直接联系，以此来促进企业的技术革新。

// 为什么需要DeSci

”在科学领域中，科研项目常因官僚主义、中介机构等无谓的摩擦而受到限制或控制，这些摩擦不仅会拖延资金，更拖慢了科研进度。最近的一个例子是新冠疫苗的研发，有的国家率先研发出疫苗，但并未与其他国家地区分享研究成果或提供可参考的研究资料，而资金及资源较少的国家和组织难以高效地获得创新成果，这意味着在疫情大流行期间会有更多人失去生命。科研的时效性很强，尤其是在高速发展的时代背景下，无论是疫苗、新药、操作技术，还是医疗试验，都呼吁着“科学需要速度和动力”，但这在中心化机构的限制下似乎是不可能。另一方面，一些研究结果往往掌握在政府或科技生产巨头的手中，受其控制并为他们的利益服务。更重要的是，最大的限制或不平等在于资金补助的获得。对于研究人员而言，获得资金本身就并非易事，而资金不足则会限制科学研究的范围。但很多时候，资金被分配给与资助方关系密切的研究人员，而不是更具有价值或更重要的项目。一个科学家或研究人员平均要花50%的时间来撰写申请资金的提案，如果无法获得资金或获得的资金不足，那么他们的所有努力都可能付诸东流。此外，政府有权削减资金，并将预算分配到其他地方。例如，英国去年减少了1.2亿英镑的科研经费，导致许多科学家放弃了他们一生的工作。DeSci或许就是这些问题的解决方案。在这里，任何人都可以获得科学研究资源或科学知识，研究人员不会因为中介机构而失去激励，而且有更多获得资助的机会。辉瑞或高通等科技公司和生产巨头也将因DeSci受益，他们能够与从事所需研究的科学团体直接接触。这就是DeSci的优势所在。

// DeSci的资金来源

”DeSci具有去中心化的特性，分布式模式使其不仅信息公开透明，而且资金也拥有更广泛的来源，例如DAO、二次方捐赠、众筹等。出版业私有化吞噬了学者和科学家的大量利润，这个问题可以通过区块链上的智能合约的道缓解。DeSci是科研人员能够直接与审核方或受众互动，而无需任何中介，从而提升对科研人员的激励。最重要的是，DeSci彻底改变了传统的资助过程。通过基于区块链构建的DAPP，科学人员可以联手探索项目，并将研究全和贡献转化为代币化的NFT，帮助科研团队获取资助的同时，构建更加可持续的科学社区。不仅如此，DeSci为众筹创造了一个更有效的模式，因为资金的分配是由公众使用二次方捐款或DAO等机制决定的。2019年，专门针对科学和技术的私人投资的市场规模估值超过170亿美元。DeSci的加入可以大大增加这一市场规模，让更多私人投资者进入这一领域。而对于生产企业而言，DeSci降低了他们的创新成本，这对参与者之间的竞争产生了积极的影响，由此产生的更直接的影响就是消费者更容易获得质量更高的产品。

// 自由与合作

”由于所有的研究数据都存储在区块链的分布式账本上，世界各地的人都可以访问DeSci上的研究成果和科学知识。这使得科研人员在传播研究成果的过程中，不用担心别有用心的审查或中心化机构对科学数据的控制。此外，研究人员可以利用区块链与来自全球各地的同行进行合作，这将使科研团队更加充满活力。除此之外，DeSci也会通过NFT激励科学家和研究人员参加学术活动。例如，他们可以培训或指导年轻的专业人士，参与同行评审并分享他们的知识和信息等。与此同时，这些NFT也可以作为评估研究人员对科学贡献程度的可信证据。因此，他们不需要再花费数年时间来通过出版作品以建立可信度，而是通过参加DeSci的增值活动来赚取NFT即可。
