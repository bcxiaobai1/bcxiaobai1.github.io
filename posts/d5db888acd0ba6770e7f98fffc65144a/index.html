我在亚马逊云平台的学习成长之路

前言

今年是亚马逊云科技成立的第16个年头，也是云计算行业诞生的16周年。作为云计算时代的“领头羊”，亚马逊2006年推出了名为Amazon Web Services(AWS)的新产品。AWS背后的想法是提供一个平台，同亚马逊内部使用的平台一样，将其作为服务向公众发布。通过将平台作为服务向公众发布，亚马逊进入了一个称为公有云的市场。作为云计算的领头公司有着得天独厚的优势，所以在亚马逊平台进行云计算相关知识的学习，是一个非常好的选择。而且现在平台还提供了重磅福利，亚马逊云科技提供了100余种产品免费套餐。其中，计算资源Amazon EC2首年12个月免费，750小时/月；存储资源 Amazon S3 首年12个月免费，5GB标准存储容量。亚马逊AWS 海外区域账户免费套餐
仅以此篇记录我在亚马逊云平台的技术成长之路，简单阐述一下是怎么从一个一无所知的小白一步一步成长起来的，是亚马逊平台陪着我一路成长和提升。

基础篇

在学习了解亚马逊云科技平台之前，我们需要先学习一个概念就是云计算。云计算通过互联网按需提供IT资源，并且采用按使用量付费的定价方式。你可以根据需要从云提供商那里获得技术服务，例如计算能力、存储和数据库，而无需购买、拥有和维护物理数据中心及服务器。
亚马逊云科技平台是帮助企业和开发人员使用Web服务来构建可扩展的复杂应用程序。
我们可以将云计算平台用于多种领域，例如应用程序的托管，内容的分发，搭建网站，企业IT，数据库等等。
从一开始的一无所知，在亚马逊官网学习之后，对云计算有了初步的了解，在我们深入学习之前我们需要明确我们为什么要学习云计算，云计算又有什么优点。

  • 敏捷性：它可以让你轻松使用各种技术，从而可以更快地进行创新，并构建几乎任何你想要的东西。您可以快速启动云资源，在云服务器进行一系列的学习和操作。
  • 节省成本：云计算可以将您的固定支出转变为可变支出，并且只需按实际使用量进行付费。由此，可变的经济支出比固定的经济支出节省很多。
  • 短时间内进行全局部署：借助云平台，您可以扩展到地理区域，并在几分钟内进行全局部署。例如：AWS的基础设施遍布世界各地，你只需要点击几下鼠标即可在多个物理位置部署应用程序。

进阶篇

对亚马逊云科技平台有了初步的了解之后，我们就可以进行一些深入的学习和操作了。我当时在亚马逊平台发现的宝藏资源就是培训和认证课程，亚马逊提供的数字培训和课堂培训。数字化培训可以让您在线学习并自己掌握学习进度，向专家老师学习最佳实践，轻松提升自身本领。除此之外，还有虚拟培训，让你可以随时随地的学习课程。课程适用范围很广，不管你是刚入门的小白还是正在提升的IT程序员，AWS培训和认证课程都可以帮助你提升自己。
而且亚马逊云平台的培训课程也是十分的细致，按角色培训(架构师、云从业者、开发人员…)，按解决方案培训(高级联网、数据分析、机器学习…)。

我当时学习机器学习就是在亚马逊平台寻找的培训课程，课程质量非常高，对于机器学习方面技能的提升有很大的帮助。课程链接放在这里，便于大家参考学习：AWS Skill Builder。如果你是新手，寻找更加明确的起点帮助你确定以后自己的职业，或者是积累机器学习相关方面的知识，建议你是从AWS学习计划开始；如果你已经掌握了一部分的相关知识，那你就可以通过AWSj进阶指南深入学习探索机器学习，并且为 AWS Certified Machine Learning – Specialty 认证考试做好准备。我呢，一开始就是一个“小白”，肯定是从头开始学起。学习课程之前需要注册一个亚马逊账号，只需要通过一个邮箱就可以了，非常简单也不需要过多的阐述。接下来就是进入我们的正题，学习机器学习课程。

本课程内容一共分为了13小节，13小节一共是36h时长，可以说是非常简短的一个课程，但是意义巨大，条理非常清晰，每一小节都有明确的主题和内容介绍，每一小节中也会分为几个小的部分。不要小看这个小小的课程，对我的帮助是非常巨大的。让我从一开始一个一无所知的小白，渐渐明白了什么是人工智能，明白了什么是机器学习，明白了什么是深度学习。机器学习是专门研究计算机怎样模拟或实现人类的学习行为，以获取新的知识或技能，重新组织已有的知识结构使之不断改善自身的性能。并不是我一开始简单认为的机器学习就是学习机器人。后来又详细学习了机器学习术语和过程，这里介绍的非常详细，让我学到了很多。

再后来就是详细学习了Amazon SageMaker（一站式机器学习平台），Amazon SageMaker通过整合专门为机器学习构建的广泛功能集，帮助数据科学家和开发人员快速准备、构建、训练和部署高质量的机器学习模型。官网也是有详细的分步指南，而且可以免费试用Amazon SageMaker，有兴趣的小伙伴完全可以尝试学习一下。
亚马逊还有一些认证考试，对自己的技能水平的一种检验方式。其中包括了AWS认证云从业者、AWS认证解决方案架构师，AWS认证数据库等等，而且也是有详细的考试认证指南，包括适合哪些人员参加，考试指南，考试样题等。
例如 AWS Certified Cloud Practitioner 认证考试面向能够有效证明AWS云知识总体掌握情况的个人。本开始验证应试者的能力包括解释AWS云的价值、了解安全最佳实践、了解 AWS 云成本、经济性和账单实践 、描述和识别核心 AWS 服务的定位，包括计算、网络、数据库和存储 等等。考察的内容AWS云概念、 AWS 云中的安全性和合规性、了解核心 AWS 服务 、了解 AWS 云的经济性。考试题型分为单选题和多选题，还有一些不计分题目，需要注意的是考试中是不会标明这些不计分的题目的。所以想要通过考试认证的小伙伴，一定要认真学习，提升自己才能顺利地通过考核。 认证的结果并不是最重要的，而是你前期的准备，你可以学到很多东西，让自己有很大的提升，毕竟知识才是无价的嘛。

番外篇

学习技术不仅仅使自己孤身一人地研究技术文档，动手实操，与他人交流也是非常重要的，可能他人很多对一方面知识独特的见解是你一直没有想到的。印象很深的是2021年亚马逊云科技峰会（上海站），虽然我只是参加的线上峰会但也是收获满满。2021年，亚马逊科技峰已经在中国举办了八年了，也算是中国云计算市场最早期的布道者。第八届亚马逊云科技峰会上，亚马逊全球副总裁、亚马逊云科技大中华区执行董事张文翊提到了很多重要的观点。第一点就是首次全面阐述了亚马逊云科技的中国战略，表明了希望利用全球优势和资源帮助中国企业上云的态度。第二点就是深入发展的规划，介绍亚马逊科技在人才培养和环境可持续发展方面的目标和计划，说白了是让更多人来把好的技术方案落地。我需要重点关注的就是亚马逊人才培养方面的发展，万一以后就入职了呢，嘿嘿~

也结识过几位亚马逊云在职者，与其中一位也有过很多的交流。他向我介绍了他的一些发展经历，遇到的一些困难和机遇。他说他的工作最吸引人的地方就是需要站在客户的角度换位思考，比如遇到一些初创公司的客户，就需要把自己当成是一名创业者，用一些通俗易懂的语言和简单的例子来向客户介绍出亚马逊云科技平台的优势，让客户能够诚心实意的选择亚马逊云科技平台。他也有一些挑战和困难，就是亚马逊的工作节奏很快，技术发展也十分迅速，自己需要在工作之余学习新的技术，跟上团队的步伐。其实他说的这些不仅仅是亚马逊在职的人需要做到的，我觉得是我们每个互联网人必备的品质，唯有不断地学习，不断地成长才能得到更好地发展，才能实现自己的价值。
