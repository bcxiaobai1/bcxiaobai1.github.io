Windows Copilot 问世

两个月前，微软推出了 Windows 365 Copilot，颠覆了互联网传统办公方式，把 Word、Excel、PowerPoint等多款产品带入了AI时代。

现在，微软又迈出了重要一步，正式上线 Windows Copilot，并将AI深度集成到Windows 11系统中。

我们可以直接在Windows系统工具栏上快速召唤Copilot，让它提供各种智能服务。

Windows Copilot

我们可以通过与Copilot对话的方式，轻松完成各种任务。为电脑切换主题，打开软件，搜索资料等等

并且集成了office全家桶，这将为我们以后工作生活中带来极大的便利

微软联合openAI已经上线了近50款插件，都可以在Copilot中调用。开发者可开发定制插件在这上面使用。

预计Copilot将在下个月开始公测，以后工作生活中真的离不开AI了。

Bing Chat 插件功能上线

继openAI对plus会员开放插件功能后，微软也宣布，接下来将在 Bing 上集成插件功能，让开发者和用户也可以自行创建插件。

目前Bing在中国区桌面端市场份额占比为37.4%，百度市场份额为27.01%。一旦Bing集成了插件功能而且Bing成为GPT的默认搜索引擎的话，将对浏览器市场产生巨大的冲击，许多用户可能会转而使用Bing。

微软这一步不光把桌面端应用市场吃死了，还把浏览器市场的桌子给掀了，希望国内赶紧追上步伐吧

Edge 与 365 Copilot 结合

前段时间365 Copilot的发布为日常办公工具带来了创新性改变。现在微软又将它与Edge结合了，以便用户更高效的使用这些工具。

这意味着为365 Copilot打通了联网功能，这也代表着365 Copilot的能力将进行飞一般的跳跃。

Copilot可根据用户打开的页面 自动生成 ppt，数据分析，行业报告等功能。

可以想到将为使用这些工具的人们带来多大的改变与创新。

搭建你的第一个AI应用（Azure AI Studio）

微软不仅整合了桌面端应用程序和人工智能技术，还发布了Azure AI Studio。现在，用户只需输入几句话，通过简单的鼠标操作，就可以生成一个AI应用程序，快速开发一个属于自己的 AI 对话模型。
