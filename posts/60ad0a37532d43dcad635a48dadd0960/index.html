如何彻底卸载 MySQL ,再可重新安装 MySQL

一、 停止已安装的 window 的 MySQL 服务


1）鼠标右键点击桌面  “此电脑”（我的脑电）->  点击管理

       

2）点击“服务和应用程序” ->  服务  ->  MySQL ;  鼠标右键点击  MySQL，点击停止服务。


二、卸载MySQL安装程序。

1）找到“控制面板”-> “程序”->“程序和功能”，卸载MySQL程序



三、删除 MySQL 安装目录下的所有文件

1、删除c盘ProgramData隐藏目录中关于MySQL的目录


1）打开C盘，  查看 ->  显示 ->  隐藏的项目


2）找到ProgramData目录




3) 删除MySQL目录

2、删除 MySQL的安装目录（根据自己安装是C盘还是D盘找或其他的自定义位置）

四、windows+R运行“regedit”文件，打开并删除MySQL注册表


找到并删除注册表：

HKEY_LOCAL_MACHINESYSTEMControlSet001ServicesMySQL

五、正常到上面的步骤就完成了。如需查看是否删除完毕，再按照第一步查看是否有 MySQL服务，如果没有说明删除完毕，如果还存在MySQL服务，

1） 点击Windows 图标，搜索命令提示符，以管理员身份运行

2）输入 sc delete mysql  删除已近安装的MySQL服务（提示：平时慎用此命令）
