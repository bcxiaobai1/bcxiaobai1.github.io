1. 关于金融科技介绍的那些事儿

1.1 互联网我都懂，金融科技到底是个啥？

在校生接触金融科技的机会感觉不多，再加上身边大部分技术佬可能都挤破脑袋只想着去互联网实习，所以对金融科技的认识不多也是人之常情。

那我引用一段定义吧：“金融科技是基于大数据、云计算、人工智能、区块链等一系列技术创新，全面应用于支付清算、借贷融资、财富管理、零售银行、保险、交易结算等金融领域，是金融业未来的主流趋势。”，这样大家应该就知道了吧，“金融科技”顾名思义就是“金融+科技”，利用科技手段给金融带来效益。具体应用场景的话，我们身边也有很多，比如银行智能柜台、智能客服、智能投顾、大数据征信系统等。

据我了解，这几年央行系统内至上而下发了很多关于金融科技发展的纲领性文件，几大国有行、地方银行等都逐步在做数字化转型的战略规划，像建行就专门成立了建信金科这样的子公司，整合了建行原有研发中心，同时还专门设有金融科技创新中心、Big Data中心、实施管理中心等直属中心，运用人工智能、区块链、云计算、量子计算等前沿技术不断出现在金融科技公司的新业务线里，光听名字都是一些非常高大上的高级技术

（希望我司也能多让我参与一下这些新赛道，升华下自己）

1.2 为什么建议应届生选择金融科技方向？

咳咳，不是王婆卖瓜自卖自夸哈，大佬和我站在统一战线，都觉得像建信金科这样的金融科技公司还是很有吸引力的：

（1）待遇在同行业的竞争力比较足、工作强度没有互联网那么大、稳定性还很高，完美反映了很有名的那句话，怎么说来着，work-life balance 。

（2） 银行的金融科技发展还是领先于券商基金保险的，发展前景较好，毕竟背靠总行（总行诶！那可是总行！！），拥有很多资源和渠道，还能接触到领先的业务场景。

（3） 而且这类公司有很多技术大佬，也有一直在追最新技术栈的。

1.3 对在校生来说，金融科技有什么岗位选择？

金融科技的招聘现在主要还是在招计算机方向的，因为不管你对接什么样的业务场景，最终落脚还是需要编程实现嘛。至于具体岗位、要求、流程和互联网应该是类似的，也很全面，只是难度相对低一些、技术上不会像互联网要求那么高的。当然，不意味着金融科技岗的程序员比互联网门槛低，金融科技岗好多都是技术拔尖、软性能力同样顶呱呱的大佬程序员！

2. 关于金融科技校招的那些事儿

2.1 金融科技的招聘流程和互联网一样吗？

我在网上搜罗了半天，发现了一个以建信金科的校招流程为例的参考答案，分享给大家，其他单位应该差不多类似吧：

笔试（非技术岗也有一道简单算法 + sql） + 心理测评 + 2轮技术面 + 一轮总裁面（无技术） + 提交材料（审核1-2周） + 体检（差额） = offer（仅供参考）

提醒一下，国企流程都比较慢，不要着急。据我了解，银行很多都是统一批次笔面试的、并不是像互联网一样反复轮动笔面试，所以可以先多看其他求职机会。

再说，现在互联网的流程我看也没多快啊，比如某华对吧，不能再说下去了，说下去号就不保了。

还有就是等工作了以后就会发现，什么事情都需要报备、请示领导、上会讨论、走用印流程等，hr也很急的、但急也没用啊，社牛大佬建议可以加一下hr小姐姐微信，保持沟通！

2.2 听说金融科技的笔面试比较简单、玄学？

公司的笔试都是内部经过考量确定的，笔试只是第一道关卡，笔试简单并不意味这技术含量低！得从多个角度来想这个问题：

（1） 难度大小不要过于在意，太难了可能通过的只剩下一些编码大神、最后还不一定愿意来，太容易了也不行。

（2） 难度大小也是相对的，对于大佬可能啥都不难、一路直通，对于准备没那么充分的同学来说、特别是想来偏国企的同学来说，这就是一大幸事啊。

（3） 还有就是，从牛客讨论帖上看，金融科技这个难度应该来说和其他同类型的企业是差不多的，有些单位甚至连技术都懒得考你了，直接看综合素质，平常心对待即可。

关于金融科技公司面试比较玄学这个情况，我也分享一下我当年的真实求职感受：

（1） 如果你考虑偏国企的地方，建议除了熟悉互联网的面试模式以外、要准备一些不用厮代码的结构化玄学面试，充分展示综合素质，这样才能更顺利通过像建信金科这样企业的面试，当然不是说不要准备技术哈哈。

（2） 面试过程中一定要稳住心态，总会有一些面试官很不符合你的预期，或者是语气上给你压力、或者是比较严肃冷漠、或者是故意理解不了你的思路...很多同学会吐槽“完了，我怎么这么倒霉，遇到了个啥也不懂、还不友好的面试官！”。对此，不管遇到什么情况，一定要沉住气、耐心交流沟通、把自己的理解全部表达即可，而不是当场表现出不耐烦、不满意、不开心，那你就真的陷入圈套了，保持面试者的心态，很多还是能顺利通过的。

2.3 应届生如何准备金融科技的校招求职？

如果感兴趣金融科技的话，我是建议除了互联网实习、可以尝试去金融行业找个实习锻炼一下，这样你也能提前了解一下情况、对校招很有帮助。

另外听大佬说，建信金科每年招聘季都会跟其他平台合作编程竞赛，大家可以关注一波。其他金融机构针对在校生的比赛（给的钱还不少）也是有的，大家可以多关注官网、公众号之类的宣传平台。

提醒一点，金融科技招聘一般比互联网晚，大家可以按照互联网的笔面试标准严格要求，通过金融科技的笔面试机会就会大一些，也会更游刃有余。

3. 关于金融科技工作的那些事儿

3.1 金融科技日常工作接触业务多、还是偏科技？

估计很多计算机的同学工作之前更多都是在互联网实习锻炼，到了金融科技公司以后其实更多还是会做开发相关的工作。

值得一提的是，因为是金融科技嘛，所以技术岗位也是会接触到很多业务场景，会关注到银行的业务生态。还是那句话，金融科技就是通过技术服务金融业务，从而带来金融服务模式和产品应用的变革。

另外补充一点，像建信金科的话，平时内部项目是分为两种，简单说就是对内和对外项目。因为前身就是建行的研发部门，所以一方面要服务于集团总部，另一方面也是有独立对外发展业务，包括与保险、理财公司等都是有对外合作项目的。

3.2 金融科技研发人员平时用的技术栈会陈旧吗？

据我了解，技术栈还是会紧跟技术发展趋势的，当然实际开发过程中，还是会结合业务场景再看如何应用和尝试，不会贸然就用最新技术、会选择最适合的。然后整体来看的话，如果满分10分，大佬认为像建信金科这样的金融科技公司从技术革新的关注度上能打9分以上、非常高的评价，当然对比互联网的技术水平还是有差距的。

对于金融科技行业的技术而言，一方面会有传统理解的开发任务，包括系统建设等，当然现在大部分也会往云端迁移重构等；另一方面也会成立团队进行人工智能、量子计算、元宇宙等新兴技术的尝试。

绝不是推销，内部的技术积累不用担心，学不到东西你来找我，我一对一辅导好吧

3.3 金融科技工作氛围、工作强度之类的如何？

以建信金科为例，大佬觉得工作氛围还是挺好的，同事们之间很友好的、不存在一些很传统国企体制内的东西。可能也是搞技术的同学们都比较纯粹、能聊到玩到一块（没有贬低非技术岗同学的意思，不要误会）。

至于工作强度，这块我看同学们估计比较关心，我补充说一下：

（1）搞技术的话版本上线肯定是会加班的、有时候是周末，这个不可避免的，只能祈祷系统上线顺利一些。

（2）像大佬也提到，国企的公司和部门肯定是不会强制要求加班之类的，还是希望敏捷开发、提升效率，避免不必要的加班时间。

（3）平时加班情况的话，会跟随岗位属性、工作内容、项目进度有一些周期性的变化，版本迭代的时候肯定会更忙碌一些、一般情况下就正常作息，强度和互联网比还是比较低的。

建议大家了解清楚、做好心理预期，这样就不会有太大落差。

4. 关于金融科技待遇的那些事儿

4.1 网上爆料的金融科技薪酬待遇准确吗？

我觉得金融科技的薪酬待遇还是很有竞争力的，像建信金科大佬透露的价格比起去厂里的白菜价、应该是个非常不错的选择，在同行业里也是很有料的。然后券商的待遇受行情影响比较大、可能模糊一些，银行的金融科技还是比较清晰的。

具体薪酬同学们可以自行了解、这里不方便说，可以说的是：

（1） 薪酬结构相对是比较固定的，应届生开的价格是根据学历、城市、事业群、岗位等进行定薪，同样条件的话应该是一样的。

（2） 应届生如果笔面试表现很优秀、愿意去的话，会开出SP、SSP这样的offer，面试过程中也可以尝试申请，薪资会高出很多、比去互联网肯定是香的，而且入职后大佬们的定级也会比较高哦，这对未来的内部发展也是很有帮助的。

（3）金融行业挺多地方有听到限薪降福利，不过大佬说建信金科暂时没有这样。

4.3 金融科技公司的内部考核、职级晋升怎么样？

根据大佬反馈，内部考核肯定还是根据表现进行综合评价的，然后如果考核比较差的话，像建信金科是会建议你进入内部的“加油站”进行培训学习，包括心理辅导、技能培训等，尽快让你适应成长、或者和人力沟通调整更合适的岗位。

职级晋升一般来说是按照绩效进行积分累加的、达到一定分值就可以晋升了，当然是需要述职的，越到后面晋升是有通过率的，不可能一路直通。然后不同类型的岗位肯定是有自己的晋升路线，个人感觉年轻人不能一口吃成大胖子，还是要专心提升自身的技术特点、业务能力等，这样该有的自然就都会有的。

5. 关于金融科技发展的那些事儿

5.1 八卦一下，建信金科是建行的子公司吗？

工作几年了，对于体制不体制、编制不编制这些，我发现身边大家已经没那么在意了、毕竟以后可能就没了。

说回来，建信金科是建行的金融科技子公司，非要从股权穿透来说的话：建信金科是建银国际持股的（之前应该是全资、现在是分出去一点点），然后建银国际又是作为中国建设银行设立于香港、并面向全球开展投资银行业务的全资子公司，所以建信金科确实是建行的全资子公司，没啥问题哈哈。

5.2 金融科技偏国企的话，稳定性方面如何？

关于稳定性问题：

（1）金融科技大多是国企，肯定不存在互联网那样的毁意向、毁约、卡实习期、裁应届生等情况，这点同学们还是可以放心的，稳定性还是杠杠的。

（2）以建信金科为例，从劳动合同来看，校招生签约的肯定是建信金科，社招有些因为是劳务派遣等形式、签约的可能是第三方公司（即外包），这点要区分清楚。

5.3 同学们应该挺关心金融科技的成长路线吧？

以前可能很多国企对于新人的培养会比较少一些，但这两年已经改善很多了。像建信金科也是，这两年随着金融科技发展，明显培训体系都建立起来了，新员工培训可以从公司战略层面了解到科技发展的规划和布局，也会有带教老师让新人更快适应吧。

未来发展的话，如果未来一直在公司成长的话，那就是走一步一步走职级，当你的技术能力不断提升、金融业务认识不断增加，能力标准和晋升的条件满足了，就有机会的。校招生应该还有金融科技管培生计划、优博计划之类的，也是非常不错的机会。

如果是想要跳槽的话，外面同行业、或者券商基金保险等都是有金融科技相关岗位的社招的，现在还真不少，很多金融机构都缺这样的复合型人才。

行业内的朋友可能感触比较深刻，你会发现很多岗位不招聘了、但金融科技类岗位一直在校招社招的。在金融科技公司不躺平好好搞的话，能力和见识肯定是能够很快提升的，不怕会丢失核心竞争力的。可能是句废话，因为哪里都需要好好努力提升竞争力
