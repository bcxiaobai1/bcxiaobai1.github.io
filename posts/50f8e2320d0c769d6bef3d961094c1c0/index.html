倾斜摄影技术构建图扑 WebGIS 智慧展馆

智慧展馆通过“云大物移智链”等技术将“物”（展品、设备、环境等）进行互联，并感知“人”（工作人员、观众等）的行为。结合 GIS、BIM、倾斜摄影进行展馆全场景复现，通过管理平台基础层、感知层、数据层、平台层、应用层之间的数据传送，将“物”的承载信息通过图扑软件可视化系统提供给“人”。对展馆生态环境质量、设施设备能耗以及人车流动向做好采集、监测与分析，提前进行风险预警。

 

倾斜摄影

图扑智慧展馆解决方案通过倾斜摄影技术勾勒精准数据模型，多维动态展示展馆外景，高频次刷新设备实时数据，支持接入多源异构数据。一站式呈现全馆运营态势，智能预测趋势，快速响应突发状况。

倾斜摄影技术以大范围、高精度、高清晰的方式全面感知复杂场景，通过高效的数据采集设备及专业的数据处理流程生成的模型数据成果直观反映地物的外观、位置、高度等属性。

模型有两种成果数据：一种是单体对象化的模型，一种是非单体化的模型数据。单体化的模型成果数据，单独的建筑物可以删除、修改及替换，其纹理也可以修改。非单体化的模型数据采用全自动化的方式生成，经过匀光匀色等步骤，通过图扑软件生成轻量化三维模型。经过多视角影像的几何校正、联合平差等处理流程，可运算生成基于影像的超高密度点云，点云构建 TIN 模型，并以此生成基于影像纹理的高分辨率倾斜摄影三维模型。

图扑软件倾斜摄影模型流畅的三维体验满足了旅游、景区等行业应用。单体化操作与表达，为应急指挥、国土安全、城市管理、房产税收等行业应用提供了基础平台；还可以进行包括高度、长度、面积、角度、坡度等的量测，应用于水利、能源开采等管理系统。

展馆概况

展馆按专业性可分为艺术收藏、绘画书法、艺术设计、摄影、卡通动漫等不同类型的主题展馆。图扑智慧展馆是集展览、会议、办公及商业服务等功能于一体的会展综合体。所有展厅之间在空间关系及人流、物流关系上相互连通、紧密联系，既是一个整体又能分割成相对独立的空间。

图扑智慧展馆的首个场景展示展馆容积率、园区总投资、建筑面积、当前在园人数、总车位、展区分类和配套服务点；采用动态柱状图显示噪音监测结果；让人们对园区有大概的了解。标注核酸检测点位，便于我们进行日常核酸检测。

结合 GIS 将 2D 标签与实际地理位置相匹配，在 3D 场景中标示不同类型展馆的位置并显示该展馆的入住率、建筑面积。

图扑软件既可整合 SuperMap、ArcGIS 来实现空间位置标示，也可在 HT for Web 基础上实现轻量快捷的 GIS 解决方案，降低了实施成本也缩短了项目实施周期。HT for WebGIS 产品支持多种类型地图服务、航拍倾斜摄影实景的 3DTiles 格式数据以及城市建筑群等不同的 GIS 数据的加载。

人车流线

展馆设计既要考虑全馆开放时人流的组织，又要考虑局部开放时不影响其他未开放区域的管理。人流线、车流线是内部路网的交通组织，可直观反映场地内的人群流动与集散。

流线设计前应充分考虑展馆地形、周边交通、出入口、功能区、景观节点、人群活动等因素，将一定的空间组织串联起来，通过流线设计划分出不同的功能区域，达到分割空间，体现设计理念等的目的。流线组织的灵活性直接影响到建筑布局及出入口的设置。

建筑流线设计规则

医疗建筑：门诊病人流线、急诊病人流线、住院病人流线、各种辅助治疗和供应服务流线。

展览建筑：观众参观流线、展品运输流线、内部工作人员流线。

体育建筑：观众流线、运动员流线、管理服务人员流线、贵宾首长流线。

图扑智慧展馆采用一个交通枢纽，把几个陈列室连接起来，参观路线既连贯也具有灵活性，可中断开放任一个陈列室而不影响其他展馆的参观。

根据图扑软件自主研发引擎渲染的动态流线，可快速找到展馆的出入口。人流线、车流线与逃生点位置可结合查看，统一规划馆内人员的逃生路线，在事故发生时让馆内人员有序撤离，缩短逃生时间，避免发生拥堵或者踩踏。

电子围栏

电子围栏是主动入侵防御围栏，可对入侵企图做出反击，并把入侵信号发送到图扑可视化系统，以保证管理人员能及时了解报警区域的情况，快速派遣安保人员处理。如今电子围栏普遍应用于园区、展馆围栏和农业围栏等，也用于加强诸如军事设施，监狱和其他安全敏感地点等敏感地区的安全。

智慧安防

安防主要包括闸机系统、监控系统、红外周界报警系统、防盗报警系统、楼宇对讲系统等为了生命、财产、环境、信息的安全防护而协同工作的一系列体系。在图扑可视化系统能查看监控分布情况和使用情况。

图扑软件可视化系统能集成多路视频，统一查看。在展馆的 3D 态势地图上标注摄像头对象并关联其视频信号源，通过场景交互来调取相应监控视频，满足运维人员对场景进行实时态势感知、历史数据回溯比对、应急处理预案等监测需求。通过查看展馆内部和周边摄像头影像，可判断展馆内外的道路拥堵情况。

交通态势

通过圈层态势分析、车辆监测、拥堵路段监测、停车监控等维度统计车流情况、路况态势并上传至图扑可视化系统，做好车辆引导、交通事故和坏车的快速处置，保证展馆周边的交通顺畅。

将展馆周边的违章点位随电子票发送给参观者，提醒参观者遵守交通规则，避免产生罚单。

智慧停车场

将车位数据和停车场监控影像接入图扑智慧展馆管控平台，结合馆内广播系统引导车辆有序停放，提高车位使用率。

使用预约车位、ETC 无感支付提升停车效率；用智慧平板锁代替道闸减少车库占用率；完善车位引导标志，从主干道到小巷均设置引导牌；周边单位、小区的停车位分时段开放。将展馆周边的停车资源统一集中到车位云平台，驾驶员可手机查看附近停车场的车位数量。

在图扑智慧展馆解决方案中显示停车场长租车位数量和位置，为进入停车场的车辆做好引导。显示每辆入场车的实时信息，防范非法入侵。

人员热力图

客流感知系统建设需要能够覆盖会展中心运营客流区域达到 90% 以上，包含所有场馆、馆间通道、圆环通道、会议区、登录区等，且主要出入口、人流高密区域、重点监控保障区域覆盖率达 100%。将客流感知系统采集的数据接入图扑软件可视化系统，统计全馆总人数、单馆总人数，实时在馆人数、在馆人流密集情况、人流在指定查询区域内的驻留时间等相关信息。

根据全馆客流情况，呈现人员热力图，帮助运维了解各区域的客流分布。人员热力图颜色使用由浅至深的色阶变化来表示客流由疏到密的变化，且人员热力图内有自动告警功能，当某个区域客流超过设定阈值，系统能够自动告警。运维及时通知现场负责人协调警力进行人员疏散，负责人可提前预判危险并指导行动。运维人员可对告警区域持续监控，降低现场人员高密所带来的风险。

数字管网

地下管网是保障展馆正常运行的“生命线”和“主动脉”，通过图扑智慧展馆解决方案实现管线集中管理，让管线敷设和管理系统化。

展馆 3D 场景可查看展馆 360° 全景，采用图扑特殊材质贴图体现展馆高级感，并用爆炸效果切换到线框模式，进行内部结构的查看。燃气、电力、水力使用不同颜色的动态线条标示，可结合每个点位的地理标签查看。

图扑软件 BIM 可视化解决方案提供了模型预处理阶段和模型加载展示阶段的轻量化方法。通过模型快照、参数化模型、BIM 组件过滤、模型三维信息智能分析功能，大幅降低模型的三角面数。通过 BIM 转 OBJ 功能和加载后通过 API 改变颜色、更换贴图功能，实现 BIM 模型的外观优化，根据应用系统采集数据实现状态展示。依托于 HT 强大的 WebGL 3D 引擎，可以用于开发各种 BIM 可视化场景，比如变电站可视化，机房可视化，工厂可视化等。

燃气管网

燃气管网是指自门站至用户的全部设施构成的系统，由门站或气源厂压缩机站、储气设施、调压装置、输配管道、计量装置、管理设施、监控系统等组成。可按用途、敷设方式、输气压力、管网形状、压力级制等加以分类。

利用图扑软件智慧展馆解决方案实现管网长度、辐射户数、换热站数量、供热面积、累计能耗、供热量、用水量、用电量、平均室温的统计，挖掘数据背后的价值，提升燃气利用率。红黄相间的动态线条显示了整个展览馆的燃气管道分布。当某段管道异常时，可结合 GIS 快速定位问题点，及时派遣人员维修。

配水管网

自水厂出来的管道称配水管网，配水管网中主要起输水作用的管道称干管，从干管分出起配水作用的管道称支管。管网的布置有枝状和环状两种基本形式。图扑智慧展览馆的管道纵横相互接通形成环状，并采用蓝绿相间的动态线条显示。为了保证供水管网的正常运行、消防和维护管理，管网上须装设各种必要的附件，例如在适当的位置安装消火栓、阀门，在管道高处装设排气阀，在管道低处设置排水阀等。可利用传感器收集附件运行数据上传图扑可视化系统，及时发现问题。

电力管道

通过图扑软件可视化系统梳理展馆电力设施现状，并对馆内历史及现状负荷特性进行研究，通过电网结构和负荷特性分析，提出馆内电网存在问题。在供电部门配网规划基础上，确定不同路段配网路由和通道需求，梳理现状电缆通道容量，预测规划需求容量，提出通道建设类型（综合管廊、电力管沟、排管或直埋等）和建设控制要求。根据绿黄相间的动态管道图，预测最佳的通道建设类型和路径，降低试错成本。

电力管道的建设情况直接影响了电压合格率、供电可靠性等质量指标；发电量、售电量、用电量等经济指标。电从发电厂到用户传输过程中损耗一般在 6%-10%，合理的管道敷设能提高电能利用率，减少电力传输损耗。将展馆当前电力负荷和电力使用峰谷值接入图扑可视化系统，并统计本地电厂和外受电比率，根据电力使用规律，运用虚拟电厂等策略实现削峰填谷。

利用图扑可视化系统统计水电、光伏、煤电、风电等保电资源类型和装机容量，掌握园区能量系统，提高能源管控效率。保障重点展厅的用电量，避免重要展会时停电。建立可再生能源电力消纳体系，减少煤电的投入，助力碳达峰和碳中和目标的实现。

通过图扑软件完整复现园区能量系统，实现分布式光伏发电系统、储能系统、太阳能+空气源热泵热水系统的综合管控。通过智慧能源管理系统，实现建筑能效管理、综合节能管理和“源网荷储”协同运行。

设备设施

电能是展览馆运行的重要能源，照明、安防、消防等设备的运转都离不开电。电也让图扑智慧展馆集中管控平台有效运行，及时发现设备故障信息，快速进行维修。

在智慧展馆的 Web 三维场景中，点击底部菜单的光照设备可模拟灯光开启效果。可通过远程控制开关在手机上对光照设备进行开关的控制，不受距离限制，不受气候因素的影响。

点击闸机按钮可拉近镜头查看闸机分布和编号；采用线框模式复现空调外机、净水器、阀门、温度感温器、水泵，点击单个设备可放大查看运行状况；点击消防栓按钮可查看消防栓的分布位置，遇到火灾时可及时调配，快速灭火。

空调机房

空调外机可将室内机排出的高压、高温气体，在室外通过风扇降温散热。冷凝后的制冷剂液体，再通过毛细管送到室内机的蒸发器中，进而蒸发变成气体以吸收室内热量。

空调集分水器起到向各分路分配水流量的作用，集水器起到由各分路、环路汇集水流量的作用。可连接各个水环路的并联管道，起到均压作用。

冷却塔是用水作为循环冷却剂，采用蒸发散热、对流传热和辐射传热等原理来散去制冷空调中产生的余热来降低水温的蒸发散热装置，以保证系统的正常运行。

在图扑智慧展馆解决方案中，以实时数据驱动的方式 24 小时监控以上设备的运行情况，发现异常及时处理。

水泵监控

水泵是输送液体或使液体增压的机械。它将原动机的机械能或其他外部能量传送给液体，使液体能量增加，主要用来输送水、油、酸碱液、乳化液、悬乳液和液态金属等。根据不同的工作原理可分为容积水泵、叶片泵等类型。在 3D 场景内结合 GIS 技术显示水泵分布，出现问题时让运维根据可视化信息迅速定位故障水泵位置，并指派工人进行维修。

智能预警

为消防设备、安防设备、电梯、空调等设置阈值，设备出现异常时及时告警，并在图扑可视化系统显示设备名称和位置。快速查找事故设备，避免火灾、电梯困人、非法入侵、二氧化碳浓度超标等事故的发生。在发生火灾及其它紧急情况时，可将公共广播系统与会展中心消防联动，发出火灾紧急广播，通知被困人员有序撤离。

当有设备在维修时可通过工单系统查看待处理工单和维修进度。

智慧展馆应用了先进的数字化多媒体产品及设备，实现对全馆的态势感知，对设备的实时监控，用数字化手段保障展馆的正常运转。同时，采用云端计算、5G、边缘计算的技术打造智慧中控系统及更便利、直观的大数据中心。

现代元宇宙展馆还涉及到智慧数字沙盘、全息投影、弧幕/球幕/环幕投影、AR 增强现实、VR 虚拟现实、裸眼 3D 等技术及产品。通过对场景或物体进行真实的“三维重现”，融合 720 度自由全景照片、遥感影像、3D 模型、3D 立体投影等多项虚拟现实技术和多媒体技术，实现场景升级再现。
