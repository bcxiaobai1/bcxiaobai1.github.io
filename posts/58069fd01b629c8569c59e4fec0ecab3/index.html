什么是SolRazr

网址https://solrazr.com
https://twitter.com/Solrazr_App

SolRazr is the first decentralised developer ecosystem for Solana
Launchpad | Accelerator | Developer Tools

Solana生态不断发展，作为Solana的第一个去中心化的开发者系统—SolRazr，为项目提供Launchpad、加速器和开发者工具。

作为一个区块链项目的创建者，既想使用区块链技术及现成的开发工具，并迅速融资，还希望享受多线程的交易并行处理系统，以及得到公平的利益分配，有这么完美的解决方案吗？SolRazr就是这样一个平台。

SolRazr已被开发成一个真正意义上的筹款和开发者平台，旨在服务那些在Solana上研发软件解决方案的项目。SolRazr将专注于提供一个高性能的网络来加速去中心化金融（DeFi）生态系统的发展。

SolRazr为 Solana 上一个智能的、去中心化的筹款（IDO）平台。借助 SolRazr这个IDO平台，开发人员可以以闪电般的速度筹集资金，促进Solana上生态发展。

SolRazr 的一些主要功能和优势

  • 可交易分配
    SolRazr通过利用 Solana 上 NFT 的力量重构代币销售白名单和分配。社区将参与平台上的交互和工作流程，铸造带有白名单和分配金额的 NFT。代币销售计划将检查 NFT 的所有权而不是 Solana 地址，这意味着社区成员可以将其白名单和分配的所有权转让给他们的朋友和家人。

  • 开发者工具
    SolRazr为刚接触区块链和 Solana 的开发人员，提供无需编写代码即可部署程序和基础设施的能力，帮助开发人员在 Solana 上快速学习和构建，帮助生态系统更快地发展。

  • 加速器
    SolRazr在 Accelerator (加速器)计划下 - SolRazr可以孵化项目、帮助好的应用上市并提供工具，而参与者可以获得各种好处，包括技术指导、资助机会和市场曝光。

  • 跨链池
    SolRazr 利用 Solana 虫洞提供跨链池，允许来自其它链的社区直接投资 Solana 项目。Solana Wormhole（虫洞） 是以太坊和 Solana 之间的双向去中心化 ERC-20 ⇄ SPL 代币桥梁。它将使现有项目、平台和社区能够跨区块链无缝移动代币化资产，从而从 Solana 的高速和低成本中受益。

背后的投资机构

SolRazr 的资本支持：有来自 Moonrock Capital、Ascensive Assets 和 Morningstar Ventures 的 150 万美元融资。

其他知名基金：包括Genesis Block Ventures、Divergence、Genblock Capital、CMS Holdings、ZBS Capital、PANONY和Skynet Trading。

天使投资人：来自Frax Finance的Sam Kazemian、来自CREAM的Leo Cheng、Santiago R Santos和Chris McCann等。
