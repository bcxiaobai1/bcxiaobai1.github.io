GitHub 官宣：弃用 trending 热榜，开发者炸锅了

trending榜单要弃用？

大家好，我是二哥呀。

昨天正在小破站上开心地刷《玫瑰叔品冰火》，嗯，一直刷道今天早上，发现真的是一个宝藏 up。

然后就收到好朋友宏哥发来的消息说，GitHub 要下架 trending 榜单了，
这一消息真让我大吃一惊啊，还没反应过来，GitHub 就为 Trending 榜单开启倒计时 30 天的模式了，30 天后就废弃了？

第一时间涌入我脑海的反应就是这些：“这怎么可能？为什么？trending 热榜挺好用的啊？为什么要下架？难道 GitHub 以后也要做推荐流？”

不知道大家有没有同感啊！

然后我今天早上登录 GitHub 再去查看 GitHub trending 热榜菜单的时候，发现：

Heads up! This Trending tab is being deprecated. Due to low usage of Trending Repositories and Trending Developers, this tab will no longer be available beginning September 30, 2022. For questions and feedback, please visit GitHub Community.

这段提示消失了？

难道说迫于压力，微软手软不砍了？

一开始还以为是浏览器缓存，我特意开了 Chrome 的无痕模式上去确认了一下，确实没有这段提示了。

到底搞什么鬼？

像微软这样体量的公司，尤其是 GitHub 这样明星级别的产品，针对一个大家耳熟能详的功能，说砍就砍，然后又突然“不砍”，不是很理解啊！

为了确认这件事，我在搜索引擎里输入 GitHub trending deprecated 关键字，就搜到了这条结果。

回复者 martinwoodward 何许人呢？一看简介大家就明白了，GitHub 的重量级人物！

来看看他是怎么回复的。

大致的意思就是，hi 伙计们，得到你们关于 trending 榜单的反馈了，鉴于你们的热情（高涨地反抗情绪），我们决定重新评估是否要弃用 trending 热榜这个计划，看看是否有其他的解决方案。

在我们搞清楚之前，暂时先去掉 trending 榜单上的警告⚠️信息。不过，有一点可以确定的是，我们的大数据显示，tending 榜单的使用率确实越来越低了，这也是我们之所以要关闭 trending 榜单的主要原因（你们不要质疑我们的动机，我们是有理有据的）。

总之，就是这么一个大概的意思。至于 trending 榜单能不能保得住，我想即便是能保住，上榜的机制（算法）也会大改，不会再像以前那样。

trending榜单的使用体验

就我个人而言，GitHub trending 榜单的使用频率还是非常高的，因为有时候确实想上去看看有什么比较有意思的新鲜项目，尤其是当自己的仓库也出现在榜单的时候，心情真的非常激动。

issue 区榜一大哥简简单单的一句话“100% for me too. I use it everyday to discover new interesting projects”也得到了非常多用户的声援。

还有这张动图应该也能反映出大部分 GitHub 用户在收到 trending 榜单要弃用时的心情。

不过，这份 trending 榜单确实也存在着一些问题，就是有些仓库会长期霸占这份名单，而有些仓库鲜有露头的机会。

毕竟用户量大的仓库，肯定是更有机会上这份榜单，而一些用户量小的新开源的仓库想要上来，就颇为不易。

不过，也有用户是支持 GitHub 弃用 trending 榜单的，因为有些仓库的作者利用这种玩法来引流，导致榜单上没有几个正经的项目：

trending榜单的优化

当然了，作为 GitHub 的一个普通用户，我们没有权力去确定 trending 榜单到底要不要被弃用，但是有一点是可以肯定的，就是微软绝对会对 trending 榜单做一个新的优化，无论最终是以什么样的心态出现。

其实 GitHub 除了 trending 榜单，也在做一些其他方面的推荐流，比如说首页的 for you（beta 中）。

点到 for you 标签页，就能发现你关注的一些仓库一些用户在 starred、forked 什么，包括你自己可能感兴趣的。

包括用户给你仓库的留言等等。

trending榜单等我们

不知道小伙伴们用 GitHub trending 榜单的多不多，反正 GitHub 确实一个卧虎藏龙的好地方，里面几乎能找到你想要的所有东西。

二哥最近也在不断优化《Java 程序员进阶之路》开源仓库里面的内容，比如说集合框架、Java IO、异常处理、Netty、辅助工具、开源轮子等。

和三个大厂朋友楼仔、一灰、小超一起准备的开源项目也在马不停蹄地准备着。大家可以看一下时间，两个小时前还在commit 呢。

也希望我们新的开源项目，能在未来上一次 GitHub trending 榜单呢
