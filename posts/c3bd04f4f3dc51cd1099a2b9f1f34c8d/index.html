一、前言

xLog是一个基于Crossbell区块链的博客解决方案，专注于Web3数据由用户掌控。Crossbell是一个基于Web3技术的去中心化博客平台，用户可以在该平台上发布文章并进行交流和创作。社区提供多种交流平台和有奖创作活动。

xLog是基于 Crossbell 区块链的一个应用产品。底层技术框架是 以太坊（Ethereum）、星际文件系统（IPFS）、专为社交内容创作打造的智能合约（Smart Contract）等。

特点：

  • 支持Web3钱包和邮箱注册
  • 内容存储在区块链上，完全去中心化

  • 可自定义域名
  • 功能丰富的后台管理
  • 支持markdown文件的导入和导出
  • 文章页拥有AI内容总结

二、正文

准备一个Web3钱包（Wallet）OR 准备一个邮箱（Email）

Web3钱包是你在 Web3 的唯一证明。 但别以为钱包里的就只是钱，在钱包里，有的还有你的数据，你的身份证明。所以，在进行注册的时候，请一定要记住钱包的助记词又或是私钥，它们是唯一能证明博客（钱包）的唯一证明，丢失了就找不回来了。xLog支持多种加密钱包的链接，大家可以选择适合自己的钱包进行连接和注册。

当然，有的朋友不熟悉区块链，又或是认为区块链太复杂太高深莫测了。不用担心，xLog同时支持邮箱的注册，登录时也可以账号（邮箱）+密码进行登录。只需要点击上述图片中下方的 I don’t have a wallet 即可进行注册和登录。

虽然最终还是需要绑定一个钱包地址（当然注册一个钱包并不会很困难，大家只需要在搜索引擎中搜索即可，十分的简单，此处就不多介绍了），但邮箱可以当做登录方式，因为在区块链上，你的每一次动作，比如发币文章等，都需要消耗$CSB，也可以比喻成钱（在下文中有说明），消耗钱，类似于支付宝微信，就需要在钱包中输入密码确定，并且，领取$CSB，也需要生成一个钱包地址才能完成，所以，钱包，是一个去中心化区块链博客必不可少的东西。

领取 $CSB

正如在现实中活动需要金钱，在区块链上进行活动（交易）需要手续费。Crossbell 区块链的手续费消耗的是 $CSB。

一个好消息是 $CSB 目前是可以免费领取一小笔的。点击后我们跳转到了 Faucet（水龙头）页面。你可以把水龙头理解为「一个救济余额不足用户的基金会」。目前该水龙头会给每个新人用户赠送 0.02 $CSB，花光后可以再领。

我们跟随页面上的指引，输入以太坊地址（为了防止滥用可能需要发送推文来验证），然后即可领取 $CSB。

三、额外介绍

1.可自定义域名

这个博客是可以自定义域名的，和GITHUB等一众网站一样，通过CNAME和TXT完成自定义域名的实现

2.基础功能已然完善的后台管理

作为一个博客系统，基础功能是最重要实现的一大板块。

xLog 提供双栏编辑器，具有实时预览功能，提供优秀的写作体验，使用标准的 Markdown 语法，支持 HTML、音视频和数学表达式。页面设计优雅，为读者提供舒适的阅读体验。具有订阅、评论、RSS 订阅和 AI 增强等功能，仅需 5 分钟即可完成。

3.安全、自由与快速

所有博客数据，包括配置、文章、订阅、评论等，都由你自己签名并安全地存储在区块链上，只有你持有的私钥才能进行控制和访问。包括 xLog 在内的任何人都无法进行更改。这保证了文章的内容无法被审核、删改与下架。

xLog提供导入和导出工具，可一键导出所有的内容，同时也提供了丰富的 API 和第三方集成。所有代码都在 GitHub 上开源，所有数据都在区块链上透明。xLog 没有任何隐瞒。

xLog通过其高效的缓存机制和众多优化，可以达到最佳性能。站点还支持PWA用于本地安装和使用。
