是个人都会，让你免费搭建个人博客，还去中心化

前言：

也许你现在没有自己的博客，但想在有必要的时候展示自我，记录身边的生活。也许你已经搭建了自己的博客，但搭建在中心化服务器上经常访问很慢，容易宕机。那么4everland可能是你的不错选择，不用花钱，访问速度快还送域名。本文就是为了帮助想拥有自己博客的新朋友搭建一套去中心化的博客或者帮原博主构建一个备站。 

介绍：

4everland: 4everland提供了一个云平台，可以优化整个项目开发和部署体验，它有很强大的功能值得去探索，个人使用是免费的，提供了域名访问，使用方便快捷。

hexo: Hexo 是一个基于 nodejs 的静态博客网站生成器，通过使用脚手架安装后，命令操作简单，直接开箱使用，支持丰富的主题，支持高度的自定义化，主要使用 markdown 语法。你可以自行开发插件，优化你的博客。 

 

搭建

在搭建博客之前，我们需要一些准备工作，首先是 4everland平台的账号，其次是安装 nodejs 以及 hexo-cli 脚手架。

先找到官网，选择Hosting

连接你的GitHub账户

创建一个新项目

安装 hexo-cli

在安装 hexo-cli 前，需要保证电脑安装了 nodejs。 nodejs 需要在node 官网下载，安装好 nodejs 后，也会相应的安装上 npm，接下来就可以安装 hexo-cli 了。

安装命令：

npm install hexo-cli -g

检测是否安装成功。

4everland + hexo 创建项目

4everland平台中支持选择多种项目模版，包括但不限于 Next.js，Nuxt.js，Hexo，Angular 等多种类型。

由于目前项目方没有提供官方的模板，需要自行去github导入git或者将本地git上传到github再通过4EVERLAND部署。

下面介绍如何将本地代码放到github上。

在确认已经安装hexo与Git的前提下初始化hexo

$ hexo init Blog

上述命令会创建一个名为Blog的文件夹，博客初始化的所有内容都将存放在这个文件夹中

接下来切换到Blog目录下

$ cd Blog

$ npm install

执行如下命令，生成静态页面

$ hexo generate

(或者直接hexo g)

安装hexo-server包

$ npm install hexo-server –save

浏览器输入https://localhost:4000,就能看到本地hexo的helloworld页面

如上，本地已经能跑起来了。

这时，只要再github里创建一个项目再将本地代码克隆到git上基本的准备工作就已经完成了

最后你只需要在4everland里登录github账户进行简单的部署就可以发现你的博客已经完美的运行在IPFS去中心化网络上。
