五、python求逆矩阵/伴随矩阵

逆矩阵的定义：

设A是数域上的一个n阶方阵，若在相同数域上存在另一个n阶矩阵B，使得： AB=BA=E。 则我们称B是A的逆矩阵，而A则被称为可逆矩阵。当矩阵A的行列式|A|不等于0时才存在可逆矩阵。  

1）创建一个方阵

A = np.array([[1, -2, 1], [0, 2, -1], [1, 1, -2]])

2）使用 linalg.det求得方阵的行列式

A_abs = np.linalg.det(A)

 3） 使用 linalg.inv 求得方阵A的逆矩阵

B = np.linalg.inv(A)

4）利用公式求伴随矩阵：

A_bansui = B * A_abs
