2021科技行业三大“神话”：元宇宙、NFT、Web3

科技以画饼为先，先造词、再造梦。

  • 苟日新、日日新、又日新，人类自第一次工业革命以来，科技发展便进入了快车道，快到了什么地步？大名鼎鼎的摩尔定律都已根本上半导体行业的发展速度了。
  • 也正因科技行业技术更迭的速度越来越快，所以每年涌现出了大量的风口与追风者（科技从业者、媒体与投资人），以至于许多未取得突破性进展的技术与概念，通过新瓶装旧酒的形式被抬上了神坛，那么即将过去的2021年科技行业出现过哪些被“神话”的科技词汇呢？

元宇宙把我们拒之门外

  • 首先提到被“神话”的科技词汇，想必大家与我一样，都会把“元宇宙”这个词给排在第一位。
  • 元宇宙被神话到了什么程度了？据悉，全球头部投资机构KKR就曾表示：未来科技巨头发展只有两条路可走，要么是融入元宇宙，要么是发展虚拟货币。由此可见元宇宙在“追风者”心中的分量。
  • 为什么大家这么看好元宇宙？元宇宙有没有被夸大？在回答这些问题之前，我们还是得重复解释一下，元宇宙就是是什么？从比较官方的解释来看，元宇宙一词源于科幻小说《雪崩》中提到的“metaverse（元宇宙）”和“Avatar（化身）”这两个概念，人们在“Metaverse”里可以拥有自己的虚拟替身，而这个虚拟的世界就叫做“元宇宙”。通俗的来说，就是能够在里面工作、社交、娱乐的虚拟系统，所以科技公司都想近水楼台先得月，成为元宇宙中的“世界政府”。
  • Facebook不仅加快了对硬件（Oculus眼镜、触觉手套）、软件（虚拟办公室Horizon Workrooms）的布局，甚至为了元宇宙把公司名字都给改成了Meta。除了Facebook外，微软、谷歌与中国的腾讯、阿里巴巴、百度、网易、字节跳动等科技巨头均加强了在元宇宙的布局，例如百度近期便推出了中国首个元宇宙平台——希壤，不过从完成度与可玩性、实用性来说，与Facebook的Horizon平台都有较大差距（都很粗糙），并且包括Facebook在内的一众科技公司在内，高喊元宇宙一年了也未拿出符合元宇宙概念的产品出来，但元宇宙今年已经卷走了数百亿美元资金，其中仅中国9月到11月期间，元宇宙便拿到了63笔融资（每笔融资规模超千万）。
  • 其实，元宇宙搭建是一个系统性的工程，它既需要以VR、AR、XR为代表的交互技术的突破，也需要区块链技术、物联网技术、网络及运算技术、人工智能技术、电子游戏技术能够对现有技术进行革新，但即便是目前最新的5G网络也难以承载电影《头号玩家》中那样沉浸式的元宇宙体验，可以说我们现在连元宇宙的门都没有摸到。

NFT已举起了手握镰刀的手

  • 如果说元宇宙是用今天的钱帮助大众完成未来梦的话，那么与元宇宙一样火爆的NFT便是用多数人的钱帮助少数人去完梦。
  • 其实，即便是NFT已经出圈了（公司的前台都会问一句NFT），但大家对NFT究竟是什么还是云里雾里的。NFT，即Non-Fungible Token，中文译名为非同质化代币，说白了就是类似于比特币一样的一种特殊的数字资产，只不过他是非同质化代币。
  • 在区块链领域，有许多类型的代币：例如以太坊的ETH、比特币、币安的BNB等。众多代币又可以分为同质化代币（如比特币、以太坊等）和非同质化代币（如NFT）两类。NFT与同质化代币（如比特币、以太坊）的区别主要有两点：第一，每一个NFT都拥有独特且唯一的标识，不可两两互换，但是常见的其他代币（比特币、以太坊等）都是同质化的，即每个代币之间没有任何区别，可以互换；第二，由于每个NFT的非同质化以及唯一性，因此NFT的最小单位是1，并且不可分割，但是常见的其他代币是可分割的。
  • 与大家对NFT是什么一样不了解的是，其实NFT早在2017年便有了第一个项目——CryptoKitties，而此后几年虽然NFT项目也有不少，如OpenSea、SuperRare等交易平台出现，耐克、F1也进入这个市场，但始终处于一个不温不火的状态，可为什么在2021年却突然就火了呢？
  • 原因不是因为NFT改变了艺术品、音乐等作品的交易方式，加强了对数字作品产权的保护，而是因为随着比特币等虚拟货币在全球遇冷（很多国家都不让炒币），币圈需要一个新的风口，于是NFT便成为了新风口，所以NFT造富的故事层出不穷，因为它需要炒起来。

Web 3又在给我们画大饼

  • 与NFT一起被炒起来的还有刚刚被路透社评为今年年度科技词汇的“Web 3”。
  • Web 3通常来说是指一个支付和资金为数字原生的互联网、一个”去中心化“应用程序与中心化应用程序竞争的互联网、一个用户能对自己的身份和数据有更多控制权的互联网，简单来说就是去中心化的互联网。
  • Web 3是互联网很快到来的下一个进化形态，它基于区块链而展开、具有去中心化的特性。它的出现，不仅会彻底打破谷歌、Meta、Twitter等科技巨头对互联网的垄断，还会催生一系列新的科技形态，与元宇宙、加密货币等新兴概念都息息相关。
  • 但是对于这样一个伊甸园般的概念，宇宙网红马斯克与推特创始人Jack Dorsey却出人意料给泼了一盆冷水。他们认为，尽管这些风投们宣扬Web3 为普通大众所有，但实际上他们早就盘算好了如何利用这个概念从普通人那赚大把的钱，最终还是会成了一个带有不同标签的中心化实体，就像是最早的博客一样。
  • 虽然Jack Dorsey的言语过激，但是也并无道理。就拿当前与Web 3理念最接近的以太坊来说，已经建立起数量繁多的去中心化应用，包括Uniswap、Compound等，用户们使用这些应用程序无需注册、认证。但是以太坊网络应用更多还停留在金融这个单一领域，并且它的许多应用也没那么去中心化，频繁发生的项目方监守自盗案便是一个例子。
  • 其实与元宇宙类似，Web 3也是一个多年前的概念，由时任以太坊CTO的Gavin在2014年提出，其核心就是去中心化，但是今年我们还基本看不到。正如马斯克所言，“现阶段，Web3只是市场营销术语，有人看过web3 吗? 我没有找到。但是我们非常想知道10年、20年、30年以后的世界会是什么样子的。
