电脑上的垃圾软件太多怎么办？教你两招轻松清理

大多数 Windows 10 计算机都安装了Windows防御程序并在后台运行。这可以防止大多数恶意软件或病毒，使他们没有机会感染您的 电脑。 但是，有的电脑可能没有正确更新或安装Windows防御程序导致的保护不足。 不管是什么原因，我们仍可以手动清理这些恶意程序。

1.用杀毒软件来删除恶意软件或病毒

要清洁您的电脑，就必须安装并运行Windows防御程序并执行扫描。 对于大多数人来说，Windows防御程序已经够用了。但是，如果您有非常顽固的病毒，您可能需要在安全模式下启动您的 电脑。

2.如何使用 Windows 防御程序扫描您的 PC

大多数情况下，Windows 防御程序会在威胁成为问题之前消除它们。 如果是这样，您可以在 Windows 安全应用程序中查看过去的威胁报告。

  1. 打开 Windows 安全
  2. 打开 Windows 设置 。 最简单的方法是单击开始按钮，然后单击齿轮图标。 或者，您可以按键盘上的 Windows 键 + i。
  3. 点击 更新和安全
  4. 从这里，您可以查看您的 PC 在 选项卡下是否有任何可用更新 Windows 更新 。 如果可用，您还可以在此处查看 Windows 防御程序的定义更新。
  5. 选择 Windows 安全性 ，然后单击页面顶部标有“ 的按钮 打开 Windows 安全性” 。
  6. 选择 病毒和威胁防护
  7. 如果您想节省时间或者您的开始菜单无法正常工作，您可以使用键盘上的 Windows 键 + R 打开运行对话框并输入“ windowsdefender ”，然后按 Enter。 ”，然后按 Enter。
  8. 从“ 保护和病毒”页面 ，您可以查看最近扫描的一些统计信息，包括最近的扫描类型以及是否发现了任何威胁。 如果存在威胁，您可以选择 链接 保护历史记录 以查看最近的活动。

如果您的恶意软件或病毒问题没有得到解决，可以运行更深入的扫描。

  1. 选择扫描选项开始。
  2. 选择旁边的单选按钮(小圆圈) Windows Defender 脱机扫描以便记住扫描，此选项将需要大约 15 分钟，并会提示您的 PC 重新启动。 在继续之前，请务必保存好你的文件。
  3. 立即点击扫描。
  4. 离线扫描的要点是您的 PC 不需要完全启动到 Windows。 如果系统文件出现问题或系统运行时无法隔离或删除的文件，则可以在离线模式下完成。
  5. 您还可以运行全面扫描 ，它还将检查驱动器上的每个文件。但是，这将需要一段时间，具体取决于您机器的速度。和以前一样，检查页面保护历史记录以查看是否发现了任何威胁。
