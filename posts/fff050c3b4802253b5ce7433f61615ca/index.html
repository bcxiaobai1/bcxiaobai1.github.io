mac charles 安装教程、使用教程

安装教程

1、mac版本的charles安装包下载

mac最新版本charles安装包dmg-其它文档类资源-CSDN下载

2、安装charles证书并进信任 

mac上下载并安装证书 Help->SSL Proxying-> Install Chrales Root Certificate

下载完成charles证书之后，在证书列表中搜索charles，双击打开，修改信任为使用信任

 

3、charles进行相关配置

第一次安装charles安装需要做一下相关配置

a、Proxy->SSL Proxying Settings

选中Enable SSL Prxying 点击add 

 b、Proxy->macOS Proxy选中

c、Proxy->Proxy Setting

 d、Charles->Preferences-Viewers

去掉勾选

 PC端抓包

PC端抓包无需在进行其他的设置了

APP端抓包

1、需要在端侧安装charles

  • help-> ssl Proxing -> Install on mobil

 

  • 设置后会弹出下面的弹出框，上面有手机需要设置的代理ip和端口

 2、手机设置wifi

  • 1.手机打开wifi设置 ，点击感叹号
  • 2.进入wifi设置后。拉倒底部，点击http代理设置
  • 3.进入后点手动，设置上面电脑端的ip和端口

3、手机下载安装证书 

  • 1.证书地址 chls.pro/ssl 在手机浏览器点击下载
  • 2.手机设置->通用->关于本机->证书信任

现在就可以进行pc端和APP端的抓包了
