什么是 NFT，它的使用情况如何？

Sept. 2022, Vincy

Data Source: Footprint Analytics - NFT Market Overview


 

NFT 代表 “非同质化 token”——独特的、可验证和不可变的数字商品所有权证明。与加密货币不同，NFTs 是唯一的和不可分割的。即：1 BTC = 0.5 BTC + 0.5，1 NFT ≠ 0.5 NFT + 0.5 NFT。



 

 

NFT 的不可逆性使其可以标记资产类别和编号，买家也可以通过 NFT 证明他们对资产的所有权。

最常见的五种 NFT 集合类型是：
 

  • 收藏品

  • 艺术

  • 游戏

  • 元宇宙

  • 实用程序

NFT 的 3 个主要协议标准是什么？

ERC-20 协议是以太坊上最早的、比较流行的 token 规格协议，是同质化 token 的协议标准。倘若以太坊平台上两种 token 都以 ERC-20 发行，好比 BTC 和 ETH 这两者，它们之间可以进行自由置换。正因为这样，同质化 token 几乎不存在稀缺性的说法。

从发展到现在，大部分的 NFT 也是在以太坊上创建的，主要形成三大标准：


 

  • ERC-721
     

ERC-721 标准由 CryptoKitties 的 CTO Dieter Shirley 创建并发布，他是第一批 NFT 创建者之一。

NFT 项目通常采用 ERC-721 的标准来签发 NFT。ERC-721与 ERC-20 具有相似的属性——它们都在同一个区块链上存储数字资产信息，但不同的是 ERC-721 具有唯一性。

ERC-721 可以是一张图片、一栋房子、一辆汽车，或是其他任何东西。它还可以促进追踪、交易和真实资产的交易和管理等。

  • ERC-1155

随着游戏虚拟资产不断普及，只在以太坊生态系统内提供资产是不够的。然而，ERC-1155 标准使资产能与其他生态系统兼容，而跨链兼容性是 ERC-1155 的主要优势。

例如，它允许一个合约中能处理几种类型的 token，包括同质化和非同质化 token，这也大大提升了 token 交易的效率。但它不能直接销毁，除非原始开发人员定期回购 token，否则它们将继续流通。由此带来的好处就是 token 的稀缺性。

  • ERC-998
     

ERC-998 是 ERC-721 标准的扩展，允许创建可合成的 token。例如，在游戏中，一个游戏角色的所有权代表着非同质化 token，而装备由代表另一个 token。如果是 ERC-721 的标准，它们就不能反映出隶属关系。ERC-998 是允许用户将它们合成一个 token，这提高了资产转移的效率。

 

NFTs 是如何工作的？

NFTs 几乎可以是任何的东西——收藏品、艺术品、GIF 和游戏内资产等。它们的价值主要也是由市场和需求决定，因此它们和现实中的实物艺术品一样买卖。

大多数 NFT 都保存在区块链上，这是一个分布式数据库，用于记录和存储有价值的信息。

此外，拥有 NFTs 的人拥有独家所有权，他们可以使用区块链技术来验证自己的所有权以及所有者之间的 token 转移。 创建者可以将特定信息存储在 NFT 的元数据中，以跟踪他们的资产转移。

有哪些 NFT 的使用案例？

NFT 的应用场景非常广泛，俗称 “万物皆可 NFT”。 开发者可以将真实的和可触摸的物体映射到NFT领域，如收藏品、虚拟世界、身份验证、时尚娱乐、摄影、电子门票、音乐、体育、游戏内物品资产、域名和动画等应用场景。

按市值排名，BAYC、CryptoPunk、MAYC 和 Otherdeed 等收藏品类 NFT 项目持续领先。在艺术类项目中，最具代表性的是 Art Blocks Curated。其中，像 CryptoPunk 这样项目不仅是创建时间早，而且是 NFT 领域中最具象征地位的头像。

主要的 NFT 交易市场有哪些？

OpenSea 是目前全球最大的 NFT 交易平台，于 2018 年 1 月成立。它涵盖广泛的 NFT 类别，并整合了各种平台的 NFT。它是目前主流 NFT 开发者和爱好者使用最多的交易平台。


 Footprint Analytics - Total Number of NFT Market Collections by Chain

后起之秀，包括有 Magic Eden、X2Y2、Looksrare 和 SuperRare 等交易平台。

Magic Eden 是 Solana 上运行（虽然扩展到 Ethereum），做了几个优化和功能。其中包括可以选择用信用卡支付，帮助新手快速熟悉加密货币市场，无需事先购买加密货币来支付。另外一个便捷的功能是他们用白名单的列表管理工具，简化了 NFT 创建者的流程，同时将 NFT 铸造与平台锁定。

从过去 90 天交易市场交易量上看，新晋的 X2Y2 于 2 月19 日上线，它以低交易手续费（0.5%）和共享质押奖励等优势吸引用户在其平台进行交易。



 Footprint Analytics - Daily NFT Market Ranked by Volume

SuperRare 是建立在以太坊上不可伪造 ERC-721 NFTs 的点对点市场，并且专注交易稀有的艺术品。艺术家可以拍卖自己的创作，也可以以固定的价格出售。该平台的特点是管理先进、体验简便、付款方式灵活，具有社交档案和移动 APP，可以进行实时拍卖。

小结

随着区块链技术不断的发展，各种类型的 NFT 和交易平台应运而生，实现数字化确权、流转和交易方式，加速数字资产化趋势。
