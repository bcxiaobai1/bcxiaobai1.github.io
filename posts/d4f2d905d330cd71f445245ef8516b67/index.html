字节工程师薪资排世界第五，中位数43万美元，2021全球程序员收入报告出炉

近日，美国科技公司数据收集网站Levels.fyi发布了2021年全球程序员收入报告。

在该报告中，Levels.fyi收集了一整年的数据情况，并根据级别和地点对各公司的薪酬进行了深入分析。

报告中的薪酬数字反映的是程序员在过去一年收入总额的中位数，包括工资、股票和奖金。

该网站成立于2017年，在全球1500多个城市拥有超过10万个数据点，数据相对权威和准确。

在报告中，将程序员分为了五个等级：初级工程师，中级工程师，高级工程师，主管工程师，以及首席工程师。

初中级程序员薪资中位数曝光

在初级工程师排名中，量化巨头公司Two Sigma以25万美元的中位数工资霸榜。

位居其后的依次是线上文件共享以及云端内容管理服务平台BOX、多人游戏平台ROBLOX、支付服务提供商Stripe。

该排行榜中占据前7位的公司，程序员的中位数薪资都超过了20万美元。与去年相比，入榜公司的总体薪资呈大幅提升。

在中级工程师中，多人游戏平台ROBLOX以中位数工资33.5万美元排名第一。

2020年，Airbnb以中位数工资29.5万美元位居榜首，而今年并未出现在排行榜中。

字节跳动程序员薪资上榜

在高级工程师中，字节跳动由去年排名第七位上升至第五位，中位数工资也从去年的40万美元提高至43万美元，职级仍然为2-2。

值得一提的是，在2020年全球程序员收入报告中，字节跳动中级工程师、高级工程师、主管工程师的中位数薪资均在榜上有名，均是排在第七位，而今年只有高级工程师薪资上榜。

在主管工程师排名中，以中位数工资64.6万美元排在全球首位的是Linkedin，去年居首位的是Stripe，中位数工资为59万美元，今年以60.8万美元排列第二。

在首席工程师中，游戏平台ROBLOX以中位数工资120万美元排名第一，其次是Facebook，中位数工资94万美元。谷歌、苹果、微软分别以80万、75万、65万美元排在第四、六、七位。

此外，Levels.fyi网站还根据地域不同对程序员的工资做了统计，同样是以中位数统计的方式。

从地域分布看，美国程序员收入最高的地区分别是旧金山（24万美元）、西雅图（21.5万美元）和纽约（19万美元）。

从全球范围看，程序员收入最高的地区是瑞士苏黎世（20万美元），其次是以色列首都特拉维夫（19万美元）、澳大利亚悉尼（14.5万美元）。

值得一提的是，在2020的报告中，中国北京以10万美元的薪资排名第十，而今年并未上榜。

薪资那么高 我能学会吗？能找到工作吗？

每个打算学，或者正在学Python的人，应该都会思考这个问题吧：学完Python找不到工作怎么办？

毕竟下定决定学习一门语言，需要付出精力跟成本。没人希望学完之后一无所获、找不到工作。但其实，这个问题多少有些问题。学完Python？达到怎样的水平才算是学完呢？入门基础知识算是学完？达到岗位招聘条件算是学完？V站上有一个点击16000+的问题：为什么纯 Python 的工作这么难找?从大家的回复中，可以总结出以下几点：

  1. Python整体上初学者较多，不好找的多是没有亮点的初学者。

  2. 单纯以Python为工作，要么是基本功深厚的Python工程师，要么是AI和数据分析等当下热门的方向，相对会好一点。

  3. 已经有基础的程序员，花几个星期，就能达到初学者学几个月的水平，想单纯靠Python一门技能找工作，竞争力不够。

所以坦白地说，这个问题有两个回答：

一、如果你不想成为Python程序员，只想用Python辅助现在的工作，拓宽自己的职业前景 —— 那你可以花一两个月的时间，学一下Python，因为它真的很简单，写一个爬虫可能只用几行代码，却可以大大提升你的工作效率。这可能是性价比最高的提升途径了，没有之一。

二、如果你想成为专业的 Python 程序员，想随便学一两个月Python，就靠它去为你赚钱的时代已经过去了，大家心里要有这个底。把Python学的很好，就如同英语里的高级翻译、同声传译等，能够单纯靠语言本身拿到较高难度，这种情况难度也较大。当然你还可以把 Python 当作基础工具，做数据分析、做机器学习等，如同英语中的商务英语、国际贸易等，也有不错的前景。学习编程是一件很公平的事，付出多少就会得到多少，没有例外。
