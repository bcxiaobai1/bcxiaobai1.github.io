人家不卡学历，是自己真的没能力

CSDN 的小伙伴们，大家好，我是二哥呀。

今天继续给大家分享一个二哥编程星球里的主题：人家不卡学历，让我去实习了，但是担心自己能力不足，二哥能给点建议吗？

除了这个问题，这位球友还私信咨询了二哥其他几个问题，我觉得都挺具有代表性意义的，这里就统一回复下，希望也能给CSDN的读者朋友们一点帮助和启发。

问题一

二哥，开发能力是做项目慢慢提升的吗？

以下??是二哥的回复。

一方面吧，看个人，我自己职业生涯前期比较依赖项目，后面比较依赖读源码、看书、看原理，输出倒逼输入啊。

就学习编程来说，一般分为两种，一种是先肝理论基础知识，然后再去实战检验，并且提升。

一种是直接实战，然后哪里不会就去补哪里。

两种方式，我认为没有优劣之分，可以根据自己的学习能力去做出选择。

就拿二哥自己来说，我就不太适合上来去肝理论知识，第一感觉枯燥，第二，不是谦虚，确实是学习能力有限，人比较普通。

所以，我职业生涯的初期，也就是刚参加工作的前三年，基本上都是靠完成公司项目，并且自己动手撸练手项目把开发能力慢慢培养起来的。

最近这两三年，撸项目少了，看源码，输出文章多了，感觉提升也非常明显，尤其是以前不明所以的，现在很多都很透彻。比如说之前生涯前期，觉得用 LinkedList 和 ArrayList 没任何差别，后来看了源码，了解了数据结构和算法，知道两者在不同场景下作用大不相同。

另外，开发能力除了需要硬编码实力，还需要业务能力，就是公司给你安排了任务，自己得能分析需求，并且通过代码去实现功能。这个转化业务的能力也非常重要，这种能力通过看书看源码不太能行得通，只能说多撸项目。

问题二

二哥，能不能咨询下，我学的数据科学与大数据技术专业，我是校企专业，但是校企一个大数据的岗位都没有推荐，全都是后端开发。而且我校企实训的时间也做的是大数据相关的岗位，我想去大数据岗位实习，但是他推荐的公司没有这个岗位，全都是测试、Java后端开发、前端、实施，我不知道怎么办了？

以下??是二哥的回复。

这个怎么说呢，很多高校，都普遍存在这些问题，学校的课程和实际的岗位需求完全不符。

你比如说，大数据岗位要求的学历其实是比 Java 后端开发高很多的，因为大数据岗位本来就少，对吧？

首先，得有大数据的公司，那一般的创业公司或者小型公司，根本就不具备这个条件，没有大数据，一个单体应用都能搞定的事情，哪来大数据。

其次，大数据岗位要求的东西，是建立在 Java 岗的基础之上的，Java 语法基础、并发编程、Java 虚拟机、Spring Boot 这些都是要学的，然后还要学习大数据相关的 Hadoop、Flume、Logstash、Filebeat、Hive 、Spark SQL 、Flink SQL、 Pig、Phoenix等等等等。

如果不是一个很好的学校，学完大数据，基本上是毕业即失业。所以，如果自己没那个实力，选 Java 后端开发就对了，千万不要高不着低不就，那就难受了。

问题三

二哥，你能帮我从网上了解一下大唐智联这家公司吗，我不知道要不要去这家公司实习？

以下??是二哥的回复。

这家公司我不太了解，网上查到的信息大概就下面这些。注册资本 3300 万，也确实在招 Java 开发，薪资 8-12k。

不过，有一点可以肯定的是，这家公司在杭州，那杭州这座城市近些年因为阿里的原因，互联网风气虽然崇尚 996 修福报，但技术环境还是非常浓烈的。

我也有不少朋友在杭州，你现在实习嘛，我觉得是可以去的，只要自己肯脚踏实地，应该是能学到东西的。

问题四

二哥，一个杭州的公司实习，一个内蒙呼市的公司实习，人家都没卡学历，杭州那个是一个老总面试我的，问我Java编程能力，我说实话自己觉得不强，然后就不想去了。还有，我就想着去呼市的小厂实习从基础做起，但家里的人想让我去杭州那边，比较那边比较发达，我也想去，但怕自己能力不足

以下??是二哥的回复。

做互联网开发，肯定去沿海一线城市啊！

城市的选择至关重要，在二哥的认知里，北上深杭>成都、苏州、南京、广州、武汉、西安>郑州、合肥、石家庄、呼特浩特等等二三线城市。

正因为你能力差，所以才能得到足够的发展和成长。去杭州，整体上的收益肯定是远大于去呼市的。

人们在发展阶段，总是希望去一些大城市，因为那里节奏快，薪资水平相对较高，技术更新迭代快，人才更多，资源更多，发展的预期也会好很多。

那你现在有这样的选择机会，肯定是要去杭州的啊！这都不用想。

人家公司还不担心你的能力问题，你反而在这里自卑的不能自已，没必要。

你不要一直揪着你不会，你实力不允许这种，社会上没有人会因为你能力差同情你的，你也不会因为得到同情和安慰就飞速成长。

只有挑战自己才能成长！！！

二哥在大学阶段也是过得浑浑噩噩，能找到实习工作就举得非常不容易，真的是差点去拜佛了！

刚到公司实习，也是担心自己实力问题，会不会直接被辞退掉，但经过为期一个月的上机考验，再加上三次笔试，幸运地留了下来。

可以说上机考试的一个月，再加上后来一个月内的三次笔试，学到的知识，比我在大学阶段都学的要多。

有强烈的危机感，就会逼迫你去争分夺秒地进步。

这么多年过来了，我很感谢我在外企工作的那三年半时间（包括一年的实习时间），让一张白纸的我，填上了五颜六色。

任何时候，机会来了，就要把握住，不要在那自怨自艾，没什么用！

二哥编程星球的 VIP 群里这位球友说得很对，我们拼的是什么，拼的是别人松懈的那一刻，我们还在努力！

就是这样啊，高考没别人考的好，上了大专、上了普通本科，想要和 985、211 院校的学生拼，靠什么？

靠自己能力不行，靠躲在自己的舒适圈，回家直接摆烂躺平？

肯定不行啊，靠的就是你比别人多付出一些，通过延长时间线来逐渐抹平差距，即便是永远也抹不平，那最起码能做到比昨天的自己好一些，对吧？

不能说一直维持在一个很低的水平线以下，那就真没得救了！

有本事就干了这碗鸡汤，有本事就把 Java 程序员进阶之路上的内容全部吃透，有本事就把星球专栏里的内容全部学完！
