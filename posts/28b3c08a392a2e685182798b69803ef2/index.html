1.怎么样知道自己Oracle是否安装成功？
要确定你的Oracle数据库是否成功安装，你可以执行以下步骤来进行确认：

（1）检查Oracle服务：打开Windows服务管理器（可以按下Win + R，然后输入services.msc），查找名为"OracleService“（其中”"是你的数据库实例名称）的服务。确保该服务的状态是“正在运行”。如果服务已启动，则表示Oracle数据库已经成功安装并正在运行。

（2）使用SQLPlus工具连接到数据库：在开始菜单中找到“SQLPlus”或者“SQL Command Line”程序，并点击运行。在弹出的命令行窗口中，输入以下命令来连接到你的Oracle数据库实例：

sqlplus username/password@hostname:port/service_name

其中，"username"是你的数据库用户名，"password"是对应的密码，"hostname"是数据库所在的主机名或IP地址，"port"是监听端口号，"service_name"是数据库的服务名。

（3）如果连接成功并显示SQL*Plus的提示符（SQL>），则表示你的Oracle数据库安装成功并运行正常。

检查监听器状态：在命令提示符窗口中输入以下命令来检查监听器的状态：

lsnrctl status

如果显示监听器正常运行并且监听到你的数据库实例，则表示Oracle数据库已成功安装并且监听器正在正常工作。

（4）测试数据库功能：可以尝试创建表、插入数据、执行查询等操作来验证数据库的功能是否正常。

常见问题

1.如何用“管理员身份”登录Oracle

别笑，真的有xd不知道。而且有的人电脑上面右键后是没管理员身份运行按键的。
点Windows图标，打开

找到安装的Oracle，打开，看到下面有个start Database->右键->更多->以管理员身份运行
输入以下命令以启动SQL*Plus工具，并连接到Oracle数据库实例：

sqlplus /nolog

在SQL*Plus的提示符下，输入以下命令以以系统管理员身份登录（SYSDBA权限）：

conn sys as sysdba

按下回车键执行登录命令。

如果你之前未设置过系统管理员（SYS）的口令，则会要求你输入一个新的密码。在提示符下输入新密码并确认。

如果提供的密码正确，并且连接信息正确，你将成功登录到Oracle数据库作为系统管理员。

2.如何解锁scott用户？
详细步骤如下
要以管理员登录后解锁scott账户、修改scott账户密码并查看其下的表，请按照以下步骤进行操作：

以管理员身份登录Oracle数据库（可以参考之前提到的登录步骤）。

输入以下命令来解锁scott账户：

alter user scott account unlock;

执行该命令后，scott账户将被解锁，可以正常登录和使用。

输入以下命令来修改scott账户的密码为"tiger"：

alter user scott identified by tiger;

执行该命令后，scott账户的密码将被修改为"tiger"。

输入以下命令来查看scott账户下的dept、emp、salgrade表：

select table_name from all_tables where owner=‘SCOTT’;
执行该命令后，将会显示scott账户下的所有表名，包括dept、emp和salgrade表（如果存在的话）。
