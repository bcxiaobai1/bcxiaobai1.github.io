Microsoft SQL Server安装方法

首先得了解什么是Microsoft SQL Server？

Microsoft SQL Server（微软结构化查询语言服务器） 是美国微软公司退出的一款关系型数据库。最新的版本是SQL Server 2019，已在美国时间2019年11月3日发布。资料库的内置语言原本是采用美国标准局和国际标准组织所定义的SQL语言，但是微软公司对它进行了部分扩充而成为作业用SQL。

区分安装版本

安装版本分别为：

  • Enterprise（企业版本）
  • Standard（标准版本）
  • Express（入门版本）
  • Developer（开发版本）

Enterprise和Standard为收费版本，Express和Developer为免费版本

所以我们学习使用的时候建议使用免费版本!!!

    最后说明

    此次安装仅仅是学习使用安装方法，如果在正式开发版本环境，不要用此安装方法。会有相当大不同。

    安装Microsoft SQL Server最好要一次成功！，因为Microsoft SQL Server的体量相当庞大，一次性卸载不干净，如果卸载再安装时，除非重新装操作系统在进行安装Microsoft SQL Server才能成功安装。本人吃过多次坑。希望大家不要多次碰壁。最好一次安装成功。
