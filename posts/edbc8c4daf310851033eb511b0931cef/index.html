在 Debian 11 上重置忘记的 root 密码的步骤

打开 Debian 11 Grub 菜单

要重置密码，首先，我们必须访问Debian 11 Bullseye的Grub 菜单。重新启动系统，同时按住键盘上的Shift键。这将使您进入 Debian 11 的 Grub 菜单。

这个步骤是重启系统的，已经关机的直接到下一步

编辑 Gurb 菜单以重置 root 密码

下一步是编辑Debian 11 的默认启动项。为此，请按键盘上的“ e”键。这将允许您编辑 Grub 的引导提示符。不要触摸或删除此处的任何内容。一旦你让编辑器移动到下一步。

添加 rw init=/bin/bash

在 Grub 编辑屏幕上，使用箭头键移动到以“ Linux ”开头的行的末尾。在此行类型的末尾 rw init=/bin/bash ，如给定的屏幕截图所示。

rw init=/bin/bash


添加给定的语法后，接下来，使用此配置启动您的系统。为此，请使用 Ctrl+X 或 F10 。

Debian 11 根外壳访问

现在，无需提供任何密码，您将拥有具有 root 访问权限的 Debian 11 shell。在这里，首先让我们检查我们的用户是否具有对安装操作系统的文件系统的读写权限。

为此使用命令：

mount | grep -w /

如果输出是：

( rw,realtime ) 则表示您对文件系统具有实时读写权限。

重置root用户密码

现在，让我们像在 Linux 系统上一样使用命令终端更改密码。

输入命令： passwd

系统会两次提示您添加新密码：

在 Debian 11 Buslleye 中重置 Root 密码

要更改root以外的其他用户的密码，我们必须提及相同的用户名。例如，如果我们的系统用户名是 Debian，并且我们想更改其密码，那么命令将是：

语法：passwd username

例子：passwd debian

重启你的系统

完成重置 Linux 密码后，重新启动系统以使用更改后的密码登录。要重新启动，请键入：exec /sbin/init

然后按下Enter键。

从这里开始，您可以使用新设置的 root 密码登录并访问您的系统。本指南对于那些因某种原因丢失或忘记了 root 密码并且现在想要更改密码的人很有用。本文中给出的步骤也适用于其他 Debian 11 版本。
