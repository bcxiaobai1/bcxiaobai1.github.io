麒麟操作系统(Kylin)—龙芯芯片—mips64架构

1.安装docker

        下载docker *.deb安装包 

        官网网站: http://www.loongnix.cn/index.php/Docker

        网络在线下载地址:

wget http://ftp.loongnix.cn/os/loongnix/1.0/docker/docker-ce_18.06.3.ce-1_mips64el.deb

官网文档地址: ShowDoc

        其中libdevmapper.so.1.02.1  可以在/lib中找

官网用户手册使用教程: Docker | 龙芯开源社区

  1. 拉取镜像

镜像查找下载地址: Quay

有的镜像在它的镜像库查找不到

有的镜像拉取不下来,异常说当前镜像与架构不匹配

        拉取镜像方式一:

                在Information中;复制镜像下载地址

docker pull cr.loongnix.cn/mips/minio

        拉取镜像方式二:

                在Tags中选择需要的镜像版本,点击Fetch Tag(下载箭头),选择Docker Pull(by Tag),最后点击"Copy Command"复制命令

docker pull cr.loongnix.cn/mips/minio:RELEASE.2022-12-12T19-27-27Z

例如:minio镜像下载

搜索minio,查找到两种镜像,选择mips架构的镜像,然后"拉取镜像方式二"

如果拉取还是失败,执行docker pull minio,选择常规linux下载镜像方式.

最总拉取还是失败,请选择宿主机安装.

注意:Elasticsearch 官方支持的 CPU 架构包括 x86_64、ARM64 和 PPC64LE，不支持 MIPS 架构。因此，在龙芯 MIPS 架构上运行 Elasticsearch 可能会存在一些问题，例如无法正常编译、启动或运行。如果需要在龙芯 MIPS 架构上使用 Elasticsearch，可能需要自行进行适配和修改。建议在官方支持的 CPU 架构上使用 Elasticsearch，以获得更好的兼容性和性能。
