USB Type-C接口会成为显示器行业的下个风口吗？

最近市场上开始陆续涌现配有USB-C接口的显示器，很多小伙伴并不知道这接口是什么用途。除一些老旧设备使用VGA外，目前显示器常用接口不外乎HDMI、DP、DVI这几种，但很多人并不知道，USB-C也是显示器接口之一。

为什么选择 USB-C？

随着 USB-C 连接的使用越来越广泛，人们的期望也将发生变化。随着新技术的发展，对更高效的数据和电力传输的需求也在增长。这包括我们喜欢的性能更高的设备，这反过来又要求提高功率和效率。

在过去的几年里，技术变革的速度甚至比 2014 年推出 USB-C 时还要快。据大多数专家称，USB-C 不会很快消失。这是我们应该计划在某个时候在外围设备中使用的技术，无论我们现在还是以后进行更改。

自问世以来，USB-C 变得越来越普遍，并已成为一系列传统端口的共识替代品。其中一些可能比其他人更熟悉，例如我们大多数人仍在使用的几乎无处不在的 USB A 型，或者与外部硬盘驱动器等设备相关的 USB B 型。

USB-C 显示器的五个主要优势。它们包括以下能力：

1. 使用一根线传输视频、数据和电源

USB-C 对我们如何使用智能电视和设备的影响是相当大的。对消费者及其日常生活有很大的好处，特别是将多根电源线（电源、数据和专用类型）整合到一根电源线上。

此外，最新的 USB type-C 连接（目前在 3.1 代和 3.2 代将于 2019 年晚些时候发布）可以提供每秒 10 吉比特 (Gbps) 的数据传输和高达 100W 的持续功率传输。与以前的 USB 充电标准相比，这远远高于正常的功率输出，以前的 USB 充电标准只能与兼容的设备一起使用，并且只能产生 2.5W。

不过，这并不是 USB-C 性能的最终上限，这要归功于英特尔® 强大的Thunderbolt 3 技术和端口。它有助于将 Type-C 的数据传输速度从 10 Gbps 提高到 40 Gbps，并且支持多种设备。

除了提供高质量的输出之外，这些功能结合起来使 USB-C 显示器从根本上更便于携带，并且在大多数情况下更易于携带。更深层次的多媒体影响Thunderbolt 3 提供对 PCI Express (PCIe) 和DisplayPort 信号的访问，而内置的 alt 模式可确保 USB-C 提供高质量流媒体所需的一切。Alt 模式是业界首选的兼容性术语，该术语描述了 USB-C 可以处理的数据类型。通过 alt 模式和正确的连接，您实际上可以输出 HDMI、DisplayPort、VGA 和许多其他受支持的格式。

从本质上讲，USB type-C 显示器已准备好支持开箱即用的高质量视觉效果。而且，如果您假设您已经为音频提供保障，那么您也有钱。但你可能没有想到，你甚至可以将 USB-C 端口用作专用耳机插孔。您可以将旧版耳机与适配器一起使用，也可以选择较新的 USB-C 耳机。

2. 无需适配器即可连接 USB-C 笔记本电脑和设备

轻松访问最现代的设备是 USB C 型显示器的另一个不可否认的优势。当您不需要时，为什么还要费心存放额外的电源线和适配器？对于办公室和公共场所来说，这是一个很大的优势，因为它们需要快速方便地访问。

它还减少了配件预算，因为您的外围设备仅依赖于一种输入/输出类型，而不是六种或更多。这意味着购买和安装到您的工作空间中的适配器更少。

但是，如果您确实需要一个，无论是用于旧设备还是不同的输出格式，USB-C 的小尺寸都不会占用您的抽屉空间。无需收集各种必要的适配器或超大的集线器，您可以使用紧凑的USB-C 多端口适配器覆盖您的许多基础。

就像 USB-C 跳转驱动器一样，它们比上一代技术或 USB-A 同等产品要小得多。根据您最常需要访问的端口，有一系列紧凑型设计。

3. 更小、更易于使用的连接类型

任何在 USB-C 出现之前就熟悉计算机的人，一旦习惯了这种新格式，就可能会喜欢它。您会注意到的第一个变化是与传统端口（如 USB type-A）或带有超大背衬的命名不当的微型变体相比，尺寸更小。

USB-C 比前几代更小，并且对于需要定期插拔的任何东西都更难使用。这在很大程度上要归功于其更方便的物理配置。

USB-C 小到足以在与智能手机或平板电脑一起使用时感到舒适，但功能强大到足以支持一系列其他并发功能。这有助于减少以前因缺乏港口标准化而需要的规划和设计。为了说明它提供帮助的最重要方式之一，包罗万象的 USB-C 可能会将设备上的六个或更多必要端口减少到一两个。

真正的小也比你想象的更重要。对于开发人员和设计人员现在如何开始为更广泛的产品标准化端口至关重要。总体而言，USB type-C 显示器紧跟最新的技术趋势，并带来更大的使用舒适度。它还通过最大限度地减少电线、连接和对配件的需求来降低对您空间的影响。

它是可逆的

Type-C 设计的另一个关键变化是它是可逆的，与 Type-A 和其他前辈相比，这是一个很大的改进。简而言之，这种可逆性仅仅意味着 USB-C 线可以以您喜欢的任何方式插入。

当您需要将其与其他设备一起使用或只是想重新连接时，您不再需要担心哪种方式是向上的，或者您是否有正确的一端。使用 USB-C 显示器，这意味着当您不得不在显示器后面钓鱼或更改连接时，压力会小一些。

更小的物理设计和可逆性的便利性都倾向于将 USB-C 置于连接的最前沿。

4. 方便为其他外接设备充电

为设备充电和剪线开辟的可能性是数据和电源标准化单线解决方案的另一个关键优势。无论您是讨厌电线造成的混乱，还是只是想要一个更好的中央解决方案来为您的设备充电，USB-C 显示器都是一个很好的解决方案。

因为 USB-C 显示器可以为设备充电提供 100W 的回馈功率，所以这里的潜力比旧标准要大得多，甚至是更新的 USB 3.0。这就是为什么许多新的笔记本电脑型号放弃了专用电源线，转而使用 USB-C。因此，您可以告别笨重且携带起来毫无乐趣的传统电源线。

为您的整个办公室奠定坚实的基础

通过 USB-C 连接，您可以通过笔记本电脑为显示器供电，反之亦然。无需担心额外的电源线或适配器，从笔记本电脑流式传输到您的显示器要容易得多。对于在办公室环境中推出的 USB-C 显示器来说，这也是一项很棒的功能，因为设置过程大大简化了。

采用最新一代技术的 USB type-C 显示器 – 如果您想在购买前确认，请寻找 USB 3.1 – 最多可以连接六个设备并为其充电。无论您选择菊花链单独还是通过多端口适配器连接，这都支持所有六个。对于任何类型的用户，从办公室工作人员到家庭，尤其是那些拥有共享媒体空间的用户，这都是一种高级的便利。

不仅仅是创建充电集线器，多端口适配器或专注于 USB-C 的工作区扩展坞也是一次组织和轻松访问所有设备的好方法。

5. 这是大多数新设备的标准

虽然我们已经讨论了 USB-C 变得越来越普遍，但我们只介绍了这一趋势的一些结果。随着我们看到 USB-C 的使用越来越多，启用的显示器是开始使用这项技术的好地方。这些型号中的大多数还具有旧端口，这意味着它们是熟悉新格式的好方法，而无需升级其余设备或购买一堆适配器。

不过，更改为 USB-C 仍然比升级到单个设备要复杂一些。我们通常会在物品老化时更换它们，但向 USB-C 的过渡意味着密切管理您的设备非常重要。否则，您可能会被配件绊倒。或者更糟糕的是，为您的其他媒体或办公套件购买了错误的产品。

目前，整合电源线和连接仍然是提高消费者可用性的重要一步。它允许制造商构建更紧凑、更高效的设备，因为在布置新的笔记本电脑设计时，整合传统端口可以节省空间。使用更少的端口，工程师可以设计更薄的笔记本电脑或装入更好的硬件。简而言之，USB-C 旨在取代各种连接，而不仅仅是我们最常用的 A 型连接。

有很多理由尽早升级到 USB-C 显示器。一方面，这通常意味着您将享受性能升级，但更重要的是，它使您领先于可能会影响您已经做的很多事情的变化。

USB 连接器和电缆是帮助我们连接设备的最有用和重要的工具之一。该技术的第一次迭代 USB-A 迅速成为保持我们的笔记本电脑、PC 和移动设备全部连接的基本功能。

2014 年，久负盛名的 USB-A 连接标准升级为 USB-C，这是一种更小、更方便、更通用的连接解决方​​案。

市场主流TYPE-C接口 PD协议芯片显示器方案

LDR6020 是深圳市乐得瑞科技推出的双 USB-C 接口的 USB PD3.1 协议通信芯片。它具备 Power Negotiation 数据包透传功能，切换 Data Role 功能，以及通过 VDM 的协商让智能设备进入 ALT Mode的功能。LDR6020的双路 USB-C 接口在功能上是对等的，每路 USB-C 都是既可以接智能设备，又可以接适配器，特别适用于双 USB-C 接口的显示器和转接器方案。也可以支持做USB-C接口台式显示器方案，LDR6020 支持 USB IF 协会发布的最新USB PD3.1 协议。

◇ 支持 USB PD2.0、USB PD3.0 USB PD3.1 协议

◇ 支持 Data Swap 切换为 UFP 角色

◇ 支持 DP Alt mode 配置

◇ 支持用户在 APP 端升级

◇ 支持 I2C 从机通信

◇ QFN-32 5x5 封装

在线升级功能
用户可以通过安装 APP 来升级 LDR6020 的固件，从而实现固件兼容性的更新。LDR6020 是通过USB来实现升级功能的，包含两个引脚：

◇ UART_RX : 串行接收脚

◇ UART_TX : 串行发送脚

◇ I2C 升级

◇ 通过USB D+, D-升级烧录。
