    C类IP地址范围从192.0.0.0到223.255.255.255。

  • 在IP地址3种主要类型里，各保留了3个区域作为私有地址，其地址范围如下：

    A类地址：10.0.0.0～10.255.255.255

    B类地址：172.16.0.0～172.31.255.255

    C类地址：192.168.0.0～192.168.255.255

  • A类网络的默认子网掩码是255.0.0.0，B类网络的默认子网掩码是255.255.0.0，C类网络的默认子网掩码是255.255.255.0。将子网掩码和IP地址按位进行逻辑“与”运算，得到IP地址的网络地址，剩下的部分就是主机地址，从而区分出任意IP地址中的网络地址和主机地址。

    子网掩码常用点分十进制表示，还可以用CIDR的网络前缀法表示掩码，即“/<网络地址位数>；”。如138.96.0.0/16表示B类网络138.96.0.0的子网掩码为255.255.0.0。

应用层 应用层协议数据单元（APDU） 应用程序 DHCP，FTP，HTTP，RPC

2.名词解释

加粗的在考试中出现过

  • CSMA/CD：载波侦听多路访问/冲突检测，应用于有线连接的局域网
  • CSMA/CA：载波侦听多路访问/冲突避免，应用于无线局域网
  • TCP：Transmission Control Protocol，传输控制协议
  • UDP：User Datagram Protocol，用户数据报协议
  • SAP：服务访问点，是一个层次系统的上下层之间进行通信的接口。
  • MTU：Maximum Transmission Unit，最大传输单元
  • MAC地址：物理地址，它是一个用来确认网络设备位置的位址。数据链路层则负责MAC位址。
  • ARP协议：（Address Resolution Protocol，地址解析协议），完成主机或路由器IP地址到MAC地址的映射。
  • OSPF：(Open Shortest Path First 开放式最短路径优先），适合在大型的、动态的互联网上使用
  • RIP：(Routing Information Protocol,路由信息协议），适合在小型的、动态的互联网上使用
  • IGP：（ Internal gateway protocol，内部网关协议 ），作用范围是自制系统
  • SNMP：（ Simple network management protocol ，简单网络管理协议）
  • ADSL：（非对称数字用户线路 ，Asymmetric Digital Subscriber Line）
  • NAT:（Network Address Translation,网络地址转换)，可以将私用的本地 IP 地址转换为合法的全球 IP 地址。
  • RFC: (Request For Comments,是一系列以编号排定的文件。)
  • DHCP：（Dynamic Host Configuration Protocol，动态主机配置协议）
  • CRC：（Cyclic Redundancy Check，循环冗余校验）
  • LLC：（Logical Link Control，逻辑链路控制）
  • HTTP：超文本传输协议

3. 练习题

  1. 如果子网掩码是 255.255.255.224，那么主机 218.22.50.40 所在的驻留子网是（ ）。

    A.218.22.50.0 B.218.22.50.224 C.218.22.50.32 D.218.22.50.40

    解：将子网掩码和主机ip 展开为二进制，进行相与。所得结果即为掩码

  2. 某公司申请到一个 C 类网络，出于地理位置上的考虑，必须切割成 5 个子网，那么子网掩码可以设置为（ ）。

    A.255.255.0.0 B.255.255.255.0 C.255.255.255.192 D.255.255.255.224

    解：由于是C类网络，所以前面是255.255.255不用解释。由于题目要求5个子网，2^3=8>5,所以至少需要3位二进制才能存放下5个子网。所以最后一个子网掩码为11100000，结果等于224,综上，结果为255.255.255.224。

  3. 要发送的数据是 101110，已知 CRC 码的生成多项式 G(X) = X^3 +1，求帧校验序列 FCS。

    答：采用 CRC 的生成多项式 G(X) = X^3 + 1，用二进制表示就是 G = 1001，这是模 2 运算中的除数。在发送的数据后面添加 3 个 0 得被除数 101110000。 模 2 运算得余数 R = 011，这就是校验序列的值。

  4. 有一个具有等距离间隔站点的基带总线型局域网，数据传输率为 10Mbps,两个相邻站点

    之间总线长度为 1000m,传播速度 200m/微秒，发送一个 1000 位的帧给另一个站，求发送开始

    到接收结束的平均时间是多少？

    ∵ 数据传输时间1000位÷10Mbps=100μS             
   线路延时时间1000m÷200m/μS=5μS               
∴ 平均时间 100μS+5μS=105μS

  5. 请问下列 IP 地址能否分配给主机？如果不能，请说明理由。

    （1）131.107.256.80 不能分配主机，因为第三个字节 256 大于 255，是非法值。

    （2）231.222.0.11 不能分配给主机，因为第一个字节 231 位于 224 和 239 之间，说明是组播地址。

    （3）126.1.0.0 不能分配给主机，因为以全 0 结尾的 IP 地址是网络地址。

    （4）198.121.254.255 不能分配给主机，因为以全 1 结尾的 IP 地址是广播地址。

    （5）202.117.34.32 可以分配给主机

  6. 一个信道的带宽是 4kHz，数字信号的电平取值存在 4 种可能，请问理想状态下该信道的最大数据传输率是多少？若考虑到信道噪声，信噪比为 30dB，则该信道的最大数据传输率是多少？（注：log21001≈10）

    理想状态下使用奈奎斯特准则求解信道最大数据传输速率

C = 2Hlog2N

将 H = 4kHz，N = 4 代入得 C = 16kbps。

有噪声干扰时使用香农公式求解信道最大数据传输速率

C = Hlog2(1+S/N)

又 10log10(S/N) = 30dB，可求得 S/N = 1000。

将 H = 4kHz，S/N = 1000 代入香农公式得 C = 40kbps。

  7. 某单位分配到一个 B 类 IP 地址 129.250.0.0，该单位有 4000 台主机，分布在 16 个不同地点。若选用的子网掩码是 255.255.255.0，请给每一个地点分配一个子网号码，并计算每个地点主机号码的最小值和最大值。（注：子网号码和主机号码均从 1 开始编号）

    对 B 类地址设置子网掩码 255.255.255.0，说明子网位为 8 位，主机位也为 8 位。
4000 台主机均匀地分布在 16 个地点，则每个地点容纳的主机数为 250 台。
子网号范围：00000001～00010000，即 1～16
主机地址范围：00000001～11111010，即 1～250
