python创建用户界面之PyQt5的使用

首先我们win+R输入cmd

有两种方法下载PyQt5

方法一：

pip install PyQt5

pip install PyQt5-tools

方法二：

pip install PyQt5 -i https://pypi.douban.com/simple

pip install PyQt5-tools -i https://pypi.douban.com/simple

下载好了之后我们打开 Pycharm

在文件里面点击设置

 进去之后我们点击加号+

注意的是程序的位置必须是designer，而且是.exe结尾的

 

 点击确定之后选择这个画笔

这里和要记得位置是：pyuic5.exe
