最全元宇宙概念分析！元宇宙为何发展于区块链？

元宇宙，Web3 时代最新热词，和 NFT、DAO 等新晋热门概念一起在 2021 年横空出世。这一概念最早诞生于 1992 年的科幻小说《雪崩》，小说中描绘了一个庞大的虚拟现实世界，人们用数字化身来控制，并相互竞争以提高自己的地位，展现了对超前的未来世界想象。

如今，伴随元宇宙概念与区块链技术交相应用，虚拟场景、社交直播等产业逐渐发展，元宇宙成为打造下一个互联网时代爆款产品的“香饽饽”。**今天，**就让我们走近元宇宙，了解元宇宙和区块链之间密不可分的关系，看看区块链上的元宇宙项目是怎么火起来的！

元宇宙是什么？

当我们听到元宇宙的时候，脑海中可能会浮现一幅关于另一个数字世界和生存空间的幻想，其实元宇宙概念在学界并没有统一权威的界定。一般我们会从两种角度出发来探讨元宇宙的概念：

  • 从时空性来看，元宇宙是一个空间维度上虚拟而时间维度上真实的数字世界；

  • 从真实性来看，元宇宙中既有现实世界的数字化复制物，也有虚拟世界的创造物。

准确地说，元宇宙更像是一个经典概念的重生，是在扩展现实（XR）、区块链、云计算、数字孪生等新技术下的概念具化。真正的元宇宙不是单独的个体，而是多重的，是现代信息技术的集成。作为互联网的下一个发展阶段和新的风向标，元宇宙集成了包括 5G、云计算、人工智能、虚拟现实、区块链等在内的技术概念，共同打造了全新的未来数字虚拟空间。

元宇宙由许多部分组成，以下是其构成的基础要素：

  • 互联网：一种分散的计算机网络。

  • 媒体的开放标准：包括文本、图像、音频、视频、3D 项目、3D 场景和几何、矢量、序列以及生成和组合任何这些内容的程序。

  • 开放编程语言标准：包括 HTML、JavaScript、WebAssembly、WebXR、WebGPU Shader Language、etc 等。

  • 扩展现实 (XR) 硬件：如智能眼镜、触觉和全方位跑步机。

  • 去中心化账本和智能合约平台（例如区块链）：用于透明、无需许可和抗审查的交易，构成所有权经济的重要基础，支持元宇宙并使其成为可行的公共产品。

在这些组成部分中，区块链占据了重要的地位，也因其自身的技术优越性不断为元宇宙的发展注入源源不断的动力。

区块链：元宇宙的技术基石

刚谈到元宇宙集合了包括区块链在内的各类新兴科技概念和元素，那么元宇宙和区块链究竟有着怎样的联系呢？

元宇宙是基于 Web3 技术体系和运作机制支撑下的可信数字化价值交互网络， 是以区块链为核心的 Web3 数字新生态。因此，我们可以把区块链技术理解为元宇宙的技术基底，高效、稳定、透明、规则化的技术优点为元宇宙的兴起提供基石，成为支撑元宇宙具体项目运作的关键。

区块链是用密码技术将共识确认的区块按顺序追加形成的分布式账本，具有技术、金融和社会属性，这三个属性要素成为了促进元宇宙健康发展的核心要素，发挥不可替代的作用。

  • 从技术属性看，BaaS、跨链、隐私计算、身份认证、分布式存储和计算等底层平台为金融、制造、教育、能源等行业的分布式应用提供了基础设施。

  • 从金融属性看，DeFi、NFT 和 xFi 通过大量的实践探索数字化金融服务模式。

  • 从社会属性看，DAO 模式为数字环境下的协作模式提供了新的思考。

元宇宙的出现是区块链技术、金融和社会属性发展的必然趋势，为元宇宙的体系的隐私保护、身份认证提供凭借，为元宇宙金融交易模式提供数字化路径，为元宇宙全新的社会交往方式提供准则，也将推动建立以区块链为核心的 Web3 数字生态。

区块链上的“元宇宙”项目

单说元宇宙的概念或许有些抽象，具体到元宇宙赛道在这两年兴起的热门项目，或许会让人惊叹其带给人们的未来畅想，今天让我们一起来看看知名公链项目上的“元宇宙”。

Flow

作为加密猫、NBA Top Shot 等热门链游背后的开发团队 Dapper Labs 专为游戏打造的区块链平台，Flow 以快速和低成本交易等优势吸引了很多合作方。Flow 和 Dapper Labs 在元宇宙项目 Matrix World 上各申领了108块土地，以建立虚拟总部，利用元宇宙概念发展新的赛道项目。

Matrix World 具有可编程性，多链连接和开放性世界等特点，允许用户与同时运行在不同区块链上的沉浸式 3D 应用程序进行交互，包括但不限于构建 3D 架构，举办虚拟会议，展示 NFT 以及更多高级功能，如使用 Matrix 内置的创建服务和计算资源创建和托管 3D DApps。

Rangers Protocol

Rangers Protocol 由链游发行商 MixMarvel 孵化，集成了跨链、NFT 和 EVM 协议，旨在为 DApp和资产提供与多条不同链（已支持以太坊、BSC 和 Tron 等）进行无摩擦交互的功能，成为元宇宙的区块链基础架构。作为可实现 EVM 多链合约互操作的高性能链群，Rangers Protocol 服务于所有想要探索区块链世界的创业者。

Rangers Protocol 的突出优势在于高性能、高安全性和高稳定性，用户可以实时获得在 Rangers 协议上进行交易的执行结果，而无需等待区块生成。此外，无摩擦交互的功能对于元宇宙建设来说至关重要，这意味着用户可以实现从不同区块链中提取数据、资产或智能合约的复杂操作。

Litentry

Litentry 作为去中心化的身份聚合器，以基于 Substrate 构建的区块链和分布式的去中心化身份（DID）验证网络为核心特色，旨在为 DID 应用程序提供去中心化、可验证的身份聚合服务。今年 8 月，Litentry 推出了元宇宙身份应用 My Crypto Profile，通过关联以太坊身份与推特账户让用户在推特上展示自己的 NFT 藏品，该游戏还与元宇宙项目 The Sandbox 等建立了合作。**My Crypto Profile 是 Litentry 推出的 Web3 身份工具，用户通过连接加密地址和社交网络身份，证明其账户所有权。**MCP Genesis NFT 是由 My Crypto Profile 发行的一套原始 NFT 作品，专属早期的 500名用户。

My Crypto Profile 的技术愿景是通过身份聚合效应的验证，为元宇宙空间的发展提供社交联结凭证，打造 Web3 时代的多功能社交平台和 NFT 艺术藏品收纳地，用跨链社交丰富 Web3 社交玩法，让用户们自如灵活地感受数字虚拟世界的社交方式。

**元宇宙未来：**多链互融、赛道合作

当我们赞叹这些将区块链技术和元宇宙发展巧妙结合的项目如火如荼地开展时，我们也需要思考区块链技术会否给元宇宙概念的“变现”带来一定的阻碍，我们是否需要提前规避一些风险？

正如多链并行时代，公链竞争日益激烈的当下，不同平台在积极开拓元宇宙赛道的同时，也在对技术爱好者、普通用户展开新一轮的“存量争夺”。加密和去中心化计算虽然在打破一些平台和跨链操作的壁垒，但是不同元宇宙项目之间的互融互通仍然不能够实现。

近期，Decentraland 作为崭新的元宇宙 3D 虚拟空间诞生了。在这个 3D 空间里有多种好玩的迷你游戏，也会给予用户诸多 NFT 奖励。但是 Decentraland 的更多功能开发却遭到了一定的限制，不同的公链不能同时为单独一个平台提供各个优势方向的功能支撑，就像无法在 The Sandbox 中探索 Decentraland 世界，也无法使用 Unreal Engine 应用打开 Unity 制作的游戏。

我们期盼着多元宇宙的未来景象，也希望元宇宙可以带领我们进入真正智能、自由、高效协作的数字化生活，但是元宇宙的基础是共享数据、共享计算和共享带宽，只有汇聚星星之火才能绽放元宇宙的高能光芒，多元宇宙需要一个全宇宙体系。

因此，区块链技术作为元宇宙设施的底层设施，理应加速自身在跨链操作、赛道拓展、项目合作等方面业务水平，为迎接 Web3 时代的元宇宙未来注入强劲动力。
