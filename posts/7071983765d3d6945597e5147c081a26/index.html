二、yes

yes也不需要安装，作用是一直输出某一句文字。

例如：

yes hello,world.

三、htop

htop是显示系统的运作的情况一个库

sudo apt install htop

运行

htop

四、fortune

作用是输出一个随机的语句（名言）。

sudo apt install fortune

运行

fortune
