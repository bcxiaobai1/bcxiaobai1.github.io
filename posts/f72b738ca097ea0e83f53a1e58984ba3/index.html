2.4、安装nginx

2.4.1、添加Nginx存储库

添加CentOS EPEL仓库，请打开终端并使用以下命令：

sudo yum install epel-release

2.4.2、安装Nginx

现在Nginx存储库已经安装在您的服务器上，使用以下yum命令安装Nginx ：

sudo yum install nginx

 2.4.3、启动Nginx

sudo systemctl start nginx
