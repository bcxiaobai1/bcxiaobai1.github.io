Mac 操作系统版本简史，让我们回到过去

几十年来，苹果一直在电脑领域不断发展创新，从 1984 年的第一个图形用户界面到 macOS Monterey，Mac 操作系统的历史漫长而多变。当我们按顺序重新访问每个 Mac 操作系统版本时，可以回到过去。

前 Macintosh 操作系统


Apple I，Apple 的第一台计算机，实际上并没有操作系统，它可以将程序保存到盒式磁带中，但 Apple II 拥有可以在软盘上组织、读取和写入的内部磁盘操作系统。

这些系统中的第一个是 Apple DOS，其继任者是 Apple ProDOS（更新后也称为 ProDOS 8 和 ProDOS 16）。

Apple 推出的第一个非磁盘操作系统是 GS/OS。GS/OS 内置了 Finder，可以支持多个磁盘文件系统。

Apple III 的操作系统是 Apple SOS，而 Apple Lisa 使用的是 Lisa OS。Steve Wozniak 将 Apple SOS 称为“有史以来最好的微型计算机操作系统”，而 Lisa OS 具有保护内存的功能。但未来的操作系统将更具影响力。

经典 Mac 操作系统


Macintosh 计算机于 1984 年发布，其操作系统称为 Macintosh 系统软件或系统 1。系统 1 帮助普及了图形用户界面，在这种界面中，人们点击图标而不是输入代码来使用他们的计算机。

系统 1 为 Apple 计算机引入了菜单栏，以及“桌面附件”应用程序，如计算器和闹钟。System 2 于 1985 年发布，并增加了对 AppleTalk 网络协议的支持。System 3 和 System 4 于 1986 年和 1987 年发布，允许 Macintosh 计算机与更多外部设备一起工作。

1987 年末，系统软件 5 终于允许 Mac 用户一次运行多个应用程序，系统软件 6 在 1988 年对其进行了改进。

1991 年 System 7 出现时发生了重大变化。它具有虚拟内存支持、内置协作多任务处理和添加的别名。它还添加了新的应用程序，并极大地改变了用户界面。

Mac OS 的命名系统更改为 System 7 的更新。此更新称为 Mac OS 7.6，“Mac OS”命名趋势将在 1997 年和 1999 年延续到 Mac OS 8 和 Mac OS 9。

Mac OS 8 与 7 没有太大区别——它被命名为 8 以终止第三方制造商对 System 7 的许可并停止制造 Mac 克隆。它确实添加了 HFS+ 和至少在后台克隆文件的能力。

Mac OS 9 改进了无线网络支持，并引入了远程网络、动态文件加密和早期版本的多用户支持。

Mac OS 9 结束了“经典”Mac OS 的时代，将功能传递给 Mac OS X（X 是 10 的罗马数字）和现代 macOS。

Mac OS X 和现代 macOS

要谈论这个操作系统时代，最简单的方法是按版本逐个介绍它们的亮点。

Mac OS X Public Beta Kodiak (2000)

Kodiak 于 2000 年发布， Apple 可以获得有关新操作系统格式的反馈。一旦猎豹进入市场，Beta 就停止工作。

Mac OS X 10.0，猎豹（2001）

尽快它的名字叫猎豹，但它是一个缓慢的操作系统，并且没有很多应用程序。不过，随着错误得到纠正，它成为新 Mac OS X 系列的坚实基础。

Mac OS X 10.1，彪马 (2001)

在 Cheetah 发布 6 个月之后发布的 Puma 增加了 10.0 中缺少的功能，例如 DVD 播放。

Puma 发布几个月后，苹果宣布 Mac OS X 将成为其电脑的默认操作系统。目前，从 Classic Mac OS 升级需要用户花钱，但至少从 Cheetah 升级到 Puma 是免费的。

Mac OS X 10.2, 捷豹 (2002)

Jaguar 具有比其前辈更好的性能和更好的合成图形，允许 iChat 和地址簿在 Mac 上运行。

“Happy Mac”面孔在此更新中退休，时隔 18 年。

Mac OS X 10.3，黑豹 (2003)

Panther 添加到 Safari 和 FileVault 中，允许更快的用户切换，并包含 Finder 更新。它还为界面添加了拉丝金属外观，在一段时间内影响了未来的设计选择。

Mac OS X 10.4, Tiger (2005)

Tiger 只能在带有内置 FireWire 端口的 Mac 上运行。Panther 没有在 Power Macintosh 和 PowerBook 上工作过；这意味着更多的苹果电脑正在失去操作系统支持。

Tiger 添加了 Spotlight、Dashboard、Smart Folders、Automator 和 VoiceOver，并更新了 Safari、Quicktime 和 Mail。随着 Apple 开始构建基于 Intel 的 Mac，Tiger 在这些新设备上的运行方式与在现有 PowerPC Mac 上的运行方式相同。

Mac OS X 10.5, Leopard (2007)

一个大的更新，Leopard 可以在 PowerPC 和 Intel Mac 上运行，但它需要一个最低时钟频率为 867MHz 的 G4 处理器和至少 512MB 的 RAM 才能安装和运行。它是最后一个支持 PowerPC 架构的操作系统。

Apple 的 Time Machine 备份软件、Spaces 和 Boot Camp 预装在 Leopard 中，并支持 64 位应用程序。新的安全功能和另一种新外观使操作系统更加完美。

Mac OS X 10.6, 雪豹 (2009)

Snow Leopard 是光盘上可用的最后一个操作系统。未来的更新是通过 Mac App Store 安装的，它是在 Mac OS X 10.6.6 中引入的。

Snow Leopard 在外观上没有太大变化，但它大大加快了 Finder、Safari 和 Time Machine 备份的速度，并且在完全安装时占用的磁盘空间比以前的操作系统少。

Mac OS X 10.7, Lion (2011)

在 Lion 中可以使用更多多点触控手势，包括让您快速访问新安装的名为 Launchpad 的应用程序导航器的手势。

Lion 还引入了 Mission Control，这是许多早期应用程序的统一体，让应用程序以与关闭时相同的状态打开。

Mac OS X 10.8，山狮 (2012)

Mountain Lion 深受 iOS 更新的影响。游戏中心和通知中心以及提醒、便笺和消息应用程序已添加到 Mac。

随着 iOS 更新，iCal 等应用程序更新为日历。所有这一切都通过 iOS 和 Mac 设备之间的更多应用同步来加入。

Mac OS X 10.9，小牛队（2013）

也许终于摆脱了大型猫科动物，Apple 将其操作系统命名约定转移到了加利福尼亚州的小牛队。

Mavericks 改进了电池寿命，并为 Mac 添加了更多 iOS 应用程序，如 iBooks 和 Apple Maps，也有更多的 iCloud 集成。

此操作系统更新是完全免费的，因为迄今为止所有 Mac 操作系统更新和升级都是如此。

Mac OS X 10.10，优胜美地 (2014)

凭借 Continuity 和 Handoff 功能，Yosemite 看到了 iOS 和 Mac 设备之间的更多集成。用户现在可以在他们想要的任何 Apple 设备上接听电话和短信以及编辑 Pages 和 Numbers 文档。

iPhoto 和 Aperture 结合到照片应用程序中，与 iOS 照片应用程序相匹配，因为 Yosemite 的图形也与 iOS 7 的图形相匹配。

Mac OS X 10.11, El Capitan (2015)

El Capitan 改进和改进了功能，而不是添加新功能。这方面的一些例子是在 Apple Maps 中添加公共交通选项和获取更新 UI 的 Notes 应用程序。

macOS 10.12, Sierra (2016)

随着 Sierra 更新，Mac OS X 正式更名为 macOS。Sierra 看到 Siri 和 Apple Pay 出现在 Mac 上，以及 iCloud 的改进，允许在具有相同 Apple ID 的 Mac 之间进行更多文件访问。

macOS 10.13, High Sierra (2017)

借助 High Sierra，Mac 现在可以支持 HEVC 视频和更多形式的 VR。更新了许多应用程序，Apple 将 Mac 转移到 Apple 文件系统 (APFS)，同时还引入了 Metal 2 API。

macOS 10.14，莫哈韦沙漠 (2018)

暗模式和动态桌面随 Mojave 一起出现，允许 Mac 的图形根据一天中的时间变化。此处还介绍了用于桌面组织的堆栈。

macOS 10.15，卡特琳娜 (2019)

Catalina 将 iTunes 拆分为音乐、播客和电视应用程序，以及重新设计的应用程序，如书籍和查找我的。它还引入了 Sidecar，允许用户将iPad 用作Mac 的第二个屏幕或图形平板电脑。

macOS 11，大苏尔 (2020)

当 macOS 更新到 Big Sur 时，停止对 32 位应用程序的支持，使一些旧的应用程序不再可用，或迫使用户更新到更高版本的应用程序。

它还最终将 macOS 的版本号从 10 更改为 11，这一变化似乎还在继续。

macOS 12，蒙特雷（2021 年）

Monterey 包含大量新功能，例如SharePlay和 Universal Control，并将快捷方式引入 Mac。测试版于 2021 年 7 月发布，它似乎比以往任何时候都更加团结 iOS、iPadOS 和 macOS 设备。

多年来，苹果电脑已经出现了大量不同的操作系统版本。它们的许多变化和发展反映了它们制造时计算机的状态，并显示了未来可能发生的变化。
