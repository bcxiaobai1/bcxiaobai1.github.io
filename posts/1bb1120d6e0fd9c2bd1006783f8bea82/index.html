3.6初始化页面样式

在app.wxss中 

/**app.wxss**/
@import "./styles/iconfont.wxss";

/* 在微信小程序中不支持通配符 * */
page,view,text,swiper,swiper-item,image,navigator{
  padding: 0;
  margin: 0;
  box-sizing: border-box;
}
/* 主题颜色 通过变量来实现
1 less 中存在变量这个知识
2 原生的CSS和wxss也是支持变量
 */
 page{
   /* 定义主题颜色 */
   --themeColor:#eb4450;
   /* 定义充一字体大小 假设设计稿大小是375px 
   1px=2rpx
   */
   font-size: 28rpx;
 }

 在那个页面要使用自定义主题颜色，就在那个页面的wxss中

{
  /* 使用主题颜色 */
  color: var(--themeColor);
}
