钱包追踪分析的 3 个使用案例

Nov. 2022, Vincy

Data Source: Footprint Analytics - Wallet Profile

钱包跟踪分析让分析师了解区块链用户的链上活动和持仓情况。 在本文中，我们将介绍钱包分析器发现的一些指标。

Footprint Analytics - Wallet Profile

Footprint Analytics  从中挑选相对比较活跃的 0x81e4xxxxxxb2e92 钱包地址为例。

  1. 投资组合统计及指标 

通过钱包地址查看各个用户的投资组合，可以观察其建仓、调仓以及持有等情况。该指标能助你快速了解大玩家在不同项目以及不同公链上的玩转。包括的指标：

  • 个人资料标签（鲸鱼、收藏家等） 

  • 购买和销售量

  • 投资回报率 

  • 盈亏情况

  • 持有价值 

据 Footprint Analytics 仪表板指标显示，该钱包地址的投资组合中含有 Game、元宇宙和 NFT 领域。游戏中持有的 NFT 比 NFT 项目中的多。

Footprint Analytics - Holding Number by Type

这个投资组合向我们展示了钱包内的 NFT 持有量，我们可以看到他们有哪些收藏、给定收藏的数量以及 ETH 的估计总价值。Decentraland LAND 当前持有的价值为 106 万美元，共有 777 个 NFT。其他 Gods Unchained、The Sandbox 和 Somnium Space 持有的比例非常小。很明显，这个用户是一个虚拟世界和卡牌游戏类型的爱好者。

此外，我们在 “4 个指标分析 NFT 收藏品的供需情况” 定义将拥有 100 万美元以上资产的地址标记为鲸鱼。因此，钱包分析器显示该用户是大鲸用户，因为他目前持有的价值超过 100 万美元。他们还通过出售游戏内的 NFT 赚取可观利润。

Footprint Analytics 是一个覆盖GameFi，DeFi，NFT 多领域的一站式链上数据分析平台。它帮助开发者，分析师以及投资者通过链上数据去获取行动指标。

目前，Footprint Analytics已经解析、清晰以及抽象整理了超过19条公链的数据，让用户可以通过使用拖拉界面以及 SQL 或者Python建立分析图表。无需编写代码即可入门的分析体验是Footprint Analytics 区别于其他链上分析工具的独有特色。平台还提供一个统一的数据 API，让用户可以快速检索多链生态的 NFT，GameFi 以及 DeFi 数据。
