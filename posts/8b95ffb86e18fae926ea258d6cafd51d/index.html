再见 Typora！这款 Markdown 神器绝了！

公众号关注 “GitHubDaily”

设为 “星标”，每天带你逛 GitHub！

大家好，我是小 G。

上个月，Typora 正式发布了 1.0.0 版本，并开启付费模式，价格为 89 元，可用 3 台设备，提供免费升级。

当时消息一出，便在技术圈引起了热议。有支持的，也有反对的。

绝大部分人吐槽的，主要还是围绕 Typora 1.0 版本的更新内容以及收费这两个点。

跟先前的 0.11.18 测试版相比而言，Typora 此次更新并没有带来太大改动升级。以致于我当时想写一篇文章，聊下新版本的更新，都无从下手。

还有一些人开始喷，Typora 吃相难看。我觉得，大可不必。

Typora 作者自产品发布以后，并没透露过产品会终身免费开放的计划。

在技术圈中，一款免费产品，如果早期需要流量和口碑来持续传播，那 GitHub 是个很好的平台。

Typora 作者没选择这么做，应该也是一早就考虑到后续的商业化。

目前，我认为 Typora 的付费用户，大体上分为两类人：

  1. Typora 曾经解决了自己的痛点，将此次付费视为打赏捐赠的；

  2. 认可 Typora 后续产品升级动向，并愿意为此付费的。

我个人，属于第 1 种。

因为在早期运营公众号时，我需要一款 Markdown 编辑器，来完成自定义排版样式，快速上传图片到图床，一键粘贴到公众号编辑等工作。

所以，我选择了Typora。

现在你们看到的这篇文章排版，就是我自己用 CSS 写的样式，在 Typora 排版好，并粘贴到公众号的。

Typora 唯一比较难受的点是，快捷键支持的太少，在进行多行编辑和多个相同词编辑时，对用户很不好友好，恰巧这个操作也是我使用最为频繁的。

在聊完 Typora 的优缺点后，我们再来谈一下付费的问题。

作为 Markdown 编辑器的重度用户，我试过很多同类型产品。比如 Sublime Text、VSCode、MWeb、Bear 等等...

Bear，是在我未用 Typora 之前，一直在用的一款 Markdown 编辑器，也是订阅付费制的，每年要从我 Apple 账户中扣掉 103 元。

现在用的很少了，还在持续付费的主要原因，是早期在这款软件上存了很多笔记。迁移成本太高，也就没去取消它的订阅套餐。

当前阶段，在进行公众号文章排版编辑时，我还是会用到 Typora，因为在这一使用场景上，它很好的解决了我的产品痛点，所以我也付费了。

不过，除了 Typora 之外，GitHub 上也有很多高质量的 Markdown 编辑器，均免费开源给开发者使用。

相关文章，我之前写过很多，今天我再重新做下整理，跟大家分享一下。

如果你不想付费的话，可以看看下面这几款，GitHub 上比较高颜值的 Markdown 编辑器。

这些是在我看了二十多个项目，做了充分调研之后，筛选过滤找到的高颜值、用户体验佳的 Markdown 编辑器。

Pine 清新优雅的编辑器

适用平台：macOS

这是一款基于 Swift 编写的 Markdown 编辑器，拥有主题定制、LaTeX 编写、自动补全、自动保存、版本控制、Touch Bar 支持等功能。

通过其设置界面，你可以很方便的挑选自己喜欢的文本字体，界面主题：

GitHub：https://github.com/lukakerr/Pine

目前还有以下几项特性仍在开发中，感兴趣的同学可以保持关注：

  • 支持编辑 CSS 文件来定制 Markdown 预览效果

  • 优化侧边栏

    • 文件夹监听同步

    • 上下文菜单中加入更多操作

Gridea 无缝对接静态博客的编辑器

适用平台：macOS、Windows

这个项目是国人开发的，19 年的时候介绍过，也火过一段时间。

作者将其产品定位为一个静态的博客写作客户端，也即是说，你可以在客户端上面写作，然后顺便将文章发布到静态博客上。

GitHub：https://github.com/getgridea/gridea

相对于其它普通的 Markdown 编辑器，它有着以下特色：

  • 对接了 GitHub、Coding Pages 的托管服务

  • 内置 Gitalk 和 DisqusJS 两个评论系统

  • 支持使用 OneDrive、Dropbox 等工具进行文章同步

  • 支持外链的封面图

从上面列举出来的几项特性可以看到，Gridea 相对于普通的 Markdown 编辑器更加强大，如果你希望写作编辑与文章同步能无缝进行，不妨试下这个编辑器。

另外作者还表示，Linux 版本正在开发中，你可以保持持续关注。

Notable 无限标签编辑器

适用平台：macOS、Windows、Linux

这是一款基于 AGPL 协议开源的 Markdown 编辑器，由于其界面清新、代码开源而受到不少程序员喜爱。

GitHub：https://github.com/notable/notable

主要有着以下几项基础功能：

  • 无限标签嵌套

  • 可导入印象笔记

  • 文章分类

  • 模糊搜索

  • 语法高亮

  • 多文章编辑

  • 多文章搜索替换

  • 支持主流桌面平台

Mark Text 多模式形态编辑器

适用平台：macOS、Windows、Linux

这是一款基于 MIT 协议开源的 Markdown 编辑器，支持 WYSIWYG（所见即所得）的编辑方式，致力于给你提供一个专注的写作环境。

GitHub：https://github.com/marktext/marktext

该编辑器还内置了插件功能，你可以为其添加 KaTeX 数学公式、emoji 表情等插件扩展。

在你编写完成后，还可以很方便的将 Markdown 导出为 HTML 或 PDF 文件。
