今年的计算机专业应届生居然也找不到工作了？

大学

“青春才几年，疫情占三年。”

PS：点击跳转文末【大彩蛋】

好似童话般美好的大学生涯，最多的，是数不清次数的核酸排查，是学校里日复一日的消毒严查。

没完没了的网课和健康打卡，每天戴口罩，打包回宿舍吃饭。

疫情以来，大学校园纯净许多，也冷清许多。

大学也就那几年，疫情就占三年，原本安排的所有设想都被打乱，不少大学生以为大学生活是他们人生中最美好的快乐时光，却要在无止境的麻木失去自由中度过。

记得19年的那个冬天，随着全国疫情的发生，临近毕业的大学生们，还没有来得及拍一张毕业照，就匆匆地离开了校园。回过头来，匆匆已是三载，不少大学生的大学生活已经结束或是面临结束。

大学生活结束了，可是来自生活的挑战还要继续…

求职

2020年全国普通高校毕业生人数874万，同比增加40万，2021年达到909万，2022年，也就是今年，预计将首次突破1000万。

此外，由于疫情持续的原因，留学生回国求职人数也不断增加，这使得大学生就业成为社会热议的话题。

可以说这届毕业生面对的将是前所未有的就业环境，甚至可以用残酷来形容。

“焦虑已经成为我们的日常。”

这是来自某双一流高校一名大四学生小吴的真实心声。

过去大部分同学都认为互联网大厂工资高，待遇好，有发展前途，一段时间以来一直是毕业生争相竞逐的目标。

经过前几年的野蛮发展，现阶段进入到了一个瓶颈阶段，发展速度慢了下来。

最近传的比较热热闹闹的就是美团和京东的裁员，另外阿里拟裁员30%，腾讯将裁员15%~30%。

像阿里、百度、滴滴这些互联网巨头部分业务都出现了一定程度的亏损，如果说这样的企业的日子也不好过的，那么那些小微企业更可想而知了。

面对以上日益变化的环境和未来源源不断的挑战，作为应届毕业生的同学们该如何自救呢？

首先，你需要充分了解自己。你真的想好自己适合做哪些方向的工作了吗？

因为语法简单，功能强大，Python 的应用越来越广泛，例如：网络爬虫、自动化办公、财务自动化、数据分析、人工智能应用等热门领域。

在 Python 的各类应用中，人工智能无疑是近几年的爆点。基于人工智能设计的自动驾驶汽车，可以自己检测周围的汽车和行人，根据天气调整驾驶状态，甚至自动规划路线！

在可以预想的将来，人工智能将占领越来越多的领域，彻底改变我们的生活。Python 是人工智能领域的主流编程语言，它方便、灵活，是很多程序员的真爱。

近两年来的另一个技术热点是大数据分析。我们每个人都生活在大数据时代：想听歌了，打开音乐软件听听“每日推荐”；忙累了想放松一下，打开抖音总能刷到有趣的视频……难怪有人会说：“推荐算法比我爸妈都了解我”。

为什么推荐算法总能正中红心？Python 发挥着举足轻重的作用。它从数据海洋中挖掘到需要的信息，再通过大数据分析的手段帮你找到感兴趣的东西。

回到日常生活中，Python 也在各个细微的角落里给我们带来各种“小确幸”。因为 Python，很多人的生活发生了意想不到的变化。

我的朋友小瑶从事财务工作，每天都会有巨量的 Excel 表格要处理，每到月末工作量就会成倍增长，加班成了常态。在我的推荐下，她学习 Python 的基本操作，自动化处理 Excel 表格，运行起来的效果也是十分的酷炫：

代码的运用，让她每天的表格处理时间从 3 个小时变成了 3 分钟，“光速”完成工作任务。这样，她就可以抽出更多的时间去学习、去提升自己了。

还不清楚能用 Python 干什么吗？没关系，各行各业的人，不管是学生还是工作党，都可以用 Python 给自己的生活带来一点惊喜。下面是我们以往课程的学员中，成功将 Python 运用在生活中的例子：

  • 大二学生小杨，用 Python 写了一个背单词的工具，用爬虫技术在网上抓取词汇的释义和例句，检查单词默写情况，并自动制定复习计划，成功拿下了六级 600+ 的高分；

  • 高中老师小朱，用 Python 写了一个学生信息统计工具，自动录入每次学生的作业完成情况与考试成绩，分析每个学生是否有进步，总结学生的优势和劣势，更好地对每一位学生因材施教；

  • 淘宝店主小王，自从学了 Python 的爬虫技术，就将网上友商的商品在不同时段的价格都下载了下来，并使用 Python 的数据分析模块分析各个商品的价格变化规律，以便自己制定出更加有竞争力的价格策略；

  • 猫奴小袁，在家养了一只不太听话的喵星人，便用摄像头观察自己家的小猫的一举一动，还特地写了个 Python 程序来分析摄像头传来的图像，判断猫咪有没有在沙发上捣乱，如果猫咪胆敢乱来，就自动让音箱播放自己录好的声音去阻止……

可以说，只有你想不到，没有 Python 做不到。Python 可以让我们尽情发挥自己的想象力，帮自己解决各种问题。

可以通过招聘网站、企业官网、校招等渠道，大量搜集目标岗位的招聘信息，关注岗位的能力要求、岗位职责、薪资、发展空间等资讯。

这样有策略地进行求职，可以避免在面试技巧还不成熟时，盲目试错，错失好机会。

在此过程中，我们也要积极准备岗位所需要的基本技能。以目前自身的能力现状，与目标岗位之间或多或少会有能力差距。

为了在求职过程中更加具有主动性和竞争力，提前做好必要的技能储备是关键。

根据不同职位的岗位描述、能力要求，可以锁定学习目标。
