NFT 离商业化还有多远？
《柯林斯词典》2021 年度的热词榜第一名：NFT。

也是 2021年，大家称其为 NFT 元年。

NFT 商业化离不开其所凭借的区块链技术支持，也在元宇宙、DeFi 等赛道的加持下不断焕发生命力。但是伴随商业版图的拓展，技术、存储、法律等方面的问题却成为了限制 NFT 商业化的关键，本文将探索 NFT 商业化发展的历程，感知商业化视角下 NFT 的繁荣业态。

NFT 商业化模式的崛起
NFT，是指非同质化通证，即基于区块链技术的非同质化数字资产。

那么 NFT 商业化又是什么呢？

**简单来说，NFT 被用来标注数字内容作品所有权。**这些内容作品有了价值标记后具备了相应的价格，进入商业转化的渠道。因此，当一个作品被打上 NFT 的标记后便被赋予了独一无二的独特编码，**成为商业交易的对象。**NFT 让原先可被大批量复制的内容具有稀缺性，被无需允许地在去中心化交易平台上自由交易、转售，这就是 NFT 商业化的基本逻辑和概念。

经过这几年的探索与发展，NFT 已基本形成了初步的盈利流通思维。以 NFT 开发者为主轴，通过数字货币交易实现产品的买卖交易，同时 NFT 拆分/抵押平台作为关键节点刺激数字资产流动和产品迭代，为创作者和用户的商业互换提供支持。

那么从方向上来看，NFT 生态系统主要通过直接出售、二级市场交易手续费、游戏内部经济交易手续费三种方式盈利，驶入商业化发展的快车道。

直接销售是 NFT 领域最常见的盈利方式，收入来自向用户销售数字商品。

二级市场交易手续费是从开发的物品二级市场交易中收费。

游戏内部经济交易手续费是从用户生成的 NFT 交易中收费（ DeFi+NFT 主要就是通过这种方式促成盈利机制）。

据相关数据统计，2020年全球 NFT 商业化市场前三大应用的市场规模分别为 1400 万美元、1290 万美元与 1290 万美元。然而，NFT 的商业化发展不仅仅在于其赋予内容交易的便捷和直接，还在于开辟了 NFT+ 社交、游戏等不同赛道的商业版图，更在于搭载区块链平台的优越特质成为**深耕 Web3 商业发展、**拓宽 Web3 商业模式的新星元素。

NFT+ 商业赛道空前繁荣
当 NFT 承载盈利属性进入各大产业链时，它也凭借其盈利模式和不同行业形成了多样的商业赛道。整体而言，NFT 产业链包含上游基础设施层、中游项目创作层和下游衍生应用层。

基础设施主要为 NFT 的铸造标记提供底层支撑；项目创作主要是 NFT 艺术藏品、NFT 游戏/元宇宙，通过创作者和消费者之间的一级交易实现商业营收；衍生应用主要是 NFT 社交、NFT 数据交易平台，以数据流量影响力实现商业变现。

在这些分赛道商业版图中，NFT+ 助力各大行业发展，带来新的商业机遇。以娱乐赛道为例，2020 年音乐 NFT 平台 ROCKI 上线，不过两年时间，平台估值高达 2100 万美元并聚集了万名独立音乐人；2022 年，Water & Music 社区确定推出近 30 个以音乐为主题的生成 / PFP NFT 项目，向 Web3 时代的音乐商业模式发起探索。

截止 2022 年 5 月，国内上线的 NFT 交易平台也多达 500 家，其中不乏一些头部互联网大厂的身影：腾讯（TME 数字藏品）与阿里先后与中国唱片集团达成合作，主流音乐产品进一步向 NFT 靠拢；网易云音乐与网易天音、网易星球联合发布网易首批音乐 NFT。《音乐 NFT 为什么火了？Polkadot 或将成为发展音乐 NFT 的最佳选择》

**此外，NFT + 元宇宙也成为各大投资方和商业人士的关注焦点。**2021 年开始，各大品牌陆续抓住了利用 NFT 销售数字商品和建立品牌资产的机会，Decentraland 举办元宇宙时装周，集合虚拟体验和时装秀，活动参与者可以数字货币购买时尚单品并获得独特的 NFT 纪念品；Dolce&Gabbana 推出四种纯虚拟设计造型，附带 NFT 以 570 万美元的价格售出，进一步塑造时尚品牌在用户心中的高贵专属感。

将元宇宙和 NFT 深度融合无疑是进一步盘活数字资产，强化与用户对 3D 虚拟世界商品贸易流通新认知的重要方式。这一发展趋势也使得 NFT 作为一种技术手段，助力元宇宙赛道商业版图的拓增与加速裂变，赋予项目创作和衍生应用层商业化发展更多机遇。

NFT 交易潜在风险与未知挑战
当我们看到 NFT 商业化发展的无穷潜力之时，也不应忽视 NFT 商业化受到的限制和挑战。

**NFT 商业化得以发展的基础设施来自区块链技术支撑的大规模基础设施。**无论是公共服务层、技术协议层，还是数据共识层都需要区块链对 NFT 交易提供可承载的技术支持。当越来越多的企业、项目、用户涌入 NFT 交易渠道和空间，无形也会增大区块链技术平台的使用成本和能耗，带来更高的技术发展要求。

除了基底技术的资源供给限制，**NFT 本身的商业交易所有权问题一直被业内人士所诟病。**目前 NFT 商品交易的物品权利类型主要包括三种：“所有权”而不包含任何著作权或著作权使用许可、“所有权”+著作权使用许可、“所有权”+著作财产权。在所有权模式下，购买者因不具备对应数字作品的著作权而无法进行商业使用，难以激励使用者后续传播行为。同时，目前尚未出台相应的法律明确在 NFT 技术背景下可以赋予数字作品所有权，相应的创设、流转、质押等配套规则的空白，使得大规模的 NFT 交易流转缺乏完善的法律保障，造成用户的不安与恐惧。

在法律权益的主体性确认之外，**用户可能还会面临 NFT 突然消失的问题。**用户购买 NFT 多数只是获得了平台的链上验证，而储存 NFT 数据的两种方式——HTTP 中心化存储和 IPFS 分布式存储，都有可能在服务平台破产倒闭、数据服务故障等情况下出现信息丢失。一旦用户大量购买、流转的 NFT 遭遇第三方平台“跑路”的情况，将会造成个人或企业的巨大商业损失，而其中的追责补偿又因法律机制的漏洞而难有后文。

不难发现，NFT 商业化版图若想获得持续发展的生命力，亟需解决技术容纳、法律确权和存储优化的优化升级问题。

携手共创 Web3 商业化图景
纵然 NFT 商业化发展面临技术、法律、权益挑战众多，但是伴随越来越多的 NFT 应用场景落地，NFT+ 已然成为 Web3 时代发展的重要赛道矩阵。不过，NFT 商业化如何在各大赛道进一步实现用户、项目下沉是发展的关键，让 NFT 成为企业品牌营销、客群营销的新手段，连接 B 端和 C 端的消费需求，或许是我们展望 Web3 时代 NFT 商业化应用的新方向。

NFT 的商业化发展依然任重道远，我们期待着 NFT 能够携手元宇宙、DAO 等理念不断优化和精进自身的商业体系和运作思路，进一步弥合与法律法规机制之间的沟壑，在满足平台用户数字资产交易的需求之外，不断扩展付费体验、艺术社交等商业版图，共创 Web3 时代的商业化图景。
