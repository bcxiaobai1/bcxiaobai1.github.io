1. Redis 客户端

Redis 给我们提供了三种客户端，分别是：

  • 命令行客户端
  • 图形化桌面客户端
  • 编程客户端

2. Redis 命令行客户端

在 Redis 安装完成之后，就有了 Redis 命令行客户端。即 Redis 安装完成后就自带了命令行客户端。

Redis 自带的命令行客户端就是 redis-cli。

2.1 redis-cli 的使用

redis-cli [option] [commonds]
  • option：选项
    • 其中常见的options有：
    • -h 127.0.0.1：指定要连接的redis节点的IP地址，默认是127.0.0.1
    • -p 6379：指定要连接的redis节点的端口，默认是6379
    • -a 123123：指定redis的访问密码
  • commonds：Redis的操作命令
    • 例如，ping：与redis服务端做心跳测试，服务端正常会返回pong
    • 不指定commond时，会进入 redis-cli 的交互控制台

2.1.1 进入 redis-cli 交互控制台方式一

连接 redis 时，直接输入密码连接 redis，进入 redis-cli 交互控制台

redis-cli -p 连接redis的IP -p redis节点的端口 -a redis访问密码

2.1.1 进入 redis-cli 交互控制台方式二

连接 redis 后，再输入密码，进入 redis-cli 交互控制台。

redis-cli -p 连接redis的IP -p redis节点的端口

AUTH [用户名] [密码]
