NFT 概念科普

作者概述：Zoey | NFTScan研究员，NFTScan是一个多链 NFT 浏览器。

2021年3月11日，一枚艺术作品——《Everydays: The First 5000 Days》的NFT，这幅作品由著名艺术家 Beeple 创作在佳士得拍卖行以 6,930 万美元（约合人民币 4.51 亿元）的竞拍价售出，也是有史以来最昂贵的艺术品之一，那么 NFT 是什么呢？

一．NFT的概念？

NFT 是 Non-Fungible Tokens 的缩写，意指非同质化代币，即为“不可互换”代币的意思，那有“不可互换”代币就有“可互换”代币。所谓“可互换”代币即同质化代币（Fungible Tokens），那什么是同质化代币？如我们常听到的比特币（BTC）、以太币（ETH）这些代币都是同质代币，所以相互之间是可互换的，每一枚 ETH 都是相同的，也就是说，一枚 ETH 和另一枚 ETH 可以任意交换，具有相同的属性、价格。

而 NFT，也是属于加密货币的一种，它们每一个都是独一无二的，每一片雪花的纹理都不一样，它们之间无法相互替换，NFT 具有独一无二、稀有、不可分割的属性。

2017 年出现了一个叫 CryptoPunks 的项目，整个系列有一万个像素风格的朋克人物，每个人物都是独一无二各有特点的，拥有以太坊钱包的人都可以免费领取，这便是第一个 NFT的由来。在部分 CryptoPunks 形象以数百万美元出售之后，目前已成为最昂贵的 NFT 之一。

 

二．为什么 NFT 有价值？

在 NFT 出现之前，对于广大网民来说，数字化资产更多时候是一个可以随意复制的文件。那么大家可能会有疑惑，我拥有一张图片，一短音频，一个视频发布到网上被人大量复制，我怎么才能说明我是这个东西的持有者呢？原创者的权益如何保护呢？那么你可以把 NFT 当做你数字资产的“身份证”，在区块链网络中，每一个 NFT 都有唯一且确定的权属证明。

独一无二性：大部分 NFT 一次只能有一个所有者，通过算法根据一系列特征生成，且带有一种存储丰富元数据的机制。所有权通过一个唯一的 ID 以及其他代币无法复制的元数据来管理，它存储了资产和所有权详细信息，因此，可以保证NFT的来源与出处，它的自身特性使得它是唯一的，具有不同的稀缺性。

稀有性：所有的创作者都可以自己决定哪些某种艺术形式作为资源，并且可以自己决定限制其数量，用确保这个作品的稀缺性。能够吸引用户购买的就是NFT 特有的稀缺性，这样的机制能够稳定和保障市场，不会出现供不应求，也给用户手上的数字资产带来更长远的发展。

不可分割性：顾名思义 NFT 的作品不像其他加密货币一般可以切割成多个小单位，NFT 以“整体”的形式来进行交易。

另一方面，由于 NFT 存在于区块链上，因而更难被操纵和篡改，每一个 NFT item 下都能做找到每一个曾经持有它、出价或转让它的地址，从而追溯其原件。NFT 的独特数据使得验证真实性和所有权变得容易，且NFT 作为数字资产，在线上可以轻松实现商品所有权兑换和交换。

三．NFT 有哪些分类？

1.个人头像（Profile Picture）

即  PFP，用来代表自己网络身份的数字化图片，还可以作为类似 pass 的功能，即一张能进入该 NFT 项目社区的“门票”，用于社交网络的头像，全球各路明星、名人也参与其中。例如库里、阿姆的都买入过 Bored Ape Yacht Club，足球巨星梅西推出名为 “Messiverse” 的 NFT 系列，余文乐的 CryptoPunks，周杰伦潮牌与 Ezek 平台联名推出的 Phanta Bear NFT 等。

 

2. 加密艺术

艺术作为 NFT 卖得最好的类型之一，也受到了大量艺术家的推崇，给到展示了他们作品的机会。其实艺术涵盖了很多领域，图像、照片、音乐、视频都可以被称为艺术藏品。如文章开头提到的艺术家 Beeple 、艺术家 Chris Torres 创作的一部视频 Nyan cat， Opensea 是艺术家展示其 NFT 作品主要平台。

 

3. 链游（Gamefi）

2017 年上线了一款名为 CryptoKitties（加密猫）的基于以太坊链上智能合约小游戏，引起了热潮，每个玩家可以拥有虚拟猫咪，可以购买、出售、驯养后代，除金钱交易外所有贩售及驯养过程皆为公开。实际NFT应用最早也在游戏上，目前，CryptoKitties 仍是区块链游戏的一座里程碑。

 

4. 会员（Pass）

类比于传统 Web 2.0，Web 3.0 用户的隐私、数据处理和所有权妥善的得到了处理。当然在 Web 3.0 领域里，真实身份是隐藏的，这时候你持有的 nft 资产就是一个媒介来证明你的身份。除了上面提到的 Bored Ape Yacht Club，Proof collective NFT 和 0xstudio 是两个非常出名的俱乐部。持有 Proof collective 俱乐部 Pass 卡的都是一些业内精英，俱乐部一共持有超过15 万个NFT，且蓝筹持有数极高持有人享受社区带来的权益当然不计取数。如 Moonbrids 发布后，该社区的玩家可以直接认购两个作为福利，羡煞旁人。

 

5. POAP

POAP 又名出席证明协议，是由 NFT 形式的加密记录支持的数字徽章，每个徽章都具有独特的设计，赋予他们可收藏性，其创建目的是为了记录和纪念活动出席。 POAP NFT 通常被称为“承载记忆的生态系统”，NFT 社区有许多专业活动，有了 POAP 能有效地证明和验证出席情况。在不久的将来，人们通过获取 POAP 收藏品，即可在区块链中同时记录自己的人生经历。

 

6.域名（Domain Names）

域名作为一种命名方法，可以用于代替难记的 IP 地址，现在可以通过区块链网络以 NFT 的形式购买域名，其通过智能合约的方式连接到区块链，用来作为新的网络拓展名，例如 .eth 或 .crypto，类似于.com。与传统域名不同的是，传统域名从根本上意义上来讲只属于注册商，而如果在 NFT 市场上购买一个域名，将能够获得该名称的独家所有权，无需中间商。NFT 域名不仅可以像 NFT 一样交易，其功能也与加密钱包地址类似，提供了可读性、安全性、隐私和所有权。

 

8.音乐（Music）

起初 NFT 是静态的，随着 NFT 市场的发展 NFT 也逐步多元化，从静止图像到 Gif 及 Mp4 的演变。2020 年夏天 Studio Nouveau 发行的第一张 NFT《Audiovisual》，由十个音乐视频组成的专辑，虽然每一个人都可以聆听，但所有权只属于一个人。一些著名的歌手也发布过 NFT，比如国饶舌创作歌手 Lil Pump、Eminem、中国台湾歌手蔡依林。在音乐行业，区块链创建的透明体系，可以追踪歌曲所有权、数字资产等，版权将自动分配给为专辑做出贡献的人。

 

9.摄影（photography）

摄影是通过将照片上传到区块链上，将照片铸成 NFT，这使摄影师有能力将照片代币化，并要求将这一特定照片的所有权作为数字资产，在区块链上的买家之间转让，使得 NFT 摄影高速增长。虽然大部分照片视频都只通过屏幕观赏，但通过佳士得和 The State Hermitage Museum 都在拍卖 NFT，说明 NFT 摄影不只是一段时间的热潮。

 

10.体育（Sports）

在美国职业篮球联赛 NBA 与 Dapper Labs 联手打造的区块链收集游戏平台 NBA TOP SHOT上，NFT 球星卡一经发行就火速引爆市场，参与买卖 NFT 球星卡的用户数更是不计其数。这款使得球迷通过游戏化方式购买和售卖正版 NBA 球星高光集锦的 NFT 收藏游戏，为球迷提供 NBA 比赛中精彩视频片段的市场交易平台。随着区块链技术的发展，美国职业橄榄球大联盟（NFL）、美国职业棒球大联盟（MLB）等体育联盟能够借助各类 NFT 项目，构建体育系列的 NFT 生态系统，从而取得经济效益。

 

11. 元宇宙（Metaverse）

元宇宙作为热门话题，其含括很广泛，那么与 NFT 相关的元宇宙有哪些呢？最有代表性的便是 NFT 虚拟土地。NFT虚拟土地是元宇宙平台上的归属自己所有的数字土地区域，备受欢迎的 NFT 土地项目有 Decentraland、The Sandbox。元宇宙项目作为一个数字世界，不仅用户可以拥有虚拟身份，还可以为链上一些活动提供场地，一些web2.0品牌也逐渐跨入，例如：The Sandbox 稍早在官方推特上宣布品牌 Adidas Originals 加入 The Sandbox 元宇宙行列。

 

四．NFT 的应用

由于 NFT 的属性，可以有效解决 web2 很多目前无法解决的问题，随着发展，NFT 的应用也逐步渗透到我们实际生活中，NFT是技术变革，能为传统行业赋能，生活逐步数字化，进一步拉近了 web3 与现实的联系。那么 NFT 能应用到现实哪些方面呢？

  1. 身份证明：比如出生证明、身份证明，如今社会与个人身份紧密相连，但难以追溯历史，而在身份中引入 NFT 可以通过允许安全机构随时验证任何人。

  2. 知识产权：所有权物体通常是有形物，而无形财产则往往通过知识产权来获得保护。而 NFT 的出现使得这些无形财产有了独一无二的代码，每一个独一无二的东西可以进行版权刊登，识别专利。

  3. 合同文书：便于及时保存、验证真伪，还能够有效溯源、防篡改等，强化合同的应用和法律效力。

  4. 票据：如演唱会门票、电影票甚至奥运会门票等，门票将使用智能合约来实现控制，并为持有者增加价值，且无法篡改，轻松查验。

  5. 会员卡：任何有价值的品牌都可以用 NFT 来标榜它的价值，其持有者可以解锁各种功能享受品牌带来的权益，并且可以通过链上记录轻松进行身份验证。

  6. 房地产：转让房地产所有权涉及大量文书工作和活动， 引入像 NFT 可以随时随地验证所有权，并且 NFT 作为点对点的交易方式，无需中介的介入。

  7. 教育行业：使用 NFT 在区块链上直观可查教育和历史，学校将不再需要使用纸质的证书，可以直接铸造毕业生的证书。

  8. 股权：将股权或特定资产 NFT 化，用 NFT 封装，进行上链和数字化流通。据悉，数字资产银行 Sygnum 已帮助泰国媒体巨头环球媒体集团（T&B Media Global）通过“股权-NFT”混合架构完成 3 亿美元的融资。

  9. 游戏：在传统游戏里，即使购买了游戏内皮肤，皮肤的所有权依旧属于运行商，而 NFT 的特性完全可以实现消费者的权益，它能确切的证明这个东西是我“一个人的”。

NFT 市场还有很多东西有待发现，并且随着时间的推移，NFT 市场仍在继续增长，NFT 的出现让我们看到了更多领域的价值产生和可能性，就像营销鬼才孙“割”说到，当所有人都在疑惑一个 NFT 凭什么这么贵的时候，更应该去理解的是，为什么一个 NFT 这么贵。目前 NFT 市场持续火热，NFT 不仅推动了产业数字化且在 NFT 带来的效应下，不能否认 NFT 在某些领域带来的价值与意义。未来 NFT 是否能成为一个趋势，未来的产业领域会进一步拓展，一切皆可 NFT。
