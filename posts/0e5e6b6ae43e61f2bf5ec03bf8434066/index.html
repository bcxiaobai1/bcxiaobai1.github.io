修改召唤神龙

最近玩召唤神龙有点上头，但是经常在快成神龙的时候一口回到解放前，实在气不过所以就试着修改一下源代码。

0x01

用谷歌浏览器打开召唤神龙游戏界面

点开network查看网络请求，发现有index.*.js这几个文件比较特别

经过观察，index.3786c.js文件中包含游戏的主要逻辑

在本地新建一个js文件，然后将index.3786c.js的代码复制到其中，用编辑器打开

0x02

这里我们需要两个插件:

  • 谷歌插件ReRes

  • vscode插件live server

ReRes的作用是将请求的文件映射为本地文件

live server的作用是在本地启动一个服务器用来预览文件

Launch a local development server with live reload feature for static & dynamic pages.

在vscode下载好live server之后点击此处打开预览（初次安装需要重新打开vscode）

点击js文件

之后复制该网络地址

在谷歌插件中打开ReRes,之后将index.3786c.js的请求路径换为本地的文件路径

index.3786c.js的路径查看方法

之后游戏加载时就会运行我们本地的js文件，我们就可以修改游戏的逻辑了。

0x03

游戏的js代码写的比较的简单，经过观察在judgeIsChange()这个函数这里可以修改进化。

做如下修改，之后刷新游戏页面，就可以在第二阶段变成小青龙?纵横天下了。
