前言

我根据知乎上一篇博主的教程来进行安装UE4和Airsim，期间遇到了一些问题，在这里总结并给出可能的解决方法

教程：（解密AirSim-初稿）第一章：初识AirSim - 知乎 (zhihu.com)

UE4下载慢

UE4是在Epic平台上的，而这个游戏平台在几家游戏平台里面虽然不是最慢的，但是速度也不稳定，我连接校园网有时速度可以达到2M/s，有时候就跟断网了一样。我也在网上查询了许多加速下载的方法，比如物理断网再插回去以及各种奇淫巧计，可能不同的网络环境以及不同的设备所适用的并不相同。

解决方法

这里记录一下我的解决方法。首先要注意的是不要动不动地就退出Epic，然后重新打开，这样会导致已经下载的会被重新扫描一次，速度也并不快，属于是白白浪费时间。但是一直下载速度极慢，也是可以这样做的。

0.重启电脑；

1.电脑连接手机热点；

2.彻底关闭Epic，打开加速器，我用的是雷神加速器（但我觉得效果一般）；

3.打开EPic，下载UE4，我下载的是4.27版本；

4.切换为WiFi网络，下载速度稳定在1~2MB/s，比之前半死不活的状态已经不错了。

ps: 好像晚上下载更加稳定，同时下载在内置硬盘中更快，我之前下载在外置SSD中了。

另外针对虚幻引擎启动不了的问题，我写的也有一篇回答，可供参考：ue5下载完打开加载到95％不动了怎么解决啊？ - 知乎 (zhihu.com)

Airsim编译过程中提示：无法打开包括文件: “Eigen/Dense”: No such file or directory [D:softwareVisual_studio20222022CommunityAirSimAir

在按照教程编译Airsim时，编译过程中报错，显示无法打开“Eigen/Dense”，在资源管理器中打开这个文件夹，发现“Eigen”文件夹是一个空的，于是搜索下载Eigen。

解决办法

eigen下载地址：eigen.tuxfamily.org/index.php?title=Main_Page#Download

下载Eigen库，并替换掉Airsim中的这个文件夹，下载要注意，本来我下载的是eigen3.4，但是问题没有解决，并出现了新的问题。

之后下载Eigen3.3.8后，问题解决。

Airsim编译过程提示：ERROR: Could not find NetFxSDK install dir; this will prevent SwarmInterface from installing. Install a version of .NET Framework SDK at 4.6.0 or higher.

提示我要下载.NET FrameWork。

解决方法

先是下载了.NET6.0.48，问题没有解决，看别人的博客，说是下载.NET Framework 4.7.2 Developer Pack，（要安装Developer Pack，而不是Runtime），问题解决。

下载地址：.NET framework 4.7.2下载地址，.NET framework所有版本目录

MSB3073 代码为6

这个问题是在启动Visual studio2022后编译Blocks环境产生的错误，我在网上搜索了好多都没有找到适合我的答案。为此Airsim我来回装和卸了好多次，也试了很多网上提出的答案，但都不行。

解决方法：

卸载重装Visual Studio，然后重试配置Airsim的过程

没错，就是这么让人无奈。运行成功了。
