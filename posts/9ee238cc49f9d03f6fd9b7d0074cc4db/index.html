曝 15 寸 iPad 或将变身 Mac？谷歌：大屏设备应具备智能手机体验

今年 6 月 Windows 11 正式官宣时，除了其“苹果味”的全新 UI，想必还有一点令许多人充满期待：支持 Android 应用。当然，此前利用各类 Android 模拟器也并非不能实现，但“原生支持”总归更令人安心，操作也更为便捷。

然而，为何微软决定在 Windows 11 中增加这一功能呢？这可能正如近期谷歌所表示的“人们渴望在大屏幕设备上获得他们最喜欢的智能手机体验”。但在这一点上，与谷歌、微软相比，苹果似乎有不同看法。

一、Android 应用在 Chrome OS 上的使用率提高了 50%

谷歌旗下，Android 和 Chrome OS 这两大操作系统在各自领域都拥有不容忽视的市场占有率。Android 系统无需多说——许多手机厂商的移动操作系统均基于 Android，近两年 Chrome OS 的发展势头也十分强劲：据今年 2 月 IDC 数据显示，2020 年 Chrome OS 的市场份额首次超过 macOS，成为全球第二大桌面操作系统（第一为 Windows）。在今年 5 月，谷歌官方也透露，相较去年同期数据，Chrome OS 的增长率已高达 92%。

要说 Chrome OS 与其他桌面操作系统不同的地方，除却简洁与速度，其自 2016 年起便宣布支持所有 Android 应用也是一大亮点。

这 5 年来，为进一步提升 Chrome OS 上的 Android 应用体验，谷歌付出了许多努力：曾在 2019 年 10 月表示，花在 Chrome OS 中 Android 应用上的时间同比增长了 4 倍；过去 2 年中，Android 应用和 Play Store 在 Chromebook 上的应用数量也在激增。

终于在昨日，谷歌分享道，今年 Android 应用在 Chrome OS 上的使用量同比增长了 50%，同时，目前全球有超过 2.5 亿台⁠活跃的大屏 Android 设备。面对这些数据的急速增长，谷歌认为这表明“人们渴望在大屏幕设备上获得他们最喜欢的智能手机体验”。基于此， 谷歌专为大屏设备推出了 Android 12L，旨在推动改善大屏设备上使用 Android 及其应用的体验。

二、明年目标：为开发者提供便利

为了总结这一年来在改进 Chrome OS 上的 Android 应用所做的工作，谷歌特为此发布了一篇文章进行概括，其中包括：

  • 推出 Android 11。稳步更新所有 Chromebook 以支持 Android 11，为确保更稳定、安全和高性能的用户体验，将 Android 应用从容器转移至虚拟机运行。

  • 改进 Android 应用的窗口管理。在支持 Android 11 的 Chrome OS 93 或更高版本中，专为移动设备设计的 Android 应用会自行在手机或平板电脑大小的窗口中运行，用户还可点击“调整大小”选项来调节窗口尺寸。而这一切更新，默认适用于 Play Store 中的大多数应用，无需开发者的任何操作。

  • Android 中的 Nearby Share ⁠功能。由于 Chrome OS 同时支持 Android 9 和 Android 11，目前运行 Chrome OS 96 或更高版本的设备中均可以使用 Android 的 Nearby Share 功能，方便与其他 Android 和 Chrome OS 用户即时共享文件、图片和链接。

  • Unity 支持 x86。谷歌与 Unity（一个广泛用于 Android 游戏的游戏引擎）合作，使不论是简单的纸牌游戏，还是 Chrome OS 的沉浸式 RPG，开发工作都能极大简化。借助 Unity 2021、2020 和 2019 LTS，开发者可支持基于 x86 和 x86_64 的 Chrome OS 设备并使用 IDE 来减少重复代码和简化发布。

除此之外，还有低延迟手写笔库、Android Studio 中可调整大小的模拟器、布局验证、适用于大屏幕的 Jetpack Compose 等相关更新，均在不同层面提升了 Chrome OS 上的 Android 应用体验。而下一步，即谷歌明年的目标，是希望能“为开发人员提供更多指导和易于使用的工具”。

三、传 15 寸的 iPad 或将支持 Mac 应用

在谷歌为桌面操作系统 Chrome OS 极力优化 Android 应用体验时，苹果却反其道而行：据上周彭博社记者 Mark Gurman 报道，苹果可能会生产一款配备 15 英寸屏幕的 iPad，而这个 15 寸的 iPad 可能将以某种方式支持运行 Mac 应用。

早在今年 6 月，Mark Gurman 就指出苹果正在开发大屏 iPad，但大概率不会在明年见到——明年苹果应该会推出新的 iPad Pro。Mark Gurman 透露，苹果正在考虑将这款大屏 iPad 做得更厚以支持更强大的扬声器——因此与其说这款 iPad 的竞争对手是 Android 平板，“智能大屏音箱”这个形容可能更为贴切。

而传说这款大屏 iPad 将支持 Mac 应用这点也令许多人摸不着头脑：虽说如果苹果想做，可能确实能使其很好地运行 Mac 应用，甚至可以像其计算机一样执行多任务处理，但这样的 iPad 似乎也就失去了其作为平板的意义。
