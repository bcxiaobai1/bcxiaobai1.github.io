在讨论不同类型的 AWS 认证及其认证考试路径之前，让我们先谈谈什么是 AWS 认证及其价值。

什么是 AWS 认证？

与任何其他云提供商相比，亚马逊网络服务提供了更多的服务和功能。亚马逊云提供 200 多项功能齐全的服务，其中 40 多项服务在其他云服务提供商中不可用。AWS认证 可以让你获得更多对你的技术技能的认可，并帮助你在使用 AWS 服务之外的知识以品牌形象脱颖而出。

AWS 认证可将你对该技术的知识转化为行业认可的证书。AWS 认证描述了候选人在部署和操作应用程序方面的技术技能、设计技能和策略。以及IT 专业人员在如何在 Amazon Web Services 平台上实施基础设施方面的稳定性。

选择正确的 AWS认证 - 有什么价值？

选择正确的 AWS 认证可以帮助你在职业生涯中成长，或者你可以在你当前的组织中跳到一个新的水平。因此，获得 AWS 认证表明你是云计算市场的熟练专业人员之一。  

如果你有兴趣继续或转向云计算领域，那么强烈建议你获得认证。因此，获得 AWS 认证尤其有助于你在竞争中脱颖而出。它不仅可以帮助你在组织中找到工作，还可以帮助你在任何组织中获得高薪工作。 

现在，我们将讨论不同类型的 AWS 认证，并提供有关每种认证的深刻信息。

AWS认证列表 和 学习路径

Amazon Web Services 认证分为三个不同的级别，分别称为 Foundational, Professional, 和 Specialty.

AWS Professional 和 Specialty 认证的区别？：

Professional 是一般性的，深入探讨了其他领域也共有的服务。AWS SAP 涵盖提供的服务，如 EC2、S3、Cloudformation、RDS 等。这些服务在所有其他领域都很常见，如大数据、DevOps、容器化等。

Specialty 特别关注特定领域，它将涵盖 AWS 中提供的所有 AWS 大数据及其相关服务。它使你为满足大数据平台要求的所有解决方案做好准备。它在 AWS 中提供的所有与大数据相关的服务（如 NoSQL、数据仓库、Hadoop）中测试你。

# 1. AWS认证 云从业者 - 基础

AWS 云从业者认证让你详细了解 AWS 云概念，例如安全性、定价、服务、架构和支持。此入门级认证考试旨在验证考生对 AWS 云的整体理解。 

云从业者认证涵盖的主题： 

  • AWS 云和全球基础设施的基础知识。
  • AWS 云架构的原理。
  • AWS 平台上的 AWS 云价值主张和关键服务及其使用案例。
  • 描述在 AWS 平台中部署和操作的特点。 
  • 确定技术援助和文档。
  • 解释 AWS 平台的安全性和合规性方面的基础知识。
  • 定义定价模型、帐户管理和计费。

推荐知识：

  • 我们建议你至少拥有六个月的任何职位经验，包括管理、技术、销售、财务或采购。 
  • 你还应该具备了解 IT 服务的基础知识及其在 AWS 云平台中的使用的能力。 

考试概览：

格式：多项选择题。

考试时间：90分钟

AWS 考试费用：100 美元 

语言：考试有英语、韩语、日语和简体中文。 

考试交付方法：在线监考考试或考试中心。

AWS 重新认证 - 如何重新认证 AWS？

AWS 认证有效期为三年，亚马逊不断为 AWS 添加功能和服务，以保持在快速发展的云市场的领先地位。AWS 重新认证是指升级到你的 AWS 证书的下一个级别，即 Associate to Professional，或者以较低的成本为你当前的证书参加重新认证考试。

在下一节中，我们将讨论 AWS 认证为何如此重要以及它将如何帮助你的职业发展。

AWS 认证的重要性

以下是说明 AWS 认证重要性的关键因素。 

1. 云是商业世界的未来： 大多数组织都在采用 AWS 服务。这是因为与其他竞争对手技术相比，AWS 服务具有快速、高级和更好的功能，并且 AWS 在单个平台中提供所有功能。 

2. 数字徽章：完成AWS认证后，你将获得一个数字徽章。你可以在社交媒体上展示你的成就，并且可以将其用作电子邮件签名来访问 AWS 赞助的认证活动。

3. 考试优惠： 在申请重新认证或任何其他考试时，你将获得 50% 的优惠券。  

4. 免费模拟考试： 你可以使用模拟考试券为你的下一个 AWS 认证做准备。 

5. AWS Certified Global Community：你还可以访问经过认证的 Linkedin 社区。它可以帮助你与同行交流，以了解有关 AWS 认证、工作机会等方面的最新更新的更多信息。 

6. 活动认可： 一旦你完成 AWS 认证，你将收到参加区域招待会的邀请。你可以使用你的数字徽章在选定的 AWS 峰会活动中使用 AWS 认证休息室。

Amazon Web Services 认证路径 - 常见问题

1. 我可以重新安排或取消 AWS 认证考试吗？

答：是的，你可以在预约后 24 小时内重新安排或取消你的 AWS 考试，而无需支付任何额外费用。

2. 如何重新安排或取消 AWS 认证考试？ 

答： 以下是取消或重新安排 AWS 考试的步骤：

Step1：登录你的 AWS 认证账户。

Step2：单击 PSI 考试或管理 Pearson VUE 考试按钮。

Step3：如果你使用 PSI 安排考试，请单击查看详细信息按钮来管理你安排的考试。或者，如果你计划使用 Pearson VUE，你可以使用即将到来的预约菜单重新安排考试。 

Step4：选择你需要的日期，并确认重新安排考试日期并点击提交按钮。 

3. 如果我错过了预定的考试，我可以退还考试费吗？

答：不，如果你错过了预定的考试，你的考试费将不予退还。而且，你必须在错过 AWS 考试时间后的 24 小时内再次注册考试。 

4. 我可以重新安排多少次考试？

答： 你可以在安排原始约会后重新安排考试日期最多 2 次。一旦达到限制，你需要在预约时间 24 小时前取消原来的预约，然后重新注册考试。 

5. 多久可以拿到AWS认证结果？

答：你将在完成考试后的五个工作日内获得结果。你将在你的 AWS 认证账户中收到 “Achieved Certification”的结果。

6. 如果我未能通过 AWS 认证会怎样？

答：万一你未能通过 AWS 考试，无需担心。你有无限的机会重考。但是，你需要等待 14 天才能重新参加 AWS 考试。

7、AWS认证的及格分数是多少？

答：  AWS 不公布考试及格分数。这是因为考试题目和及格分数会随着内容的更新而更新，以反映考试表格中的变化。 

8. 我可以在多长时间内参加我的模拟考试？

答：你可以参加 180 天的模拟考试。一旦开始考试，你需要在 30 天内完成，否则考试将过期。 

9. AWS 难学吗？

答： 不，学习 AWS 并不难；如果你找到了优秀的资源和学习路径，就很容易获得考试资格。

10. 需要哪种 AWS 认证？
