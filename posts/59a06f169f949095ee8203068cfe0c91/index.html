cd命令

语法：cd 目录名
功能：改变当前所在目录，将当前工作目录改变到指定的目录下
常用用例：
cd …:返回上一级目录
cd ~:返回用户家目
cd -:返回最近访问的目录

ls命令

用法:ls[ 选项] [目录名]
功能：能够列出当前目录中都包含了哪些文件

ls的常用选项：
-l :以列表的方式来显示内容（ls -l 可以缩写为ll）

-t :目录下的文件以时间排序

pwd命令

功能：显示当前用户所在的目录

touch命令

用法：touch [选项] 文件名
功能：可以更改文档和目录的时间或者新建一个不存在的文件。

cat命令

语法：cat [选项] [文件]
功能：查看目的文件中的内容

常用选项：-n对输出的行进行编号

mkdir命令

语法： mkdir [选项] [swpu]
功能：在当前目录下创建一个名字为swpu的目录

常用选项：-p 可以创建多级目录

rm命令

语法：rm [-f-i-r-v] [名字]
功能:删除文件和目录

注意：删除目录时需要加常用选项-r才可以直接删除，不过一级一级询问你是否删除，此时再加-f便可直接删除

cp命令

语法：cp [选项] 源文件/源目录 目标文件/目标目录
功能：将源文件/源目录复制到目标文件/目标目录（若不存在目标目录/目标文件则就创建一个）

mv命令

语法：mv [选项] 源文件或目录 目标文件或目录
功能：

  1. 视mv命令中第二个参数类型的不同（是目标文件还是目标目录），mv命令将文件重命名或将其移至一个新的目录中。
  2. 当第二个参数类型是文件时，mv命令完成文件重命名，此时，源文件只能有一个（也可以是源目
    录名），它将所给的源文件或目录重命名为给定的目标文件名。
  3. 当第二个参数是已存在的目录名称时，源文件或目录参数可以有多个，mv命令将各参数指定的源
    文件均移至目标目录中。


man命令

语法：man [选项] 命令
功能：查看命令功能

less命令

语法：less [参数] 文件
功能：查看文章内容，不会里把内容加载到内存中，也能进行查找。

常用选项：q 退出less

head命令

语法：head [参数] [文件]
功能：显示开头的行数，一般开头为十行
选项：-n显示的行数

tail 命令

语法： tail [必要参数] [选择参数] [文件]
功能：用于显示指定文件末尾内容，不指定文件时，作为输入信息进行处理。常用查看日志文件。

选项：-n<行数> 显示行数

date命令

date 指定格式显示时间： date +"%Y-%m-%d %H:%M:%S"
date 用法：date [OPTION]… [+FORMAT]
在显示方面，使用者可以设定欲显示的格式，格式设定为一个加号后接数个标记
%H : 小时(00…23)
%M : 分钟(00…59)
%S : 秒(00…61)
%X : 相当于 %H:%M:%S
%d : 日 (01…31)
%m : 月份 (01…12)
%Y : 完整年份 (0000…9999)
%F : 相当于 %Y-%m-%d

grep命令

语法：grep [参数] [文件]
功能：用于查找文件中是否包含指定字符串, 并显示对应的行.

ps命令

功能：用于才看系统上的进程

netstat命令

语法：netstat [参数]…
功能：查看系统上的网络状态.
netstat -anp | grep “进程名”
netstat -anp | grep “端口号”
