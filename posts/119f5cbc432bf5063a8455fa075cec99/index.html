CC0模式如何在NFT市场中成为主流？探索CC0 NFT的市场反馈

知识产权和版权是一个有着悠久而曲折历史的复杂现象。追溯到 18 世纪，版权法已经发展并以许多不同的方式在几乎同样多的法律管辖区得到应用。

随着 NFT 生态系统中的人们寻求一种既能避免混乱的版权问题，又能合法地赋予收藏者权力的方法，一些项目开始使用一种称为 CC0 版权许可的东西，这是一种被一些人视为解决这些问题的法律工具

CC0 NFT是什么

CC0，也称为知识共享，仅表示对知识产权“不保留任何权利”。这是一种版权形式，允许创作者放弃对其作品的合法权益，并尽可能将其移入公共领域。

CC0和NFT结合后相当于是建立在区块链溯源功能上的开源知识产权，创作者允许他人拥有其NFT并以任意方式用于商业目的，且无需将收益归属于原始创作者或团队。这使得CC0 NFT能高速地累积流量和关注度，从而吸引大量资金和创作人才，这些资金和人才又能促进项目更具新意的发展。

为什么要用CC0

在NFT领域引入CC0之前，区块链中数字版权的争议最初爆发于Yuga Labs和Larva Labs的版权限制。

Larva Labs限制旗下Crypto Punks的持有者们制作衍生产品或使用其品牌，被认为是为了保护其自身品牌，以便Larva Labs自身直接与其他企业和品牌进行合作获得更大收益。这激起了持有者们的不满，导致其中一些持有者放弃了该项目。

相比之下，Yuga Labs授予旗下BAYC NFT的持有者们较大的知识产权自由和商业权利，让他们可以使用购买的NFT艺术品本身创造和售卖衍生产品，但不能使用BAYC的品牌名称和标识.

例如：购买了BAYC NFT的人可以把该NFT的图印在T恤上去售卖，但如果把BAYC的logo和品牌名称也印上，这就是不被允许的。也就是说，首先，创造衍生品的权利仅授予给了BAYC NFT持有者；其次，持有者被限制使用BAYC的品牌名称和标识。

对诸如Yuga Labs和Larva Labs这类版权限制的不满，引发了CC0 NFT的爆炸式增长.

把CC0和NFT结合起来更进一步地解放了创作者和消费者，让他们自己决定作品的所有权模式，但仅限于虚拟网络或者元宇宙中，这是因为不可纂改的所有权只存在于数字领域，无法控制现实世界的市场

而如果在NFT项目上使用CC0许可证，就意味着：该项目向持有人提供一种额外保证——其作品的共享所有权模式可以延展到现实世界中。这是一个激进的承诺，即放弃对现实世界市场中的最后一点控制权，让大众共享一切作品，从而创建一个完全去中心化的品牌，任何人都能以任何方式修改、使用CC0项目的知识产权并从中获利，无需获得许可。

CC0 NFT项目都有哪些

CrypToadz

CrypToadz项目是 6969 种类似蟾蜍的生物的集合。在 CrypToadz 世界中，这些独特的生物正试图逃离邪恶的国王 Gremplin，他们的暴虐统治者。ETH 区块链 NFT 是使用 ERC-721 代币标准创建的。这个售罄项目的巨大成功归功于它是一个CC0项目。这些 NFT 上的艺术属于公共领域，不保留任何权利，因此其所有者和社区可以根据 CC0 许可证自由使用它们。

CryptoDickButts

CryptoDickButts是另一个 CC0 NFT 项目，具有非常独特的艺术风格，最初是一个笑话。开发人员使项目保持简单。该项目没有路线图或实用程序，只有在 CC0 许可下可供公众使用的 NFT 艺术。以每 NFT 0.052 ETH 的价格出售，拥有 5200 NFT 的 Series 3 系列现在（在撰写本文时）以底价约为 1 ETH。该项目的成功是 CC0 NFT 吸引 NFT 社区的一个很好的例子。

Goblintown.wtf

Goblintown.wtf是 10,000 个 NFT 丑陋生物的集合，其中主要由地精组成，还包括其他生物，例如居住在地精镇的龙、巨魔、巫师等。该项目是由一组匿名开发人员发起的，他们被推测与无聊猿游艇俱乐部的创建者 Yuga Labs 有关。CC0 项目也没有针对其社区的实用程序、路线图甚至是 Discord。相反，它有免费铸币，这为那些刚接触该领域的人提供了拥有 NFT 的低成本障碍。

Mfers

从表面上看，Mfers可能看起来像是一个 meme 项目——它只是在 OpenSea 上有一个页面，没有精心制作的网站和一个现已不存在的 mint 页面。尽管如此，当 10,000 个 NFT 的集合最初以 0.069 ETH 铸造，NFT 社区知名的艺术家 Sartoshi 的项目没有像我们列表中的许多其他人那样的实用程序或路线图。它的吸引力只是它的 CC0 艺术品和有机社区。

Nouns

Nouns是最早的 CC0 NFT 项目之一，由一群在 NFT 领域享有盛誉的 NFT 收藏家和开发者发起。最初是一个实验性 CC0 项目，是最成功的项目之一，迄今为止总共赚了至少 1 亿美元。每日拍卖的所有 ETH 收益都将捐给 NounsDAO，这是该项目的管理组织，代币持有者可以对资源的使用方式进行投票。近日，NounsDAO 签约了一支专业的 DOTA 电竞战队，并有太阳镜、咖啡等商品的计划，未来计划开设一家真实世界的咖啡馆。Nouniverse 拥有稳固的团队、蓬勃发展的社区和领先的 DAO，绝对值得一看。

Rektguy

Rektguy背后的故事对 NFT 世界来说太熟悉了——它是一家 NFT/加密货币经纪人，因最近的价格暴跌而受到影响。NFT 的支持者 Ovie Faruq 或 OSF 与Alien Queen一起是该项目的幕后推手。OSF 是 NFT 领域的知名品牌，在 Twitter 上拥有 18 万粉丝——这对 rektguy 的成功非常有利。此外，该项目还得到了在这些圈子中也很有名的 Cozomo de' Medici 和 Snoop Dogg 的支持。

Tiny Dinos

Tiny Dinos自称是一个全链 CC0 NFT 项目，“没有路线图，只有小恐龙”。这个由 10,000 只简单的像素化恐龙组成的项目免费铸造，并提供了一些有趣的福利，这些福利对于 NFT 来说是相对较新的趋势。全链 NFT 可在链之间转移——所有者只需将 NFT 包装起来即可在不同的链上进行交易。所有者还可以更改 NFT 的背景，从而实现一定程度的定制。这个项目中艺术品的简单性是有时简单更好的一个典型例子。

CC0 NFT对市场的意义

CC0 项目可能会面临失败跑路的风险，但社区成员和 NFT 所有者围绕其 NFT 进行商业化和构建的能力为已经建立的 CC0 项目提供了巨大的弹性。如果团队离开、死亡或变得太有钱而无法照顾，社区可以继续建立和提供价值。个人还可以组建 DAO，创建自己的愿景和努力来维持和支持项目。

概括：

  • NFT 项目具有建立强大和持久社区的强烈动力

  • 与非 CCO 项目相比，CCO 项目对网络效应的摩擦更小，更有可能创建可持续社区

  • CCO 项目将吸引为社区增加持续价值的建设者

  • CCO 项目的所有者将从相邻项目中获得价值

  • CCO 项目将更普遍、更容易地集成到元宇宙世界中

  • 与非 CCO 项目相比，CCO 项目的“团队风险”更小，更有弹性

总结

许多成功的 CC0 项目都是通过引入可以在一系列不同背景下灵活使用的知识产权而成功的，减少商业纠纷，有效地将竞争转变成合作，NFT玩家可以自由地对艺术品做任何他们想做的事情，这弥补了实用性和路线图的不足。

CC0 NFT让艺术充满多样性和二次乃至多次创作的可能性，活力让艺术充满创造力。区块链天然的去中心化和开源许可让NFT项目在CC0的模式下得到广泛的关注和支持，相应的CC0授权模式让社区为所有成员在价值创造的深入发展上提供了便利。
