一、$attrs

我们在使用组件的过程中，有时需要给被封装的子组件传递Prop
例如：

<template>
    <el-input v-model="innerVal" placeholder="placeholder"
    @input="$emit('input',$event)"
    >
    </el-input>
</template>

<script>
  export default {
    name: "MyInput",
    data(){
      return{
        innerVal:this.value
      }
    },
    watch:{
      value(){
        if(this.value!==this.innerVal){
          this.innerVal=this.value
        }
      }
    }
  }
</script>
