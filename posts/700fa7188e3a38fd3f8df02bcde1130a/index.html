YUM源的配置及安装图形化用户管理器（红帽子）

红帽子的YUM源的配置（敲重点！！）及安装图形化用户管理器

一、挂载iso镜像

首先-》编辑虚拟机设置-》CD/DVD(SATA)-》使用ISO映像文件-》确定



检查是否挂载成功可以看桌面（图形化界面）右下角是否有光盘亮着或者看桌面是否有一个光盘显示（二者可选择看一个）
ps:正常情况两个都会是显示的


二、blkid查看镜像文件的UUID，并复制下来

将UUID=2017-07-11-01-39-24-00复制下来

三、mkdir 创建一个目录用于镜像文件的挂载

四、vim打开/etc/fstab,写入

写法一：/dev/sr0 /挂载点 iso9660 defaults 0 0
写法二:UUID=XXXX /挂载点 iso9660 defaults 0 0
（我采用的是写法二）

ps:这里的挂载点就是我们第三步创建的目录

五、保存退出后，使用mount -a刷新挂载点

使用ESC，输入：wq保存退出
使用mount -a刷新挂载点

六、配置YUM源：

①首先cd /etc/yum.repos.d/
②其次删除目录下的所有以*.repo结尾的文件
③最后添加以XXX.repo结尾的文件
XXX.repo文件内容：

[dvd]
name=dvd
baseurl=file:///挂载点（挂载点是自己创建的目录）
gpgcheck=0
enabled=1

七、保存退出后，检查yum库源：yum list all

esc+:wq保存退出
使用yum list all检查yum库源

八、安装用户组图形化管理工具：

使用以下命令安装用户组图形化管理工具

yum -y install system-config-users

安装完成后，使用以下命令打开用户管理器，即可进行用户和组的添加

system-config-users
